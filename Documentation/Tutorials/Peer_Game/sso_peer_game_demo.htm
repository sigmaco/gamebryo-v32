Peer Game Demo Overview

The Peer Game demo is a fully-functional demonstration of the online features of the Emergent Game Framework. The demo contains a world that is created in World Builder and a sample application that loads this world and lets the user interact with it. Peer Game expands the Base Game demo by adding the ability to host games and have other users connect to them.

Peer Game demo

Source Location

By default, the pre-compiled binary executables install to the following location:

%EMERGENT_PATH%\Build\Samples\[Platform]\[Compiler]\PeerGame

For example, the Windows binary that is built using Visual Studio 2008 uses "Win32" for the platform and "VC90" as the compiler. Several versions of the Peer Game executable are installed by default and additional versions can be compiled using the Game Framework Samples solution file in the parent directory (one directory up from the executable location).
