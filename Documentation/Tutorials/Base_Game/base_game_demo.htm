Base Game Demo Overview

The Base Game demo is a fully functional demonstration of the Emergent Game Framework. It contains a world that is created in World Builder and a sample application that loads this world and lets the user interact with it. It expands the Hello World demo by adding PhysX triggers, collision, and Lua scripted AI behaviors.

Some of the features of Base Game include:

  • Puzzles to solve via triggers
  • Gameplay Camera
  • Triggers
  • Walkable surfaces
  • Behavior-driven AI entities
  • Behaviors invoked by key press
  • LUA and C++ Behaviors

Source Location

By default, the pre-compiled binary executables install to the following location:

%EMERGENT_PATH%\Build\Samples\[Platform]\[Compiler]\BaseGame

For example, the Windows binary that is built using Visual Studio 2008 uses "Win32" for the platform and "VC90" as the compiler. Several versions of the Base Game executable are installed by default and additional versions can be compiled using the Game Framework Samples solution file in the parent directory (one directory up from the executable location). Use the release version or compile a debug version if you'd like to experiment with rapid iteration.
