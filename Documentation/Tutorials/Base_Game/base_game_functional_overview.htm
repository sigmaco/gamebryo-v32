Base Game Demo Functional Overview

The Base Game demo is driven by the Gamebryo Game Framework, a few C++ files that are specific to the application, and some application-specific data, including script files for implementing game logic. It expands on the Hello World demo by adding PhysX triggers and collisions, multiple rooms, AI waypoints, and behaviors.

The application contains one level that includes the following:

  • Four rooms (one hidden)
  • Three computer-controlled characters (called AIJack)
  • One player-controlled character (called Avatar)
  • Multiple PhysX-based triggers
  • Doors that open and close
  • A box the player can pick up

The level and the entities that are included in the demo are created in Toolbench. The entity behaviors are written in Lua. The primary purpose of Base Game is to illustrate a more advanced possible Game Framework-based application that demonstrates:

  • Programmer features:
    • PhysX triggers and collision
    • Player-focus cameras
    • Walkable/non-walkable areas
  • Designer and Scripter features:
    • Implementing game logic using Lua scripted behaviors
    • Setting animation frames from script
    • Creating AI waypoints
    • Updating AI goals from script
    • Creating and removing Entities at runtime
