Rapid Iteration: Resetting the Game

You can reset the game (Hello World, in this case) when the state of the simulator is invalid. You can also reset the system to repeat a behavior that is triggered only when an entity is created. A full system reset causes the game to pause, flush all entities, and reload the world(s).

Sending a Full System Reset Message to the Hello World Game

  1. Double-click HelloWorld.block in Toolbench to open World Builder.
  2. In Toolbench, select Game > Reset Game from the menu.
  3. Watch the application window. All entities reset to their default states specified in the world file.

Moving on...

Resetting the game is not a rapid iteration feature per se, but it is important that you know how to reset the game simulation without restarting the game. Next, let's make some changes in a behavior script. See Script Changes.
