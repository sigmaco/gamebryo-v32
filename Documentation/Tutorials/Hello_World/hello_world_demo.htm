Hello World Demo Overview

The Hello World demo is a fully-functional demonstration of the Emergent Game Framework that highlights key features of Gamebryo LightSpeed's entity modeling and runtime simulation systems. The demo contains a world that is created in World Builder and a sample application that loads this world and lets the user interact with it.

Hello World demo

Source Location

By default, the pre-compiled binary executables install to the following location:

%EMERGENT_PATH%\Build\Samples\[Platform]\[Compiler]\HelloWorld

For example, the Windows binary that is built using Visual Studio 2008 uses "Win32" for the platform and "VC90" as the compiler. Several versions of the Hello World executable are installed by default and additional versions can be compiled using the Game Framework Samples solution file in the parent directory (one directory up from the executable location).
