Added Features, Changed Behavior, and Fixed Bugs

Gamebryo LightSpeed 3.2

  • None in this release.

Gamebryo LightSpeed 3.1

  • None in this release.

Gamebryo LightSpeed 3.0

  • The directory structure has changed. Third Party Libraries are now in Source\3rdParty\.
  • Updated Lua to version 5.1.4 and LNUM to 20081022.
  • Updated DockPanel Suite to version 2.2.0.0.
  • Several have been added. See the Note above for an up-to-date list.

Gamebryo 2.5

  • Updated OpenEXR to version 1.6.1, including IlmBase 1.0.1.
  • Updated LibPNG to version 1.2.16.
  • Updated OpenEXR to version 1.4.0, and switched to shipping built binary libraries rather than shipping source code and recompiling.
  • Updated TinyXML to version 2.5.2
  • Changed directory structures inside the distributions in the ThirdPartyCode directory to facilitate future version updates.
  • Removed PNGQuant utility from the distribution.

Gamebryo 2.3

  • Updated Bison to version 2.1.

Gamebryo 2.2.1

  • Updated the Weifen Luo Dock Panel suite to version 1.0.0.0.
  • Added Xalan-C++ 1.10 and Xerces-C++ 2.7 to the ThirdPartyCode directory.

Gamebryo 2.1.0

  • Updated all third party libraries to make use of the latest available released versions as of the engine release. This includes ZLib 1.2.3.
  • Fixed the name of the Bison directory from Bison 1.8.5 to Bison1.875.

Gamebryo 2.0.0

  • Updated all third party libraries to make use of the latest available released versions as of the engine release. This includes LibPNG 1.2.8 and ZLib 1.2.2.
  • Added the OpenEXR library for support of OpenEXR image files.

Known Issues and Limitations

  • There are no known issues for this release.
