Third Party Code - DragDropHelper

Version

No explicit version provided. http://bea.stollnitz.com/blog/?p=53

Description

This code assists with dragging and dropping items between WPF data bound controls.

Usage

The DragDropHelper code is used to support drag and drop operations in the Toolbench Project Explorer. It has been heavily modified for Toolbench use.

Location: %EMERGENT_PATH%\Source\Tools\Toolbench\Plugins\Emergent.Toolbench.UI\DragDropHelper.cs
