Third Party Code - NvTriStrip

Version

NvTriStrip 2/17/2004 http://developer.nvidia.com/object/nvtristrip_library.html

Description

NVTriStrip is a library for vertex cache aware stripification of geometry.

Usage

NiStripify uses NvTriStrip to generate vertex-cache aware triangle strips from the exporter pipeline. Shipping games do not need to link in this library. The nVidia code is embedded within the library. Modifications were made to this code to remove hard-coded numbers and allow for greater configuration of stripification options.

Location: %EMERGENT_PATH%\Source\CoreRuntime\ToolLibs\NiStripify\*.*
