Third Party Code - Effects11

Version

Effects11 from August 2009 DX SDK

Description

Effects11 is a library provided by Microsoft in the DX SDK that loads and executes FX shaders through D3D11.

Usage

One of the libraries in Gamebryo's shader system, ecrD3D11EffectShaderLib, contains the code for Effects11, which is compiled into the library directly. Some minor modifications were made to the code, and are marked as such in the distributed source.
