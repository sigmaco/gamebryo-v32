Third Party Code - WPF Model-View-ViewModel Toolkit

Version

0.1. http://wpf.codeplex.com/Wiki/View.aspx?title=WPF%20Model-View-ViewModel%20Toolkit

Description

A toolkit to introduce the Model-View-ViewModel pattern in WPF.

Usage

Toolbench uses modified versions of CommandReference and DelegateCommand classes that were originally generated by the WPF MVVM Toolkit. These classes are used in support of WPF UI, including portions of Entity Modeling Tool and World Builder.

Location: %EMERGENT_PATH%\Source\Tools\Toolbench\Plugins\Emergent.Toolbench.UI\MVVM
