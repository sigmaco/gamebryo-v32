Third Party Code - OpenEXR

Version

OpenEXR 1.6.1 (including IlmBase 1.0.1): http://www.openexr.com/

Description

OpenEXR is a high dynamic-range (HDR) image file format developed by Industrial Light & Magic for use in computer imaging applications.

Usage

The Gamebryo half float code found in efd and NiSystem files:

%EMERGENT_PATH%\Source\Foundation\efd\Float16.*

%EMERGENT_PATH%\Source\CoreRuntime\EngineLibs\NiSystem\NiFloat16.*

is based upon Industrial Light & Magic's half class found in the OpenEXR distribution. The specific license information for that class is provided here as well as in the source.

Emergent also uses the OpenEXR library to parse OpenEXR images in the NiOpenEXRReader library.

%EMERGENT_PATH%\Source\CoreRuntime\ToolLibs\NiOpenEXRReader\*.*
