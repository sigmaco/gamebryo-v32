Third Party Code - Quaternion

Version

"Hacking Quaternions" in Game Developer Magazine, March 2002.

Description

This code is a performance optimization of the quaternion slerp algorithm.

Usage

The Gamebryo quaternion interpolation code in NiMain files:

%EMERGENT_PATH%\Source\CoreRuntimeEngineLibs\NiMain\*\NiQuaternion_*.cpp
