Version

Description

Xalan-C++ is an XSLT processor for transforming XML documents into HTML, text, or other XML document types. It is licensed under the Apache 2.0 license, a copy of which is included with the distribution and can be found at http://www.apache.org/licenses/LICENSE-2.0.txt.

Usage

Xalan is used to convert COLLADA material files to NSF files (%EMERGENT_PATH%\Source\Tools\Collada2NSF\).
