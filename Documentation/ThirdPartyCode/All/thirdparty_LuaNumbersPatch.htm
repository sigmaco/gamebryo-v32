Third Party Code - Lua Numbers Patch (LNUM)

Version

lua514-lnum-20081022 http://luaforge.net/projects/lnum

Description

The LNUM patch allows easy and wide configuration of Lua number types, with the use of simple compiler defines. It provides added integer accuracy (32- or 64-bit), enhances speed on non-FPU systems and allows use of complex numbers (C99 required).

Usage

The egfLua library in GameFramework already has the LNUM patch applied to it.

The following defines that control LNUM behavior are set in the egfLua build solutions. Complex number support is not enabled by default.

  • LUA_USELONGLONG
  • LNUM_DOUBLE
  • LNUM_INT64

LNUM Integration Location: %EMERGENT_PATH%\Source\GameFramework\egfLua\*.*
