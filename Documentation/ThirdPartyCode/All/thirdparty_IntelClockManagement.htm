Third Party Code - Intel Clock Management

Version

Originally published in Game Programming Gems 3.

Description

The clock classes provide a mechanism for generating events in a game simulation.

Usage

The Gamebryo LenguinsOnIce demo includes code that was published in the article "Scheduling Game Events" of the book Game Programming Gems 3. The LenguinsOnIce source files affected are: Clock.cpp, Clock.h, and Clock.inl.
