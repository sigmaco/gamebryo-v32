Third Party Code - SWIG (Simplified Wrapper and Interface Generator)

Version

1.3.34 http://www.swig.org/index.php

Description

SWIG is a software development tool that connects programs written in C and C++ with a variety of high-level programming languages.

Usage

SWIG is used by the GameFramework to bind C++ built-ins to the supported scripting languages.
