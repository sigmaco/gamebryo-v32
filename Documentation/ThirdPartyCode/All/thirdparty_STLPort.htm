Third Party Code - STLPort

Version

5.1.5 http://sourceforge.net/projects/stlport/

Description

STLPort is a portable implementation of the C++ Standard Template Library.

Usage

The Foundation and GameFramework code bases make extensive use of STLPort containers.

A copy of STLPort that has been ported to all supported platforms can be found at:

%EMERGENT_PATH%\Source\3rdParty\STLPort\
