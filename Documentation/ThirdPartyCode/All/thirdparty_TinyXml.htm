Third Party Code - TinyXml

Version

TinyXml 2.5.3: http://sourceforge.net/projects/tinyxml/.

Description

TinyXml is a simple, small, minimal, C++ XML parser that can be easily integrating into other programs. It reads XML and creates C++ objects representing the XML document. The objects can be manipulated, changed, and saved again as XML.

Usage

Emergent embeds TinyXml in its efd library to facilitate DOM XML parsing. The library has been modified to support SAX style XML parsing as well.

The modified source can be found at:
