Third Party Code - Autodesk FBX

Version

FBX 2010.2: http://usa.autodesk.com/adsk/servlet/pc/index?id=6837478&siteID=123112

Description

Autodesk FBX is an interchange format SDK for 3D content.

Usage

The following libraries interact with the FBX SDK to export NIF geometry and entities to FBX representations:

  • Source/Tools/Toolbench/Plugins/Emergent.Toolbench.WorldBuilder.FBX/
  • Source/Tools/Toolbench/Plugins/Emergent.Toolbench.WorldBuilder.UI.FBX/
  • Source/CoreRuntime/ToolLibs/NiFBXConverter/

Note that these libraries are not intended to be linked into a customer's executable and are provided to enhance the content designer's workflow.

Emergent Game Technologies believes that it is in full compliance with the terms of the license for the portions of the engine that include and make use of this code.
