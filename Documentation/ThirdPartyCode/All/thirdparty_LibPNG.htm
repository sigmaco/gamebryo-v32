Third Party Code - LibPNG

Version

LibPNG 1.2.16: http://www.libpng.org/pub/png/libpng.html

Description

libpng is the official PNG reference library. It supports almost all PNG features, is extensible, and has been extensively tested for over 13 years.

Usage

NiPNGReader uses LibPNG to parse PNG files into NiPixelData objects.

The following code depends on LibPNG: %EMERGENT_PATH%\Source\CoreRuntime\ToolLibs\NiPNGReader\*.*
