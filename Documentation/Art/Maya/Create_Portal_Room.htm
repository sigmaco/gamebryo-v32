Create Portal Room

Click the Create Portal Room button to create a room and name it appropriately. During the export, the portal conversion code detects a room by locating "_NDLRM" in the name.

Create Portal Room button

Depending on what is selected during the export, you will see three different types of behavior.

  • Nothing selected—an empty room node named "Room_NDLRM" will be created in Maya
  • One transform selected—The "_NDLRM" tag will be appended to the end of the selected object's name
  • Multiple transforms selected—The selected objects will be grouped together under a new room node named "Room_NDLRM"
