Create Proxy Spherical Collider

Select both the particles and an object and then click the Create Proxy Spherical Collider button to automatically attach that object as a collider in Maya. "Extra Attributes" will also be added to the newly created collider. The Gamebryo Maya Exporter uses these attributes during export.. The proxy sphere will create a spherical collider in Gamebryo based off the bounding sphere of the collider object.

Create Proxy Spherical Collider button

Proxy spherical collider

The following extra attributes are added to the collider:

  • Ni_SphericalCollider—This attribute indicates to the exporter that the sphere collider should be created based on the bounds of this object. This option is on by default.
  • Ni_ColliderDeleteOnExport—This attribute indicates that children of this transform should be removed from the export after the proxy sphere has been created. This is useful when the geometry used by the collider is unnecessary. This options is off by default.
