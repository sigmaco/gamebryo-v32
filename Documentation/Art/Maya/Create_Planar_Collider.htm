Create Planar Collider

Select the particle system and then click the Create Planar Collider button to automatically create a plane and associate it with the particle system as a collider. "Extra Attributes" will also be added to the newly created plane. These are used by the Gamebryo Maya Exporter.

Create Planar Collider button

Planar collider

The following extra attributes are added to the collider:

  • Ni_ColliderDeleteOnExport—This attribute indicates that children of this transform should be removed from the export after the planar collider has been created. This is useful when the geometry used by the collider is unnecessary. This options is off by default.
  • Ni_PlanarCollider—This attribute indicates to the exporter that the planar collider should be created. This option is on by default.
