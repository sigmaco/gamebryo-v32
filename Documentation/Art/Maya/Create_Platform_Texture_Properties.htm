Create Platform Texture Properties

After selecting a "File" texture node in Maya, the Gamebryo category within the attribute window may be opened to reveal the "Create Platform Texture Properties" button. You can use this button to modify the texture properties. These properties are stored as "Extra Attributes" and allow you to specify the texture format, size, and location. This allows you to set properties on a per texture basis.

Create Platform Texture Properties button

Add Gamebryo Attributes - Platform texture options

Different platforms have different requirements so the platform texture properties allows you to customize your texture based on the destination platform.

Texture formats

During export, the format of this texture can be converted from the original format found in the file to another format. This allows you to keep you original art in a large high fidelity format and then convert to a format better suited for the destination platform. 8bit palletized textures use 32bit palette entries which include alpha. The default platform format is specified in the Texture Options of the Export All Options dialog.

Note

It is important to remember this is only a preferred format. Not all formats are supported on all platforms and during the export process the destination renderer will determine the final destination format.

Available texture formats include the following:

  • Platform Default (Match Default Platform Format)
  • Default (Match Source Image)
  • 24 / 32 bit
  • 16 bit
  • 8 bit palletized
  • Compressed DXT 1
  • Compressed DXT 3
  • Compressed DXT 5

Texture sizes

During Export, the texture can automatically be reduced in size on a per platform basis. The default platform reduction is specified in the Texture Options of the Export All Options dialog.

Available texture sizes include the following:

  • Platform Default (Match Default Platform Reduction)
  • Default (Match Source Image)
  • 1/2
  • 1/4
  • 1/8
  • 1/16
  • 1/32
  • 1/64

Texture locations

The final location of the texture can be set on a per platform basis. The default location is specified in the Texture Options of the Export All Options dialog.

Available texture locations include the following:

  • Scene Default
  • Internal to NIF
  • External as NIF
  • External Original Format

Tip

The PC platform supports loading of textures directly by Direct X. In essence textures are loaded by the renderer directly into VRAM and it will completely circumvent Gamebryo. This option is only valid for textures which are loaded in the External Original Format. This option is only a hint and the renderer is not obligated to honor it.
