Overview

Gamebryo's particle system implements the following high-level features:

  • Sprite Particles
  • Mesh Particles
  • Point, Volume, Surface, and Curve Emitters
  • Colliders (planar and spherical)
  • Forces
  • Particles that spawn particles

This document describes the Gamebryo-specific particle system features as exposed in Maya. It is not meant as a replacement for the Maya documentation.

Gamebryo Particles in Maya

In Maya, Gamebryo particle systems are represented by five custom nodes:

  • NiParticleSystem
  • NiEmitter
  • NiCollider
  • NiForce
  • Ni*Spawner (includes Collision, Living, and Death)

These nodes are created and associated with each other using the Gamebryo Particle shelf.

The Gamebryo Maya Plug-in includes a real-time viewer that can be launched from the Gamebryo Particle shelf. The Remote Viewer uses the Gamebryo renderer so that the artist will see an accurate representation of how the particle system will look in-game. In addition, the particle systems rendered by the Remote Viewer are updated immediately when attributes are changed in the Maya UI, enabling rapid particle system iteration.

The Gamebryo particle nodes are not the same as the standard Maya particle nodes that are created using the Maya Dynamics shelf. The Gamebryo-specific nodes represent the features implemented by the Gamebryo NiParticle runtime library. Thus, unlike the standard Maya particle system nodes, every attribute on the Gamebryo particle system nodes will affect the exported particle system. As a convenience, the Gamebryo Particle shelf provides a button that will make a best-effort attempt to convert all Maya particle systems in a scene to Gamebryo particle systems. These converted particle systems may require additional tweaking by the artist.
