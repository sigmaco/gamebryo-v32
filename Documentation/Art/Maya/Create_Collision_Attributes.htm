Create Collision Attributes

You can use the Gamebryo Maya Plug-in to create proxy geometry that Gamebryo can use in collision detection. For example, you can set up collision volumes using Maya. The proxy geometry, also known as Alternate Bounding Volumes (ABVs), are simpler approximations of the volume occupied by objects that will be used in collision tests. ABV.mb shows an example of ABV Attributes. You generate collision volumes by setting Extra Attributes in the Attribute Editor. You do this by clicking the Create ABV Attributes button.

You must also ensure that you use the Create ABV process script when you export. This option is available when you press the Optimization Settings button on the Export options dialog.

Create ABV Attributes button

ABV proxy geometry

It is much easier to test for a collision between two spheres than it is to test between two complex objects consisting of many hundreds or thousands of triangles.  The same applies to other simple geometric shapes, such as a half-space (one side of a plane), a capsule, and a box.  Alternate Bounding Volumes (ABVs) enable you to wrap your complex geometry with these simple shapes in order to significantly increase collision performance.

Gamebryo enables you to create proxy boxes inside your modeling package, so that you can carefully define and align these ABVs to your model geometry. Select (check) the Enable Collision Data check box if you want to enable the creation of ABV geometry for an object.

Triangle Collisions and Mesh Profiles

Meshes used for Triangle collision shapes have special requirements on their data. The sample Mesh Profile named PosNormPicking.xml sets the appropriate access for both triangle collisions and picking, and any triangle collisions should use that mesh profile or another like it. The profile must be extended if color or texture information is required from picking.
