Supported Formats

Gamebryo natively supports the following image file types:

  • NIF—the Gamebryo-specific format
  • TGA—a good choice for images with alpha
  • BMP—24 bits
  • PNG
  • DDS—DirectX compression. nVidia supplies an Adobe Photoshop plugin for importing and exporting .dds textures. See http://developer.nvidia.com/object/nv_texture_tools.html.
  • SGI
  • HDR—Radiance High Dynamic Range
  • EXR—A subset of the OpenEXR format is supported for High Dynamic Range

Other image types, including JPG, are not officially supported.
