Create Billboard Node

Many objects in the world are symmetric around their vertical axis. Light posts are a great example -- they look the same from all sides. You can simplify the scene by creating these as billboards. Billboards are a special type of object, which by default always faces the camera.

Create Billboard Node icon

To create a "Billboard Node", select the object you want to billboard and click the Icon. A Maya group is created with a special "Extra Attribute" which determines the operation of the billboard. Alternately you can click the icon when nothing is selected. This will automatically create the billboard node and a polygon plane already oriented along the correct axis.

Extra Attributes

Use this field... To...
Always Face Camera Direction Make the object always follow the camera. Minimizes the difference between the "up" and the Y-axis.
Rotate About Up Make the object follow the movement of the camera, but keeps its "up" vector up in the world.
Rigid Face Camera Direction Make billboards accept the coordinate axes of the camera. In this mode, billboards appear locked (or rigid) in screen space.
Always Face Camera Center Make the object always face the center of the camera. Minimizes the difference between the "up" and the Y-axis.
Rigid Face Camera Center Make billboards point directly to the center of the camera.
