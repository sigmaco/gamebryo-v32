Applying a Shader

You use the NiMultiShader material to apply shaders in Maya. Any object that has that material applied will also have the shader applied.

Applying a shader (click image to enlarge)

The attribute editor UI is split into two sections—the top section that is non-scrollable and the bottom scrollable section.

The top section is used to display the current shader options. This includes what shader will be referenced by this node, plus what texture to currently display in the Maya viewport. For more information please see Base Shader Panel.

The bottom section or the scrollable section displays all the shader attributes. The interface is designed to adapt to the current shader.

The scrollable section is divided into four sections.

  • Material
  • Geometry
  • Vertex Colors
  • Alpha Transparency
  • Shader Attributes (Standard Material or Custom Shader)
