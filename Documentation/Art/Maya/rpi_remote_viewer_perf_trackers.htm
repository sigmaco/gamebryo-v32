Gamebryo Remote Viewer Visual Performance Trackers

The Remote Viewer has built-in performance trackers that can display renderer performance data in real-time as an graph overlay. The visual trackers can be activated by pressing the F11 key when the Remote Viewer window is selected or by pressing the right stick down on a game controller. Refer to NiApplication and NiVisualTrackers section of Gamebryo documentation for a description of the graphs.

Then Sync performance in the Timing Tracker is unique to the Remote Viewer. This graph represents the amount of time spent synchronizing the scene with Maya.
