Particle introduction

Note

The ability to directly export Maya particle system nodes is marked for deprecation in a future Gamebryo release. This functionality is replaced by the new Gamebryo-specific particle system nodes that were added as part of the Rapid Particle Iteration feature. A tool shelf button is provided that makes a best effort to convert existing Maya particle system nodes to Gamebryo particle system nodes.

Gamebryo has a powerful particle system for creating both billboard and geometry particles. Using the Gamebryo Maya Plug-in, you can create different types of emitters, animate particle values, apply fields, and created colliders.  As we go through each different aspect of particle systems, we will discuss what features are implemented and how they correspond to the Gamebryo Maya Plug-in.  Since there are differences in the features of Gamebryo and Maya, the Gamebryo Maya Plug-in provides an icon to automatically create a particle system with the necessary extra Gamebryo functionality.

Create Emitter and Particles button

Supported types

The Gamebryo Maya Plug-in supports Directional, Omni, Volume, and Surface Emitters. Curve Emitters are not supported within the Gamebryo Maya Plug-in.

Examples

Look in the Particle sub-directory of the Maya Artist Examples directory for examples of particle systems modeled in Maya for Gamebryo.
