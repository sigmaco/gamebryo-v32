Create Spherical Collider

Select the particle system and then click the Create Spherical Collider button. This automatically creates a sphere and associates it with the particle system as a collider. "Extra Attributes" that are used by the Gamebryo Maya Exporter are added to the newly created sphere.

Create Spherical Collider button

The following extra attributes are added to the collider:

  • Ni_ColliderDeleteOnExport—indicates that children of this transform should be removed from the export after the collision sphere has been created. This setting is useful when the geometry used by the collider is unnecessary. By default, the setting is OFF.
  • Ni_SphericalCollider—indicates to the exporter that the sphere collider should be created based on the bounds of this object. By default, this option is ON.
