Maya Plug-in: Supported Versions

This Gamebryo release includes support for the following 32-bit versions of Maya:

  • Maya 2008
  • Maya 2009
  • Maya 2010

If you plan to create PhysX content for use with Gamebryo, you must also install the PhysX Maya plug-in. Gamebryo 3.2 is intended for use with a pre-release PhysX Maya plug-in, found in the installation media at PhysX\NVIDIA_PhysX_Plug-in_Maya_x86_2.01.0529.1836.msi. Note the license agreement found in the same directory.
