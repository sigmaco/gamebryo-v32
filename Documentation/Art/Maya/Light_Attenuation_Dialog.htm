Light Attenuation Dialog

The light attenuation dialog is a custom interface to make setting the attenuation values of lights easy. With this dialog, you can specify the use of Maya's decay rate or a decay rate through Gamebryo. When specifying the attenuation through distance, it is important to remember that light intensity drops away rapidly, but never actually reaches zero. Through the interface, you specify a "Max Distance", but in reality, this distance is only where the light is 1% of its original intensity. The table below, "Light Distance By Intensity", shows distances and intensities so you can see the general shape of the curve. There is a great deal of difference between "Linear", and "Quadratic" decay rates, so experimentation can be very useful.

Light Attenuation dialog

Use this field... To...
Use Maya Decay Use standard Maya decay rate instead of Gamebryo settings during export
Set Attenuation By Set attenuation by maximum distance or by custom attenuation. Setting maximum distance will automatically calculate the attenuation over distance.
Decay Rate Specify the decay rate. This helps the custom window determine your attenuation settings. You can choose between Linear and Quadratic. This field is unavailable (grayed out) when you are using Custom Attenuation.
Max Distance Specify the Max Distance that you want your light to shine.
Note: It is important to remember that light intensity drops away rapidly, but never actually reaches zero. Through the interface, you specify a "Max Distance", but in reality this distance is only where the light is 1% of its original intensity.
This field is unavailable (grayed out) when you are using Custom Attenuation.
Dimmer Control the base intensity of the light, before applying any attenuation (1.0 = normal brightness, 0.5 = half brighness, 0.0 = no light at all).
Constant Attenuation Moderate the effect that linear and quadratic attenuations produce. Specifically, this is the value of C in the lighting equation at the bottom of the NiPointLight reference.
Linear Attenuation Give a falloff with distance (1/2 brightness at twice the distance). Specifically, this is the value of L in the lighting equation at the bottom of the NiPointLight reference.
Quadratic Attenuation Give a falloff with distance squared (1/4 brightness at twice the distance). Specifically, this is the value of Q in the lighting equation at the bottom of the NiPointLight reference.
Light Distance By Intensity View the general shape of the curve.
Tip: There is a great deal of difference between "Linear" and "Quadratic" decay rates, so experimentation can be very useful.
