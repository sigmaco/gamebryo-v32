Maya Mel Munger

During installation, the Maya Mel Munger generates files that allow Gamebryo to augment the existing Attribute Editor. Because it is possible that users might have custom versions of the Attribute Editor templates, the Attribute Editor is augmented by aliasing the existing Attribute Editor templates. Gamebryo modifies the template associated with the given type to add the Gamebryo custom interfaces.

If there is an existing file, the Maya Mel Munger will create a backup of that file. Backups are saved in the following location:

<Maya_Install_Directory>\scripts\Gamebryo Backup
