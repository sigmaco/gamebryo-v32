Mesh Packing Profile Options

You can use the Mesh Packing Profile Options dialog box to select a default mesh profile for the scene. This is the mesh profile that every mesh will use, unless the mesh has a mesh profile attribute.

Mesh Packing Profile Options

When you click the Reload Profiles button, the plug-in reparses the %EMERGENT_PATH%\Media\ToolPluginsData\MeshProfiles directory for new and modified mesh profiles. Values in the Mesh Packing Profile drop-down list are also refreshed.
