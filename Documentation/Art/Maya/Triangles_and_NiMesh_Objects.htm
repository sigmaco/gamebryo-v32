Triangles and NiMesh Objects

NiMesh objects are Gamebryo's logical groups of triangles. Each NiMesh object incurs a fixed overhead when it is transformed and rendered. Hence, the aim is to keep the NiMesh object count as low as possible while still obtaining the desired visual result.

One way to reduce the number of NiMesh objects is to collapse meshes with the same materials that are close together in a scene. This collapsed mesh will be converted to a single NiMesh object instead of several separate NiMesh objects. The Scene Graph Optimization Tool Plug-in can perform this automatically if the meshes are all children of the same group.

It should be noted that putting all the triangles into one NiMesh object could prevent culling and create a clipping bottleneck.
