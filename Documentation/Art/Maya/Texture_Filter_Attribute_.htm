Texture Filter Attribute

The Gamebryo Maya Plug-in can customize the texture filter options of Maya textures: Phong, Lambert, or Blinn. Select the material, and click the Texture Filter button. By doing this step, you can override the global texture filter setting found in the export options. In the Attribute Editor, you can adjust the "Extra Attribute" to select "Bilerp", "TriLerp", "Anisotropic", or any of the other Gamebryo texture filter types.
