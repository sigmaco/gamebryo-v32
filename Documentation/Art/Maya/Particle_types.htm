Particle types

The Gamebryo Maya plug-in supports both billboard and instanced geometry particles.  "Max Count" indicates the maximum number of particles that can be in existence at any one time. "Lifespan" is the maximum age of any particles while "Lifespan Random" will modify the maximum particle age for each particle.

Maya Particle Attributes:

  • Max Count
  • Lifespan
  • Lifespan Random

Billboard Particles

The Gamebryo Maya Plug-in creates billboard sprite particles that normally point toward the camera. "Sprite Scale X,Y" will set the default size of each sprite. Gamebryo only supports uniform scale. As a result, X and Y should be the same value. If these values are not the same, an error will be shown in the Maya output window during export.

Maya Billboarded Particles Attributes:

  • Sprite Scale X,Y

Geometry Particles

The Gamebryo Maya plug-in will detect geometry associated with a particle system and export it accordingly. Please note that using Geometry particles are much more expensive then Billboard particles.
