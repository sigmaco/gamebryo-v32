Custom Shaders

The Shaders can be viewed in the Gamebryo viewers and exported directly in NIF files. Shaders will not be visible in the Maya render views. From within the NiMultiShader, you will be able to view and adjust any Shader custom attributes.

Parameters

This section is populated by all non-texture attributes. The control that is used is dependent on what type of attribute is desired.

Textures

This section contains all the textures that the shader requires. The "Texture" field accepts links to Maya texture nodes. The texture option frame layout is used to separate the options for each of the individual textures. See the section on texture options for more information.
