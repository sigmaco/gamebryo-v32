Launching the Gamebryo Remote Viewer

When the Start Remote Viewer button is pressed, the following dialog will appear. The Platform drop down allows the user to select a platform (e.g. PC, Playstation 3, or Xbox 360). Once the platform is selected, the available targets of that type are listed in the Target drop down. Press the OK button to launch the viewer on the specified target.

Start Remote Viewer dialog

Launch Settings Descriptions

Setting Description
Show Logging Window If the Remote Viewer does not successfully launch, use the logging window and Maya Output Window to view status and error messages that may be generated while starting the viewer. This is especially useful when launching the viewer on console dev kits.
DX Settings Dialog Enabling this option will open a Windows-specific renderer settings dialog before the Remote Viewer is initialized. This dialog allows the user to set many options such as window size and DX9 or DX10. Refer to Renderer Settings Dialog in Gamebryo Documentation for additional information.
Note: This setting only affects the PC viewer.
Window Always on Top If this checkbox is set, the Remote Viewer window will always be placed on top of other windows on the desktop, including Maya.
Note: This setting only affects the PC viewer.
