Gamebryo Remote Viewer Options

The Remote Viewer's runtime behavior can be changed using the options dialog. This dialog is launched from the Gamebryo Particle shelf. These settings take effect immediately after the OK button is pressed for already running viewers and for new viewers when they are launched.

The Playback settings represent how time and animations are handled by the viewer. The Remote Viewer will display the time range defined by the Maya Range Slider. The artist can use the slider to set the playback range start and end times.

The Range Slider can be enabled in the Maya Windows, Settings/Preferences, Preferences, Interface, UI Elements dialog. The Remote Viewer will only display animations within the Maya Playback range frames.

Remote Viewer Options Dialog

Playback Options Descriptions

Setting Description
Playback Mode This setting controls how animations are played by the Remote Viewer:
  • Once — Play the animations in the scene once and then stop
  • Loop — Continuously restart the animation between the Maya Range Slider Start and End frames
Simulate Particles past Playback End Time This setting is only available when Playback Mode is set to Once. When this option is checked, all the particle systems in the scene will continue to simulate past the Maya Range Slider End Time. This setting will cause particle systems to simulate forever.
Restart Playback after Any Change If this option is checked, all the animations and particle systems in the scene will restart beginning at the Maya Range Slider Start Time whenever any change is made to the scene. This includes moving the camera.
Note: If the Run Up Particles after Edit option is enabled, then particle systems will not restart, but other animations will.
Run Up Particles after Edit When this option is checked, particle systems will appear as a continuous stream of particles as they are modified in Maya. When this checkbox is not checked, particle systems in the Remote Viewer will restart when they are modified in Maya.

Miscellaneous Options

Setting Description
Add Default Lights Adds lights to the scene displayed in the Remote Viewer.
Hide Viewer UI Turns off the blue UI on the Remote Viewer.
Note: This does not hide the wait icon that indicates the scene is being updated.

Example Option Uses

If editing the walk sequence of a character that has footstep particles, enable the following settings:

  • Playback Mode = Loop
  • Simulate Particles past Playback End Time = Not applicable
  • Restart Playback after Any Change = Unchecked
  • Run Up Particles after Edit = Unchecked

If editing a continuous particle system, like a fountain or fire, enable the following settings:

  • Playback Mode = Once
  • Simulate Particles past Playback End Time = Checked
  • Restart Playback after Any Change = Unchecked
  • Run Up Particles after Edit = Checked

If editing an explosion, enable the following settings:

  • Playback Mode = Once
  • Simulate Particles past Playback End Time = Unchecked
  • Restart Playback after Any Change = Checked
  • Run Up Particles after Edit = Unchecked
