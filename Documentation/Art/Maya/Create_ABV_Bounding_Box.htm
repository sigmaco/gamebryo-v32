Create ABV Bounding Box

You can click the Create ABV Bounding Box button to generate ABVs automatically in Maya. More specifically, this command will create an axis-aligned minimum bounding box around the object selected in Maya. The bounding box geometry will also have appropriate ABV "Extra Data" added, which will specify a "BOX_BV" collision bounding volume and a propagation mode of "PROPAGATE_NEVER".

Create ABV Bounding Box button

With this default set up, if something collides with the collision bounding volume (the box), the collision detection system will consider it as a collision with the selected object, and it will stop traversing downward in the scene graph for more collisions. The properties can be changed on a per-object basis using the Create Collision Attributes button on the Transform Attributes.
