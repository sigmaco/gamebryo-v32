General Animation Notes

Gamebryo supports Inverse Kinematics (IK) animation out of Maya by automatically baking the controlled joints during export. These joints will be baked from the animation start time to the animation end time on Maya's range slider.

Animation Looping Controls button

Animations fall into two categories: looping and non-looping. By default, the Gamebryo Maya Plug-in creates all animations as looping. However, this choice is not always desirable. If you need to change the default animation behavior, click the Animation Looping Controls button and then modify the looping behavior as necessary. Gamebryo will create special "Extra Attributes", which the exporter will use to determine the loop settings.

Animation Looping

Use this field... To...
Looping Type Specify the type of looping to apply: Gamebryo supports three different looping types:
  • Loop Animation—animations are in constant motion, looping from the beginning to the end of the animation
  • Reflecting Loop—animations play once. Before the first key time, their values are consistent with the first key value. After the last key time, their values are consistent with the last key value.
  • No Looping—loops continuously, first playing forward, then reflecting backwards
Propagate to Children Propagate the looping type down to all children lower in the scene graph.
