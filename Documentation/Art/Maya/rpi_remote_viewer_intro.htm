Gamebryo Remote Viewer: Introduction

The Remote Viewer renders the current scene being edited using the Gamebryo renderer. This tool can be launched from the Maya Gamebryo Particle shelf for any supported platform that the user has a valid Gamebryo license for. In addition, it can run on multiple platforms simultaneously. The Remote Viewer does not require restarting between edits, which results in faster visual feedback (and lower iteration times) than other Gamebryo tools.

The Remote Viewer allows an artist to quickly see how the scene will look when rendered on different consoles and to iterate on particle systems in real-time. In addition, the Remote Viewer can be used as a rapid scene viewer for editing other scene entities such as meshes or materials. When editing non-particle system nodes, the Refresh button must be pressed to export and update the scene in the Remote Viewer.

The Remote Viewer uses TCP port 13215. If the viewer is being run on a different workstation or a console development kit, it may be necessary to add an exception for this port to the Maya workstation’s firewall.
