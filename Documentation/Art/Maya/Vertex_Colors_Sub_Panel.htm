Vertex Colors

Use this field... To...
Override Default Value Specify how vertex properties will be applied to objects with this material. If you select (check) this option, new vertex properties will be created for objects with this material. However, there is an overhead cost associated with changing the vertex properties during rendering. If you deselect (uncheck) this option, the objects will use the shading model that is currently used by the parent. We recommend that you leave this unchecked unless necessary.
Source Mode Specify the source mode. Options include the following:
  • Ignore—The emissive, ambient and diffuse are all specified by the corresponding material color
  • Emissive—The material emissive color is replaced by the source vertex color
  • Ambient and Diffuse (Default)—The material ambient and diffuse colors are both replaced by the source vertex color
Light Mode Specify the light mode. Options include the following:
  • Emissive Only—Only the emissive component is used to light the object
  • Emissive, Ambient, and Diffuse—All of the values are used to light the object
