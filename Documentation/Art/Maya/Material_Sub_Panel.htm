Material Panel

You can use the Material sub-panel to set properties that affect material colors, shininess, shading, and texture.

Material sub-panel

Use this field... To...
Material Diffuse, Specular, Ambient, and Emittance Color Control the material colors of the object. These colors are used in conjunction with the different textures.
Note: The Diffuse color will modulate and darken your base map if you have one. To get pure texture color, the diffuse color should be white when a base map is loaded. It should be noted that changing the specular color will add a specular property, which incurs a performance penalty.
You can animate these values by selecting the material and setting keys for Color, Material Specular, Material Ambient, and Material Emittance in the channel box. Additionally right clicking in the attribute editor will allow the user to select the Set Key option from the context menu.
Material Shininess Set the shine or "gloss" of the material. Higher values tend to make the material appear to have a more mirror-like finish (tighter specular highlights).
Material Alpha Set the material alpha. 0.0 represents fully translucent; 1.0 represents fully opaque. You can animate this value by selecting the material and then keying the Out Transparency in the Channel Box. Animated Material.mb shows an example of animated material values
Shading Choose from the three shading settings that the Gamebryo Maya Plug-in supports:
  • Default
  • Flat Shading
  • Gouraud Shading
Shading can be set on a per material basis for this object. The Default setting will use the shading model currently used by the parent. There is an overhead cost associated with changing the shading mode during rendering, so it is best to use "Default" unless necessary.
Texture Apply Mode Determine how the vertex colors (whose generation are controlled via the NiVertexColorProperty) are blended with the filtered texture color.
  • Replace—Basic texturing without any dynamic or static vertex-based lighting
  • Decal—Decal or bullet-hole textures on a base map.
  • Modulate—Texture and final vertex colors and alpha values are multiplied component-by-component. (Default)
