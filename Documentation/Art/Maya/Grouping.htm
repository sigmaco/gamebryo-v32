Grouping

Whenever collections of objects in Maya are grouped, the Gamebryo Maya Plug-in adds another NiNode object to the scene graph as the "group" object. This additional node adds run-time overhead to rendering the scene graph. Try to use groups only when they are needed for animation or some other run-time purpose. Avoid them for static groups of objects.

If groups are used that are not needed for animation, the Scene Graph Optimization Tool Plug-in should be able to collapse the hierarchy.
