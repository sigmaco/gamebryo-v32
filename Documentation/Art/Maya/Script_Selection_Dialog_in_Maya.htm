Script Selection Dialog in Maya

When you use the Gamebryo Maya Plug-in to export a scene, it presents the Gamebryo Script Selection dialog. Use this dialog to select the export script that you want to use during export. Alternately, you can launch the Script Editor dialog from within the Script Selection dialog. Use the Script Editor dialog to create or edit scripts.

Script Selection dialog

The Script Selection drop-down at the top of the dialog displays the name of the currently selected script. Below that is a list of all the tool plug-ins that the selected script executes.

If you want to customize your export pipeline, you can click the Script Editor button.
