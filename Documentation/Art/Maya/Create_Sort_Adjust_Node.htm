Create Sort Adjust Node

After you select a transform, click the Create Sort Adjust Node button to add an attribute called NiData_NiSortAdjustNode to the transform.

Create Sort Adjust Node button

A new section in the interface called Sort Adjust displays. In this section's Sort Adjust Node drop-down, choose from one of the following options:

  • Sorting Off—the object and its children will not get sorted and will be drawn immediately
  • Sorting Inherit—the object is sorted as if it were a regular NiNode
