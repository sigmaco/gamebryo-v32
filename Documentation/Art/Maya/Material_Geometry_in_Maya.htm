Geometry Panel

Use the BiNormals & Tangents drop-down to choose the method for generating BiNormal and Tangent information for all objects with this material. If this option is left at the default value of "Auto" but either a normal or parallax map is assigned or a Gamebryo Shader that specifies an NBT method is selected then NBT frames will be generated.

Geometry panel

If you specify an NBT method in this field, it will override the NBT method selected by a Gamebryo Shader.

If you are using a normal map that was generated in Maya, then we recommend that you select the Maya NBT method. This method should reduce UV seam issues, and it should prevent you from having to invert the green color channel.
