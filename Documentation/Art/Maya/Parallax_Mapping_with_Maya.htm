Parallax Map

Parallax maps are height field textures that are used to offset the lookup for other textures using the same UV coordinate set. This offset is meant to give the illusion of increased surface detail where there is none. In this way, parallax maps appear to provide the same visual results as a displacement map. However, no new triangles are created. Instead, the view vector is used to adjust the UV mapping so that it follows the contours of the height field.

The following attribute is created when using the parallax map:

  • Offset—A scalar value that adjusts the amount that the pixels are offset.
