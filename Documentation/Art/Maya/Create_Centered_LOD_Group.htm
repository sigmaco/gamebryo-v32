Create Centered LOD Group

One of the differences between Gamebryo and Maya comes with the function of LOD groups .  In Gamebryo, the LOD group uses the distance from the camera to determine which child to display. When Maya creates a LOD group, by default, it is created at the origin with the LOD levels offset from there.  After export, it is the distance to the LOD group (i.e. origin) that is used in determining the current level of detail.  Create Centered LOD Group will create a Maya LOD group, but the group will be centered in the middle of the children.  This setting removes a typical stumbling block many people encounter when using LOD.

Create Centered LOD Group button

Range LOD uses the range from the camera to the LOD node to determine which LOD level to be displayed. The Range values are determined by the Threshold attributes in Maya's LOD group.

LOD thresholds

Using the Create Centered LOD Group icon will automatically create the necessary "Extra Attributes" for Screen LOD. Screen LOD uses a size of the objects bound on the screen to determine which LOD level to display. In the example SwordScreenLOD.mb, the highest detail LOD will be displayed when sword bound is 30% (0.3) the size of the screen. The medium detail LOD will be displayed if it the bound is larger then 10% (0.1) of the screen size. When the bound is smaller then 10% (0.1) the lowest detail LOD will be displayed.

Screen LOD

Note

To use Screen LOD, you must check "Ni_Use_ScreenLOD". This attribute is NOT checked by default.
