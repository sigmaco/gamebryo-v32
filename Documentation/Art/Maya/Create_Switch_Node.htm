Create Switch Node

You can use Gamebryo to create switch nodes. Switch nodes are special nodes within the scene graph that only display one of their children based on an internal value. To create a Switch node, select the children that will be part of this group and click the Create Switch Node button.

Create Switch Node button

Most often you use switch nodes for holding the different damage stages of an object. For example, you may create three different damage levels for an object. The game programmer with whom you are working will be responsible for setting the child to be displayed. When you create a switch node, a group will be added to Maya and a number of "Extra Attributes" will be added to the group.

Use this field... To...
Enable Switch Tag this group so that the exporter will export it as a switch node. This value is stored in the attribute Ni_SwitchNode.
Update All Define how this node will update. In general, only the active child of a NiSwitchNode needs to be updated because only the active child is visible. Checking this attribute will cause all of the children to be updated. The default value is OFF for performance reasons because in general it is pointless to update the children you can't see. This value is stored in the attribute Ni_SwitchUpdateAll.
Child Index Specify which child is the visible child. Setting the index to 1 indicates the first child will be visible. This value is stored in the attribute Ni_SwitchChildIndex.
