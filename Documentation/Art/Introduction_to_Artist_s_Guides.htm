Artist Level

The Artist Level includes information for artists who are new to Gamebryo LightSpeed and for those who have used previous Gamebryo versions. It includes conceptual information about the production pipeline and the artist's role within the pipeline. It also includes task and reference documentation that explains how to use Emergent's digital content creation plug-ins to increase artists' speed and efficiency.

This level also includes information about how to use scripts to customize facets of the production pipeline, and it includes a reference section that lists the shaders that ship with Gamebryo.

Artist tool details are outlined in the following sections:

  • Gamebryo 3ds max Plug-in
  • Gamebryo Maya Plug-in
  • Gamebryo XSI Plug-in
  • General Tool Pipeline Topics
  • Gamebryo Shader Reference
