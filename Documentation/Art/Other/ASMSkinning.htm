ASMSkinningLighting

<ASMSkinning.NSF>

Requirements

  • NSF Shader Library
  • Implementation VS11:
    • Vertex Shader 1.1
  • Implementation VS11_PS11:
    • Vertex Shader 1.1 and Pixel Shader 1.1

Description

This shader performs skinning in the vertex shader using 20 bones. It uses assembly shaders referenced by an NSF shader.

ASMSkinningLighting is not as optimized as ASMSkinningNoLighting, but it can calculate normals to perform lighting calculations. This shader applies a base map lit with a single light.

ASMSkinningNoLighting uses a more optimized method than ASMSkinningLighting by removing a CPU per-bone matrix multiplication, but it cannot perform any lighting calculations. This shader does apply a base map without lighting.

Art Requirements

  • Skinned model
  • Base map (Using first UV set)
  • One light of any kind

There are no artist editable variables in this shader.
