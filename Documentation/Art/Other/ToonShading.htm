ToonShading

<ToonShading.NSF>

Requirements

  • NSF Shader Library
  • Vertex Shader 1.1 and Pixel Shader 1.1

Description

ToonShading is a modified version of LuminanceTransfer that produces cartoon lighting. It uses two dynamic lights and a base map before creating the toon effect.

Art Requirements

  • Base map (Using first UV set)
  • Two lights of any kind

Custom Attributes

  • ToonRamp—The 1D texture ramp that maps luminance to a color. This works with toon shading when only one or two colors are used in the ramp.

Global Variables

There are no global variables for this shader.

Object Variables

  • Light1—A generic light whose diffuse color and direction are used to light the model.
  • Light2—A generic light whose diffuse color and direction are used to light the model.
