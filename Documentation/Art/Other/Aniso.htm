Aniso

Programmatic shader from ShaderLibrary.nl*

Requirements

  • ShaderLibrary.nl9 or ShaderLibrary.nlx
  • Vertex Shader 1.1
  • Vertex Shader 1.1 and Pixel Shader 1.1 on Xenon

Description

Aniso is a programmatic shader that is stored in the ShaderLibrary library, which is a part of the ShaderSample sample. It performs a simple anisotropic lighting effect.

Art Requirements

No additional art requirements beyond the Custom Attributes.

Custom Attributes

  • AnisoTexture - This is the texture that is used to achieve the anisotropic effect.

Global Variables

There are no global variables in this shader.

Object Variables

There are no object variables in this shader.
