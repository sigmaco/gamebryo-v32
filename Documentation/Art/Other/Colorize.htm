Colorize

<Colorize.NSF>

Requirements

  • NSF Shader Library
  • Implementation VS11_PS11:
    • Vertex Shader 1.1 and Pixel Shader 1.1
  • Implementation VS11:
    • Vertex Shader 1.1

Description

Colorize is a programming sample shader. It takes the color from the custom attribute and modulates it by the base map.

Art Requirements

  • Base map. (Using first UV set)

Custom Attributes

  • SelectedColor - This is the color that will be modulated with the base map.

Global Variables

There are no global variables in this shader.

Object Variables

There are no object variables in this shader.
