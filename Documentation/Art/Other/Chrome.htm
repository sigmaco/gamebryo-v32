Chrome

<Chrome.fx>

Requirements

  • FX Shader Library
  • Implementation Chrome:
    • Vertex Shader 1.1 and Pixel Shader 2.0
  • Implementation ThreePassChrome:
    • Vertex Shader 1.1 and Pixel Shader 1.1

Description

This shader produces a chrome effect using a base map, diffuse cube map, specular cube map, and normal map (that contains gloss in alpha channel). There are multiple versions of this particular effect, including skinned and static versions of the geometry.

Art Requirements

  • Base map (Using first UV set)
  • Normal, binormal, and tangent arrays (automatically calculated during export)

Custom Attributes

  • NormalMap - The normal map texture in tangent space. The glossiness of the specular component should be stored in the alpha channel of this texture. (Using first UV set)
  • LightingMap_cube - The cubic environment map used for diffuse lighting.
  • ReflectionMap_cube - The cubic environment map used for specular lighting.
  • bumpiness - Scalar value used to bias the normal map.
