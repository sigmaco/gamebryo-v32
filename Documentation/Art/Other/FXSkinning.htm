FXSkinning

<FXSkinning.FX>

Requirements

  • FX Shader Library
  • Vertex Shader 1.1 and Pixel Shader 1.1

Description

Minimal FX-based skinning shader with 20 bones. Like ASMSkinningNoLighting, it is not capable of transforming normals to perform lighting calculations.

Art Requirements

  • Skinned model
  • Base map (Using first UV set)

Custom Attributes

There are no artist editable variables in this shader.
