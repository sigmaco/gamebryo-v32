PerPixelLightingPS

<PerPixelLighting.NSF>

Requirements

  • FX Shader Library
  • Vertex Shader 1.1 and Pixel Shader 1.1

Description

This shader applies a base map and performs per-pixel lighting using a normal map and a single point light. This shader differs from the PerPixelLightingNoPS shader in that it uses a pixel shader to perform the texture lookups. It does not add in the ambient lighting component.

Art Requirements

  • Base map (Using first UV set)
  • Point light
  • Normal, binormal, and tangent arrays (automatically calculated during export)

Custom Attributes

  • normalMap—The tangent-space normal map for this shader. (Using first UV set)

Global Variables

  • AmbientIntensity—Global scalar for ambient light intensity.

Object Variables

  • MSLightPos—Model-space light position.
