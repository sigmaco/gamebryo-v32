LuminanceTransfer

<LuminanceTransfer.NSF>

Requirements

  • NSF Shader Library
  • Implementation TransferWithBaseMap:
    • Vertex Shader 1.1 and Pixel Shader 1.4
  • Implementation TransferWithMaterial:
    • Vertex Shader 1.1 and Pixel Shader 1.1

Description

The LuminanceTransfer function demonstrates how to implement a variety of techniques at the object level with a simple luminance conversion and a texture lookup of a 1D ramp texture based off of the luminance value. The default texture, IRRamp.bmp, simulates an infrared effect for the shader. The 1D ramp could be used to get sepia tones, night vision effects, or toon shading.

There are two implementations for this shader. TransferWithBaseMap uses the base map in the luminance calculation and hence would work better for something like toon shading or sepia tones. TransferWithMaterial simply uses the material in the lighting calculations and is better suited for effects like infrared. Both shaders use vertex shader version 1.1.

TransferWithBaseMap uses pixel shader 1.4 since it does a true dependent texture read in the pixel shader after the luminance conversion. TransferWithMaterial uses pixel shader 1.1.

Art Requirements

  • Implementation TransferWithBaseMap:

    • Two lights of any kind
  • Implementation TransferWithMaterial :
    • Two lights of any kind

Custom Attributes

  • TransferRamp—1D texture map used as lookup ramp based off of the luminance of a pixel.

There are no global variables for this shader.

Object Variables
