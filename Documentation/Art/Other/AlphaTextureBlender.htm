AlphaTextureBlender

<AlphaTextureBlender.NSF>

Requirements

  • NSF Shader Library
  • Implementation FixedFunction:
    • Hardware capable of 3 textures per pass
  • Implementation Programmatic:
    • Vertex Shader 1.1 and Pixel Shader 1.1

Description

The Alpha Texture Blender shader blends between two textures using the alpha channel of a third texture. The value in the third texture's alpha channel represents the fraction of the second texture's color value to use. Thus, an alpha channel of all 1.0 would produce only the second texture while an alpha channel of 0.0 would produce only the first texture.

Art Requirements

No additional art requirements beyond the Custom Attributes.

Custom Attributes

  • Texture1 - The first texture to blend. This should use the first UV set.
  • Texture2 - The second texture to blend. This should use the second UV set.
  • AlphaBlendMap - The alpha channel of this image indicates what percentage of Texture2 to blend on top of texture 1. This should use the third UV set.

Global Variables

There are no global variables in this shader.

Object Variables

There are no object variables in this shader.
