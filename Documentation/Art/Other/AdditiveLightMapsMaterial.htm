AdditiveLightMapsMaterial

<NiCommonMaterial>

Requirements

  • NiCommonMaterial Library
  • Implementation PerPixel:
    • Pixel Shader 2.0
  • Implementation PerVertex:
    • Vertex Shader 2.0

Description

This additive light maps material enables projected light maps (up to 3) to be added in after the base map has been applied, with the result that projected lights lighten instead of darken the base map.

Other than this difference, the texturing pipeline remains essentially the same as that of NiStandardMaterial, since this additive light maps material is derived from the standard material. Thus, it supports texture transforms, lighting, etc. See the NiStandardMaterial texturing pipeline for more details.

More specifically, projected light maps in the Gamebryo standard material act like dark maps. Projected light maps are multiplied with the base texture. The resulting combination is then multiplied by vertex colors or dynamic vertex lighting to get the final lit result. As with dark maps, if a portion of the projected light map is black, that area of the final result will be black, and where the projected light map is white, the base texture will show through. However, this additive light maps material alters the behavior by resequencing when projected lights are added in, with the result that projected lights lighten instead of darken.
