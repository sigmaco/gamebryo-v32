Changing The Export Script in 3ds Max

You can use export scripts (in conjunction with process scripts) to customize and automate how the Gamebryo 3ds Max Plug-in exports your Max art assets. For example, you can use a script to export a NIF file to disk. You can use another script to export keyframes into a KF file before exporting a NIF file.

To change the export script

  1. Make sure that the Gamebryo Maxscript Toolbar is loaded and available in 3ds Max.

    See Gamebryo MaxScript Toolbar Introduction.

  2. In 3ds Max, create a teapot (or any other object). You should now have something similar to the image below.
  3. Click the Export to Gamebryo button (see image below).
  4. In the Export File dialog box, type the name of the file that you want to save and then click Save.
  5. In the Export Scene to NIF File dialog box, click the Edit button beside the File Export Script drop-down.
  6. In the Script Editor dialog, click the New button next to the Current Script (Template File Name) drop-down.
  7. In the New Script dialog, type the name of the script and then click OK. (In this example, we chose to call the export script "Tutorial Export Script.)

    In the Script Editor dialog, the name of the script that you just specified in the New Script dialog appears in the Current Script (Template File Name) drop-down.

  8. In the Script Editor dialog, double-click NIF Exporter to add the tool plug-in to the Plugin List for Current Script listbox.
  9. In the Script editor dialog, click the Save button. In the Save Script dialog that appears, click Save.
  10. In the Script Editor dialog, click OK.
  11. In the Export Scene to NIF File dialog, click Save to export the file the NIF, using the settings that you just specified in the export script.
