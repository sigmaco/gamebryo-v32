Multi-Sub Object Materials and Shell Materials

Multi-Sub Object Materials

Gamebryo cannot create multiple materials per NiMesh object. This important note must be made about Multi-Sub materials. Each Sub-material becomes its own NiMesh. In the image below, there are 10 sub-materials. Upon conversion to Gamebryo, this material will be converted into 10 NiMesh objects.

Multi-Sub Object Materials

Be careful about how you use Multi-Subs. In many cases, careful UV coordinate placement within a single material will allow you to bypass the need for another sub-material.

It is a good idea to look at the scene graph after export when using multi-sub objects. If you see "NiMesh 'MY_MODEL' " plus a number repeated several times at the same level in a hierarchy, you have serious problems with performance due to multi-sub objects.

Shell Materials

Gamebryo supports 3ds max shell materials by exporting the Baked Material, in all cases.

Shell Material Parameters

Advanced Lighting Override Material

Gamebryo ignores the settings in the Advanced Lighting Override Material and exports using the Base Material listed in the panel.

Other Max Materials

Most post rendering materials like Composite, Double Sided, Ink n' Paint, Morpher, Raytrace, Shellac and Blend are not supported.
