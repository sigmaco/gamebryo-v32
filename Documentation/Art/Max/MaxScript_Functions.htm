MaxScript Functions

GamebryoExport PathToFile

This will bring up the export dialog initially pointing to the PathToFile file. Example: GamebryoExport C:\temp\testfile.nif

GamebryoQuickSave

This will save the max file and export the file to the same folder with the same name but different extension.

DoGamebryoView WhichViewer

This will convert the scene and launch the corresponding viewer. The WhichViewer option values are:

Value Description
-1 The last viewer that was run from the Gamebryo Utility
0 This launches the DirectX-based viewer
1 This launches the SceneViewer application
2 This launches the AnimationTool application
4 This launches the Xbox 360 viewer

GetSourceForShader stringShaderName

This returns a string that defines the custom attributes for the input shader name.
