Custom Attributes for Your Custom Shaders

When the user presses the Apply button on the Custom Shader panel in the Gamebryo Shader, additional user interface items can be added as custom attributes to the current material. The following image is a sample of the Custom Attributes that can be added by a shader:

An Example of Custom Attributes

All of the custom attributes shown above are animatable and will be properly exported on the geometry on which this material is applied. The effects of manipulating these variables are not directly visible in the Max viewports. Instead, you will have to view the effects using one of the Gamebryo viewers available in the Gamebryo Utility plug-in.
