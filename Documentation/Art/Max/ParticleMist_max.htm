ParticleMist.max

The ParticleMist.max samples uses a Max Super Spray particle system to create a mist effect. Gravity and deflector space warps are used to cause the particles slowly float down then hover on a plane.

The material applied to the particle system uses a chaotic monochrome texture and gets the coloring through vertex colors. The vertex colors change over the lifetime of each particle according to the Particle Age map in the material's Particle Color map slot. The vertex colors are set to E for emissive and alpha blending is set to Additive. Because additive alpha blending is used, the Particle Age map uses black as the color at times when particles should fade out.

The scene also contains hidden geometry that can be used for picking or collision detection.
