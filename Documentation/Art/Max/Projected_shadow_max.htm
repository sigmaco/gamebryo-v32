Projected shadow.max

The projected shadow.max sample demonstrates a projected shadow texture. Note that only the objects that are included in its light Include/Exclude dialog receive shadows.

projected shadow.max sample

This sample demonstrates how a projected shadow map can be used with normal dynamic lighting. The Direct light named "Projected Shadow Map" casts a shadow map onto Plane01 (using an inclusion list). The shadow map is intended to mimic the shadow that would be cast by Torus01 as it is animated along the path Circle01. The "Projected Shadow Map" light is animated so that the shadow map correctly tracks Torus01.

Note that the "Projected Shadow Map.Target" is used to point the "Projected Shadow Map" light at Torus01 and a roll controller is used to rotate the shadow map appropriately about that axis.
