Adding Custom Content to 3ds Max

Custom Attributes

Gamebryo supports the max custom attributes system through our NiExtraData derived classes. The custom attribute system is an extremely powerful way to add custom information to any max object, including materials and nodes. The ability to do add such custom information can turn 3ds max into a very flexible editor for your game-specific options. This data may also be used to drive your game-specific plug-ins to the art pipeline.

The image listed below shows a sphere object's modifier panel with custom attributes added. The attributes were added via a Maxscript utility plug-in.

Take a look at the MaxScript help file distributed with 3ds max under the section "Scripted Custom Attributes" for more details.

User-Defined Properties

This approach is the traditional method for entering additional information into your 3ds max data. Have your artists or generate user interfaces that enter in the text to the user-defined properties panel of any max object. This data will be stored in a NiStringExtraData object called "User-Defined Properties".

Parameter Editor

Another way to add custom attributes is to use the Parameter Editor tool built into Max. This tool can be found in the animation menu and is an easy way to get named and animatable extra data into Gamebryo without having to write script. This method has the advantage of giving the artist a UI for editing the parameters that matches the type of data they are authoring. Consult the 3ds Max user's manual for more information on using the parameter editor.
