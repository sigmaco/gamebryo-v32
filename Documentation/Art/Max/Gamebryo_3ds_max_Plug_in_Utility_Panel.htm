Gamebryo 3ds Max Plug-in Utility Panel

The Gamebryo 3ds max Plug-in panel gives the user several exporting and viewing controls. It provides the user with information about what version of the plugin is running, whether it is running in debug or release mode, what version of Gamebryo it is using, and when it was last built. The additional rollouts and their functionality is covered in the following sections:

Quicksave Panel

Viewer Selection Panel

Options Panel

Default DX9 Viewer User Interface

If you don't see the Gamebryo 3ds max Plug-in in the Utilities rollout, click the "More..." button or click the "Sets" button to add it to your rollout.
