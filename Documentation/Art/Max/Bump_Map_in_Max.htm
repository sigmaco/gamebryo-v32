Bump Map in Max

The Bump Map slot is used to offset texture lookups into a reflective environment map. The map works best with a monochrome image representing a height map with black representing the low heights and white representing the high heights. The maximum height of the bump map is set by the 'Magnitude' setting in the Bump mapping sub-panel.

Bump maps in Gamebryo do not have the same effect as bump maps in Max. In Max, the bump maps are used to get per-pixel lighting, but in Gamebryo, this effect is accomplished by applying a Normal Map.
