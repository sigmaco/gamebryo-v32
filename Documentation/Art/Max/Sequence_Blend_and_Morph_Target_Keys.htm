Sequence Blend and Morph Target Keys

In addition to the "start" and "end" keys, you may want to specify target keys that indicate when to start a blend or a morph for a sequence. Please see the Gamebryo Animation Tool documentation for more information about how to choose between using blends and morphs.

Specifying Blend and Morph Text Keys

Blend target text keys are used to tag the frames in each animation that will be blended to or from. You can indicate a blend target with any note track text key. It is a good idea to tag frames in the animations with text keys that indicate the position of the character, such as "right foot down" or "knees together". Then, when specifying a blend transition in Gamebryo Animation Tool, you can match up like positions to minimize visual artifacts in the blend.

Morph targets that indicate frames at which the animations are "in sync" should be tagged in both of the animations to be morphed. For example, it may be a good idea to set morph targets when the right or left foot is down and the knees are together. A note track text key is designated as a morph target by the word "morph:" preceding the name of the key, for example "morph: knees together". The text of the key must keep the ":" after the word "morph" for it to work. Morph targets must occur in identically named pairs in the two animations. At least one pair of morph targets must be present in order to specify a morph transition in the Gamebryo Animation Tool. All defined morph targets in the animations are used when performing a morph, so it is not necessary to specify which ones to use in the Transitions dialog of the Gamebryo Animation Tool.
