Environment Maps

Environment maps can be used to simulate the reflection of a static environment off of a shiny object or to show different kinds or colors of metal. They can also be used in combination with gloss maps to specify where the reflections appear (i.e. scuffs on a shiny object will not reflect the environment as much as the polished parts).

There are two methods for specifying environment maps in Gamebryo: the Reflection map slot on a material or by overloading a Max light's settings. This document will detail how to adjust a light's settings so that the exporter will use it as an environment map.

First make the Bitmap you want reflected or cast. Load it into an Omni light as the Map texture under the Shadow Parameters rollout. You will need to turn the effect On. You will also need to set the Light Affects Shadow Color to on.
