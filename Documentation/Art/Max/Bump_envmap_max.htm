Bump envmap.max

This 3ds Max sample includes a shiny, bumpy teapot that uses an environment map (envmap04.bmp) that is modulated by a black and white bump-map image (BRKWEA_B.bmp). This model was created using the standard primitive Max teapot. Brkwea.JPG is used in the base map slot, while grayscale.tga is an r,g,b map without alpha and is used in the bump map slot. An environment map (envmap04.bmp) is assigned to Omni 01. By creating a texture with various amounts of gray and also a solid area of white, representing high, flat areas, we were able to create a pot with both smooth surfaces and bumpy ones, with a subtle brick pattern shining through the smooth areas.

Note that the bump map cannot be seen without using the environment map. The environment map appears in the white area of the bump map texture and shines on top of the base texture. However, the environment map disappears in the varying gray regions of the bump map. The environment map effect is added via the Direct Light "Env Light" by adding a map to the Shadow Parameters->Map field and by checking the Light Affects Shadow Color box. (Keep in mind that you cannot view the final product in the Gamebryo 3ds Max Plug-in viewer unless you have a GeForce3 or higher card.)

By default, this sample installs to the following location:

%EMERGENT_PATH%\Samples\Models\Max Artist Examples\bump envmap.max
