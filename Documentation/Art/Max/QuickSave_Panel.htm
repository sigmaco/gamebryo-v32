QuickSave Panel

You can use the QuickSave panel to quickly export your Max scene to a NIF file. When you click the QuickSave button, the Gamebryo 3ds Max Plug-in saves the current Max scene. It then saves a .nif file in the directory that contains the Max file.

For example, if you are working on a Max scene that's encapsulated in the C:\DataSets\foo.max file, the quick save operation will create C:\DataSets\foo.nif.
