Custom Shader Sub-Panel

When you select a Shader type over the "Default Pipeline" you are overriding any default parameters this material has.

Custom Shader Sub-Panel

Selected Shader

This dropdown list displays all of the known custom shaders. When you select a shader, it is not actually used until the Apply button is pressed.

Description

This text field contains the text description of the currently selected custom shader.

Apply

When this button is pressed, the currently selected shader from the Selected Shader dropdown list will be applied to the current material. This command will add a set of custom attributes at the bottom of the material that provide effect-specific user interfaces.

Reset

When this button is pressed, the currently applied shader's custom attributes are reset to their original values, losing any keys and their current values.

Viewer Technique

This dropdown indicates all of the various rendering techniques available for the currently applied shader. These techniques can be options like wireframe mode, hardware compatibility modes, etc. The viewers will use the currently selected technique.

Export Technique

This dropdown indicates all of the various rendering techniques available for the currently applied shader. These techniques can be options like wireframe mode, hardware compatibility modes, etc. The final source output will use the currently selected technique.
