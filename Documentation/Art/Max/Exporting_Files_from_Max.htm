Exporting Files from Max

Save your Max file as something after you assign all your notes and keys in case you need to re-export later. Then choose File > Export. In the Gamebryo 3ds max Plug-in Export Options dialog box, you can select which export script you wish to run. Make sure that the Controller Extractor is included in the export script before the NIF Exporter plug-in. Several scripts are included that will extract animations to a single KF file or multiple KF files, either saving out a NIF or not saving one out. You can edit the Controller Extractor plug-in settings by clicking the Edit button, highlighting the Controller Extractor plug-in in the script editor dialog, and clicking the Edit Plugin button.
