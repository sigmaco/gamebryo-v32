Gamebryo Shader - Per-Texture Settings Dialog

Note that this dialog is deprecated. You can still access it from the Gamebryo shader for the purpose of backwards-compatibility with older art assets. If you wish to assign texture export settings on a per-texture basis, the proper method is to use the texture conversion custom attributes.

These settings define the default content quality for the maps used by this shader. The possible settings are as follows:

Global Settings: This setting uses the content quality defined by the export dialog.

True Color (32-bit): This setting exports a 32-bit version of the texture.

High Color (16-bit): This setting exports a 16-bit version of the texture.

Compressed: This setting exports a DirectX compressed version of the texture.

Note

These settings can be overridden at export time by checking the "Override Per-Texture Settings" checkbox.
