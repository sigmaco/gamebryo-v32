Texture MipMap Custom Attributes

These custom attributes have been designed to allow artists to create a completely custom mip-map pyramid. These attributes can be assigned by using the Gamebryo toolbar icons discussed in the Gamebryo MaxScript Toolbar Introduction.

Auto-Generate Mip-Maps

This option enables/disables the custom mipmap icons.

Mip-Map Levels

Each level must be specified by a bitmap image of precisely half the prior image size in each dimension. In the case of non-power of two textures, if the dimensions of a level are odd, the result of the division is rounded up. It is possible to view each level using the "View" button.
