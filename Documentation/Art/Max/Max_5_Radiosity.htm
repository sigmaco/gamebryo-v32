Max Radiosity

3ds max's Radiosity feature (found under Rendering / Advanced Lighting) is a great way to calculate complex lighting situations ahead of time and then "bake" that information into the scene geometry by applying the resulting image as a Dark map texture (in the Dark map slot of the Gamebryo Shader) or in the vertex colors.

How to create the Radiosity solution is covered in the Max documentation. Once you have the solution it can be applied by using the "Render to Texture" dialog to create Dark Maps or by using the vertex colors of the object. (By checking the "Use Vertex Colors" dialog in the Gamebryo Shader.)

The drawback to using Dark Maps is it increases the amount of textures you need to use while the drawback to vertex burning is you need to have a fairly tessellated mesh for detailed shadows. If you use vertex colors as your final solution, it is suggested you tessellate your mesh by hand before burning the solution and not enabling the "Global Subdivision Settings" under the Radiosity Meshing Parameters as this can cause extreme tessellation if you are not cautious.

Two examples of this static lighting method are included in the Max samples directory as Radiosity darkmap and Radiosity- after vertex colors.max.
