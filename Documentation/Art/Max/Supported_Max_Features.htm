Supported 3ds Max Features

Note

The following tables are intended to represent the suite of Max functionality that has been tested. Max functionality that is not listed below is untested and is most likely unsupported.

See the Gamebryo 3ds Max Plug-in Release Notes for updates/known issues on specific features.

Animations

Feature Limitations
Float controllers (TCB, Bezier & Linear) Ease-in/out parameters of TCB keys are not supported
Position controllers (TCB, Bezier & Linear) Ease-in/out parameters of TCB keys are not supported. See Position in Animating Position, Rotation, And Scale in 3ds Max for more details.
Rotation controllers (TCB, Bezier & Linear) Ease-in/out parameters of TCB keys are not supported. See Rotation in Animating Position, Rotation, And Scale in 3ds Max for more details.
Scale controllers (TCB, Bezier & Linear) Ease-in/out parameters of TCB keys are not supported. Only uniform scaling supported. If non-uniform scaling is required to be animated, use a morph target. See Scale in Animating Position, Rotation, And Scale in 3ds Max for more details.
Color controllers (TCB, Bezier, & Linear) Ease-in/out parameters of TCB keys are not supported
LookAt Controllers Only Look-At Transform controllers are supported. LookAt constraints are sampled.
Roll Controllers  
Visibility animation Only binary visibility is supported
Note tracks Attached to converted nodes/objects as NiTextKeyExtraData
Out of Range Types Linear and Relative Repeat types Loop in Gamebryo

Hierarchy

Feature Limitations
Full matrix hierarchy is converted All scales except animated scales are pushed down the tree to the geometry.

Animated non-uniform scaling is not supported. Instead, use a morph target modifier. Morph targets can be easily generated from the Tools / Snapshot menu option.
