Bump Map Sub-panel

The Bump Map sub-panel allows the user to set the Magnitude parameter of the bump map applied in this shader.

Bump Magnitude

The Bump Magnitude is the scale factor applied to the bump map image. Changing this value changes the magnitude of the offset into the environment map that the bump map causes. The effect can be seen below. The images represent the extremes of the Bump Magnitude values. The left image is set with a value of 0.0. The right image has a value of 2.0.
