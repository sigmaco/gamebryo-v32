Detailmap.max

Detail maps are r,g,b,a maps on all platforms. The detailmap.max sample adds a detail map to the walls of the darkmap.max sample. It does this by placing a Bitmap shader in the "Walls" material's Detail slot. The detail.bmp image was generated (in Photoshop) by adding noise to a middle gray (128, 128, 128) base image. This approach results in the random noise pattern when the camera is brought close to one of the walls. Middle Grey has no effect on the base texture. Anything lighter than (128, 128, 128) will lighten the base texture; anything darker will darken the base texture. A third "UVW Mapping Modifier" was added to each of the walls to provide a set of UVs for the detail map. The detail map uses this UV channel to repeat sixteen times along each edge of a wall.

By default, this sample installs to the following location:

%EMERGENT_PATH%\Samples\Models\Max Artist Examples\detailmap.max
