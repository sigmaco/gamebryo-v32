Script Selection Sub-panel

Process Script: This dropdown list shows all of the known process scripts. Process Scripts are a set of plug-ins that manipulate the scene graph in some fashion. Provided plug-ins include Stripification, Optimization, and Creating ABVs. Please see the Scripting the Art Pipeline section of this manual for more details. Clicking 'Edit' will bring up the Script Management dialog for the current script.

File Export Script: This dropdown list shows all of the known export scripts. Export Scripts are a set of plug-ins that save the scene to disk. Provided plug-ins include the Controller Extractor and NIF Exporter. Please see the Scripting the Art Pipeline section of this manual for more details. Clicking 'Edit' will bring up the Script Management dialog for the current script.

Use Original Script Templates If Available: When a Max file is loaded and its internal copy of the script is out of sync with the original template, checking this box will use the original template script instead of the temporary version stored in the Max file. Please see the Scripting the Art Pipeline section of this manual for more details.
