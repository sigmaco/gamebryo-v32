Default DX9 Viewer User Interface

The user can define additional keystrokes or change the default key bindings by clicking the "Option" button next to the viewer you are using. This will bring up the options dialog for your particular viewer and by hitting the "Create New Map" button; you can add new map set for custom key bindings under the Configurable UI Map Editor. Gamebryo 3ds max Plug-in ships with a default scheme saved in the file "InputMap.uim" and is specified below. Please see the documentation on Editable User Interface for more information.

Keyboard Input

Keystroke Action
'UP' Translate camera forward
'DOWN' Translate camera back
'LEFT' Translate camera left
'RIGHT' Translate camera right
'PAGE UP' Translate camera up
'PAGE DOWN' Translate camera down
'Control + UP' Orbit camera forward
'Control + DOWN' Orbit camera back
'Control + LEFT' Orbit camera left
'Control + RIGHT' Orbit camera right
'NUMPAD-8' Look up
'NUMPAD-2' Look down
'NUMPAD-4' Turn left
'NUMPAD-6' Turn right
'INSERT' Roll counter-clockwise
'DELETE' Roll clockwise
'C' Change to next camera
'Shift + C' Change to previous camera
'P' Toggle animation on/off
'R' Reset camera to original position and rotation
'F1' Toggle Screen Text output
'F2' Toggle Frame Rate
'+' Increase input speed
'-' Decrease input speed
'W' Toggle Wireframe mode
'O' Toggle Overdraw mode
'F' Increase Far Plane 10.0
'V' Decrease Far Plane 10.0
'G' Increase Near Plane 10.0
'B' Decrease Near Plane 10.0

Mouse Input

Mouse Buttons and Modifiers (Event Type) Action
Left Button (Double Click) Reset camera
Left Button (Movement) Translate camera up/down & left/right
Right Button (Movement) Mouse-look
Middle Button (Movement) Translate camera forward/backward & left/right
Wheel (Movement) Mouse translate up
Left + Right Buttons (Press) Orbit Forward
Control + Left Button (Press) Orbit Right
Control + Right Button (Press) Roll clock-wise
