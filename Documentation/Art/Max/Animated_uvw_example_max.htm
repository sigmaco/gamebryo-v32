Animated uvw example.max

In this 3ds Max sample, a texture is moved via UV Offset in the map rollout dialog. This demonstrates animating a texture by moving its position on an object. The moving texture is a glow map; the base map is stationary. By default, this sample installs to the following location:

%EMERGENT_PATH%\Samples\Models\Max Artist Examples\animated uvw example.max
