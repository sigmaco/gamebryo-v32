Switch Node Utility

This MAXScript utility was designed to allow users to specify nodes whose children are specific states of the scene graph.

Switch Node Utility

Refresh Switch Node Sets: This button will update the entire Switch Node utility plug-in with the contents of the 3ds max scene. This step is required if you rename objects or destroy objects.

Recenter Objects when they are added to the set: This checkbox tells the plug-in whether or not to move all objects in a Switch Node Set to a common origin. The objects are moved according to their pivot.

Set Name: Dropdown list that shows all of the known Switch Node sets in the current Max scene graph. If you delete or manipulate the User Property buffer of an item in the set, please hit the 'Refresh Switch Node Sets' button.

Active Child Name: Dropdown list that shows all of the known "states" of the switch node. The currently selected item is the active state at export time.

Child Node List: Displays all of the Switch Node states for this particular Switch Node Set. The list is sorted by index in the state list.

Move Up/ Move Down: Moves the selected state up or down in the state list.

Show All: This option shows all of the items in the state list. These items are all made visible.

Show Selected Only: Shows only the selected state in the state list. This item is made visible, but all others in the state list are hidden.

Add Selection To Set: Takes the currently selected objects and adds them to the current Switch Node set.

Remove Selection From Set: Removes the current state selected in the state list from the state list.
