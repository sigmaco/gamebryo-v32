Animation Compression Reference and Description

Compression Utility

This utility provides the artist with the ability to edit the compression settings on a per-actor, per-sequence, per-object, or per-sequence and object level.

Global Compression Settings

This panel gives the compression settings for the current actor root. These settings will be used for every object in every sequence unless otherwise specified.

Selection Control

In some cases, it may be necessary to override the global behavior of the compression settings. This may be due to unacceptable compression in the hierarchy or due to tighter memory needs on commonly used sequences. The Selection Control group provides all of the UI necessary to override the global behavior.

Global Compression Settings
Don't Use Compression Toggle whether or not compression is attempted on the sequence. Applies to every object in every sequence unless otherwise specified.
Actor Root Select which actor root is currently active for the global settings. Applies to every object in every sequence unless otherwise specified.
Actor Compression Ratio Select the percentage of the original keys to use when fitting to a spline. Applies to every object in every sequence unless otherwise specified.
Use Compact Keys Toggle whether or not the B-Spline control points are themselves compressed. These compressed control points may take more processing at runtime, but can often reduce animation memory overhead. Applies to every object in every sequence unless otherwise specified.
Selection Control
Sequence Only To apply the same compression settings to all keys in the sequence. An idle sequence can often be represented with far fewer keys than a walk animation, for instance. Overrides global compression settings.
Object Only To adjust a track of animation in all sequences. For instance, if the pelvis is the root of the bone hierarchy, it might be necessary to shift its compression ratio higher in quality than its children. This will affect all sequences containing that bone. Overrides global compression settings.
Object and Sequence To specify an override for a specific object and only apply the override for a specific sequence. For instance, if the pelvis is the root of the bone hierarchy, it might be necessary to shift its compression ratio higher in quality than its children for the run cycle, but the idle animations will work well with the global settings. Overrides global compression settings.
Sequence To select the sequence that will be edited. The field is active only if you select Sequence Only or Object and Sequence. Overrides global compression settings.
Object To select the object(s) that you want to edit. Objects that have been overridden appear with a trailing asterisk (*). Overrides global compression settings.
Compression Override
Override global for this combo To specify that the current selection should override the global settings. This setting must be true in order for the following settings to take affect.
Don't compress this combo To override the global "Don't use compression" setting for the current selection.
Compression Ratio To override the global "Actor Compression Ratio" setting for the current selection.
Use compact keys To override the global "Use compact keys" setting for the current selection.
