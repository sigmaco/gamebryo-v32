Gamebryo MaxScript Toolbar Introduction

All of the MaxScripts listed in the utility panel can also be launched from a toolbar, menu, or mouse menu.

In order to access these utilities from the toolbar, the following steps must be taken:

  1. Open the Customize menu and select Customize User Interface. The dialog below will appear:
  2. Select the Toolbar tab.
  3. Select the GamebryoTools Category from the Category drop down.
  4. You can now drag the listed actions to an existing toolbar or create your own with the New button.
  5. Alternatively, you could load the GamebryoTools.cui file from the Max\UI folder. Be advised that this will override any existing toolbars that you have defined.

If you wish for this toolbar to appear at application start-up, please save the configuration to MaxStart.cui in the Max\UI folder.

Additionally, options are available here not available in the utilities panel.

Gamebryo Toolbar Icons

  1. Export to Gamebryo—This icon allows you to export a file to Gamebryo's format, NIF.
  2. Export Selected to Gamebryo—This icon allows you to export only the selected items in the scene to Gamebryo's format, NIF.
  3. Set Export Options—This icon allows you to set the Gamebryo Export dialog settings without actually exporting.
  4. Quick Preview—This icon allows you to preview with the last selected viewer set in the Gamebryo Utility Panel.
  5. Quick Preview Selected—This icon allows you to preview the selected items in the scene with the last selected viewer set in the Gamebryo Utility Panel.
  6. SceneViewer Preview—This icon allows you to preview with the Asset Viewer plug-in.
  7. Animation Tool Preview—This icon allows you to preview with the AnimationTool plug-in.
  8. Animation Management Utility—This icon launches the Actor Manager Utility.
  9. Animation Compression Utility—This icon launches the Animation Compression Utility.
  10. Bone LOD Utility—This icon launches the Bone LOD Utility.
  11. LOD Utility—This icon launches the Gamebryo LOD Utility.
  12. Portal Utility—This icon launches the Portal Utility.
  13. Billboard Utility—This icon launches the Billboard Utility.
  14. Optimization Keep Utility—This icon launches the Optimization Utility.
  15. Z-Buffer Utility—This icon launches the Z-buffer Utility.
  16. Add/Remove Texture Conversion Custom Attributes—This icon adds or removes the Texture Conversion and Mip-Map custom attributes to the currently selected texture map in the material browser.
  17. Add Texture Conversion Custom Attributes to All Maps and Sub-Materials —This icon adds the Texture Conversion and Mip-Map custom attributes to the currently selected material and all sub-materials and their maps in the material browser.
  18. Remove Texture Conversion Custom Attributes from All Maps and Sub-Materials—This icon removes the Texture Conversion and Mip-Map custom attributes to the currently selected material and all sub-materials and their maps in the material browser.
  19. Add/Remove Multi-Nif Custom Attributes—This icon adds the Multi-Nif attributes used by the Multi-Nif exporter plug-in.
  20. Add/Remove Mesh Profile Custom Attribute—This icon adds the custom attribute used to specify platform-specific geometry streams. See the Mesh Profile Custom Attribute documentation for more details.
  21. See INI File Settings—This icon brings up a simple dialog box with the settings saved in the Gamebryo INI file.
  22. See Root Node Settings—This icon brings up a simple dialog box with the settings saved in the root node.

Additional Gamebryo Toolbar Icons

Gamebryo comes with some utilities that are not on the Gamebryo toolbar by default. If you wish to use them, simply customize your toolbar by dragging and dropping these utilities from the customize user interface dialog mentioned above to an existing toolbar.

  1. Quick Save—This icon allows you to quickly save the current file and export to the last location.
  2. Launch Xbox 360 Viewer—This icon allows you to preview with the Xbox 360 Viewer plug-in.
  3. Launch PS3 Viewer—This icon allows you to preview with the PS3 Viewer plug-in.
  4. Show PC Materials in viewport—This icon changes the currently active materials containing Gamebryo XPlatform Materials to switch to PC for the viewport renderer.
  5. Show Xbox 360 Materials in viewport—This icon changes the currently active materials containing Gamebryo XPlatform Materials to switch to Xbox 360 for the viewport renderer.
  6. Show PS3 Materials in viewport—This icon changes the currently active materials containing Gamebryo XPlatform Materials to switch to PS3 for the viewport renderer.
  7. Add/Remove Precache flags—This icon adds the custom attributes used to control how an object is precached. See the Precache Custom Attributes documentation for more details.
