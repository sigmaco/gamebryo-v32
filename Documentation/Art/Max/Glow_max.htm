Glow.max

Glow maps are useful when textured self-illumination is required. This 3ds Max sample demonstrates this effect by adding a self-illuminated leopard pattern onto a partially lit sphere. The glow map file, glow-orange.bmp, illuminates and colorizes a base map (L1_0.bmp). Wherever glow-orange.bmp is black, the base map remains unaffected. Where the glow texture has color, the material becomes self-illuminating.

By default, this sample installs to the following location:

%EMERGENT_PATH%\Samples\Models\Max Artist Examples\glow.max
