Special Purpose Gamebryo 3ds max Plug-in INI Options

There is a setting in your Gamebryo 3ds max Plug-in INI file which you may want to edit. This file is located at ${3dsmax}/plugcfg/mimmerse.ini. To use this option, use a text editor to add the line to the INI file.

Option Description
UseAreaSuperSpray=1 Will cause SuperSpray to emit from an area based on the emitter icon size, instead of from a point [default: UseAreaSuperSpray=0].
