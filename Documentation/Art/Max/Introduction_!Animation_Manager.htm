Introduction to the Animation Manager

To take advantage of all of Gamebryo's animation capabilities to drive multiple characters in an interactive scene you need to animate the actors or characters in sequences. These sequences can then be activated by the gamepad, keyboard or mouse.

The animation manager handles the organization of these sequences by allowing you to name them, set the length of each sequence, and set how each blend is controlled by the system. The information gets stored in a note track that it creates on the base node of each actor and gets exported as KF files, ready to use in Gamebryo. You can specify additional Note Tracks lower in the character hierarchy than the root, such as on an arm or leg. This approach will cause additional KF files to be exported for just that sub-hierarchy. All sequence names in a particular MAX file must be unique.
