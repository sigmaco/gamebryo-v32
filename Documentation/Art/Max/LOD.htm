Level Of Detail and 3ds Max

The Gamebryo Max plug-in supports several different methods for LOD in a game.

Static Scene Graphs & Geometry

Gamebryo allows artists to use either the LOD Utility that ships with Max or the Gamebryo LOD utility to set up LODs. The Max LOD utility allows the LOD to switch based off of final pixel coverage. This conversion is not exact, but will approximate the Max utility. The Gamebryo LOD utility allows the LOD to switch based of distance between the pivot point of the camera and the LOD node. Neither system will support skinned characters and their bone hierarchies.

Skinned Characters & Bone Hierarchies

Gamebryo supports LOD for skinned characters with the Bone LOD system. This system allows the artist to specify subsets of the bone hierarchy to affect the mesh at different LOD levels. Additionally, the artist can specify different skins for each LOD. This allows the artist to use a single bone hierarchy for skinning multiple character LODs. This system does not allow automatic transitions for the LOD. Usage requires the application to programmatically change the LOD.
