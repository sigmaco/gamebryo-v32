Scene Mesh Profile Selection

The scene mesh profile drop down allows a user to select a default mesh profile for the entire scene. This is the mesh profile that every mesh will use, unless the mesh has a mesh profile custom attribute.

The Reload Profiles button will refresh the profiles listed in the mesh profile drop down box. Errors associated with loading mesh profiles, after pressing the Reload Profiles button, can be viewed in the MaxScript Listener window.

For additional information on mesh profiles see Introduction to Mesh Profiles.
