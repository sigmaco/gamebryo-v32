Using the Gamebryo Shader with 3ds Max

You can use the Gamebryo Shader to customize your Max art assets for Gamebryo.

To use the Gamebryo Shader

  1. Make sure that the Gamebryo Maxscript Toolbar is loaded and available in 3ds Max.

    See Gamebryo MaxScript Toolbar Introduction.

  2. In 3ds Max, create a teapot (or any other object). You should now have something similar to the image below.
  3. Open up the Material Editor. If the currently selected material is not a Standard material, select a Standard material.
    />
  4. In the Shader Basic Parameters rollout, select "Gamebryo Shader" from the drop-down.
  5. Scroll down to the Maps rollout. Open it and you should see a rollout like the one below. Note that these maps are different from ones that you may be accustomed to in the Blinn or Phong Standard Materials.
  6. Drag the current material to the teapot. This applies the Gamebryo shader to the object.
  7. In the Maps rollout, click the button (currently labeled "None") beside the Base map.
  8. In the Texture Selection dialog, select Bitmap from the list. Under the Bitmap Parameters rollout, click the Bitmap filename button.
  9. In the file selection dialog, navigate to %EMERGENT_PATH%\Samples\Models\Max Artist Examples\Textures, select a texture file (in this example, we select bricks-red.bmp), and click the View Image button.

    The texture image file opens within the Specify Cropping/Placement dialog.

  10. In the Maps rollout, click the button (currently labeled "None") beside the Bump map.
  11. In the Texture Selection dialog, select Bitmap from the list. Under the Bitmap Parameters rollout, click the Bitmap filename button.
  12. In the file selection dialog, navigate to %EMERGENT_PATH%\Samples\Models\Max Artist Examples\Textures, select a texture file (in this example, we select grayscale.tga), and click the View Image button.

    The texture image file opens within the Specify Cropping/Placement dialog.

  13. In the Maps rollout, click the button (currently labeled "None") beside the Reflection map.
  14. In the Texture Selection dialog, select Bitmap from the list. Under the Bitmap Parameters rollout, click the Bitmap filename button.
  15. In the file selection dialog, navigate to %EMERGENT_PATH%\Samples\Models\Max Artist Examples\Textures, select a texture file (in this example, we select envmap04.bmp), and click the View Image button.

    The texture image file opens within the Specify Cropping/Placement dialog.

    The Maps rollout should now appear as follows.

  16. Close the Material Browser window and then click the Asset Viewer Preview button (see image below).

    The teapot art asset displays within the Asset Viewer with the three maps combined to form the impression of a bumpy teapot. You can manipulate the asset within the viewer by using several different keyboard and mouse commands. The left mouse click or UP, DOWN, LEFT, and RIGHT keyboard buttons move the camera forward, backward, left, and right. If you press and hold down the CTRL key while you click the left mouse button or any of the previously mentioned keyboard keys, you can orbit the camera around the origin of the world.
