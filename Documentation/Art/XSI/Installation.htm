XSI Plug-in installation

The Gamebryo XSI Plug-in is packaged as an XSI Workgroup that you can connect to using the Plug-in Manager in XSI. This makes the installation and removal of the XSI Plug-in clean and  easy.

Caution

If you have a prior version of the Gamebryo XSI Plug-in installed, read the Uninstalling a prior version section before you start to install this latest version of the plug-in.

Prerequisites

The Gamebryo XSI Plug-in uses Softimage's Crosswalk technology to read dotXSI files. Before you install the plug-in, you must install the prerequisite packages listed below.

Note

The following instructions assume that Gamebryo is installed in %EMERGENT_PATH% and that XSI is installed in C:\Softimage\XSI_X.X, where X.X is the version of XSI being used (6.5 or 7.0; version 7.5 uses a different path). If your installation paths are different, substitute paths that are appropriate for your environment.

These instructions also assume that the Gamebryo XSI Plug-in has been installed to C:\Softimage\XSI_X.X\GamebryoAddons. If you install the plug-in into a different location, adjust references to this folder accordingly.
  1. Softimage XSI vX.X 32-bit for Windows XP
  2. Emergent Game Technologies Gamebryo
  3. Softimage Crosswalk SDK v3.3
  4. Windows XP 32-bit
  5. NVidia Cg

Caution

It is important that XSI be installed BEFORE the Crosswalk SDK for the plug-in to work properly. Although the plug-in will connect regardless of which order the two are installed, errors will occur during export if Crosswalk is installed first.

The Softimage Crosswalk SDK v3.3 is available for download from the Softimage web site:

http://www.softimage.com/downloads/Crosswalk/default.aspx

More recent releases of the Crosswalk SDK may work but have not been tested with the Gamebryo XSI Plug-in.

Windows XP 64-bit, as well as 32-bit and 64-bit Windows Vista may work, but the 32-bit version of XSI is required even on 64-bit operating systems.

Uninstalling a prior version

If you had a version of the Gamebryo XSI Plug-in prior to Gamebryo 2.6, you must completely uninstall any prior installations of the Gamebryo XSI Plug-in before you install this latest version.

To uninstall a prior version of the XSI Plug-in

  1. Launch an XSI Command Prompt.
  2. Execute the following commands in the XSI Command Prompt:

    CD "C:\Softimage\XSI_X.X\Application\spdl"
    XSIBATCH -u GamebryoMaterial.spdl

  3. Delete the following file:

    <Prior Gamebryo Install Directory>\SDK\Win32\ToolPlugins\Lib\VC80\XSIImportPlugin.dll

  4. Delete the following folder:

    C:\Softimage\XSI_X.X\Addons\Gamebryo

  5. Delete the following file:

    <Prior Gamebryo Install Directory>\SDK\Win32\ToolPlugins\Lib\VC80\GamebryoShaders.dll

  6. Delete the following file:

    C:\Softimage\XSI_X.X\Data\DSPresets\Shaders\RealTime\GamebryoMaterial.Preset

  7. Delete the following file:

    C:\Softimage\XSI_X.X\Application\spdl\GamebryoMaterial.spdl

Installation Steps

To install the XSI Plug-in, you use the XSI Plug-in Manager.

  1. Copy the C:\Softimage\XSI_X.X\GamebryoAddons folder to a location of your choice. This location can be on your hard drive or on a network share. You will connect to this location as an XSI Workgroup.
  2. In XSI, click File > Plug-in Manager. On the Workgroups tab, click the Connect button and choose the GamebryoAddons folder that you copied in step 1. The plug-in is now installed.  At this time, disconnect any plug-ins from previous versions of Gamebryo if you have not already (see the "Uninstallation Steps" section of this document for more details).
  3. After installing, two toolbars will be added to XSI: the GamebryoExportToolbar toolbar and Gamebryo Material toolbar. These can be accessed via XSI's View->Toolbars menu item. The toolbars contain a number of useful tools that will be discussed at various points in this documentation.

If you want to use the Gamebryo CubeMapMaker command-line tool, it is located in the Gamebryo installation path under %EMERGENT_PATH%\Source\Tools\DeveloperTools\CubeMapMaker.

PATH environment variables

XSI typically adds some paths to the PATH environment variable upon installation. However, in some cases these paths fail to be added. When such path failures occur, you will experience problems when you try to use the XSI Plug-in. Before using the plug-in, ensure that the following paths exist in your PATH environment variable:

  • C:\Softimage\XSI_X.X\Application\bin
  • %CommonProgramFiles%\Softimage (or something similar to c:\Program Files\Common Files\Softimage)

Uninstalling the XSI Plug-in

We recommend that you uninstall the Gamebryo XSI Plug-in before you install a newer version or attempt to re-install.

To uninstall the XSI Plug-in

  • In XSI, click File > Plug-in Manager. On the Workgroups tab, right-click the Gamebryo workgroup and choose Disconnect. The plug-in is now uninstalled.

We recommend that you exit XSI and restart before connecting to a new workgroup.
