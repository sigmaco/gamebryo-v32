Controlling vertex coloring in the XSI Plug-in

You can control how vertex colors are applied to an object and thus how they interact with the lighting pipeline in your exported NIF. The exporter defines a custom property set named GamebryoVertexColor for assigning vertex color metadata to objects in the scene.

You can assign the GamebryoVertexColor property set to the selected object(s) in either of the following ways:

  • Use the CreateGamebryoVertexColor custom command
  • Click the Create Gamebryo Vertex Color button on the exporter toolbar.

The Vertex Color Handling property specifies how the vertex colors are applied versus how the material colors are applied during lighting. The property can take the following values.

  • Ignore - The emissive, ambient and diffuse are all specified by the corresponding material color.
  • Emissive - The material emissive color is replaced by the source vertex color.
  • Ambient and Diffuse - The material ambient and diffuse colors are both replaced by the source vertex color.

The Lighting Mode property specifies how the object is lit. The property can take the following values.

  • Emissive Only - Only the emissive component is used to light the object.
  • Emissive, Ambient, and Diffuse - All of the values are used to light the object.
