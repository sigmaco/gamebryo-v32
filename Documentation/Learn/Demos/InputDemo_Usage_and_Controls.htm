InputDemo Usage and Controls

The InputDemo sample features different controls depending on the info screen currently being displayed.

Common Controls

Each screen allows for the following controls to switch to the next info screen.

Keyboard Game Pad Function
TAB SELECT + START Switch to the next info screen.

Immediate Mode Screen Controls

The immediate mode screen for game pads polls each possible input and displays its state on screen. The same is done for the keyboard and mouse if they are present. Note that the TAB key will trigger the screen switch on the keyboard though, so make sure not to press it unintentionally.

Rumble Screen Controls

Only game pads support rumble, so they are the only devices whose input is listed for this screen.

Game Pad Function
Left trigger (L2) Percentage of primary rumble (motor 0)
Right trigger (R2) Percentage of secondary rumble (motor 1)

Action Map Mode Controls

The Action Map Mode screen displays several different actions which can be mapped. Keep in mind that the axis range for the sample is set to [-30,30]. Also, when an input combination is listed as

<A> + <B> (for example, R1 + SELECT, or L1+R1+SELECT)

it indicates that <A> must be held when <B> is pressed.

'Horz.' indicates the horizontal axis of an analog stick. 'Vert.' indicates the vertical.

Each mapped action will generate a string which is displayed on-screen under the device which generated it. The mappings are listed below:

Callback Action Map Controls

The Callback Action Map Mode screen displays two actions which have been mapped using a callback method. Each mapped action will generate a string which is displayed on-screen under the device which generated it. The mappings are listed below:

Keyboard Mouse Game Pad Displayed String
ESCAPE Left START START/ESC/LEFT button pressed!
ENTER Right LUP Displays the current click count.
