Operation

The colliders can be selected using the space bar and moved using the controls specified in the user interface. Various display options and the FindCollision frequency can be changed using the UI menu in the demo. The current frame rate is displayed in the upper left corner, and the current FindCollision frequency (attempts per second) is displayed underneath the frame rate.

At startup, both soldier models will be animated and a collider will be selected (as indicated by a black wireframe box around it). If the current collider is moved such that it intersects another collider or collidee in the scene, a red wireframe ABV will be displayed for both objects as long as contact is maintained. If one of the animated soldiers is involved in a collision, it is likely that this contact will be periodically broken, causing the red wireframe to "flash".
