CharacterPerformance Demo Characters

The following characters are included with CharacterPerformanceDemo:

  • Zombie_Standard.KFM—A zombie with 3 levels of detail (51, 16, 12 bones respectively).
  • BadSushi_Fish_Standard.KFM—Bruce the Bad Sushi from the BadSushi demo. Morphed character with no skinning.
  • LB-SelfLit.KFM—A 26-bone character with no normals and static lighting. Bone LOD level 1 has 9 bones.
  • LB-DynaLit.KFM—same as LB-SelfLit.KFM, but the character has normals and dynamic lighting. Add a light to the scene via the menu to make the character visible.
  • LM-SelfLit.KFM—a 28-bone character with no normals and static lighting.
  • LM-DynaLit.KFM—same as LM-SelfLit.KFM, but the character has normals and dynamic lighting. Add a light to the scene via the menu to make the character visible.
  • LM-DynaLit4BonePartition.KFM—same as LB-DynaLit except that each partition uses only 4 bones. The increased number of partitions causes a severe performance degradation.
