Overview of ProfileSample

ProfileSample demonstrates how to gather performance data from a Gamebryo application using NiMetrics. This sample demonstrates two different ways to analyze metrics. It demonstrates how to output metrics into an XML file via the NiXMLMetricsOutput class. It demonstrates how to show metrics on-screen using NiVisualTracker and the associated NiVisualTrackerOutput class. It demonstrates adding a custom metric into the application by recording a pseudo-random value each frame.
