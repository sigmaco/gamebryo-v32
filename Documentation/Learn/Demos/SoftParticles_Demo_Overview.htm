SoftParticles Demo Overview

The SoftParticles sample demonstrates a graphical effect using the NiParticle system. Soft particles are a solution to the problem of the hard edges that are created when billboarded particles intersect with other geometry.

In addition, this demo serves as an example about how to extend NiStandardMaterial to create custom shaders. See the code overview for more details.
