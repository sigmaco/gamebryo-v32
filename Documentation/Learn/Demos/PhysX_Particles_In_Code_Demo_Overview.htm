PhysX Particles-In-Code Demo Overview

The PhysX Particles-In-Code sample demonstrates the construction of a PhysX-based particle system in code. While most PhysX-based particle systems will be created in art tools, there are some situations in which software-created particles may be required, such as when using custom emitters.

The demo simulates particles rising up a column constrained by collision planes that are not rendered. They push a cube around as they rise. The system consists of a particle emitter, mesh-based particles, and the minimal set of particle modifiers. The PhysX actors for the particles are small spheres. If the demo detects PhysX hardware at startup then the simulation will occur using hardware, otherwise software simulation is used.
