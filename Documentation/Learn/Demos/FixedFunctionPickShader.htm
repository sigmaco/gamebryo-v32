FixedFunctionPickShader

This shader, derived from NiD3DShader, makes use of the fixed function pipeline to handle the vertex transforms, so it only needs to set up the passes and texture stages in its constructor to write out the value in the D3DRS_TEXTUREFACTOR render state unmodified. Then, in the overridden SetupShaderPrograms function, GetColorForObject is called on the stored ColorPicker object and the result is passed into the render state.
