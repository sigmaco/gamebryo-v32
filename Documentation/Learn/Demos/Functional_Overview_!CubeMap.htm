Functional Overview

In basic terms, the demo is quite simple. Other than loading the mobile room there is only one other additional object in the scene—the sphere. The sphere is simple geometry, loaded from a NIF, which has its transform and scale adjusted to fit around planets. The interesting part is the environment map that is applied to the sphere. Rather than using a sphere map, a cube map texture is applied. Also, each frame the cube map texture (or part of it) is re-rendered to represent the scene in that given frame. This is the step that makes the reflection appear "live". The helper class, CubeMapCamera is designed to make it easy for an application to render the images needed to update rendered cube maps.
