Setup

The demo is built from a single NIF file, "Collides.nif". The NIF file contains not only the geometric descriptions of the basic colliders in the scene (Sphere, Box, Capsule), but also proxy boxes generated in 3ds max to define Alternate Bounding Volumes (ABVs) that will approximate the geometry of these objects in the Collision system in an efficient manner. The NIF also contains the skin-and-bones soldier character and the proxy boxes used to create the ABV tree for it. The same model is used for the collider and collidee versions of the dragonmen.
