MMEndBattleTrigger Model

Found in project: MangledMetal

Mixes In: EntitiesDeadSource, RemoteBoolPropertyResponse, MultiTargetResponse, MMMesh

Description

An example trigger that mixes in multiple trigger responses to control a series of events. When all entities of a particular mixin have been destroyed, it will change a remote entity's bool property to true
