MangledMetal HUD Service

The MangledMetal HUD Service (MmHUDService) is responsible for parsing the HUD definition file and rendering the splash screen and gameplay HUD elements.

On initialization, the service reads the file MangledMetalHUD.xml (located in the Data\HUD subdirectory) which contains HUD elements describing the visual components to be managed by the service. This information is then used to prepare render steps for the NiRenderFrame of the currently active RenderSurface.

As the service is ticked it monitors the elapsed service manager time to determine when to perform render frame changes according to the HUD element descriptions.

There are two types of HUD elements:

  • Splash
  • Texture

Splash elements describe demo introduction screens. They are presented in the order that they are defined in the HUD elements file. There are attributes for modifying display duration, fading and scale transitions.

Texture elements describe textured HUD components displayed during gameplay. There are attributes for size, position and alignment. A texture element can also be associated with a Player entity property via the "property" attribute so that the texture can be dynamically changed based on the value of the entity property.
