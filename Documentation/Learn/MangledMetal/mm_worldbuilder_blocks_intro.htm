MangledMetal World Builder blocks

MangledMetal has several block files that it ships with, for different purposes. Block files are basically different "levels" that can be loaded separately. You can choose which block file gets loaded via the Configuration= command-line option. See MangledMetal configuration options for more information.

The following block files are shipped with MangledMetal.

File Name Description
EmptyBattlefield The EmptyBattlefield block is simply an open area in which you can test new models and behaviors. The background is uninteresting but there's enough space to try out modifications to the system.
Level01 Level01.block is a simple block file using the MangledMetal assets.
Level03 Level03.block is the main block file for the demo, and the default file to be loaded on most platforms. It is a fully populated level with environment props, terrain, enemies, spawn areas, and so forth. For more information about the layout of the MangledMetal block, view MangledMetal Blocks Detail.
Level04Terrain Level04Terrain.block is a large multi-sector block file showcasing the latest terrain features.
PhysXTest_01 The PhysXTest_01 block is a test block used to demonstrate PhysX support. Some simple PhysX objects are created and initialized.
