MangledMetal Terrain

Terrain for this demo has been authored in World Builder. The terrain files can be found at: %EMERGENT_PATH%\Samples\GameDemos\MangledMetal\Data\MangledMetal\GamebryoData\Generic\Terrain\.

MangledMetal terrain also takes advantage of tagging and PhysX. The ability to move over the terrain is set using custom tagging, which is described in the MangledMetal: Walkables, NonWalkables & Terrain topic. PhysX data has been cooked using the PhysX Terrain Entity.
