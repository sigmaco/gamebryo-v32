CollisionSpawnPoint Model

Found in project: TriggersLibrary

Mixes In: SpawnResponse, CollisionTriggerSource, Mesh

Description

This model is a complete definition of a triggered spawn point which spawns enemies when a player "collides" with it, i.e. when a player enters its activation zone. It mixes in a trigger source model and a trigger response model to form a complete trigger.
