MangledMetal Maya art assets overview

All Maya source art assets can be found at: %EMERGENT_PATH%\Samples\GameDemos\MangledMetal\Art Files\Export

All Maya source art assets were authored in Maya version 8.5.

Axis Orientation

In order to be easily compatible with assets produced in 3D Studio Max, all assets produced in Maya were developed in a Z-up environment. If you open a MangledMetal source art file in Maya and it shows up sideways:

  1. Go to Window -> Settings/Preferences -> Preferences
  2. Click on Settings
  3. Change the Up axis to Z

Walkable/Nonwalkable

In order to work with the MangledMetal picking service, all MangledMetal assets contain one or more meshes called Walkable or Nonwalkable. This allows an artist to define in the scene file where players cannot walk, or what they will walk on top of.

To work properly, these meshes must:

  • Have the PosNormPicking Mesh Packing Profile applied
  • Be directly under the mesh they are associated with in the hierarchy
  • Be hidden

It is important to note that the picking service calculates from the center of the character. As such, all Nonwalkable meshes should extend beyond the edge of their associated mesh by approximately the same distance as the radius of the character (about 150 centimeters for the Player Character).

PhysX

All assets that interact with the PhysX simulation (as either active or passive rigid bodies) also have an associated Rigid Body in the scene file. It is important to note that, once exported, these Rigid Bodies will not scale along with their parent geometry inside of World Builder.
