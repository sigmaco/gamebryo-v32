EntitiesDeadSource Model

Found in project: TriggersLibrary

Mixes In: AbstractTriggerSource

Mixed Into: MMEndBattleTrigger

Description

A trigger source which fires when a set of entities is dead. When the BeginChecking() behavior is called, this model will periodically count the number of instances of a given model; when the count reaches zero, the trigger will fire.

Properties

ModelToCheck String EntitiesDeadSource Enemy Name of the model to count. When zero instances of this model are active, the trigger will fire.
CheckFrequencyMs UInt32 EntitiesDeadSource 500 How often to perform the check, in milliseconds.

TriggerID String EntitiesDeadSource EntitiesDead Identifier for this trigger. This can be useful to script code if it wants to track which trigger source caused an OnTrigger call.

MaxActivations UInt32 AbstractTriggerResponse 1 Maximum number of times this trigger is allowed to be activated. If the TriggerSource detects something that would have triggered, it should not call OnTrigger if ActivationCount >= MaxActivations.
