CollisionTrigger Model

Found in project: TriggersLibrary

Mixes In: CollisionTriggerSource, MultiTargetResponse

Description

This model defines a complete trigger model which informs multiple entities when a collision has occurred. The trigger has a mesh (to be collided with).

Properties

Name Type Source Model Default Value Traits Description
UseModelFilter Boolean CollisionTriggerSource True If 'true', the model filter will be used to determine whether a colliding entity is eligible to cause the trigger to fire.
ModelFilter String CollisionTriggerSource Player If 'UseModelFilter' is true, the colliding entity must mix in this model to cause the trigger to fire.
TriggerCooldown Float64 CollisionTriggerSource 1 Time, in seconds, before this trigger can fire a second time.
LastTriggerTime Float64 CollisionTriggerSource -1 Used internally to remember the last time the trigger fired.
TriggerID String CollisionTriggerSource Collided Identifier for this trigger. This can be useful to script code if it wants to track which trigger source caused an OnTrigger call.
ActivationCount UInt32 AbstractTriggerResponse 0 Number of times this trigger has been activated.
MaxActivations UInt32 AbstractTriggerResponse 1 Maximum number of times this trigger is allowed to be activated. If the TriggerSource detects something that would have triggered, it should not call OnTrigger if ActivationCount >= MaxActivations.
TriggerOnEnter Boolean PhysXTrigger True If true, callbacks will be invoked when a shape enters the trigger shape.
TriggerOnStay Boolean PhysXTrigger False If true, callbacks will be invoked when a shape is penetrating the trigger shape.
TriggerMinInterval Float32 PhysXTrigger 0.1 The shortest time between trigger invocations when TriggerOnStay is active.
TriggerOnLeave Boolean PhysXTrigger True If true, callbacks will be invoked when a shape leaves the trigger shape.
AnchorAtBase Boolean PhysXTrigger True Hidden When True, the anchor is at the base of the 3D object. When False, the anchor is at the center of the 3D object.
ShapeTranslation Point3 PhysXShape 0, 0, 0 The translation offset of the shape from the entity's position.
ShapeRotation Rotator PhysXShape 0, 0, 0 The rotation to apply to the shape on top of the entity's rotation.
ShapeCollisionGroup UInt16 PhysXShape 0 The collision group for the shape.
ShapeType String PhysXShape box One of: box, sphere or capsule
ShapeDimensions Point3 PhysXShape 1, 1, 1 The side length in X, Y and Z for a box. The radius of a sphere is X and Y and Z are unused. The radius of the capsule is X, Y is unused and Z is the height not including the end caps.
SceneName String PhysXShape PhysXDefaultSceneName Names the PhysXScene that this terrain object should be placed in.
Position Point3 Placeable 0, 0, 0 The world-space position of the entity.
Rotation Rotator Placeable 0, 0, 0 The world-space rotation of the entity.
Scale Float32 Placeable 1 The world-space scale of the entity.
TargetList EntityRef List MultiTargetResponse List List of entities to inform. These entities should all implement the OnTrigger behavior.
