SoulCage Model

Found in project: MangledMetal

Traits: Active

Mixes In: DestructibleObject

Mixed Into: SoulCage_forGate, SoulCage_forSpawnTemple

Description

Instance of a DestructibleObject with a specific appearance (soul cage). Used to control various components of gameplay including opening gates and disabling spawn temples.
