RegionChangeTrigger Model

Found in project: MangledMetal

Traits: Active

Mixes In: RegionChangeResponse, CollisionTriggerSource, 3DProxy, MultiTargetResponse

Description

A trigger that changes the visibility of some regions of the world when collided with

ProxyScale Float32 RegionChangeTrigger 0.01 Proxy Scale is the actual scale to render the geometry at. For proxies this scale always overrides the "Scale" property for rendering in the editor because that property may have another meaning that is not visual (for the proxy). This property can also be used to correct art authored at a different scale from the rest of the world. The Proxy Scale will be multiplied by the World Scale setting in a block to arrive at the actual rendered scale of the proxy. The Proxy Scale value should represent the scale to apply to an object inserted into a meter scale world to make it look correct relative to other meter scale objects. So if your proxy art were authored in centimeters, this value would be 0.01.
