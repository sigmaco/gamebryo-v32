Core Runtime Tutorials: Introduction

The Core Runtime tutorials are designed to present important Core Runtime concepts. The tutorials should be completed in order because each tutorial builds upon its predecessor. By the end of Tutorial 9, we will have created a simple application in which a remote control controls a helicopter in a war-torn village. On the remote control, a display shows what the helicopter sees.
