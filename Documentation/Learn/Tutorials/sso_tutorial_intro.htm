Online Tutorials: Introduction

The following tutorials are specific to Online functionality:

  • Peer Game—intended to be a bare-bones example of a working peer-to-peer game. It demonstrates functionality from the Base Game demo, reimplemented to support multiple players, including support for rapid iteration. The accompanying tutorial walks you through all of the code changes that are required to convert Base Game into the online-enabled Peer Game.
  • Multi-player Rapid Iteration—demonstrates how you can rapidly iterate on multiplayer games, just as you can on standalone games. This tutorial walks through some examples of rapidly iterating on Peer Game with multiple clients connected.
