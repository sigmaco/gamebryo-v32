PC enet Reliable UDP Code Overview

Module Description
NetLibEnet.cpp
  • The network library class.
  • Overrides some NetLib calls.
  • Creates and registers transports.
  • Creates and listens to connections.
NetTransportEnet.cpp
  • Implements the transport layer.
  • Creates new outgoing connections.
  • Listens for incoming messages.
  • Sends and receives messages over reliable UDP
ConnectionEnet.cpp
  • Implements the IConnection class.
  • Accepts and closes connections.
  • Creates enet peer connections.
  • Listens and connects from and to connections.
  • Sends and receives messages from the underlying network.
  • Validates and checks data in the message.
