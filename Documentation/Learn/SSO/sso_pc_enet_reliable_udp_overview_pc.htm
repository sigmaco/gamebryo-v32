PC enet Reliable UDP Overview

This sample integrates the open source enet library with Foundation Messaging. There is no matchmaking in this sample because the enet library does not provide any. Connections must be established by manually specifying hostnames or IP addresses of the peers to which you want to connect.

You can substitute efdNetLibEnet for the default NetLib that is used in the PeerGame Sample and Tutorial. However, if you do this, you need to be aware that rapid iteration will not be available.
