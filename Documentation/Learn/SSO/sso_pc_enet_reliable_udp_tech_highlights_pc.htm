Integrating PC enet Reliable UDP

You can integrate the efdNetLibEnet sample network library into your application by following these steps.

  1. Link against efdNetLibEnet. In Visual Studio, add the following to your project settings (Linker > Input > Additional Dependencies). 
    %EMERGENT_PATH%/SDK/Win32/VC90/Debug/efdNetLibEnet.lib
  2. Add headers and register/unregister NetLib Factory for NetLibEnet. 
    #include <efdNetLibEnet/NetLibEnet.h>

// Register NetLibEnet so that it can be factoried and used
INetLib::RegisterNetLib<NetLibEnet>("NetLibEnet");
  3. Configure each process to use the enet NetLib. 
    [NetService]
NetLibType=NetLibEnet

[QualityOfService]
Default=kQOS_Reliable
