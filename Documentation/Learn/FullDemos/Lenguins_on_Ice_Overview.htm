Lenguins on Ice Application Overview

The Lenguins on Ice sample shows off many features of the Gamebryo engine, including advanced rendering effects and integration with the NVIDIA PhysX system.

Minimum Recommended System Specifications

Because the Lenguins on Ice sample was designed to demonstrate high end rendering effects and large scale physical interactions, it imposes substantial system requirements. The sample makes use of pixel shader and vertex shader model 2.0, so it requires the use of hardware compatible with these features. At startup, the sample will check the capabilities of the graphics hardware being used, and if the graphics hardware is insufficient, the sample will display an appropriate error message box, before exiting.

The minimum recommended PC system used with this application is as follows:

  • Pentium IV 3.X GHz CPU
  • 1 GB RAM
  • nVidia GeForce 5800, ATi Radeon 9700, or equivalent shader model 2.0 capable graphics card
  • Windows XP
  • DirectX 9.0
