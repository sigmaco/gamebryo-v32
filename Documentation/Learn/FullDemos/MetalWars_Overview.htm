Metal Wars Application Overview

The Metal Wars sample shows off many of the features of the Gamebryo 2 engine including: advanced, next-generation rendering effects; full-control animation playback and blending; as well as complex, high-detail scenes and characters.

Minimum Recommended System Specifications

Because the Metal Wars sample was designed to demonstrate the limits of "next generation" technology found in the Gamebryo 2 release, when run on the PC platform, it imposes substantial system requirements. The sample makes use of pixel shader model 2.0 and floating point textures and therefore requires the use of DX9 and hardware compatible with these features. At startup, the sample will check the capabilities of the graphics hardware being used, and if insufficient will prompt the user with an error message box, after which the application will exit.

The minimum recommended PC system used with this application is as follows:

  • Pentium IV 3.X GHz CPU
  • 1 GB RAM
  • Graphics card capable of shader model 2.0 and multiple render targets, such as nVidia GeForce 6800 or ATi Radeon 9700.
  • Windows XP
  • DirectX 9.0
