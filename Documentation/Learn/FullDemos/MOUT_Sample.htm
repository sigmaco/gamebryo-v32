MOUT Sample Overview

The MOUT (Military Operations on Urbanized Terrain) sample shows the use of the Gamebryo engine for a military-style visualization application. MOUT gives an example of a bipedal character that can be navigated in an outdoor environment that consists of hilly terrain populated with a small number of buildings.

Minimum Recommended System Specifications

The details of the minimum-specification system used when testing this application are as follows:

  • Pentium III 1GHz
  • 256 MB RAM
  • nVidia GeForce3 or ATi Radeon 8500
  • Windows XP
  • DirectX 9
