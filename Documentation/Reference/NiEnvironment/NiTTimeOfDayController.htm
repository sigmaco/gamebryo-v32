Inheritance graph for NiTTimeOfDayController

Description

This class is a templated TimeOfDay controller class that allows any variable on any object to be controlled by the TimeOfDay system through an exposed "Set" function on the target.

The type of the target.

Public Methods

NiTTimeOfDayController(Functor kFunctor)

NiTTimeOfDayController Constructor.

This creates a time of day controller that is bound to a particular variable on a target through a simple "set" function.

Parameter Description
kFunctor the functor used to set the property's value. This functor MUST implement a templated void SetValue(Value toSet) function. See NiTTimeOfDayDefaultFunctor for more information.

virtual void Update(float fScaledTime)

Update function called once per frame if an interpolator has been attached to animate this controller's value.

Parameter Description
fScaledTime the current point in time of the animation.
