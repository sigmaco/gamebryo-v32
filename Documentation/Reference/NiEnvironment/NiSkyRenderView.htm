Inheritance graph for NiSkyRenderView

Description

This Ni3DRenderView class is specially designed to render sky objects to the view port.

The main customization of this class is it automatically ignores sending camera translation data to the renderer whilst rendering objects in it's scene. This allows sky boxes and sky domes to always be centered about the camera without needing to explicitly move the boxes every frame.
