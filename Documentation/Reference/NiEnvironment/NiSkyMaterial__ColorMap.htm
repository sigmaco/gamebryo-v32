NiSkyMaterial::ColorMap

<NiSkyMaterial.h>

Description

A structure to encapsulate the selection of a color map modifier.

Public Types

NiSkyMaterial::ColorMap::Value

Name Description
CUSTOM Specifies a custom color map source.
SKYBOX Specifies to use a cubemap texture.
GRADIENT Specifies to use a gradient color map.
FOG Specifies to use the fog color to blend.
ORIENTED_SKYBOX Specifies to use a cubemap which has an up vector (ie, is rotated in the pixel shader).
