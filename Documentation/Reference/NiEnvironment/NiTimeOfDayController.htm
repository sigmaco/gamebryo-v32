: public NiRefObject

Inheritance graph for NiTimeOfDayController

Description

This interface is used in the NiTimeOfDay system to allow time of day animations to be applied to various values throughout the scene.

Public Methods

virtual ~NiTimeOfDayController()

Virtual destructor for this base interface.

virtual void Update(float fScaledTime) = 0

Update function called once per frame if an interpolator has been attached to animate this controller's value.

This function should only be called by the NiTimeOfDay class.

Parameter Description
fScaledTime the current point in time of the animation.

virtual void SetInterpolator(NiInterpolator* pkInterpolator) = 0

Sets the interpolation object that will control the animation of the variable this controller is tied to.
