Description

This class encapuslates a bit field used to describe a NiSkyMaterial shader.

The NiSkyMaterial generates shaders based on a user configuration of a NiSky object, this allows generic skys to be quickly added to a scene. The NiSkyMaterialDescriptor provides an easy way of describing all the information that is needed to generate a shader.
