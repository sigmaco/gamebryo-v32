NiRendererTabController

: public NiBaseTabController

<NiRendererSettings.h>

Description

This class controls a Windows dialog with one combo box, which allows the user to select the desired renderer. It also instantiates each NiBaseRendererOptionsView that has registered with NiSettingsDialog. These objects control all the options for creating their respective renderer.

Virtual Member Functions

virtual NiWindowRef InitDialog(NiWindowRef pParentWnd)

In this method class creates dialog as a child of pParentWnd, and places the new dialog into a tab. The HWND for newly-created dialog is returned.

virtual char* GetCaption()

Returns a pointer to string with name of tab.

virtual unsigned int SetBasicHeight()

Resize child dialog to basic height and return new height in pixels. Width must be constant. Called when user presses Advanced button to change size of tab form advanced to basic.

virtual unsigned int SetAdvancedHeight()

Resizes child dialog to advanced height and return new height in pixels. Width must be constant. Called when user presses Advanced button to change size of tab form basic to advanced.
