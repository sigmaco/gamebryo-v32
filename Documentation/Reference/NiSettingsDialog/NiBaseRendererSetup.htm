NiBaseRendererSetup

: public NiMemObject

<NiBaseRendererSetup.h>

Description

This is an abstract base for classes that expose an NiBaseRendererOptionsView-derived object to the NiSettingsDialog library. The NiBaseRendererSetup-derived class then creates a renderer of the appropriate type based on the NiRendererSettings object that is produced.

Static Member Functions

NiBaseRendererSetup* GetFirstRendererSetup(NiTListIterator& kIter)

Returns the first NiBaseRendererSetup object that has been registered. The list will be iterated with kIter.

NiBaseRendererSetup* GetNextRendererSetup(NiTListIterator& kIter)

Returns the next NiBaseRendererSetup object that has been registered, based on the iterator kIter.

NiRendererPtr CreateRenderer( NiRendererSettings* pkRendererSettings, NiWindowRef pWndDevice, NiWindowRef pWndFocus)

Creates a new renderer instance based on the settings in pkRendererSettings.

Virtual Member Functions

virtual NiBaseRendererOptionsView* GetRenderOptionsView( NiRendererSettings* pkSettings)

Returns a new instance of the NiBaseRendererOptionsView-derived class that corresponds to the renderer that this NiBaseRendererSetup-derived object manages.

virtual void ReleaseRendererOptionsView(NiBaseRendererOptionsView* pkROV)

Release the NiBaseRendererOptionsView-derived object that this NiBaseRendererSetup-derived object maintains.
