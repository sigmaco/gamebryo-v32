Description

This class associates a NiObject with a NiSceneGraphUpdateObjectId.

This class is used by the NiSceneGraphUpdate class to syncrhonize scene graph objects between an editor and viewer application. The editor is responsible for generating the unique ID. The IDs are used to identify objects to be updated or removed in the viewers.
