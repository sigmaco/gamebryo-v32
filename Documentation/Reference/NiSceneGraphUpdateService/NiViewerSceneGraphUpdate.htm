Inheritance graph for NiViewerSceneGraphUpdate

Description

This singleton is used by a viewer application to cache a scene graph that is replicated from an editor.

The singleton can be used to access the scene graph root, current camera, and config settings sent by a NiEditorSceneGraphUpdate object. In addition, this object interprets the timeline related config settings sent by the editor to start, restart, and stop animations in the replicated scene graph.

Public Static Methods

static void InitializeViewer(
NiViewerSceneGraphUpdate* pkViewerSceneGraphUpdate = 0,
efd::ServiceManager* pkServiceManager = 0)

Initialize NiSceneGraphUpdate singleton with a NiViewerSceneGraphUpdate object.

This method invokes the NiSceneGraphUpdate::Initialize() method to initialize object-based network messaging.

In addition, this method registers a new NiSceneGraphUpdateService object with the ServiceManager to enable sending and receiving messages between the singleton and the network messaging system.

Parameter Description
pkViewerSceneGraphUpdate Optional NiSceneGraphUpdate object to be used as singleton. If this parameter is NULL, a new NiViewerSceneGraphUpdate will be instantiated.
pkServiceManager Optional ServiceManager object to use when registering the NiSceneGraphUpdateService and other services. If this value is NULL, a new ServiceManager will be created.

static void ShutdownViewer()

Halts replication and invokes shutdown on the NiSceneGraphUpdate singleton.
