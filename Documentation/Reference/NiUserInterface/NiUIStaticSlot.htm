NiUIStaticSlot*

: public NiUIBaseSlot*

<NiUISlot.h>

Description

NiUIStaticSlot0, NiUIStaticSlot1, and NiUIStaticSlot2 implement slot functionality for member static functions as well as C-style functions. The only differences in signature between these classes and their parent classes (NiUIBaseSlot0, NiUIBaseSlot1, and NiUIBaseSlot2, respectively) are the constructor and the Initialize function.

Constructors

NiUIStaticSlot1()

NiUIStaticSlot2()

This initializes an empty slot. It will not be valid until after initialize is called.

NiUIStaticSlot0(void (*pfnCallback)())

NiUIStaticSlot2(void (*pfnCallback)(Arg1Type, Arg2Type))

These constructors provided for convenience. Slots initialized with these constructors are valid.

Supported Member Functions

void Initialize(void (*pfnCallback)())

void Initialize(void (*pfnCallback)(Arg1Type))

void Initialize(void (*pfnCallback)(Arg1Type, Arg2Type))

This function must be called at least once on slots created through the default constructor. It may be called any time after that when a new callback is desired for the function.
