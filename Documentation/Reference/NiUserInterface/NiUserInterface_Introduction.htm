NiUserInterface Introduction

The NiUserInterface library was created to unify the look and feel of all Gamebryo demos. While it does provide much of the basic functionality required of a UI system (inter-element communication, basic UI elements, navigation controls, etc.) it is not meant to be a full system used by our customers nor is it intended as a reference for building a complete UI system. For such a system, customers are referred to Anark or Scaleform.

The NiUserInterface library is built on top of the NiInput, NiCursor, NiSystem, and NiMain libraries, and requires all of these for proper functioning. For an example of proper initialization and shutdown of the UI system, see NiSample. The various elements are initialized in most demos, although some elements are more frequently used than others.
