NiUILabel

: public NiUIBaseElement

<NiUILabel.h>

Description

NiUILabel is a multiline text output dialog. It handles its own word-wrapping, ensuring that text fits within the allotted space. If too much text is input for the size of the label, the label fits as much text as possible.

Constructors

NiUILabel(const char* pcLabelText = "", const NiColorA& kBackgroundColor = NiColorA::WHITE, const NiColor& kTextColor = NiColor::BLACK)

Note that while the background color may contain opacity, the foreground color must be fully opaque.

Supported Enumerations

The NiUILabel class allows different alignment options.

NiUILabel::LabelVerticalAlign

Options include: TOP, BOTTOM, VERT_MIDDLE.

NiUILabel::LabelHorizontalAlign

Options include: LEFT, RIGHT, HORIZ_MIDDLE.

Supported Member Functions

void SetText(const char* pcLabelText)

This function changes the label text as well as finding all locations where a line must wrap.

void SetBackgroundColor(const NiColorA& kBackgroundColor)

Sets the background color.

void SetTextColor(const NiColor& kTextColor)

Sets the text color.

virtual void Draw(NiRenderer* pkRenderer)

Render this element.

void SetAlignment(NiUILabel::LabelVerticalAlign eVert = TOP, NiUILabel::LabelHorizontalAlign eHoriz = LEFT)

This function allows the application programmer to change the alignment of the label text. However, it is important to note that in the case of horizontal alignment of multiple line labels, the lines all begin at the same left-most point but the entire text is shifted so that the widest line is right aligned or centered.
