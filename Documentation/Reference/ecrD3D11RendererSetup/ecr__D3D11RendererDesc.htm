Description

D3D11RendererDesc is the NiBaseRendererDesc that manages renderer settings for D3D11.

This class defines an interface for communicating renderer settings to and from a D3D11RendererOptionsView object. It provides a list of available renderer settings, modifies those settings in response to user input, and saves the settings in an NiRendererSettings object.
