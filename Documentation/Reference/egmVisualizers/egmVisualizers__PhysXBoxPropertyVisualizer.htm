Description

A visualizer for a PhysX box.

This is only different from the Cube visualizers in that the origin of the visualizer is different.

By default, the Position, Rotation, and Dimensions properties on the entity are used to create the visualizer mesh. The properties used to drive the visualizer can be overridden by the extra data passed to the constructor.

Construction and Destruction

PhysXBoxPropertyVisualizer(egf::ExtraDataPtr spExtraData)

Constructor.

The extra data can be used to override which properties drive the visualizer. The string value of the extra data is used to find the property. If that entry does not exist or it is specified as "Default", then the default property on the entity will be used.

"Anchor" entry overrides default position property. "Orientation" entry overrides default rotation property. "Dimensions" entry overrides default dimensions property. "Color" entry specifies the color of the visualizer (no default). "Offset" entry specifies the position offset of the visualizer (no default).

Public Methods

virtual PropertyChangeAction UpdateFromEntity(egf::Entity* pEntity)

Check which entity properties exist that are driving this visualizer, cache those values, and return the action required to update the visualization mesh based on which properties have changed.
