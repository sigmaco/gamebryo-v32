Description

A visualizer for a PhysX box.

This is only different from the Cube visualizers in that the origin of the visualizer is different.

By default, the Position, Rotation, and Dimensions properties on the entity are used to create the visualizer mesh. The properties used to drive the visualizer can be overridden by the extra data passed to the constructor.
