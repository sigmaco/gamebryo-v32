Introduction to NiMeshProfileXMLParser

The NiMeshProfileXMLParser library is an internal use only library that provides a means for parsing an XML-based representation of a mesh profile and converting it into an in-memory mesh profile that may be consumed by the NiMeshProfileProcessor library.

As the NiMeshProfileXMLParser library is the only library currently employed by Gamebryo's exporters and tools for providing authoring capabilities, those interested in authoring their own XML-based representations of mesh profiles should read the section on Introduction to Authoring XML Mesh Profiles.

Developers interested in the internals of this library should read the section entitled NiMeshProfileXMLParser Internals.
