NiMeshProfileComponent::NiRemap

<NiMeshProfileComponent.h>

Description

This structure is used to store the data needed to perform a remap.

This class does not do any of the remap functionality but serves solely as the data storage to be used to convert to a NiOpRemap operational node.

remap="src-a=x src-b=y dst-a=z dst-b=w"
