Introduction to Authoring XML Mesh Profiles

In order to author XML mesh profiles that can be parsed by the NiMeshProfileXMLParser library (the XML profile parser used by Gamebryo's exporters and tools) and utilized to transform exported meshes appropriately, an understanding of the information found in the following sections should be considered prerequisites:

  • The Tool Plug-in Framework
    This documentation gives an overview of the tool plug-in framework - the context under which the mesh profile system operates.
  • The Mesh Profile System
    This documentation offers a high level description of the Mesh Profile System, how profiles are associated with mesh objects, how they are selected, and how the mesh profile system fits within the context of the tool plug-in framework.
  • The NiMeshProfileProcessor
    The documentation found in this section provides information on the in-memory mesh profiles (distinct from any XML-based representation). Details can be found here on how these mesh profiles operate and are used to transform and manipulate exported mesh objects.

The NiMeshProfileProcessor library does not provide methods for streaming its in-memory profiles to and from disk, but instead relies entirely on external construction of any mesh profiles that it might use. This is where the NiMeshProfileXMLParser library enters the picture. The NiMeshProfileXMLParser provides a specification (based in XML) for defining a mesh profile. This specification is implicitly found within the library's parsing capabilities.

Under the %EMERGENT_PATH%\Media\ToolPluginsData\MeshProfiles directory, there exist a schema entitled "NiMeshProfileSchema.xsd" that may be used to help author valid XML that conforms to the implicit definition. It should be noted, however, that the NiMeshProfileXMLParser doesn't directly make use of that schema file. One final note before moving on is that Gamebryo provides a tool called MeshProfileCompiler which may be helpful to authoring XML mesh profiles. This tool is based on the NiMeshProfileXMLParser and can be used for testing and debugging authored XML profiles against syntax errors.

The remainder of this documentation describes the implicit specification defined by the NiMeshProfileXMLParser library and how it is used to author mesh profiles in more detail.
