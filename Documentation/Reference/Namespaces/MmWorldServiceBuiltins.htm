The entities must not have a boolean property called "IsStatic" set to true.

Parameter Description
entity The entity at the center of the proximity check

Return Value the list of entities within the radius

efd::list<egf::EntityID> GetEntitiesInArc(
egf::EntityID entity,
efd::Float32 radius,
efd::Float32 arc)

Retrieve all entities within a specified arc.

The entities must have a float property called "Radius", do not have a boolean property called "IsStatic" set to true, and touch an arc defined by the given arguments. The arc originates from the base entity and goes along the entity's negative y axis. The distance computation considers depth, but the arc computation does not, resulting in an orange slice shaped volume of influence.

Parameter Description
entity the origin of the query arc. This entity is excluded from the returned results
radius The distance from the base entity to consider other entities
arc The size of the arc to search for entities in degrees
Return Value the list of entities whose radius came within the arc volume
