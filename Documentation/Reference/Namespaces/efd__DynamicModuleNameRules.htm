efd::DynamicModuleNameRules Namespace

Private Methods

EE_EFD_ENTRY const char* GetPlatformPrefix()

Queries for the platform dependent prefix of a shared/dynamic library.

Parameter Description
Return Value The platform dependent prefix. (e.g. lib under Linux/Unix)

EE_EFD_ENTRY const char* GetPlatformExtension()

Queries for the platform dependent file extension of a dynamic library.

Parameter Description
Return Value The platform dependent file extension including the "dot". (e.g. "dll" under Windows, "so" under Linux)

EE_EFD_ENTRY const efd::utf8string GetPlatformSpecificName(const efd::utf8string& i_strBaseName)

Convenience method which constructs a string representing the full name of the shared library.

Pass in just the root name with no directory information.

Parameter Description
i_strBaseName the base name of the module.
Return Value utf8string A string containing the platform specific name of the form: "<GetPlatformPrefix()><i_strBaseName>.<GetPlatformExtension()>"
