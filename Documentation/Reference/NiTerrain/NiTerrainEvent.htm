NiTerrainEvent

: public NiRefObject

Inheritance graph for NiTerrainEvent

Description

This class defines an event object that when "Raised" will callback to a set of arbitrary handler objects that have been registered upon it.

Public Types

Destructor.

bool AttachHandler(HandlerType* pkHandler)

Attach a handler to the event to be notified when the event is raised The event will take over ownership of the handler object that is passed in.

Parameter Description
pkHandler The handler object.

void DetachHandler(const HandlerType* pkHandler)

Detach a handler from the event.

The passed in handler will be compared to the handlers attached to the event and if a matching one exists, one instance of it will be removed.

Parameter Description
pkHandler The handler object to look for and detach.

HandlerType* FindHandler(const HandlerType* pkHandler) const

Query the event to discover if a handler matching the passed in one has been registered on this event.

Parameter Description
pkHandler The handler object to look for
Return Value A pointer to the existing handler if a matching instance exists on the event.
