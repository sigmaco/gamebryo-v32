NiTerrainStreamingTask::SpecialLevelOfDetail

<NiTerrainStreamingTask.h>

Description

An enumeration to indicate special types of levels of detail parameters.

Public Types

NiTerrainStreamingTask::SpecialLevelOfDetail::VALUE

Name Description
UNCHANGED When an LOD parameter is this value, it means the value should be unchanged.
UNLOAD When an LOD parameter is this value, it means to unload all of the LOD's.
BASE The lowest level of detail that can be loaded.
