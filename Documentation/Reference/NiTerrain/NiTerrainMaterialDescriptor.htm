Inheritance graph for NiTerrainMaterialDescriptor

Description

This class manages the bit field for the terrain material.

This bit field is then used to construct the individual descriptors for both the vertex and pixel shader. The bit field encodes the layers affecting the terrain, lights affecting the terrain, and various other attributes.
