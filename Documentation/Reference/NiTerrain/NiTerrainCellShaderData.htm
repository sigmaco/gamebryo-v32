NiTerrainCellShaderData

: public efd::MemObject

Inheritance graph for NiTerrainCellShaderData

Description

This class is used to manage and store the shader data for each Cell in the terrain.

Public Types

NiTerrainCellShaderData::RenderMode

Render modes.

Name Description
RENDER_DISPLAY Render the terrain for display to the user.
BAKE_DIFFUSE Render the terrain for baking diffuse textures.

NiTerrainCellShaderData::DebugMode

Shader debug modes.

Name Description
DEBUG_OFF Disable all debug functionality.
DEBUG_SHOW_NORMALS Colorize the terrain according to the normal calculated at that pixel.
DEBUG_SHOW_MORPH_VALUE Colorize the terrain according to the geo-morphing value.
DEBUG_SHOW_NUM_SURFACES Colorize the terrain to highlight the number of surfaces used on the terrain.
DEBUG_SHOW_UNPAINTED_AREA Colorize the terrain according to the total mask value.
DEBUG_SHOW_GLOSSINESS Colorize the terrain according to the glossiness at the current pixel.
DEBUG_FLAG_MASK Debug flags section - for disabling specific parts of the shader.
DEBUG_DISABLE_NORMAL_MAPS Disable the use of normal mapping on the terrain.
DEBUG_DISABLE_PARALLAX_MAPS Disable the use of parallax mapping on the terrain.
DEBUG_DISABLE_SPECULAR_MAPS Disable the use of specular mapping on the terrain.
DEBUG_DISABLE_DETAIL_MAPS Disable the use of detail mapping on the terrain.
DEBUG_DISABLE_DISTRIBUTION_MASKS Disable the use of distribution masks on the terrain.
DEBUG_DISABLE_LIGHTING Disable the effect of lights on the terrain.
DEBUG_DISABLE_HIGH_DETAIL Disable rendering of high detail data. Only use baked maps.
DEBUG_DISABLE_BASE_NORMAL_MAP Disable low detail normal mapping.

Public Attributes

NiPoint4 m_kStitchingInfo

Used when morphing between cell lods in order to keep the borders from cracking.

NiPoint3 m_kAdjustedEyePos

Modified camera coordinates used for lod calculations.

NiPoint2 m_kLowDetailTextureSize

Size of the low detail texture in pixels.

NiPoint2 m_kLowDetailTextureOffset

Offset to apply to uv lookups when sampling the low detail texture map.

X coordinate stores the offset of the low detail diffuse map, y coordinate stores the offset of the low detail normal map.

NiPoint2 m_kLowDetailTextureScale

Scale to apply to uv lookups when sampling the low detail texture map.

X coordinate stores the size of the low detail diffuse map, y coordinate stores the size of the low detail normal map.

NiPoint4 m_kLayerScale

Each component stores the scale for the given surface material layer.

NiPoint2 m_kBlendMapScale

Scale used to modify the incoming uv coordinates for properly accessing the blend map for a given cell.

NiPoint2 m_kBlendMapOffset

Offset used to modify the incoming uv coordinates for property accessing the blend map for a given cell.

NiPoint4 m_kDistributionRamp

Strength used to modify the distribution map effect on a given layer.

NiPoint4 m_kParallaxStrength

Strength used to modify the parallax map effect on a given layer.

NiPoint4 m_kSpecularPower

For a given layer, defines how tight the specular highlights on on the surface.

NiPoint4 m_kSpecularIntensity

Defines the overall strength of the specular lighting for a given layer.

NiPoint4 m_kDetailTextureScale

Scale value for the detail map on a given layer.

float m_fMorphThreshold

Threshold for which morphing starts.

float m_fMorphDistance

Distance over which morphing occurs.

NiUInt32 m_uiMorphMode

Type of LOD calculations to perform on the terrain. (2D, 2.5D, 3D).
