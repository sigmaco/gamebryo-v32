Inheritance graph for NiTerrainSurfacePackageFile

Description

This class is used to stream a surface package's data to/from disk.

The class provides basic versioning support to allow backwards compatibility with older package formats.

This function must be called on all successfully opened files to signal that the file is no longer used.
