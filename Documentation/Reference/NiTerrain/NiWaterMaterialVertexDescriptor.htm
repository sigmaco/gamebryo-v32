Description

This class encapsulates a bit field used to describe a vertex shader for NiWaterMaterial.

The vertex material descriptor provides a way of easily describing all the information needed to produce a vertex shader that matches the requirements of a provided NiWater object. The descriptor is used by the NiWaterMaterial to determine how to build the vertex shader for a NiWater object.
