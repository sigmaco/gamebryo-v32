Description

This class encapsulates a bit field used to describe a pixel shader for NiWaterMaterial.

The pixel material descriptor provides a way of easily describing all the information needed to produce a pixel shader that matches the requirements of a provided NiWater object. The descriptor is by the NiWaterMaterial to determine how to build the pixel shader for a NiWater object.
