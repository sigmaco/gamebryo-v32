NiTerrainStreamingTask::Status

<NiTerrainStreamingTask.h>

Description

Task status. At any one point in time, the task will be in one of these states

Public Types

NiTerrainStreamingTask::Status::VALUE

Name Description
INVALID A state for when the task has just been created and not initialized for execution yet.
CANCELLED A state to show that the task has been cancelled.
PRELOADING A state to indicate that the task is preloading data from disk.
LOADING A state to indicate that the task is decoding data from disk.
UNLOADING A state to indicate that the task is unloading data in a sector.
REMOVING A state to indicate that the task is removing a sector from the terrain.
BUILDING_HEIGHTS A state to indicate that the task is building the heightmap of a sector.
BUILDING_LIGHTING A state to indicate that the task is building the lighting of a sector.
BUILDING_MAPS A state to indicate that the task is building the textures of a sector.
LOCKING A state to indicate that the task is locking its streams/textures.
POPULATING A state to indicate that the task is populating the sector data structures.
UNLOCKING A state to indicate that the task is unlocking its streams/textures.
