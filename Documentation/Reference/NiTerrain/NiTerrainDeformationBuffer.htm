Description

A class used to hold a 2D array of values used in the painting of the terrain.

At present two types of buffer are defined. A heightmap buffer and a surface mask buffer. These objects are designed to be initialized by a NiTerrain object.

The buffer manages it's own memory allowing a single buffer object to be reused many times with minimal memory allocations.
