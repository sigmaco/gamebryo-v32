NiTerrain::MetaDataVisitor

<NiTerrain.h>

Description

This class is used to provide a base class for custom visitor objects to be passed in when querying metadata.

Public Methods

virtual ~MetaDataVisitor()

Virtual destructor.

virtual void Visit(
const NiSurface* kSurface,
const NiMetaData& kData,
float fWeight) = 0

Visitor function.

This function is called for every surface encountered on the terrain by the visitor and the corresponding weight of that surface at that point.
