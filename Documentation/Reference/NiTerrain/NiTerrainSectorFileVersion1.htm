virtual bool NextBlock()

Read in the next block of information from the file.

Can only be called on a file opened for read access

Parameter Description
Return Value True if the next block was successfully read.

virtual bool PushBlock(NiInt32 iChildID)

Jump to the [iChildID]th child of the current block.

Can only be called on a file opened for read access

Can be useful for jumping to a specific level of detail region of the file

Parameter Description
iChildID the ID of the block to jump to
Return Value True if the block was successfully read, false otherwise.

virtual bool PopBlock()

Jump to the parent of the current block.

Can only be called on a file opened for read access

Using this function after reading a LOD region of the file (i.e. sequentially reading more than the 4 children of the parent block) will cause any further GetBlockID calls to return incorrect information

Parameter Description
Return Value True if the block was successfully read.
