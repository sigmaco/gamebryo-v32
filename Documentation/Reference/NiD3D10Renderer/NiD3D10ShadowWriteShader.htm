NiD3D10ShadowWriteShader

: public NiD3D10FragmentShader

<NiD3D10ShadowWriteShader.h>

Description

NiD3D10ShadowWriteShader is a specialized shader used by the shadowing system. The only difference from this shader and a normal NiD3D10FragmentShader is that in the PreProcessPipeline function this shader overrides the standard back face rendering state and sets it to back face rendering state the shader has been supplied.

Constructors

NiD3D10ShadowWriteShader(NiMaterialDescriptor* pkDesc)

This constructor simply passes the NiMaterialDescriptor up to the NiD3D10FragmentShader.

Supported Virtual Member Functions

virtual unsigned int PreProcessPipeline(const NiRenderCallContext& kRCC)

The function passes the Render Call Context to its parent's PreProcessPipeline function and then manually sets the culling mode base on the render back face state the shader has been supplied.

Supported Static Member Functions

static void SetRenderBackfaces(bool bRenderBackfaces)

Sets the back face rendering state to be used when rendering geometry to a shadow map.
