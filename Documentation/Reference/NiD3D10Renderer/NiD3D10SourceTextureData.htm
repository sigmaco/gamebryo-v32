NiD3D10SourceTextureData

: public NiD3D10TextureData

<NiD3D10SourceTextureData.h>

Description

NiD3D10SourceTextureData contains the D3D10 representation of a source texture, as well as the ID3D10ShaderResourceView that describes how the data should be interpreted.

Constructors

This class contains no public constructors the static Create function below is used to create NiD3D10SourceTextureData objects.

Static Member Functions

NiD3D10SourceTextureData* Create(NiSourceTexture* pkTexture)

Creates the renderer data for the specified NiSourceTexture. NiSourceCubeMap objects are also valid to be passed in; NiD3D10SourceTextureData represents both. The D3D10 source texture renderer data will be created from the associated NiPixelData object.

bool SetMipmapSkipLevel(unsigned int uiSkip)

unsigned int GetMipmapSkipLevel()

Get and set the number of mipmap levels to skip with loading the D3D10 representation from the texture.

Supported Member Functions

virtual void Update()

Forces the D3D10 representation of the texture to update itself based on the source texture.
