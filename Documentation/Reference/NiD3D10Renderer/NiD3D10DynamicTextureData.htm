NiD3D10DynamicTextureData

: public NiD3D10TextureData

<NiD3D10DynamicTextureData.h>

Description

NiD3D10DynamicTextureData contains the Direct3D10 representation of a dynamic texture. In general, an application will not have to manipulate objects of this class. Instead, an application will perform operations via an NiDynamicTexture object. If the current device does not directly support dynamic textures, then this class will approximate them by keeping a texture in local system memory and uploading it to the device when necessary.

Constructors

There are no publicly defined constructors for this class. The static Create function below is used to create NiD3D10DynamicTextureData objects.

Static Member Functions

NiD3D10DynamicTextureData* Create(NiDynamicTexture* pkTexture)

Creates the texture data for the specified NiDynamicTexture.

Supported Member Functions

virtual void* Lock(int& iPitch)

This function will return a pointer to a system memory block that is used to simulate the behavior of a dynamic texture. The Pitch of that system memory is returned by reference in iPitch. This call will return NULL if the lock operation fails.

virtual bool UnLock() const

This function unlocks the dynamic texture. The return value will be false if the unlock operation fails..

virtual bool IsLocked()

Returns whether or not the dynamic texture is currently locked.
