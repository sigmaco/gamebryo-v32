NiD3D10SystemDesc

: public NiMemObject

<NiD3D10SystemDesc.h>

Description

NiD3D10SystemDesc is a class that is designed to represent all of the adapters that are installed and configured in the machine. Its main purpose is to serve as a container class for all of the system's Gamebryo adapter descriptors. See the notes below for a hierarchy diagram of the classes that are attached to this system descriptor.

Constructors

There are no public constructors, as only the Gamebryo D3D10 renderer should create instances of this class.

Supported Member Functions

bool IsEnumerationValid() const

Returns a flag indicating whether or not the current enumeration is valid. The enumeration of adapters may have failed if the DXGI library failed to be loaded, or the IDXGIFactory failed creation, or if the enumeration itself failed.

bool Enumerate()

Cause the NiD3D10SystemDesc to perform an enumeration of the adapters. It should not be necessary for applications to call this unless they know the current enumeration is not valid.

unsigned int GetAdapterCount() const

Returns the number of adapters attached to the system. Valid adapter indices range from 0 to the return value of this function minus one.

const NiD3D10AdapterDesc* GetAdapterDesc(unsigned int uiIndex) const;

Returns the descriptor for the given adapter as attached to the system. The valid values are given by the adapter count function above.

IDXGIFactory* GetFactory() const

Returns a pointer to the IDXGIFactory.

Static Member Functions

static void GetSystemDesc(NiD3D10SystemDescPtr& spSystemDesc)

Returns an instance of the NiD3D10SystemDesc. If none exists, a new one will be created and enumerated.
