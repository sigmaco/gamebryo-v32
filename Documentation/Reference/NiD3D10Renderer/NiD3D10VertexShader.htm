NiD3D10VertexShader

: public NiD3D10ShaderProgram

<NiD3D10VertexShader.h>

Description

NiD3D10VertexShader is the shader program derivation for vertex shaders.

Constructors

NiD3D10VertexShader()

The default constructor creates an initialized NiD3D10VertexShader instance.

NiD3D10VertexShader(ID3D10VertexShader* pkVertexShader, ID3D10Blob* pkShaderByteCode)

This constructor creates an initialized NiD3D10VertexShader instance using the provided vertex shader and an ID3D10Blob containing the shader's byte code.

Supported Member Functions

ID3D10VertexShader* GetVertexShader() const

Returns the ID3D10VertexShader pointer.

void SetVertexShader(ID3D10VertexShader* pkVertexShader)

Sets the ID3D10VertexShader pointer. Note that the vertex shader and the shader byte code need to match up, so this function should not be called without a corresponding NiD3D10ShaderProgram::SetShaderByteCode.

virtual void DestroyRendererData()

Releases the ID3D10VertexShader reference.

virtual void RecreateRendererData()

Recreates the ID3D10VertexShader from the shader byte code, if it exists.
