NiD3D10RenderedTextureData

: public NiD3D10TextureData

<NiD3D10RenderedTextureData.h>

Description

NiD3D10RenderedTextureData contains the D3D10 representation of a source texture, as well as the ID3D10ShaderResourceView that describes how the data should be interpreted.

Constructors

This class contains no public constructors the static Create function below is used to create NiDX9SourceTextureData objects.

Static Member Functions

NiD3D10RenderedTextureData* Create(NiRenderedTexture* pkTexture, Ni2DBuffer::MultiSamplePreference eMSAAPref)

Creates the renderer data for the specified NiRenderedTexture with the preferred multisample antialiasing level. NiRenderedCubeMap objects are also valid to be passed in; NiD3D10RenderedTextureData represents both.
