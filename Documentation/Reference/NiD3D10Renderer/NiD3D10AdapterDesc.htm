NiD3D10AdapterDesc

: public NiMemObject

<NiD3D10SystemDesc.h>

Description

NiD3D10AdapterDesc is a descriptor of the capabilities of a piece of 3D rendering hardware. It describes the supported display devices attached to the adapter (as NiD3D10OutputDesc objects) , as well as the various supported driver types (as NiD3D10DeviceDesc objects).

Constructors

There are no public constructors, as only the Gamebryo D3D10 renderer should create instances of this class.

Supported Member Functions

const NiD3D10DeviceDesc* GetHWDevice() const

Returns an NiD3D10DeviceState that describes this adapter using the HW drivers.

const NiD3D10DeviceDesc* GetRefDevice() const;

Returna an NiD3D10DeviceState that describes this adapter using the Reference drivers.

unsigned int GetOutputCount() const

Returns the number of display devices attached to the device.

const NiD3D10OutputDesc* GetOutputDesc(unsigned int uiIndex) const

Returns a description of the display device at the given index.

const DXGI_ADAPTER_DESC* GetDesc() const

Returns the DXGI_ADAPTER_DESC that describes this adapter.

IDXGIAdapter* GetAdapter() const

Returns the IDXGIAdapter represented by this NiD3D10AdapterDesc.

unsigned int GetIndex() const

Returns the index of this NiD3D10AdapterDesc.

bool IsPerfHUD() const

Returns whether or not the adapter is the one reported by nVidia's drivers for use with the NvPerfHUD tool.
