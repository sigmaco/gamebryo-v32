NiD3D10DepthStencilBufferData

: public NiD3D102DBufferData

<NiD3D102DBufferData.h>

Description

NiD3D10DepthStencilBufferData is a Gamebryo class that represents a 2D surface that can act as a depth/stencil buffer. Gamebryo creates instances of this class internally to represent depth/stencil surfaces, but applications can also create instances if they have an ID3D10Texture2D that they want to use as a depth/stencil buffer.

Constructors

This class contains no public constructors. The static Create function below is used to create NiD3D10DepthStencilBufferData objects.

Static Member Functions

static NiD3D10DepthStencilBufferData* Create(ID3D10Texture2D* pkD3DTexture, NiDepthStencilBuffer*& pkBuffer, D3D10_DEPTH_STENCIL_VIEW_DESC* pkDSViewDesc = NULL)

Creates a new instance of an NiD3D10DepthStencilBufferData object. The pkD3DTexture parameter should be the texture that will be used as the depth/stencil buffer. This texture must have been created with the D3D10_BIND_DEPTH_STENCIL bind flag. The pkBuffer parameter will be the pointer that holds the NiDepthStencilBuffer object that references the NiD3D10DepthStencilBufferData. If pkBuffer is not empty, then it will be filled in with the newly created NiD3D10DepthStencilBufferData object. If it is empty, then when the function returns, pkBuffer will point to a new NiDepthStencilBuffer object. This new object will need to be assigned to a smart pointer to prevent it from getting leaked accidentally. This function will add a reference to the passed-in ID3D10Texture2D object. The pkDSViewDesc parameter contains the view desc that will be used to create the D3D10 depth stencil view. Passing NULL for this parameter will cause the view to be of mipmap level 0 of the entire resource.

Supported Member Functions

ID3D10DepthStencilView* GetDepthStencilView() const

Returns the depth stencil view.

ID3D10Texture2D* GetDepthStencilBuffer() const

Returns the texture for this depth stencil buffer.
