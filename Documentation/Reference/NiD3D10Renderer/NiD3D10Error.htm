NiD3D10Error

: public NiMemObject

<NiD3D10Error.h>

Description

NiD3D10Error is a Gamebryo class that handles error reporting in the NiD3D10Renderer library. It allows applications to programmatically query the library to determine the cause of any failure so that it can respond in an appropriate manner.

Constructors

There are no public constructors. Only the Gamebryo D3D10 renderer should create instances of this class.

Static Member Functions

static ErrorMessage GetLastErrorMessage()

Returns the enumeration of the last error message reported through NiD3D10Error. Possible enumerations are described here.

static const char* const GetErrorText(ErrorMessage eMessage)

Returns a text string describing the error message.
