NiD3D10ShaderProgramCreatorHLSL

: public NiD3D10ShaderProgramCreator

<NiD3D10ShaderProgramCreatorHLSL.h>

Description

NiD3D10ShaderProgramCreatorHLSL is the class that generates NiD3D10ShaderProgram objects from code written in D3D10 HLSL, stored either in a file or in a buffer in memory.

Constructors

This class has no public constructors.

Supported Member Functions

virtual unsigned int GetShaderCreationFlags() const

virtual void SetShaderCreationFlags(unsigned int uiFlags)

Set and retrieve an unsigned int value that, after getting OR-ed with the universal shader creation flags obtained from the base NiD3D10ShaderProgramCreator class, will be passed into the Flags parameter of the D3D10CompileShader function. The renderer-specific implementations will ensure that only flags that are appropriate for the shader assembly function are used.

Notes

File Extensions

By default, NiD3DShaderProgramCreatorHLSL is registered as the NiD3DShaderProgramCreator function to be used with files whose extension is .hlsl, .vsh, .gsh, or .psh. This registration can be changed using the function NiD3D10ShaderProgramFactory::RegisterShaderCreator, so that applications can specify their own extensions. Refer to the documentation of NiD3D10ShaderProgramFactory for more details.
