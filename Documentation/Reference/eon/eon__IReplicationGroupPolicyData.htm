Description

A class for use by IReplicationGroupPolicy instances for storing per-entity data.

These objects can be stored in and retrieved from the entity instance itself. As there can be hundreds of replicating entities care should be taken to ensure instances of this class are as small as possible. The producer entity will own these objects and will delete them when the entity is destroyed. IReplicationGroupPolicy implementations should lazy create these objects as needed. It is not required that policy data be used, if a policy does not need any per-entity data then it should not use this class.
