virtual bool GetCommandLineArgs(
int& argc,
char **& argv)

Get the command-line arguments for this invocation, if available.

On Windows the command line is limited to MM_WIN32_MAXARGS arguments (default 20). The entire command line must be less than MM_MAX_CMDLINE bytes (default 256). These constants can be adjusted in MangledMetal_Win32.h.

Parameter Description
argc (output) number of arguments
argv (output) the returned list of arguments
