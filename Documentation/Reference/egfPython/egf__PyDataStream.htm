egf::PyDataStream

<PyDataStream.h>

Description

The storage structure for a custom Python class.

The PyObject_HEAD macro defines all the python-specific goo and must come first. After that you simply list whatever data you need. If a PyObject* returns true from PyDataStream_Check then it is safe to cast the PyObject* to a PyDataStream*.
