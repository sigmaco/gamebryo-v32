Binary Stream Macros

#define EE_DeclareDerivedBinaryStream ()

This macro should be used within the class declaration of a class that derives from binary stream.

#define EE_ImplementDerivedBinaryStream(classname, rawread, rawwrite)

This macro should be used inside the implementation .cpp file of a class that derives from binary stream.

Parameter Description
classname The name of the class
rawread The function pointer for the raw read function
rawwrite The function pointer for the raw write function
