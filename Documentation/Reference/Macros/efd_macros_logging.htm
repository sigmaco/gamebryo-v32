Logging Macros

#define EE_LOG(module, level, args)

Macro to log a message and record the module, level, file, and line number information.

Parameter Description
module Module name
level Log level
args Message to log

#define EE_LOG_AT(module, level, args, file, line)

Macro to log a message and record the module, level, file, and line number information.

Note

This version is rarely used, use EE_LOG instead to automatically use the current file and line number. This version is used in the rare case where you wish to log a message from an external source that provides internal file and line number information.

Define EE_DISABLE_LOGGING to disable logging completely.

Parameter Description
module Module name
level Log level
args Message to log
file Filename where the log message originated.
line Line the log message occurred on.

#define EE_LOG_ONCE(module, level, args)

Macro to log a message only once while recording the module, level, file, and line number information.

This version prevents a log from filling up with duplicate messages.

Parameter Description
module Module name
level Log level
args Message to log

#define EE_OUTPUT_DEBUG_STRING(x)

Writes a string to standard debug output.

Parameter Description
x String to write out.

#define WriteToStdErr(x)

Writes a string to standard error.

Parameter Description
x String to write out.
