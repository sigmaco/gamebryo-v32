Property Macros

#define EE_DECLARE_PRIMITIVE_ASSOCIATIVE_ARRAY_PROPERTY(Typename, Type, ClassID)

Macro to create associative array properties using the TPropertyAssoc template class.

Parameter Description
Typename The name of the property
Type The type to use
ClassID The classID where the property will exist

#define EE_DECLARE_PRIMITIVE_SCALAR_PROPERTY(Typename, Type, ClassID)

Macro to create scalar properties using the TPropertyScalar template class.

Parameter Description
Typename The name of the property
Type The type to use
ClassID The classID where the property will exist
