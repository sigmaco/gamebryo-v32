NiMessageBox

<NiMessageBox.h>

Description

A platform-independent class to present a dialog box to the user. On the consoles, the contents of the dialog box are written to the debug output.

Static Member Functions

static void SetMessageBoxFunction(MessageBoxFunction pfnMessageBox)

static MessageBoxFunction GetMessageBoxFunction()

Gets and sets a callback function pointer that will be used to present a message box. This allows an application to completely control the process of displaying a message box. If the callback is set to NULL, the default Gamebryo-provided callback function will be used.

static unsigned int DisplayMessage(const char* pcText, const char* pcCaption, void* pvExtraData = NULL)

Calls upon the NiMessageBox system to display a message box with the provided text and caption. The callback provided by in SetMessageBoxFunction will be used to display and handle the message box. If the callback is set to NULL, the default Gamebryo-provided callback function will be used.

Callback Function

typedef unsigned int (*MessageBoxFunction) (const char* pcText, const char* pcCaption, void* pvExtraData)

This is the prototype of the callback function that can be used to construct and display the message box. It must handle the entire process of creating, showing, responding to, and deleting the message box.

Notes

Backwards compatibility

For backwards compatibility, the following function (not declared in the scope of NiMessageBox) is still supported:

unsigned int NiMessageBox(const char* pcText, const char* pcCaption, void* pvExtraData = NULL)

Calls upon the NiMessageBox system to display a message box with the provided text and caption. The callback provided by in SetMessageBoxFunction will be used to display and handle the message box. If the callback is set to NULL, the default Gamebryo-provided callback function will be used.
