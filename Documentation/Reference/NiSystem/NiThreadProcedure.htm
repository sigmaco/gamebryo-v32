NiThreadProcedure

: public efd::ThreadFunctor

<NiThreadProcedure.h>

DescriDescription

This class exists for backwards compatibility as part of the bridge layer, providing a wrapper for the Foundation class efd::ThreadFunctor.

When an efd::Thread is executed, the virtual method Execute will be called on the associated ThreadFunctor. However, NiThreadProcedure implements Execute as a call to the virtual method NiThreadProcedure::ThreadProcedure.

Because of this, either method may be overridden to implement a thread procedure, but it is recommended that all new code make use of efd::ThreadFunctor and implement Execute.

Supported Member Functions

virtual unsigned int ThreadProcedure(efd::Thread* pkThread)

This virtual function is called by the efd::Thread framework to do all useful work associated with the NiThreadProcedure
