Information Log and System Output Routines

NiLogger

A configurable logging system that supports formatted output to the debug window or a file on platforms that support file writing. Refer to the NiLogger class reference documents for usage and details.

void NiWriteToStdErr(char* pcText)

Writes the given text to stderr (or the closest platform-dependent equivalent).

void NiOutputDebugString(const char* pcOut)

Sends the given text to the NiLogger system via NIMESSAGE_GENERAL_0. By default, this sends a message to the debug window (or the closest platform-dependent equivalent) in Debug builds.

void NiMessageBox(const char* pcText, const char* pcCaption)

Displays a message box (or the closest platform-dependent equivalent), titled pcCaption, containing text pcText. When possible, the box will have an OK button that the user can dismiss to allow the application to continue.
