NiDebug

<NiDebug.h>

Description

The macros in this file have been renamed and moved into the Foundation library header <efd/Asserts.h>. The header file remains as part of the bridge layer between the Core Runtime and Foundation, presenting an equivalent interface (via renaming macros) for backwards compatibility.

Registering a custom assert handler is now handled via the class efd::AssertHelper.

All new code should use the Foundation names instead of the legacy names.
