NiBinaryLoadSave

<NiBinaryLoadSave.h>

Description

This is a standalone header file that defines a set of utility functions for streaming. Some of these functions are templated, with T representing the templated class. These functions are useful for writing the streaming functionality in classes that wish to use Gamebryo's streaming system. Because these functions all use NiBinaryStream, they all respect the endianness settings of their binary stream parameters.

Static Functions

void NiBinaryStreamLoad (NiBinaryStream& is, T* pValue, unsigned int uiNumEls = 1)

This function loads uiNumEls elements from the binary stream is into the array located at pValue. pValue must be allocated before this call.

void NiBinaryStreamLoad (NiBinaryStream& is, void* pvValue, unsigned int uiNumEls, unsigned int* puiComponentSizes, unsigned int uiNumComponents = 1)

This function loads an array of uiNumEls elements, each of which is a potentially heterogeneous structure of components into the memory location pointed to by pvValue. This memory must be allocated before the function call. Each element is made up of a set of uiNumComponents components, whose sizes are listed in order in the array puiComponentSizes.

void NiBinaryStreamLoadEnum (NiBinaryStream& is, T* pValue)

Load a single enumerated value from the binary stream is into the memory location pointed to by pValue. pValue must be allocated before this function call.

void NiBinaryStreamSave (NiBinaryStream& os, const T* pValue, unsigned int uiNumEls = 1)

This function reads uiNumEls elements from the array located at pValue and saves them into the binary stream os.

void NiBinaryStreamSave (NiBinaryStream& os, const void* pvValue, unsigned int uiNumEls, unsigned int* puiComponentSizes, unsigned int uiNumComponents)

This function saves an array of uiNumEls elements, each of which is a potentially heterogeneous structure of components, into the binary stream os. The array that is saved out is located in the memory location pointed to by pvValue. Each element is made up of a set of uiNumComponents components, whose sizes are listed in order in the array puiComponentSizes.

void NiBinaryStreamSaveEnum (NiBinaryStream& os, const T& value)

Save a single enumerated value from the memory location referred to by value into the binary stream os.
