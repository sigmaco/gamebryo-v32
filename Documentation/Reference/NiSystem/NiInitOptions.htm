NiInitOptions

: public efd::InitOptions

<NiInitOptions.h>

Description

A platform-independent class designed to encapsulate any arguments to NiInit. It also contains arguments that will be passed to EE_INIT from within NiInit, as it derives from efd::InitOptions.

Constructors

NiInitOptions()

Create an NiInitOptions object that matches the Gamebryo defaults.

Supported Member Functions

Supported PS3-Only Member Functions

unsigned int GetJobWorkloadCount() const

void SetJobWorkloadCount(NiUInt32 uiJobWorkloadCount)

Gets/sets the number of job workloads that Floodgate will reserve on the SPURS instance. The default is 14, but a value between 1 and 15 can be specified.
