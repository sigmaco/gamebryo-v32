NiBinaryShaderLibrary

: public NiD3DShaderLibrary

<NiBinaryShaderLibrary.h>

Description

NiBinaryShaderLibrary is the derived NiD3DShaderLibrary class that provides applications with an access point to binary shader (NSB) files. The library will create NiBinaryShader objects for shaders found from the NSBShader objects returned NSBLoader component of NSBShaderLib.

Constructors

The constructor is protected the access point for NiBinaryShaderLibrary instances is the static Create function.

Member Typedefs

typedef NiBinaryShader* (*CREATENIBINARYSHADER)(const char* pcClassName)

This is the prototype for the callback function that is made when the NiBinaryShaderLib library is creating an NiBinaryShader instance. The callback receives the class name, pcClassName, which was associated with the implementation that is being converted into a run-time shader. A registered callback should use the name to determine whether an NiBinaryShader-derived class is required and if so create and return it. Otherwise, they should return NULL and the system will create a standard NiBinaryShaderinstance.

Supported Static Functions

static NiBinaryShaderLibrary* Create(int iDirectoryCount, char* apcDirectories[], bool bRecurseSubFolders)

This function will create an instance of the NiBinaryShaderLibrary. The function LoadNSBShaders is then called, which passes the parameters to the library's NSBLoader object so it can load the NSB files it finds. The NSBLoader will search for all NSB files, loading them and adding them to the library's list of available shaders. If the load function fails, the instance is deleted and NULL is returned, otherwise the library instance is returned.

static void Shutdown()

This function will shut down the static instance of the NiBinaryShaderLibrary.

static NiBinaryShaderLibrary* GetLibrary()

Retrieve the static instance of the NiBinaryShaderLibrary.

static CREATENIBINARYSHADER SetCreateNiBinaryShaderCallback(CREATENIBINARYSHADER pfnCallback)

This function is used to set the CreateNiBinaryShader callback the application wishes to use. This MUST be set prior to loading any NSB shaders. Passing in NULL will result in the default callback function, NSBShader::DefaultCreateNiBinaryShader, being set as the callback.

static CREATENIBINARYSHADER GetCreateNiBinaryShaderCallback()

This function returns the currently set CreateNiBinaryShader callback.

static NiBinaryShader* DefaultCreateNiBinaryShader(const char* pcClassName)

This is the default implementation of the CreateNiBinaryShader callback. If a valid class name is passed to it, it simply reports the occurrence through the NiD3DRenderer::Warning function.
