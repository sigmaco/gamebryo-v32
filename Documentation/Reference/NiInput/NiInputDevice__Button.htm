NiInputDevice::Button

<NiInputDevice.h>

Description

The NiInputDevice::Button sub-class provides a representation for a button of a device. It is important to note that every button is treated as analog with a raw range of 0-255. If a button is physically a digital one, its value will be set to 255 when it is pressed. With respect to modifiers (buttons that are held prior to a button being pressed), if a button is greater than or equal to an analog threshold (typically 128), then the modifier is considered on.

The button sub-class should be considered internal to the NiInputDevice class, but is described here to give some technical information about the make-up of a device in the system.

Notes

No application should ever create or access an instance of an NiInputDevice::Button class. The entire class should be considered an 'internal use only'.
