NiD3D10BinaryShader Class Reference: Introduction

The NiD3D10BinaryShader library demonstrates a fully developed NiD3DShader derivation. The library provides the basis for Gamebryo's binary shader (NSB) format. Developers should feel free to have their own custom written parser libraries generate NSB format files to remove the need to develop a custom binary format as well. The NSB format is provided to allow for platform-independent binary storage of shaders.

The NiD3D10BinaryShader library defines the following classes:

  • NiD3D10BinaryShader
  • NiD3D10BinaryShaderLibrary
