Description

The KFMFactoryRequest is used to request an asynchronous flat model file load.

The resulting NiActorManager instance is returned to the caller in an KFMFactoryResponse.

This is the only type of request that the KFMFactory will respond to.

The accumulation parameter can be used to specify whether or not the actor created is set up to use accumulation.
