Description

The KFMFactoryResponse is used to return an NiActorManager from a KFMFactoryRequest sent to a KFMFactory.

If GetStatus() == ALR_Success, then GetActor() will return the successfully completed actor.

If GetStatus() == ALR_PartialSuccess, then GetActor() will return the actor, but some of the animation sequences may not have loaded. The actor will display on screen, but may not animate properly.

If GetStatus() is some error, then GetActor() will return false.
