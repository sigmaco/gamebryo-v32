Description

The ActorModel class provides the built-in model functionality for the Actor model in the Standard Model Library.

Entites that mix in Actor use this built-in, and the Animation Service makes use of its properties. This model is primarily a data container. However, it is responsible for notifying the egm::AnimationService that an entity with the Actor model has been created or deleted. This is done in the OnAdded and OnRemoved methods.

Public Static Methods

static egf::IBuiltinModel* ActorModelFactory()

The factory class for the Actor built-in model.

It is public to allow the egmAnimation::AnimationService to register the factory in its PreInit method.
