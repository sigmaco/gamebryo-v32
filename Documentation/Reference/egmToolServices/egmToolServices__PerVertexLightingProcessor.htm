Inheritance graph for egmToolServices::PerVertexLightingProcessor

Description

A render list processor which removes the normal, parallax, and bump maps from an object's texturing property to cause NiStandardMaterial to use per-vertex lighting as opposed to per-pixel.

This processor caches the created NiTexturingProperties to avoid the speed penalty of creating new NiTexturingProperties for each render.
