NiScriptInfo

: public NiRefObject

<NiScriptInfo.h>

Description

This class is the container for specifying a set of plug-ins to be executed by the NiPluginManager. The order in which plug-ins are to be executed and their respective parameters are contained within an internal NiPluginInfo.
