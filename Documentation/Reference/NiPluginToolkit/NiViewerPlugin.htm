NiViewerPlugin

: public NiPlugin

<NiViewerPlugin.h>

Description

This class is an extension of NiPlugin designed to support viewing information.

Constructors

NiViewerPlugin(const char* pcName, const char* pcVersion, const char* pcShortDescription, const char* pcLongDescription)

The constructor takes the same inputs as the NiPlugin constructor: the display name of the plug-in, its version, a short description, and a long description.

Overridden Virtual Functions

virtual bool CanExecute(NiPluginInfo* pkInfo)

This function is overridden from NiPlugin to check if pkInfo is an NiViewerPluginInfo object. If it is, the function returns the result of CanView. Otherwise, the function returns false.

virtual NiExecutionResultPtr Execute(const NiPluginInfo* pkInfo)

This function is overridden from NiPlugin to check if pkInfo is an NiViewerPluginInfo object. If it is, the function returns the result of View. Otherwise, the function returns failure.

Pure Virtual Functions

The following virtual functions must be overridden in any NiViewerPlugin-derived class:

virtual bool CanView(NiViewerPluginInfo* pkInfo) = 0

This function takes a NiViewerPluginInfo script and returns whether or not the plug-in can execute that script.

virtual NiExecutionResultPtr View(const NiViewerPluginInfo* pkInfo) = 0

This function executes the provided NiViewerPluginInfo script and returns a NiExecutionResult indicating success or failure.

virtual bool SupportsRenderer(NiSystemDesc::RendererID eRenderer) = 0

This function returns whether or not the plug-in supports the specified renderer.
