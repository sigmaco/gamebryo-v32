NiBatchExecutionResult

: public NiExecutionResult

<NiBatchExecutionResult.h>

Description

This class encapsulates the results of the execution of a series of plug-ins.  It contains an array of NiExecutionResults.  It also contains a function to return all the results in the form of a NiScriptInfo script.

Constructors

NiBatchExecutionResult(ReturnCode eReturnCode)

The constructor takes the main return code to set for this result.  This return code should be EXECUTE_SUCCESS if all the results in its array are EXECUTE_SUCCESS.  If any plug-ins failed to execute successfully, eReturnCode should be EXECUTE_FAILURE.

Supported Member Variables

NiTObjectArray<NiExecutionResultPtr> m_pkPluginResults

This is the array of NiExecutionResults representing the results from an array of plug-ins.  It is a public variable.

Supported Member Functions

bool WasSuccessful()

This function returns true if all the NiExecutionResults in the array are EXECUTE_SUCCESS and false otherwise.
