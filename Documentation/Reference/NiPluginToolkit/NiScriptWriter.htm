NiScriptWriter

:public NiRefObject

<NiScriptWriter.h>

Description

This class is the interface for the encoding of NiScriptInfo objects to character strings. An NiScriptReader is usually complemented with an NiScriptWriter.

Supported Member Functions

NiString Encode(NiScriptInfo* pkScriptInfo)

This method is called to write an NiScriptInfo object to a character string.

NiString GetFileExtensions()

This method returns the file extensions that this writer supports.  The format for this string is "*.EXTENSION;*.EXTENSION2".
