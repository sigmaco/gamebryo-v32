NiProgressDialog

: public NiDialog

<NiProgressDialog.h>

Description

This simple dialog is the standard progress dialog used by the tool plug-ins. It contains two lines of text and a progress control.

Constructors

NiProgressDialog(NiString strTitle)

Default constructor.

Supported Member Functions

virtual int DoModal()

This method always returns IDCANCEL. This method should not be called. Instead, call Create and Destroy.

void SetLineOne(NiString strFullText)

This method sets the first line of text on the output.

void SetLineTwo(NiString strFullText)

This method sets the second line of text on the output

void StepIt()

This method updates the progress bar control one position

void SetRangeSpan(unsigned int uiSpan)

This method sets the overall number of ticks on the progress bar control.

void SetPosition(unsigned int uiPos)

This method sets the current tick of the progress bar control.
