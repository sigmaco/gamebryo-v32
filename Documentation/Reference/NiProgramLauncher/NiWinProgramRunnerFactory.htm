Inheritance graph for NiWinProgramRunnerFactory

Description

This Windows-specific factory is used to create NiWinProgramRunner objects and query information about the platform.

Public Methods

~NiWinProgramRunnerFactory()

Virtual destructor.

virtual efd::SystemDesc::PlatformID GetPlatformID()

Returns NiSystemDesc::NI_WIN32.

virtual const char* GetPlatformDisplayName()

Returns "PC".

virtual const char* GetPlatformInternalName()

Returns "Win32".

virtual const char* GetPlatformExecutableFileExt()

Returns "exe".

virtual bool GetAvailableTargetNames(
NiStringArray& kTargetNames,
NiString* pkErrorInfo = NULL)

Returns all available Windows machines that can be used to launch an executable.

NOTE: At this time, NiWinProgramRunnerFactory only supports the local PC (not remote PCs)

Parameter Description
kTargetNames Returns a list of machine names associated with the platform. The NI_PROGRAMRUNNER_LOCAL_TARGET_NAME constant represents the name of the local PC.
pkErrorInfo Optional string used to return error messages
Return Value False if an error was detected. Pass string in as pkErrorInfo for additional details.

virtual NiProgramRunner* Create()

Create an NiWinProgramRunner object that can be used to launch a Windows exe.

It is the caller's responsibility to delete the object returned by this method.

Parameter Description
Return Value NULL if a program runner could not be created
