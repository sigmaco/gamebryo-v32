Inheritance graph for NiWinProgramRunner

Description

This class is used to launch Windows applications on the local PC.

Public Methods

virtual ~NiWinProgramRunner()

Virtual destructor.

Destroying the runner object does not terminate the program that was launched by it.

virtual bool Start(
const char* pcTargetName,
const char* pcExecutableFilename)

Launch windows exe on the specified target machine.

At this time, the NiWinProgramRunner only supports the local workstation, not remote PCs.

Parameter Description
pcTargetName The name of target where the executable is to be launched. This can be queried from the NiProgramLauncher singleton or a NiWinProgramRunnerFactory object
pcExecutableFilename The path and filename to the exe file on the local workstation
Return Value True if the executable was successfully launched. Check output window if the launch failed for error messages.
