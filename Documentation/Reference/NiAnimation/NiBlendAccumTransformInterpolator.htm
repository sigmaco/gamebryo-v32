NiBlendAccumTransformInterpolator

: public NiBlendInterpolator

<NiBlendAccumTransformInterpolator.h>

Description

NiBlendAccumTransformInterpolator is responsible for blending NiQuatTransform values together and accumulating the results of that blend.

Constructors

NiBlendAccumTransformInterpolator(bool bManagerControlled, float fWeightThreshold, unsigned char ucArraySize)

The constructor creates an NiBlendAccumTransformInterpolator using the specified weight threshold and array size. Stored times for the sub-interpolators are used if bManagerControlled is true. Also note that NiBlendInterpolators do not stream their sub-interpolators when the bManagerControlled flag is true, they are instead streamed by their parent NiControllerSequence.
