NiPoint3Evaluator

: public NiKeyBasedEvaluator

<NiPoint3Evaluator.h>

Description

NiPoint3Evaluator uses NiPosKey objects to animate an NiPoint3 value over time. This NiKeyBasedEvaluator only supports one key channel. A posed (i.e. constant) channel is stored using the key array. As a result, such posed evaluators are handled more efficiently as an NiConstPoint3Evaluator.

Evaluators are primarily intended for sequence-based animation. Refer to the Animation Sequences section in the Introduction to Animation for more details.

Supported Member Functions

NiPosData* GetPoint3Data() const

These functions get and set the NiPosData for this NiPoint3Evaluator.

NiPosKey* GetKeys(unsigned int& uiNumKeys, NiPosKey::KeyType& eType, unsigned char& ucSize) const

This function returns the keys currently held by this evaluator. The number of keys, key type, and key stride are returned in uiNumKeys, eType,and ucSize.

void ReplaceKeys(NiPosKey* pkKeys, unsigned int uiNumKeys, NiPosKey::KeyType eType)

This function specifies the key data to store. Note that NiPoint3Evaluator will not copy the NiPosKeys -- it assumes ownership of the keys and deletes any previous keys.

void SetKeys(NiPosKey* pkKeys, unsigned int uiNumKeys, NiPosKey::KeyType eType)

This function specifies the key data to store. Note that NiPoint3Evaluator will not copy the NiPosKeys -- it simply overwrites the pointers to the old keys. This action can easily cause memory leaks, so calls to SetKeys should generally be avoided in favor of ReplaceKeys.
