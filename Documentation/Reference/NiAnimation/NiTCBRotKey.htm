NiTCBRotKey

: public NiRotKey

<NiTCBRotKey.h>

Description

NiTCBRotKeys are used to represent a smoothly interpolated series of rotations. Each key contains a time value (the parameterization), a quaternion and TCB control values.

Constructors

NiTCBRotKey()

NiTCBRotKey(float fTime, const NiQuaternion& kQuat)

The default constructor creates an uninitialized NiTCBRotKey. The second form of the constructor supplies the time and specifies the rotation via an angle axis representation. The third and final form supplies a time and a quaternion.
