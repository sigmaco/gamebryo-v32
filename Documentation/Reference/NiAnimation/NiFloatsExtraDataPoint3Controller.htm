NiFloatsExtraDataPoint3Controller

: public NiExtraDataController

<NiFloatsExtraDataPoint3Controller.h>

Description

NiFloatsExtraDataPoint3Controller animates three contiguous values simultaneously on an NiFloatsExtraData object attached to an NiAVObject in the scene graph.

Constructors

NiFloatsExtraDataPoint3Controller(const NiFixedString& kFloatsExtraDataName, int iFloatsExtraDataIndex = -1)

The constructor takes the name of the NiFloatsExtraData object to control and the index of the desired three floats in the extra data to update.

Virtual Function Overrides

virtual void Update(float fTime)

Overridden from NiInterpController. Implemented to update the single interpolator.

virtual bool UpdateValue(float fTime, const NiPoint3& kPoint3, unsigned short usIndex = 0)

Overridden from NiInterpController. This function sets the associated NiPoint3 attribute to the specified value. This function always returns true.

Supported Member Functions

int GetFloatsExtraDataIndex() const

void SetFloatsExtraDataIndex(int iFloatsExtraDataIndex)

These functions get and set the index of the first of three contiguous values in the NiFloatsExtraData object that the controller animates.
