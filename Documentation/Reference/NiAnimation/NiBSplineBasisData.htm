NiBSplineBasisData

: public NiObject

<NiBSplineBasisData.h>

Description

NiBSplineBasisData contains an NiBSplineBasis for use in interpolation of open, uniform B-Splines. This class is separate from NiBSplineData because NiBSplineBasis caches its values internally so that subsequent calls will save calculations. Since multiple NiBSplineInterpolator and NiBSplineEvaluator objects may have the same NiBSplineBasis, this value is encapsulated in an NiObject container, allowing it to be cloned and streamed appropriately.
