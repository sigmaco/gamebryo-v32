NiLookAtEvaluator

: public NiEvaluator

<NiLookAtEvaluator.h>

Description

NiLookAtEvaluator rotates a "Driven" object so that it always faces a "Looked At" object. This class is mainly used to make a camera or spot light track a specific node in a character. Dependency cycles (e.g. one look-at controlled object looking at another look-at controlled object that looks back at the first) are broken arbitrarily. In order to provide a complete NiQuatTransform, an NiLookAtEvaluator may contain up to three sub-evaluators. These optional sub-evaluators control the translation, scale, and roll angle of the Driven object.

The AlwaysUpdate function for this class returns true by default since the Looked At object may move between successive update calls.

Note that Gamebryo does not guarantee that the Looked At object will be updated before the NiLookAtEvaluator. During sequence animation the driven transform computed by NiLookAtEvaluator will lag one frame behind. This behavior has minimal visual impact when frame rates are sufficiently high.

Evaluators are primarily intended for sequence-based animation. Refer to the Animation Sequences section in the Introduction to Animation for more details.

Constructors

NiLookAtEvaluator()

This constructor creates an NiLookAtEvaluator with no Driven or Looked At objects. +X is the default axis. Supported member functions can be used at a later time to specify the Driven and Looked At objects as well as other attributes.

NiLookAtEvaluator(const NiFixedString& kLookAtObjectName, const NiFixedString& kDrivenObjectName, Axis eAxis = X, bool bFlip = false)

This constructor creates an NiLookAtEvaluator with the specified Looked At and Driven objects. If the evaluator is contained within a sequence, these names must refer to nodes within the character on which the sequence is played. See the supported member functions for a description of the other parameters.

Virtual Function Overrides

Overridden from NiEvaluator.

Supported Member Functions

const NiFixedString& GetDrivenObjectName() const

void SetDrivenObjectName(const NiFixedString& kName)

These functions get and set the name of the object driven by the NiLookAtEvaluator. If the evaluator is contained within a sequence, this name must refer to a node within the character on which the sequence is played.

const NiFixedString& GetLookAtObjectName() const

void SetLookAtObjectName(const NiFixedString& kName)

These functions get and set the name of the object the NiLookAtEvaluator forces its Driven object to follow (i.e. look at). If the evaluator is contained within a sequence, this name must refer to a node within the character on which the sequence is played.

Axis GetAxis() const

void SetAxis(Axis eAxis)

Get and set the axis of the Driven object that will be made to face the Looked At object.

bool GetFlip() const

void SetFlip(bool bFlip)

Get and set whether the sense of the axis the Driven object looks down is reversed. This parameter allows all six primary axes to be specified.

Get and set the sub-evaluator controlling the position of the Driven object.

NiEvaluator* GetRollEvaluator()

Get and set the sub-evaluator controlling the roll angle of the Driven object.

NiEvaluator* GetScaleEvaluator()

void SetScaleEvaluator(NiEvaluator* pkEval)

Get and set the sub-evaluator controlling the scale of the Driven object.

Notes

Axis

The axis down which the Driven object should look is specified by the enumeration Axis, which has possible values of X, Y, and Z.

Channel

The Channel enumeration is outlined as follows. These enumerations are useful for querying information about a specific animation channel in an NiLookAtEvaluator.

Value Meaning
POSITION This value specifies the position animation channel.
ROTATION This value specifies the rotation animation channel.
SCALE This value specifies the scale animation channel.
