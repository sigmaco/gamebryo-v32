NiTextureTransformController

: public NiFloatInterpController

Description

NiTextureTransformController is used to animate a single member of an NiTextureTransform.

Constructors

NiTextureTransformController(NiTexturingProperty* pkTarget = NULL, NiTexturingProperty::Map* pkMap = NULL, TransformMember eMember = TT_TRANSLATE_U)

The constructor takes the NiTexturingProperty target of this controller, the Map in that property the contains the texture, and the TransformMember indicating which aspect of the texture transform to modify.

Virtual Function Overrides

virtual const char* GetCtlrID()

Overridden from NiInterpController. The ID indicates which TransformMember is being modified.

virtual void Update(float fTime)

Overridden from NiInterpController. Implemented to update the single interpolator.

virtual bool UpdateValue(float fTime, float fFloat, unsigned short usIndex = 0)

Overridden from NiInterpController. This function sets the associated float attribute to the value specified by fFloat. This function always returns true.

Supported Member Functions

NiTexturingProperty::Map* GetMap()

bool SetMap(NiTexturingProperty::Map* pkMap, NiTexturingProperty* pkTarget = NULL)

These functions get and set the NiTexturingProperty::Map and NiTexturingProperty associated with this NiTextureTransformController.

TransformMember GetAffectedMember()

void SetAffectedMember(TransformMember eMember)

These functions get and set the NiTextureTransform member associated with this controller. The possible members are outlined in the Notes section.

Notes

TransformMember

The TransformMember enumeration is outlined as follows.

Value Meaning
TT_TRANSLATE_U Controls the translation of the U coordinates.
TT_TRANSLATE_V Controls the translation of the V coordinates.
TT_ROTATE Controls the rotation of the coordinates.
TT_SCALE_U Controls the scale of the U coordinates.
TT_SCALE_V Controls the scale of the V coordinates.
