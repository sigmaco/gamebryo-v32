NiLookAtInterpolator

: public NiInterpolator

<NiLookAtInterpolator.h>

Description

NiLookAtInterpolator rotates an object so that it always faces a "Looked At" object. It is mainly used to make a camera or spot light track a character. Dependency cycles (e.g. one look-at controlled object looking at another look-at controlled object that looks back at the first) are broken arbitrarily. In order to provide a complete NiQuatTransform, an NiLookAtInterpolator may contain up to three sub-evaluators. These optional sub-evaluators control the translation, scale, and roll angle of the NiLookAtInterpolator.

Note that Gamebryo does not guarantee that the "Looked At" object will be updated before the NiLookAtInterpolator. During scene graph updates the target transform computed by NiLookAtInterpolator may lag one frame behind. This behavior has minimal visual impact when frame rates are sufficiently high.

Constructors

NiLookAtInterpolator()

This constructor creates an NiLookAtInterpolator with no target object to follow. +X is the default axis. Supported member functions can be used at a later time to specify the target as well as other attributes.

NiLookAtInterpolator(NiAVObject* pkLookAt, Axis eAxis = X, bool bFlip = false)

This constructor creates an NiLookAtInterpolator with the specified target object to follow. See the supported member functions for a description of each of these parameters.

Virtual Function Overrides

virtual bool Update(float fTime, NiObjectNET* pkInterpTarget, NiQuatTransform& kValue)

Overridden from NiInterpolator. Remember to check each of the three values in an NiQuatTransform for validity before using them. This call causes the NiLookAtInterpolator to return the transform that will cause the target object to face the "Looked At" object.

virtual bool IsTransformValueSupported() const

Overridden from NiInterpolator. Returns true to indicate that NiQuatTransform values are supported.

virtual void Collapse()

Overridden from NiInterpolator.

Notes

Axis

The axis down which the NiLookAtInterpolator should look is specified by the enumeration Axis, which has possible values of X, Y, and Z.
