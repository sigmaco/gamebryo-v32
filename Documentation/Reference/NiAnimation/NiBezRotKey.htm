NiBezRotKey

: public NiRotKey

<NiBezRotKey.h>

Description

NiBezRotKeys are used to represent a smoothly interpolated series of rotations. Each key contains a quaternion.

Constructors

NiBezRotKey()

NiBezRotKey(float fTime, float fAngle, const NiPoint3& kAxis)

NiBezRotKey(float fTime, const NiQuaternion& kQuat)

The default constructor creates an uninitialized NiBezRotKey. The second form of the constructor supplies the time and specifies the rotation via an angle axis representation. The third and final form supplies a time and a quaternion.
