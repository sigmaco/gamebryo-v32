NiConstTransformEvaluator

: public NiEvaluator

<NiConstTransformEvaluator.h>

Description

NiConstTransformEvaluator supports a constant (i.e. posed) NiQuatTransform value. There are three animation channels on this evaluator: position, rotation, and scale. It is important to note that a valid NiQuatTransform returned from this evaluator may have an invalid position, rotation, or scale. Please see the documentation on NiQuatTransform on how to query the class for its valid values. Please refer to the notes to see the enumerations for the NiEvaluator channels supported by NiConstTransformEvaluator.

Evaluators are primarily intended for sequence-based animation. Refer to the Animation Sequences section in the Introduction to Animation for more details.

Constructors

NiConstTransformEvaluator()

This constructor creates an NiConstTransformEvaluator containing invalid data.

NiConstTransformEvaluator(const NiQuatTransform& kPoseValue)

This constructor creates an NiConstTransformEvaluator with the specified posed value.

Supported Member Functions

This function sets the posed transform value for this evaluator.

void SetPoseTranslate(const NiPoint3& kTranslate)

This function sets the posed translate value for this evaluator.

void SetPoseRotate(const NiQuaternion& kRotate)

This function sets the posed rotation value for this evaluator.

void SetPoseScale(float fScale)

This function sets the posed scale value for this evaluator.

Notes

Channel

The Channel enumeration is outlined as follows. These enumerations are useful for querying information about a specific animation channel in an NiConstTransformEvaluator.

Value Meaning
POSITION This value specifies the position animation channel.
ROTATION This value specifies the rotation animation channel.
SCALE This value specifies the scale animation channel.
