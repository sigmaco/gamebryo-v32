NiBSplineColorInterpolator

: public NiBSplineInterpolator

<NiBSplineColorInterpolator.h>

Description

NiBSplineColorInterpolator uses B-Splines to animate an NiColorA value over time. This NiBSplineInterpolator only supports one control point channel.

Constructors

NiBSplineColorInterpolator()

This constructor creates an NiBSplineColorInterpolator containing invalid data.

NiBSplineColorInterpolator(NiBSplineData* pkData, NiBSplineData::Handle kColorACPHandle = NiBSplineData::INVALID_HANDLE, NiBSplineBasisData* pkBasisData = NULL)

This constructor creates an NiBSplineColorInterpolator with the specified NiColorData.

NiBSplineColorInterpolator(const NiColorA& kPoseValue)

This constructor creates an NiBSplineColorInterpolator with a pose value of kPoseValue.
