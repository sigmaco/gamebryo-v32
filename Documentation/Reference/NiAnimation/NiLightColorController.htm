: public NiPoint3InterpController

Description

NiLightColorController animates the ambient, diffuse and specular colors of an NiLight-derived object. If in ambient mode, it will animate the ambient color while forcing the diffuse and specular colors to be black. In non-ambient mode it will force the ambient color to be black while animating the diffuse and specular colors. Since only three color components are used by this controller, it is derived from NiPoint3InterpController instead of NiColorInterpController.
