NiBSplineColorEvaluator

: public NiBSplineEvaluator

<NiBSplineColorEvaluator.h>

Description

NiBSplineColorEvaluator uses B-Splines to animate an NiColorA value over time. This NiBSplineEvaluator only supports one control point channel. If the channel is not animated, the posed evaluator can be handled more efficiently as an NiConstColorEvaluator.

Evaluators are primarily intended for sequence-based animation. Refer to the Animation Sequences section in the Introduction to Animation for more details.

Constructors

NiBSplineColorEvaluator()

This constructor creates an NiBSplineColorEvaluator containing invalid data.

NiBSplineColorEvaluator(NiBSplineData* pkData, NiBSplineData::Handle kColorACPHandle, NiBSplineBasisData* pkBasisData)

This constructor creates an NiBSplineColorEvaluator with the specified data objects and control point handles.

Virtual Function Overrides

virtual unsigned short GetChannelCount() const

Overridden from NiBSplineEvaluator. This value should be 1 for NiBSplineColorEvaluator objects.

virtual unsigned int GetDimension(unsigned short usChannel) const

Overridden from NiBSplineEvaluator. This value should be 4 for NiBSplineColorEvaluator objects.

virtual unsigned int GetDegree(unsigned short usChannel) const

Overridden from NiBSplineEvaluator. This value should be 3 for NiBSplineColorEvaluator objects.

virtual NiBSplineData::Handle GetControlHandle(unsigned short usChannel) const

Overridden from NiBSplineEvaluator. This function returns the control point handle for the specified channel. This handle is used when interpolating the value in the NiBSplineData object.

virtual void SetControlHandle(NiBSplineData::Handle kControlHandle, unsigned short usChannel)

Overridden from NiBSplineEvaluator. This function sets the control point handle for the specified channel. This handle is used when interpolating the value in the NiBSplineData object.

Notes

Channel

The Channel enumeration is outlined as follows. These enumerations are useful for querying information about a specific animation channel in an NiBSplineColorEvaluator.

Value Meaning
COLORA This value specifies the color animation channel.
