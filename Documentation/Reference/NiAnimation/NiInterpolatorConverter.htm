NiInterpolatorConverter

<NiInterpolatorConverter.h>

Description

The NiInterpolatorConverter static class is used to convert an NiInterpolator to an equivalent NiEvaluator. Equivalency is determined by the contents of the animation channel and not necessarily the RTTI of the specified interpolator. For instance, an NiFloatInterpolator containing posed animation data will be converted to an NiConstFloatEvaluator.

Evaluators perform a similar role as interpolators; however, evaluators are more efficient for sequence animation. Interpolators are automatically converted into equivalent evaluators using NiInterpolatorConverter when sequences are generated with the Gamebryo art export tools. Refer to the Animation Sequences section in the Introduction to Animation for more details regarding sequence animation and evaluators.

Constructors

As a static class, NiInterpolatorConverter has no supported constructors.

Supported Static Member Functions

static NiEvaluator* Convert(NiInterpolator* pkInterpolator, const NiEvaluator::IDTag& kIDTag, bool bCollapseInterpolator)

Converts the specified interpolator to an equivalent evaluator with the specified tag. The animation channels in the interpolator will be collapsed before conversion if bCollapseInterpolator is true. Collapsing the interpolator in this manner ensures any interpolator with posed (i.e. constant data) channels to be converted to an equivalent const evaluator. NULL is returned if the specified interpolator cannot be converted.
