NiBSplineFloatEvaluator

: public NiBSplineEvaluator

<NiBSplineFloatEvaluator.h>

Description

NiBSplineFloatEvaluator uses B-Splines to animate a float value over time. This NiBSplineEvaluator only supports one control point channel. If the channel is not animated, the posed evaluator can be handled more efficiently as an NiConstFloatEvaluator.

Evaluators are primarily intended for sequence-based animation. Refer to the Animation Sequences section in the Introduction to Animation for more details.

Constructors

NiBSplineFloatEvaluator()

This constructor creates an NiBSplineFloatEvaluator containing invalid data.

NiBSplineFloatEvaluator(NiBSplineData* pkData, NiBSplineData::Handle kFloatACPHandle, NiBSplineBasisData* pkBasisData)

This constructor creates an NiBSplineFloatEvaluator with the specified data objects and control point handles.

Virtual Function Overrides

virtual unsigned short GetChannelCount() const

Overridden from NiBSplineEvaluator. This value should be 1 for NiBSplineFloatEvaluators.

virtual unsigned int GetDimension(unsigned short usChannel) const

Overridden from NiBSplineEvaluator. This value should be 1 for NiBSplineFloatEvaluators.

virtual unsigned int GetDegree(unsigned short usChannel) const

Overridden from NiBSplineEvaluator. This value should be 3 for NiBSplineFloatEvaluators.

virtual NiBSplineData::Handle GetControlHandle(unsigned short usChannel) const

Overridden from NiBSplineEvaluator. This function returns the control point handle for the specified channel. This handle is used when interpolating the value in the NiBSplineData object.

virtual void SetControlHandle(NiBSplineData::Handle kControlHandle, unsigned short usChannel)

Overridden from NiBSplineEvaluator. This function sets the control point handle for the specified channel. This handle is used when interpolating the value in the NiBSplineData object.

Channel

The Channel enumeration is outlined as follows. These enumerations are useful for querying information about a specific animation channel in an NiBSplineFloatEvaluator.

Value Meaning
