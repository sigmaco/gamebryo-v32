NiQuaternionInterpController

: public NiSingleInterpController

<NiQuaternionInterpController.h>

Description

NiQuaternionInterpController is the base class for all NiInterpControllers that use an NiInterpolator to animate their target NiQuaternion value. The interpolator set for this controller must support NiQuaternion values.

Constructors

This is an abstract base class and therefore has no public constructors.

Supported Member Functions

There are no public supported non-inherited member functions in this class.
