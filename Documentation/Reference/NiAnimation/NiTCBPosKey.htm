Description

NiTCBPosKeys are used to represent a 3-D TCB spline. Each key contains a time value (the parameterization), a position and the TCB control values.

Constructors

NiTCBPosKey()

NiTCBPosKey(float fTime, const NiPoint3& kPos, float fTension, float fContinuity, float fBias)

The default constructor creates an uninitialized NiTCBPosKey. The second form of the constructor supplies the time, position and TCB controls.
