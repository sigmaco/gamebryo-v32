NiPoint3Interpolator

: public NiKeyBasedInterpolator

<NiPoint3Interpolator.h>

Description

NiPoint3Interpolator uses NiPosKeys to animate an NiPoint3 value over time. This NiKeyBasedInterpolator only supports one key channel.

Constructors

NiPoint3Interpolator()

This constructor creates an NiPoint3Interpolator containing invalid data.

NiPoint3Interpolator(NiPosData* pkPosData)

This constructor creates an NiPoint3Interpolator with the specified NiPosData.

NiPoint3Interpolator(const NiPoint3& kPoseValue)

This constructor creates an NiPoint3Interpolator with a pose value of kPoseValue and a NULL NiPosData.

Virtual Function Overrides

virtual bool Update(float fTime, NiObjectNET* pkInterpTarget,NiPoint3& kValue)

Overridden from NiInterpolator. If there are animation keys in this interpolator, the value will be the interpolated value from the Update call. Otherwise, this value will be the pose value if there are no keys and the pose has been set.

virtual bool IsPoint3ValueSupported() const

Overridden from NiInterpolator. Returns true indicating that NiPoint3 values are supported.

virtual void Collapse()

Overridden from NiInterpolator.

virtual NiAnimationKey::KeyType GetKeyType(unsigned short usChannel) const

Overridden from NiKeyBasedInterpolator. The channel should always be zero for an NiPoint3Interpolator.

Overridden from NiKeyBasedInterpolator. The channel should always be zero for an NiPoint3Interpolator.

Overridden from NiKeyBasedInterpolator. The channel should always be zero for an NiPoint3Interpolator.

Supported Member Functions

NiPosData* GetPoint3Data() const

void SetPoint3Data(NiPosData* pkPosData)

These functions get and set the NiPosData for this NiPoint3Interpolator.

void SetPoseValue(const NiPoint3& kPoseValue)

This function sets the pose value for this interpolator. If the interpolator has an NiPosData object, it is set to NULL.

NiPosKey* GetKeys(unsigned int& uiNumKeys, NiPosKey::KeyType& eType, unsigned char& ucSize) const

This function returns the keys currently held by this interpolator. The number of keys, key type, and key stride are returned in uiNumKeys, eType,and ucSize.

void ReplaceKeys(NiPosKey* pkKeys, unsigned int uiNumKeys, NiPosKey::KeyType eType)

This function specifies the key data to store. Note that NiPoint3Interpolator will not copy the NiPosKeys -- it assumes ownership of the keys and deletes any previous keys.

void SetKeys(NiPosKey* pkKeys, unsigned int uiNumKeys, NiPosKey::KeyType eType)

This function specifies the key data to store. Note that NiPoint3Interpolator will not copy the NiPosKeys -- it simply overwrites the pointers to the old keys. This action can easily cause memory leaks, so calls to SetKeys should generally be avoided in favor of ReplaceKeys.
