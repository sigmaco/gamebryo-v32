NiTransformInterpolator

: public NiKeyBasedInterpolator

<NiTransformInterpolator.h>

Description

NiTransformInterpolator supports the animation of position, rotation, and/or scale using an NiQuatTransform. This NiQuatTransform can be an unchanging pose or interpolated from animation key channels. There are three animation key channels on this interpolator: position (using NiPosKeys), rotation (using NiRotKeys), and scale (using NiFloatKeys). It is important to note that a valid NiQuatTransform returned from this interpolator may have an invalid position, rotation, or scale. Please see the documentation on NiQuatTransform on how to query the class for its valid values. Please refer to the notes to see the enumerations for the NiKeyBasedInterpolator key channels supported by NiTransformInterpolator.

Notes

Channel

The Channel enumeration is outlined as follows. These enumerations are useful for querying generic animation information in this class' implementation of NiKeyBasedInterpolator methods.

Value Meaning
POSITION This value specifies the position animation key channel.
ROTATION This value specifies the rotation animation key channel.
SCALE This value specifies the scale animation key channel.
