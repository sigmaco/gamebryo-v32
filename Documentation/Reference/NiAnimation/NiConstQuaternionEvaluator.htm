NiConstQuaternionEvaluator

: public NiEvaluator

<NiConstQuaternionEvaluator.h>

Description

NiConstQuaternionEvaluator supports a constant (i.e. posed) NiQuaternion value. This NiEvaluator only supports one channel.

Evaluators are primarily intended for sequence-based animation. Refer to the Animation Sequences section in the Introduction to Animation for more details.

Constructors

NiConstQuaternionEvaluator()

This constructor creates an NiConstQuaternionEvaluator containing invalid data.

NiConstQuaternionEvaluator(const NiQuaternion& kPoseValue)

This constructor creates an NiConstQuaternionEvaluator with the specified posed value.

Supported Member Functions

void SetPoseValue(const NiQuaternion& kPoseValue)

This function sets the posed value for this NiConstQuaternionEvaluator.

Notes

Channel

The Channel enumeration is outlined as follows. These enumerations are useful for querying information about a specific animation channel in an NiConstQuaternionEvaluator.

Value Meaning
ROTATION This value specifies the rotation animation channel.
