NiGeomMorpherConverter

<NiGeomMorpherConverter.h>

Description

The NiGeomMorpherConverter class is used to convert assets that use the NiGeomMorpherController class to use the NiMorphMeshModifier class. Applications should never create one of these objects - one is created automatically as needed.

Static Member Functions

static void Convert(NiStream& kStream, NiTLargeObjectArray<NiObjectPtr>& kTopObjects)

Converts all of the assets in the scene graphs rooted at the elements of kTopObjects. The stream kStream is the one the object was loaded from.
