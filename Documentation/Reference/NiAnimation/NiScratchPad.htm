NiScratchPad

: public NiObject

<NiScratchPad.h>

Description

NiScratchPad holds instance-based data for evaluators in an active sequence (i.e. NiControllerSequence). Many evaluators cache their current animation curve segment in the scratch pad which allows the evaluators to reuse this localized data across successive update times thereby increasing runtime performance. Evaluators and their associated scratch pad data are primarily intended for sequence-based animation.

Supported Member Functions

There are no supported member functions as NiScratchPad acts as an Emergent internal extension to NiControllerSequence.

Notes

NiScratchPadBlock enumeration

<NiScratchPadBlock.h>

The NiScratchPadBlock enumeration lists the types of data blocks supported by NiScratchPad. These values are utilized exclusively by Emergent internal functions.
