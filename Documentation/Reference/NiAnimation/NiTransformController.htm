NiTransformController

: public NiSingleInterpController

<NiTransformController.h>

Description

NiTransformController uses an NiInterpolator to animate its target's translation, rotation, and/or scale using an NiQuatTransform value. This class replaces the NiKeyframeController class from previous versions of Gamebryo. The interpolator set for this controller must support NiQuatTransform values.

NiTransformController objects are not utilized for sequence animation. Instead, the NiMultiTargetPoseHandler propagates animated transform data directly to scene graph nodes. NiTransformController objects may still be used for non-sequence based animation.

Constructors

NiTransformController()

This constructor creates a default transform controller with no interpolator.
