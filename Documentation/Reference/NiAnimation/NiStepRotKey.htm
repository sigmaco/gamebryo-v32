NiStepRotKey

: public NiRotKey

<NiStepRotKey.h>

Description

NiStepRotKeys are used for NiQuaternion value animation. There is no interpolation performed between keys. Instead, the value of the last key encountered is used when interpolating.

Constructors

NiStepRotKey()

NiStepRotKey(float fTime, const NiQuaternion& quat)

NiStepRotKey(float fTime, float fAngle, const NiPoint3& axis)

The default constructor creates an uninitialized NiStepRotKey. The second form of the constructor supplies the time and quaternion while the third specifies the rotation using an angle axis representation.
