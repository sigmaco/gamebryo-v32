NiAnimationConstants

<NiAnimationConstants.h>

Description

The NiAnimationConstants class contains static functions which provide access to constants shared by several classes in the animation system. Most often these constants are related to strings found within KF and NIF files.

Constructors

As a static class, NiAnimationConstants has no supported constructors.

Supported Static Member Functions

Functions returning names of controller types

static const NiFixedString& GetFlipCtlrType()

static const NiFixedString& GetGeomMorpherCtlrType()

static const NiFixedString& GetMorphWeightsCtlrType()

static const NiFixedString& GetPSEmitParticlesCtlrType()

static const NiFixedString& GetTransformCtlrType()

These functions return the strings found in KF and NIF files which specify the controller types of NiFlipController, NiGeomMorpherController, NiMorphWeightsController, NiPSEmitParticlesCtlr, and NiTransformController objects.

Functions returning interpolator IDs

static const NiFixedString& GetBirthRateInterpID()

static const NiFixedString& GetEmitterActiveInterpID()

These functions return the strings found in KF and NIF files which specify the interpolator IDs for Birth Rate and Emitter Active interpolators referenced by the particle system controller NiPSEmitParticlesCtlr. Evaluators utilize these same strings as evaluator IDs.

Functions concerning the name of the non-accumulation child node of the accumulation root

static const NiFixedString& GetNonAccumSuffix()

This function returns the suffix which the Controller Extractor export plug-in concatenates onto the name of the accumulation root node to form the name of the non-accumulation child node.

static int GetNonAccumSuffixLength()

This function returns the length of the string returned by GetNonAccumSuffix.

Functions dealing with common strings contained within text keys

static const NiFixedString& GetStartTextKey()

static const NiFixedString& GetEndTextKey()

These functions return the strings of text keys that occur at the start and end of each sequence.

static const NiFixedString& GetMorphTextKey()

This function returns the common prefix used by all morph text keys.

static int GetMorphTextKeyLength()

This function returns the length of the string returned by GetMorphTextKey.

static NiTextKeyMatch* GetEndOfSequenceMatch()

This function should only be used internally by NiActorManager to both determine which keys are the end keys and which text key callbacks are end of sequence callbacks.
