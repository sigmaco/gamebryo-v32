NiBlendTransformInterpolator

: public NiBlendInterpolator

<NiBlendTransformInterpolator.h>

Description

NiBlendTransformInterpolator is responsible for blending NiQuatTransform values together.

Constructors

NiBlendTransformInterpolator(bool bManagerControlled, float fWeightThreshold, unsigned char ucArraySize)

The constructor creates an NiBlendTransformInterpolator using the specified weight threshold and array size. Stored times for the sub-interpolators are used if bManagerControlled is true. Also note that NiBlendInterpolators do not stream their sub-interpolators when the bManagerControlled flag is true, they are instead streamed by their parent NiControllerSequence.

Virtual Function Overrides

virtual bool Update(float fTime, NiObjectNET* pkInterpTarget, NiQuatTransform& kValue)

Overridden from NiInterpolator. Returns true to indicate that the returned NiQuatTransform value is valid.

virtual bool IsTransformValueSupported() const

Overridden from NiInterpolator. Returns true to indicate that NiQuatTransform values are supported.
