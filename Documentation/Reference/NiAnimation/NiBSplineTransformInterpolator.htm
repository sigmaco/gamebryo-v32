NiBSplineTransformInterpolator

: public NiBSplineInterpolator

<NiBSplineTransformInterpolator.h>

Description

NiBSplineTransformInterpolator supports the animation of position, rotation, and/or scale using an NiQuatTransform. This NiQuatTransform can be an unchanging pose or interpolated from B-Spline control point channels. There are three animation control point channels on this interpolator: position, rotation, and scale. It is important to note that a valid NiQuatTransform returned from this interpolator may have an invalid position, rotation, or scale. Please see the documentation on NiQuatTransform for how to query the class for its valid values. Please refer to the notes to see the enumerations for the NiBSplineInterpolator channels supported by NiBSplineTransformInterpolator.

Notes

Channel

The Channel enumeration is outlined as follows. These enumerations are useful for querying generic animation information in this class' implementation of NiBSplineInterpolator methods.

Value Meaning
POSITION This value specifies the position animation key channel. This value is of dimension 3 and degree 3.
ROTATION This value specifies the rotation animation key channel. This value is of dimension 4 and degree 3.
SCALE This value specifies the scale animation key channel. This values is of dimension 1 and degree 3.
