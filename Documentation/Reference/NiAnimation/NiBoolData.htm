NiBoolData

: public NiObject

<NiBoolData.h>

Description

The NiBoolData class provides a shareable wrapper for Boolean animation keys.

Constructors

NiBoolData()

The default constructor creates an NiBoolData with no animation data.

Supported Member Functions

unsigned int GetNumKeys() const

This call simply returns the number of keys the NiBoolData is currently storing.

NiBoolKey* GetAnim (unsigned int &uiNumKeys, NiBoolKey::KeyType &eType, unsigned char &ucSize) const

This call returns the key data that the NiBoolData currently stores.

void ReplaceAnim(NiBoolKey* pkBoolKeys, unsigned int uiNumKeys, NiBoolKey::KeyType eType)

This call gives the NiBoolData animation data to store. Note that the NiBoolData will not copy the NiBoolKey objects—it will use them directly and delete any existing keys.

void SetAnim(NiBoolKey* pkBoolKeys, unsigned int uiNumKeys, NiBoolKey::KeyType eType)

This call gives the NiBoolData animation data to store. Note that the NiBoolData will not copy the NiBoolKey objects—it will use them directly but does not delete any existing keys. This method is used to transfer ownership of keys.
