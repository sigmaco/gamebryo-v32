Description

NiBezFloatKeys, in spite of their name, are used to represent a 1-D Hermite spline. Each key contains a float value and two tangents.

Constructors

The default constructor creates an uninitialized NiBezFloatKey. The second form of the constructor supplies the time, value and tangents.
