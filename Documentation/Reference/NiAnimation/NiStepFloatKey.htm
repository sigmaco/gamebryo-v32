NiStepFloatKey

: public NiFloatKey

<NiStepFloatKey.h>

Description

NiStepFloatKeys are used for floating point value animation. There is no interpolation performed between keys. Instead, the value of the last key encountered is used when interpolating.

Constructors

NiStepFloatKey()

NiStepFloatKey(float fTime, float fValue)

The default constructor creates an uninitialized NiStepFloatKey. The second form of the constructor supplies the time and float value.
