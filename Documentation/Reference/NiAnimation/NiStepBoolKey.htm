NiStepBoolKey

: public NiBoolKey

<NiStepBoolKey.h>

Description

NiStepBoolKeys are used for standard boolean animation. There is no interpolation performed between keys. Instead, the value of the last key encountered is used when interpolating.

Constructors

NiStepBoolKey()

NiStepBoolKey(float fTime, bool bBool)

The default constructor creates an uninitialized NiStepBoolKey. The second form of the constructor supplies the time and boolean value.
