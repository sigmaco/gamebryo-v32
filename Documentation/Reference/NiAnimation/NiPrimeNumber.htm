NiPrimeNumber

<NiPrimeNumber.h>

Description

The NiPrimeNumber static class contains several functions dealing with the first 170 or so prime numbers. These static functions are intended for use with algorithms such as hash functions which perform better when utilizing prime numbers of a given magnitude. The animation system makes use of such hash functions to locate tagged objects more quickly.

Constructors

As a static class, NiPrimeNumber has no supported constructors.

Supported Static Member Functions

static unsigned int GetPrimeArrayCount()

This function returns the number of primes handled by the NiPrimeNumber class. These correspond to the first 170 or so primes starting at 2.

static unsigned int GetPrimeAt(unsigned int uiIndex)

This function returns the prime number at the specified index. uiIndex must be less than GetPrimeArrayCount.

static unsigned int GetMaxPrime()

The function returns the largest prime number supported by the NiPrimeNumber class.

static bool GetNearbyPrime(unsigned int uiNumber, unsigned int& uiPrime)

The function finds the smallest prime number that is greater than or equal to uiNumber. The found value is returned via uiPrime. This function returns true when uiNumber is less than or equal to GetMaxPrime, and false otherwise.
