KFM Viewer

Type: VIEWER

Version: 1.1

<KFMViewer.dll>

Description

The KFM Viewer plug-in runs the Controller Extractor and the NIF Exporter plug-ins on the scene graph and saves the NIF file, a single KF file, and a KFM file in a temporary location. It then spawns the viewer that handles KFM files, passing it the newly created KFM file as input. The result of this sequence of steps is that the Gamebryo Animation Tool is launched with a file that contains all the animation sequences contained in the scene. If the scene has no animation sequences defined, this plug-in will fail to execute successfully.

Options

KFM Viewer Options

Renderer Type

This section allows you to choose the renderer type that will be used in the Animation Tool when it is run. The choices are DX9, D3D10, or D3D11. If D3D10 or D3D11 is specified, the Animation Tool is run with "-D3D10" or "-D3D11" as a command-line parameter.

Artist Notes

Please see the documentation on the Controller Extractor plug-in and the Gamebryo Animation Tool ( Gamebryo Animation Tool User Interface) to take full advantage of this plug-in.
