Remove Tagged Objects

Type: PROCESS

Version: 2.0

<RemoveTaggedObjects.dll>

Description

Removes objects that are tagged with "NiRemoveObject" from the scene graph. If an NiNode is tagged to be removed, it and all of its children that can be removed will be. To tag an object, add an NiStringExtraData that has the string "NiRemoveObject". This can be done in Maya by adding a string extra attribute. In Max this can be done by adding the text "NiRemoveObject" to the User-Defined properties of an object. Be careful to double check the capitalization because the test for "NiRemoveObject" is case sensitive. This plug-in will not remove skinned geometry, bones, or objects that are referenced by other objects in the scene graph.

Options

There are no options for this plug-in.
