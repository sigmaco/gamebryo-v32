NIF Importer

Type: IMPORT

Version: 1.2

<NifIO.dll>

Description

The NIF Importer plug-in reads a NIF file from disk and stores it in the NiSceneGraphSharedData object in the shared data list, creating that shared data object if one does not exist. Any additional objects at the top-level of the NIF file are placed in a NiExtraObjectsSharedData object in the shared data list.

Options

There are no options for this plug-in.
