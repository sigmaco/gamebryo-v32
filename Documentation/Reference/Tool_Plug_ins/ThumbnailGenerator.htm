Thumbnail Generator

Type: PROCESS

Version: 2.0

<ThumbnailGeneratorPlugin.dll>

Description

This plug-in is designed to generate a thumbnail image that will be embedded within a NIF file. When combined with the NiThumbnailExtractor plug-in for Windows Explorer, this allows users to view thumbnail previews of their NIF and KFM files in Windows Explorer. This plug-in writes out a temporary NIF file and then runs the ImageGenerator application on the NIF file.

To view the thumbnails, you will need to install the ThumbnailExtractor COM DLL.

Controlling the thumbnail camera

In order to adjust the default camera for thumbnails, artists can create a scene camera called "ThumbnailCam". The generator will use this camera for generating the thumbnail if it exists. This camera will not appear in the NIF file when using the ThumbnailGenerator plug-in.

Options

Users can adjust the export targets that generate the thumbnail images. This may be desirable to reduce file size for platform-ready assets. Additionally, users can adjust the coordinate system (ImageGenerator CoordType) and size of the thumbnail.
