NIF Viewer

Type: VIEWER

Version: 1.2

<NIFViewer.dll>

Description

This plug-in runs the NIF Exporter plug-in on the scene graph and saves a NIF file in a temporary directory. It then spawns the viewer that handles NIF files, passing it the newly created NIF file as input. The result of this sequence of steps is that the Gamebryo Asset Viewer is launched with the NIF file representing the current scene graph.

Options

NIF Viewer Options

Renderer Type

This section allows you to choose the renderer type that will be used in the Asset Viewer when it is run. The choices are DX9, D3D10, or D3D11. If D3D10 or D3D11 is specified, the Asset Viewer is run with "-D3D10" or "-D3D11" as a command-line parameter.
