NIF Exporter

Type: EXPORT

Version: 1.2

<NifIO.dll>

Description

The NIF Exporter plug-in grabs the scene graph found in the NiSceneGraphSharedData object in the shared data list and saves it out to a NIF file.

The root node of the scene graph and all cameras in the scene graph are saved as top-level objects in the NIF file. If a NiExtraObjectsSharedData instance is present in the shared data, any objects in it that are marked as streamable are also exported as top-level objects in the NIF.

Options

There are no options for this plug-in.
