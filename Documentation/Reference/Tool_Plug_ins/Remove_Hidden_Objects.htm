Remove Hidden Objects

Type: PROCESS

Version: 1.0

<RemoveHiddenObjects.dll>

Description

This plug-in will delete AppCulled objects from the scene graph. Typically this would be used during character export form Max where you want to delete the unused bone geometry. It is important to remember that this plug-in does not consider if an objects visibility is animated or if other objects in the scene reference the hidden object.

Options

There are no options for this plug-in.
