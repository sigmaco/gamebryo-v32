Normal Visualizer

Type: PROCESS

Version: 2.0

<NormalVisualizer.dll>

Description

This plug-in is designed to give detailed feedback on the normals of all the geometry in the scene. A short line representing each normal is added to the scene. It does not work for morphed or skinned geometry. It is simply a snapshot at time zero of the normals.

Options

You can save the contents of the dialog to a text file by clicking the Save As button.
