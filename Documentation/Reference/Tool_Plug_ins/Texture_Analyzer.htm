Texture Analyzer

Type: PROCESS

Version: 1.0

<TextureAnalyzer.dll>

Description

This plug-in is designed to give detailed feedback on the textures used by the scene. For each texture in the scene graph, the file name, dimensions, pixel format, pixel mask sizes, palette information, total size in kilobytes, and number of mip-map levels are displayed.  Careful observation of these numbers can highlight trouble spots.

Options

You can save the contents of the dialog to a text file by clicking the Save As button.
