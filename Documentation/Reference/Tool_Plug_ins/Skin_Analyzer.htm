Skin Analyzer

Type: PROCESS

Version: 2.0

<SkinAnalyzer.dll>

Description

This plug-in is designed to give detailed feedback on hardware partitioning of a skinned model. For each skin in the scene graph, a list of all of the partitions are generated, including the triangle count for each partition. The higher the triangle count per partition, the better your performance will become. The fewer partitions you have, the better your performance will become. The second listing for a skinned mesh details the exact bones in each partition, their mean and maximum weights in the partition. Careful observation of these numbers can highlight trouble spots.

Options

You can save the contents of the dialog to a text file by clicking the Save As button.
