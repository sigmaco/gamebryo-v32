Mesh Finalize

Type: PROCESS

Version: 1.0

<MeshFinalize.dll>

Description

This plug-in is used by the mesh profile system and is an instrumental part of the tool plug-in pipeline. Users who wish to have their exported scenes affected by profiles should ensure that this is the last process plug-in the pipeline. This library is built upon the NiMeshProfileProcessor and it is expected that exporters and other tools have properly initialized that system at the beginning of the application and properly shut it down when the application terminates. This plug-in also supports calling out to platform specific pre- and post-finalize functions in DLLs that perform additional mesh manipulation.

Options

There are no options for this plug-in.
