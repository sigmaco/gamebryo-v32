Animation Analyzer

Type: PROCESS

Version: 1.0

<AnimationAnalyzer.dll>

Description

This plug-in is designed to give detailed feedback on the animation keys used in the scene graph. Note that it does not support compressed animation sequences since the compression process occurs in the export phase instead of the process phase.

Options

You can save the contents of the dialog to a text file using the "Save As" button.
