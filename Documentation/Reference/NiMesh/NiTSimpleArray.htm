NiTSimpleArray

<NiTSimpleArray.h>

Description

Templated fixed-size array class.

Useful when using an array on the stack. It is also useful to describe a nested array for use with an NiTStridedRandomAccessIterator for vertex elements like blend indices or blend weights.
