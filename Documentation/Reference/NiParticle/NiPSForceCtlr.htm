Inheritance graph for NiPSForceCtlr

Description

Abstract base class for all particle force time controllers.

Contains functions for resolving a force name to an NiPSForce pointer.

Public Methods

virtual void SetTarget(NiObjectNET* pkTarget)

Overridden from the base class to resolve the force name into an NiPSForce pointer.

const NiFixedString& GetForceName() const

Returns the force name (used to find the force pointer).

void SetForceName(const NiFixedString& kForceName)

Sets the force name.

If the force pointer has already been resolved from the name, changing the name with this function will have no effect. This function should only be called on a controller before its target has been set.

NiPSForce* GetForcePointer() const

Returns a pointer to the NiPSForce object that this controller animates.

If the controller has no target, NULL is returned.
