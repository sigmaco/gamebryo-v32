NiPSForceDefinitions::BombData

<NiPSForceDefinitions.h>

Description

Data required by the NiPSBombForce class.

Public Types

NiPSForceDefinitions::BombData::DecayType

The type of decay used by the bomb.

Name Description
DECAY_NONE The force will not decay over distance.
DECAY_LINEAR The force will decay linearly with distance: (Decay - Distance) / Decay.
DECAY_EXPONENTIAL The force will decay exponentially with distance: NiExp(-Distance / Decay).
NUM_DECAY_TYPES The maximum number of decay types. This should be increased if more decay types are added.

NiPSForceDefinitions::BombData::SymmType

The type of symmetry employed by the bomb.

Name Description
SYMM_SPHERICAL The force is applied spherically about the position of the bomb object.
SYMM_CYLINDRICAL The force is applied perpendicular to the direction specified by the bomb axis and the rotation of the bomb object but centered about the position of the bomb object.
SYMM_PLANAR The force is applied parallel to the direction specified by the bomb axis and the rotation of the bomb object.
NUM_SYMM_TYPES The maximum number of symmetry types. This should be increased if more symmetry types are added.
