NiPSSimulatorForcesKernelStruct

<NiPSSimulatorForcesKernel.h>

Description

A structure containing data required by the NiPSSimulatorForcesKernel Floodgate kernel.

Public Attributes

float m_fCurrentTime

The current simulation time.

size_t m_stDataBufferSize

The size of the force data buffer in bytes.

size_t m_astDataRanges[NiPSForceDefinitions::NUM_FORCE_TYPES]

An array containing range values in bytes for the force data sections in the data buffer.

A data range is provided here for each type of force. The data range includes from the first byte of the force data section through the 16-byte-aligned end of that section. It does not include the 16 byte header preceding the data block that specifies the force type.
