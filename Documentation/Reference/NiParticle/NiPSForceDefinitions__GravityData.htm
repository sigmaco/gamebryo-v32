NiPSForceDefinitions::GravityData

<NiPSForceDefinitions.h>

Description

Data required by the NiPSGravityForce class.

Public Types

NiPSForceDefinitions::GravityData::ForceType

The type of the gravity force.

Name Description
FORCE_PLANAR The force operates parallel to the direction computed from the gravity object and the gravity axis.
FORCE_SPHERICAL The force operates spherically about the position of the gravity object.
NUM_FORCE_TYPES The maximum number of force types. This should be increased if more force types are added.

Public Attributes

NiPoint3 m_kGravityAxis

A unit vector specified in gravity object space that determines the direction of the gravity force.

float m_fDecay

How the gravity force is diminished by distance.

float m_fStrength

The acceleration of the gravity force.

float m_fTurbulence

The amount of turbulence to apply to particles.

float m_fTurbulenceScale

The amount by which the turbulence value will be scaled prior to being applied.

ForceType m_eForceType

The type of gravity force to apply.
