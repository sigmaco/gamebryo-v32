egmTerrain::TTimeOfDayEntityFunctor

<TTimeOfDayEntityFunctor.h>

Description

Functor class allowing egf::Entity object to be registered with the NiTimeOfDay object.

This is a templated class that can be associated with any property types supported by the NiTimeOfDay. It will use the member variable set by the constructor to allow the NiTimeOfDay object to animate the appropriate property values.

NiTTimeOfDayDefaultFunctor for more information.
