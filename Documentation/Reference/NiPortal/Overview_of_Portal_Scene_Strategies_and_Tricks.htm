Overview of Portal Scene Strategies and Tricks

There are an almost infinite number of ways to organize a dataset for rooms, walls and portals. Some will work better than others, but sometimes the tradeoffs are quite subtle. This section sets out to describe one major approach to dividing a dataset. This method has its pros and cons, and it is up to the team of artists and developers to decide which exact system is likely to produce the most optimal dataset for a particular application. Following this, some specific performance topics are discussed. This document assumes that the reader is familiar with the basics of portals, walls, rooms and room groups as described in the Portals Basics document.
