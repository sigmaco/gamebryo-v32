Portals and Avoiding Backbuffer Clears

A set of convex rooms representing an interior scene tend to form closed environments (other than their portals, which in turn connect to other rooms). Because the viewer is certain to always see something in such an environment, most applications using the portals module to draw interiors can turn off backbuffer clears, since the scene will already be drawing to every pixel on the screen. Note, however, that applications using read-write Z-buffering (i.e. just about all modern applications) will still have to clear the depth buffer.
