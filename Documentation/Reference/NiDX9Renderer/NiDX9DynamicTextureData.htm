NiDX9DynamicTextureData

: public NiDX9TextureData

<NiDX9DynamicTextureData.h>

Description

NiDX9DynamicTextureData contains the DX representation of a dynamic texture. In general, an application will not have to manipulate objects of this class but will instead perform operations via an NiDynamicTexture object. If the current device does not directly support dynamic textures, then this class will approximate them by keeping a texture in local system memory and uploading it to the device when necessary.

Constructors

There are no publicly defined constructors for this class. The static Create function below is used to create NiDX9DynamicTextureData objects.

Static Member Functions

NiDX9RenderedTextureData* Create(NiDynamicTexture* pkTexture)

Creates the renderer data for the specified NiDynamicTexture.

Supported Member Functions

virtual D3DBaseTexturePtr GetSysMemTexture() const

If the current device does not directly support dynamic textures, this function will return the local system memory texture that is used to simulate the behavior of a dynamic texture. If the devices does support dynamic textures, then this function will return NULL.

virtual bool IsLocked()

Returns whether or not the dynamic texture is currently locked.
