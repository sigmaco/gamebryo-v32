NiD3DHLSLPixelShader

: public NiD3DPixelShader

<NiD3DHLSLPixelShader.h>

Description

NiD3DPixelShader is the shader program derivation for pixel shaders that have been compiled from DirectX HLSL source.

Constructors

NiHLSLPixelShader(NiD3DRenderer* pkRenderer)

The default constructor creates an initialized NiD3DHLSLPixelShader instance.

Supported Member Functions

virtual void DestroyRendererData()

Called when the renderer is being recreated, this function will call NiD3DRenderer::PurgePixelShader, which will release the device-specific data relative to the shader program.

virtual void RecreateRendererData()

Called when the renderer has completed being recreated, this function will call NiD3DRenderer::RecreatePixelShader, which will restore the device-specific data relative to the shader program.

virtual bool SetShaderConstant(NiShaderConstantMapEntry* pkEntry, const void* pvDataSource = NULL, unsigned int uiRegisterCount = 0)

Allows the shader program to set the pixel shader constant described in pkEntry, using the data in pvDataSource and the register count in uiRegisterCount. If pvDataSource is NULL or uiRegisterCount is 0, the function will use the appropriate values from pkEntry. The function is provided so that shader programs in different formats can set their constants in a manner they expect.

virtual bool SetShaderConstantArray(NiShaderConstantMapEntry* pkEntry, const void* pvDataSource, unsigned int uiNumEntries, unsigned int uiRegistersPerEntry, const unsigned short* pusReorderArray = NULL)

Allows the shader program to set an array of pixel shader constants described in pkEntry, using the data in pvDataSource, the number of entries in uiNumEntries, the value from uiRegistersPerEntry, and the array pusReorderArray. The number of entries corresponds directly to the number of values provided in pvDataSource, and the number of registers per entry indicates how those entries will be packed into registers. The reorder array allows the shader program to take data that is in a contiguous block, and reorder the registers according to how the shader program expects the data to be arranged. The function is provided so that shader programs in different formats can set their constants in a manner they expect.

virtual const char* GetShaderTarget() const

virtual void SetShaderTarget(const char* pcShaderTarget)

Get and set which shader model to use when compiling the function. Possible values for the shader model can be found in the DirectX9 documentation for D3DXGetPixelShaderProfile.

virtual void SetConstantTable(LPD3DXCONSTANTTABLE pkTable)

Get and set the constant table for the shader, which maps shader variables to pixel shader constant registers.
