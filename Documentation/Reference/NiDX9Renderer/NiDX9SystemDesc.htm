NiDX9SystemDesc

<NiDX9SystemDesc.h>

Description

NiDX9SystemDesc is a class that is designed to represent all of the adapters that are installed and configured in the machine. Its main purpose is to serve as a container class for all of the system's Gamebryo adapter descriptors. See the notes below for a hierarchy diagram of the classes that are attached to this system descriptor.

Constructors

There are no public constructors, as only the Gamebryo DX9 renderer should create instances of this class.

Supported Member Functions

unsigned int GetAdapterCount() const

Returns the number of adapters attached to the system. Valid adapter indices range from 0 to the return value of this function minus one.

const NiDX9AdapterDesc* GetAdapter(unsigned int uiAdapter) const

Returns the descriptor for the given adapter as attached to the system. The valid values are given by the adapter count function above.

Static Member Functions

static char* GetD3D9FormatString(D3DFORMAT eD3DFormat)

This helper function converts the D3D surface format enumerations into a printable character string.

static unsigned int GetBPPFromFormat(D3DFORMAT eD3DFormat)

This helper function returns the size of a D3D surface format enumeration in bits-per-pixel value.
