NiDX9AdapterDesc::ModeDesc

<NiDX9SystemDesc.h>

Description

NiDX9AdapterDesc::ModeDesc is a nested class of NiDX9AdapterDesc, which is designed to represent a single display mode supported by the adapter (width, height and format), as well as a list of all refresh rates supported by the adapter in this mode.

Constructors

There are no public constructors, as only the Gamebryo DX9 renderer should create instances of this class.

Supported Member Functions

unsigned int GetClosestRefreshRate(unsigned int uiRefreshRate) const

Returns the closest available refresh rate (in frames per second) to the given parameter for this display mode. If the Direct3D standard values of D3DPRESENT_RATE_UNLIMITED (0x7fffffff) or D3DPRESENT_RATE_DEFAULT (0x00000000), they are returned directly. Only the rates available in this mode will be returned.

Supported Member Data

The following elements of member data have been made public to ease access to the important values in the class.

unsigned int m_uiWidth

The width of the display mode in pixels.

unsigned int m_uiHeight

The height of the display mode in pixels.

unsigned int m_uiBPP

The color depth of the mode, in bits per pixel.

D3DFORMAT m_eD3DFormat

The Direct3D format of the mode (D3DFMT_*).

NiTPointerList<unsigned int> m_kRefreshRates

A list of all refresh rates (in integer frames-per-second), each of which are valid with this display mode. These are not guaranteed to be in any specific order.
