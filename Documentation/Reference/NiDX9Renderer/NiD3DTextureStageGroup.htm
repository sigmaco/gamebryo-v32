NiD3DTextureStageGroup

: public NiMemObject

<NiD3DTextureStageGroup.h>

Description

NiD3DTextureStageGroup is a collection of stage and sampler states to be set on the device. There are two sets of states in the group, ones which are 'fire-and-forget' and ones which are saved and then restored later.

Constructors

NiD3DTextureStageGroup()

The default constructor creates an initialized NiD3DTextureStageGroup instance. In general, it is a good idea to not directly create NiD3DTextureStageGroup objects, but to call GetFreeTextureStageGroup to obtain one from the pool of available objects.

Supported Static Member Functions

static NiD3DTextureStageGroup* GetFreeTextureStageGroup()

Retrieves an instance of the class from the pool maintained by the renderer. This is provided to prevent fragmentation.

static void ReleaseTextureStageGroup(NiD3DTextureStageGroup* pkGroup)

Returns the given instance to the pool.

Supported Member Functions

void SetStageState(unsigned int uiState, unsigned int uiValue, bool bSave = false)

Sets the given stage state, uiState, to the given value, uiValue. If bSave is true, the state will be restored when RestoreAllStageStates is called.

void RemoveStageState(unsigned int uiState)

Removes the given stage state from the group.

bool GetStageState(unsigned int uiState, unsigned int& uiValue, bool& bSave)

Retrieves the given stage state, uiState, from the group, setting the value and save parameters passed in. Returns true if the state was present, false if not.

void SetSamplerState(unsigned int uiState, unsigned int uiValue, bool bSave = false)

Sets the given sampler state, uiState, to the given value, uiValue. If bSave is true, the state will be restored when RestoreAllSamplerStates is called.

void RemoveSamplerState(unsigned int uiState)

Removes the given sampler state from the group.

bool GetSamplerState(unsigned int uiState, unsigned int& uiValue, bool& bSave)

Retrieves the given sampler state, uiState, from the group, setting the value and save parameters passed in. Returns true if the state was present, false if not.

void RemoveAllStageStates()

Removes all stage states from the group.

void RemoveAllSamplerStates()

Removes all sampler states from the group.

NiD3DError SetTextureStage(unsigned int uiStage)

Sets both the stage and sampler states on the device for the given stage.

NiD3DError RestoreTextureStage(unsigned int uiStage)

Restores saves stage and sampler states for the given stage.

NiD3DError SetAllStageStates(unsigned int uiStage)

Sets the stage states on the device for the given stage.

NiD3DError RestoreAllStageStates(unsigned int uiStage)

Restores the stage states on the device for the given stage.

NiD3DError SetAllStageSamplers(unsigned int uiStage)

Sets the samplers states on the device for the given stage.

NiD3DError RestoreAllStageSamplers(unsigned int uiStage)

Restores the sampler states on the device for the given stage.
