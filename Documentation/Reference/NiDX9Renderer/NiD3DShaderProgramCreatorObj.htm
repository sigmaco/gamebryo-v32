: public NiD3DShaderProgramCreatorAsm

Description

NiD3DShaderProgramCreatorObjis the class that generates NiD3DShaderProgram objects from compiled assembly shader code or compiled HLSL shaders that are output from an off-line shader assembly program like fxc.exe, vsa.exe or psa.exe. The object code can be stored either in a file or in a buffer in memory. Although it inherits from NiD3DShaderProgramCreatorAsm for issues related to file loading, NiD3DShaderProgramCreatorObj will create the appropriate HLSL shaders if a constant table is detected in the precompiled object.

Constructors

This class has no public constructors.

Supported Member Functions

This class has no supported member functions.

Notes

File Extensions

By default, NiD3DShaderProgramCreatorObj is registered as the NiD3DShaderProgramCreator function to be used with files whose extension is .vso or .pso. These registrations can be changed using the function NiD3DShaderProgramFactory::RegisterShaderCreator, so that applications can specify their own extensions. Refer to the documentation of NiD3DShaderProgramFactory for more details.
