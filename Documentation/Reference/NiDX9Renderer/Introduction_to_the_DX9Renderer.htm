Introduction to the DX9Renderer

The Gamebryo DX9-based Direct3D renderer is designed to give developers access to the full set of features available through the D3D9 API. NiApplication links in the NiDX9Renderer by default.

The NiDX9Renderer requires the same version of the DX SDK to be installed on development machines that was used originally to build the library. It also requires the DirectX9.0c runtime corresponding to this version of the SDK to be installed on end user machines. Refer to the documentation on DirectX Support for more details.
