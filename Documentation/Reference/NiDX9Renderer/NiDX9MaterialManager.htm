NiDX9MaterialManager

: public NiMemObject

<NiDX9MaterialManager.h>

Description

NiDX9MaterialManager is the class that manages the state of DirectX materials.

Constructors

NiDX9MaterialManager()

This function creates a new instance of a NiDX9MaterialManager. Since the renderer creates the default one automatically, no application should need to use this function.

Supported Member Functions

D3DMATERIAL9* GetCurrentMaterial()

Returns the DirectX representation of the currently active material.

void SetCurrentMaterial(NiMaterialProperty& kMat)

Set the current material using the specified NiMaterialProperty.

void UpdateToDevice(LPDIRECT3DDEVICE9 pkD3DDevice9)

Send the current material down to the device.

void PurgeMaterial(NiMaterialProperty* pkMaterial)

Release the DirectX representation of the specified NiMaterialProperty.
