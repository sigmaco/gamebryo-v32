Antialiasing in DX9

The NiDX9Renderer supports full-scene antialiasing via multisampled backbuffers. Applications wishing to use antialiasing should specify a multisampled framebuffer mode to renderer creation (FBMODE_MULTISAMPLES_*). The NiDX9Renderer will automatically fall back to a default framebuffer format if the given multisampling mode cannot be supported. Applications wishing to use multisample antialiasing on the widest range of hardware should use FBMODE_MULTISAMPLES_2 or FBMODE_MULTISAMPLES_4. Having created a framebuffer that supports multisampling, the application need not take further action to enable antialiasing. The renderer will automatically enable the required render states.
