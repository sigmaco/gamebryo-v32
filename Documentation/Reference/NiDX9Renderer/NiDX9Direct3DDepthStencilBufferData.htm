NiDX9Direct3DDepthStencilBufferData

: public NiDX9DepthStencilBufferData

<NiDX92DBufferData.h>

Description

NiDX9Direct3DDepthStencilBufferData is a Gamebryo class that represents a depth/stencil buffer that the application manually creates as an LPDIRECT3DSURFACE9. This allows applications to create depth/stencil surfaces through the Direct3D interface and use them with Gamebryo.

Constructors

This class contains no public constructors the static Create function below is used to create NiDX9Direct3DDepthStencilBufferData objects.

Static Member Functions

static NiDX9Direct3DDepthStencilBufferData* Create(LPDIRECT3DSURFACE9 pkD3DDepthStencilSurface, NiDepthStencilBuffer*& pkBuffer)

Creates a new instance of an NiDX9Direct3DDepthStencilBufferData object. The pkD3DDepthStencilSurface parameter should be the surface that will be used as the depth/stencil buffer, and pkBuffer will be the pointer that holds the NiDepthStencilBuffer object that references the NiDX9Direct3DDepthStencilBufferData. If pkBuffer is not empty, then it will be filled in with the newly created NiDX9Direct3DDepthStencilBufferData object, but if it is empty, then when the function returns, pkBuffer will point to a new NiDepthStencilBuffer object. This new object will need to be assigned to a smart pointer to prevent it from getting leaked accidentally. This function will add a reference to the passed-in LPDIRECT3DSURFACE9 object.

Supported Member Functions

void PurgeBufferData()

In response to a lost device, it is necessary to release all non-managed resources, which will include all depth/stencil surface. This function releases the LPDIRECT3DSURFACE9 object that the NiDX9Direct3DDepthStencilBufferData was created with so the device can be reset. After this call is made, it is not legal to use the NiDepthStencilBuffer object that refers to this data object until RecreateBufferData has been called on it.

bool RecreateBufferData(LPDIRECT3DSURFACE9 pkD3DDepthStencilSurface)

After a device has been successfully reset, a new depth/stencil surface may be created and assigned to the NiDX9Direct3DDepthStencilBufferData using this function. If the function succeeds, it will return true. This function will add a reference to the passed-in LPDIRECT3DSURFACE9 object.
