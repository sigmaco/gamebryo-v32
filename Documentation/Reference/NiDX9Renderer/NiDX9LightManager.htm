NiDX9LightManager

: public NiMemObject

<NiDX9LightManager.h>

Description

NiDX9LightManager is the class that manages the state of DirectX lights.

Constructors

NiDX9LightManager(NiDX9RenderState* pkRenderState, LPDIRECT3DDEVICE9 pkD3DDevice9)

This function creates a new instance of a NiDX9LightManager. Since the renderer creates the default one automatically, no application should need to use this function.

Supported Member Functions

void NextFrame()

Indicates to the NiDX9LightManager that the frame has changed, and so existing lights should be queried when they are used to determine whether their parameters have changed.

void SetState(const NiDynamicEffectState* pkState, const NiTexturingProperty* pkTex, const NiVertexColorProperty* pkColor)

Sets the state of the DirectX lights based on the lights contained in pkState. The ApplyMode of the NiTexturingProperty and the NiVertexColorProperty are also considered when preparing the lighting.

void RemoveLight(NiLight& kLight)

Release the DirectX representation of the specified light.

void ReleaseLights()

Release all DirectX lights. This function is called by NiDX9Renderer::Recreate when resetting the device.

bool IsLit()

Returns whether lighting is active. This function only returns a relevant value after SetState has been called.
