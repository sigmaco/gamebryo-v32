NiDX9Direct3DTexture

: public NiTexture

<NiDX9Direct3DTexture.h>

Description

NiDX9Direct3DTexture represents a texture that has been created by the renderer to directly wrap a Direct3D texture provided by the application. This allows applications to make use of unusual textures for specific purposes. Refer to the documentation on this feature for more details.

Constructors

This class contains no public constructors the static Create function below is used to create NiDX9Direct3DTexture objects.

Supported Member Functions

static NiDX9Direct3DTexture* Create(NiRenderer* pkRenderer)

Creates a new instance of an NiDX9Direct3DTexture. This function should not be called by applications, since NiDX9Renderer::CreateNiTextureFromD3DTexture is a more complete function.

virtual unsigned int GetWidth() const

Returns the width of the texture.

virtual unsigned int GetHeight() const

Returns the height of the texture.
