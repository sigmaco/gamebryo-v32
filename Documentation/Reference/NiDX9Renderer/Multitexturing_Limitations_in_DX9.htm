Multitexturing Limitations in DX9

  • The DX9 renderer supports only 8 texture coordinate sets per vertex. Any additional texture coordinate sets will be ignored. There is no limit on the number of textures applied.
  • Note that the NiDX9Renderer assumes (without checking) that two-texture single-pass hardware can support at least basic multitexturing modes (ADD, MODULATE, MODULATE2X, BLEND). Hardware that cannot support some of these modes may see dropped textures or even dropped objects. Also, the renderer does not currently check if different blending and clamping modes in different stages of the same rendering pass will cause multitexturing to fail on the hardware. No known recent cards have this limitation, but first-generation multitexture hardware (such as the ATI RagePro and 3dfx Voodoo cards) may have problems.
  • Not all hardware cards can support perspective-projected textures. As a result, not all hardware can support NiTextureEffects with WORLD_PERSPECTIVE texture coordinate mode.
