: public NiD3DShader

<NiDX9ErrorShader.h>

Description

NiDX9ErrorShader is a special error shader meant to clearly identify when a shader otherwise fails to be applied to an object. The NiDX9ErrorShader will oscillate 10% of the radius of the geometry's bounding volume, and will randomly change colors every frame. This behavior is very irritating to look at, and should immediately visually identify when there a problem.
