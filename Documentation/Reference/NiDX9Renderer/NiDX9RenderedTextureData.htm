Description

NiDX9RenderedTextureData contains the DX representation of a rendered texture.

Constructors

This class contains no public constructors—the static Create function below is used to create NiDX9RenderedTextureData objects.

Static Member Functions

NiDX9RenderedTextureData* Create(NiRenderedTexture* pkTexture, NiDX9Renderer* pkRenderer)

Creates the renderer data for the specified NiRenderedTexture using the specified NiDX9Renderer.
