NiD3DShaderLibrary

: public NiShaderLibrary

<NiD3DShaderLibrary.h>

Description

NiD3DShaderLibrary implements the D3D-specific shader library class. The class introduces a directory member, ms_pcDirectory, which is intended to provide derived libraries with a common place to look for shaders. The platforms the library is valid on are also identified in the static member ms_uiPlatform. Valid shader program versions are also exposed via the ms_uiPSVersion and ms_uiVSVersion members. Finally, there is a user-definable version that is held in the ms_uiUserVersion member.

The description of the library is held in the m_spShaderLibraryDesc member, which is an instance of an NiShaderLibraryDesc class.

Constructors

NiD3DShaderLibrary(char* pcName)

The constructor creates an NiD3DShaderLibrary names pcName.

Supported Static Member Functions

Sets the static directory string, ms_pcDirectory, to the given pcDirectory.

Returns the static directory string.

static void SetPlatform(unsigned int uiPlatform)

Sets the static platform flags member to the given uiPlatform.

static unsigned int GetPlatform()

Returns the static platform flags member.

static void SetVersionInfo(const char* pcVersionName, unsigned int uiVersion)

Sets the version member of the given pcVersionName to the given version value. Valid names are:

Version Name String Corresponding Member
PSVersion ms_uiPSVersion - pixel shader version.
VSVersion ms_uiVSVersion - vertex shader version.
UserVersion ms_uiUserVersion - user defined version.

static unsigned int GetVersionNumber(const char* pcVersionName)

Returns the corresponding static version member. See table above for name mappings.

Supported Member Functions

virtual void SetRenderer(NiD3DRenderer* pkRenderer)

Called by the system to set the local renderer pointer for the shader. This function is virtual to allow for any additional processing the shader requires to be done when it gets a valid renderer pointer.

NiD3DRenderer* NiD3DShaderLibrary::GetRenderer()

Returns the currently set renderer pointer for the shader.

virtual NiShaderLibraryDesc* GetShaderLibraryDesc()

Returns a pointer to the NiShaderLibraryDesc object for this shader library.
