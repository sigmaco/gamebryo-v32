NiDX9TextureData

: public NiTexture::RendererData

<NiDX9TextureData.h>

Description

NiDX9TextureData contains the D3D representation of the data in a texture.

Constructors

NiDX9TextureData(NiTexture* pkTexture, NiDX9Renderer* pkRenderer)

Creates the renderer data for the specified NiTexture using the specified NiDX9Renderer.

Supported Member Functions

unsigned int GetLevels() const

Returns the number of mipmap levels in the texture.

D3DBaseTexturePtr GetD3DTexture() const

Returns a pointer to the D3D representation of the texture. The D3DBaseTexturePtr type can be cast directly to an LPDIRECT3DBASETEXTURE9.

NiDX9RenderedTextureData* GetAsRenderedTexture()

If this texture is an NiRenderedTexture, this function will return a pointer to this object, or else it will return NULL.

NiDX9SourceTextureData* GetAsSourceTexture()

If this texture is an NiSourceTexture, this function will return a pointer to this object, or else it will return NULL.

NiDX9DynamicTextureData* GetAsDynamicTexture()

If this texture is an NiDynamicTexture, this function will return a pointer to this object, or else it will return NULL.

bool IsRenderedTexture() const

Returns whether this texture is an NiRenderedTexture.

bool IsSourceTexture() const

Returns whether this texture is an NiSourceTexture.

bool IsDynamicTexture() const

Returns whether this texture is an NiDynamicTexture.

Static Member Functions

const NiDX9PixelFormat* FindClosestPixelFormat(const NiTexture::FormatPrefs& kPrefs, NiPixelFormat* const* ppkDestFmts)

This function will return an NiDX9PixelFormat representing the format preferences found in kPrefs. This function should only be called from the NiDX9Renderer itself.

void ClearTextureData(NiDX9Renderer* pkRenderer)

Release all texture data in the specified renderer.
