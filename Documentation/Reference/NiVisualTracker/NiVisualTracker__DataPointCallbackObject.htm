NiVisualTracker::DataPointCallbackObject

: public NiMemObject

<NiVisualTracker.h>

Description

A callback for scatter graph data points. Each data point in the graph is added separately by calling NiVisualTracker::AddDataPoint method and each point can update its position via the TakeSample method, which is called at the update frequency. Past values are not displayed; only the most recent position is displayed.

Constructor

There is no base class constructor. Each derived class must define a constructor.

Destructor

virtual ~DataPointCallbackObject()

The destructor does nothing.

Supported Methods

virtual bool TakeSample(float fTime, efd::Point2& kPointOut, efd::UInt32& uiSymbolInOut) = 0

Take a sample of the data for this point the the given time and put it in kPointOut. You can optionally change your symbol by changing the uiSymbolInOut value. Return false to have your callback object (the rendered data point) removed, or true to continue to display it.
