Inheritance graph for efd::ThreadFunctor

Description

ThreadFunctor is an interface class that defines the interface between an efd::Thread and the function that thread invokes.

This treats the inheriting class instance into a function object that can be invoked by efd::Thread.

Public Methods

virtual efd::UInt32 Execute(efd::Thread* pArg) = 0

Classes that inherit from ThreadFunctor implement this method which is then invoked by efd::Thread.

Parameter Description
pArg A pointer to the efd::Thread object that invoked this method.
Return Value An arbitrary integer return value for the thread. efd::Thread will store this value as the return value from execution.
