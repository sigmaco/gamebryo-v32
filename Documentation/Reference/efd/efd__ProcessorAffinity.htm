efd::ProcessorAffinity

<ProcessorAffinity.h>

Description

A platform-independent wrapper class for setting processor affinities.

NiProcessorAffinity supports both an ideal processor and a processor affinity mask. Note that different platforms may only support one of these methods for specifying target processors.
