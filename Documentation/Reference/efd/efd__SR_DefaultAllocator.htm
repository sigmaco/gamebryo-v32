efd::SR_DefaultAllocator

<SerializeRoutines.h>

Description

This routine does a default allocation.

It exists to allow overrides if you use a custom allocator or specific heaps for specific types of data. The default assumes the object being allocated is an efd::MemObject and thus uses our special memory manager.
