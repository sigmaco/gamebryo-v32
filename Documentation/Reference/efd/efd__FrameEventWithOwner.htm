Inheritance graph for efd::FrameEventWithOwner

Description

A FrameEventWithOwner is an IFrameEvent implementation where the event handler functions accept a pointer to the owning object, typically an ISystemService derived class, as a parameter.

The event is initialized with the owning object pointer.
