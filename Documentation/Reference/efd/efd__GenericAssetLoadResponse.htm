Description

A response message to a generic Asset load request.

The response contains the original request URN, a result, the size of the asset, and buffer containing the asset data.

Construct a new GenericAssetLoadResponse.

Each response contains the original request URN, a result code, the size of the asset, and the asset data.

Parameter Description
urn original request URN.
responseCategory original request response category.
result result of the load request.
assetPath Fully qualified path to the asset.
isReload hint to the factory indicating this is a reload.
assetSizeInBytes the size of the asset, in bytes. 0 if the asset failed to load.
assetData buffer, of size assetSizeInBytes, contain the Asset data from disk. 0 if the asset failed to load. The GenericAssetLoadResponse takes ownership of this data.
