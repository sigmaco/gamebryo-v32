Description

IAssetImporter is a virtual interface class used to define asset importing extensions in the Asset Controller application.

Each import extension that is registered gets invoked for every new asset when it is first discovered. A child class can process the asset to determine whether certain tag or property values ought to be set for the asset. Tags that are created in this way become part of the generated metadata for the asset web.
