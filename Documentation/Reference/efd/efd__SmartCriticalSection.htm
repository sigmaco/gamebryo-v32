efd::SmartCriticalSection

<SmartCriticalSection.h>

Description

A SmartCriticalSection is a convenience wrapper around a CriticalSection.

The main purpose of the SmartCriticalSection is to Unlock the CriticalSection when the SmartCriticalSection goes out of scope.

Construction and Destruction

SmartCriticalSection(CriticalSection& criticalSection)

Default constructor. Does not lock the CriticalSection.

SmartCriticalSection(const SmartCriticalSection& )

Default constructor. Does not lock the CriticalSection.

~SmartCriticalSection()

Destructor. Unlocks the CriticalSection if it is locked.

Public Methods

void Lock()

Passthrough call to m_criticalSection.

void Unlock()

Only calls Unlock if CriticalSection is actually locked.

efd::UInt32 GetOwningThreadID() const

Passthrough call to m_criticalSection.

efd::UInt32 GetCurrentLockCount() const

Passthrough call to m_criticalSection.
