efd::SelectStorageForBytes

<TemplateHelpers.h>

Description

SelectStorageForBytes is for use in templated methods that need to devine the correct signed or unsigned efd integral type to hold a certain byte size worth of data.

This can be helpful if, for example, you need to cast templated parameters to equivalent signed or unsigned types in order to perform certain mathematical operations. An example usage might look something like:

template<typename T = efd::UInt32> class Example { typedef SelectStorageForBytes<sizeof(T)>signedType MySignedType; typedef SelectStorageForBytes<sizeof(T)>unsignedType MyUnsignedType;

MySignedType m_storage; };
