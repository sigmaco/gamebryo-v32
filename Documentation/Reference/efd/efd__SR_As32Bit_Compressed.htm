efd::SR_As32Bit_Compressed

<SerializeRoutines.h>

Description

This routine saves room in the storage media by compressing a 32bit type.

Its very common to have enums or container sizes that are technically 32 bits long but are expected to contain small values. If the value is less than 64 it will pack to a single byte. Of course in the worst case it will take 5 bytes so this might not be a good choice if negative values are expected.
