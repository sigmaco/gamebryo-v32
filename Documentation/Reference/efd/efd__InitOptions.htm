efd::InitOptions

Inheritance graph for efd::InitOptions

Description

A platform-independent class designed to encapsulate any arguments to EE_INIT.

In addition, this class contains the user created Allocator object and any additional system level services that may be overridden by the user.

bool GetUseUnhandledExceptionFilter() const

Returns false if the default unhandled exception filter should be disabled.

By default an unhandled exception filter is registered that logs the exception record, a stack trace if stack traces are enabled via the EE_USE_EXCEPTION_STACKTRACE define and flushes the logger. If no Logger is present the default filter will perform a EE_OUTPUT_DEBUG_STRING instead.

void SetUseUnhandledExceptionFilter(bool shouldUse)

Set to true to enable the default unhandled exception filter, or false to disable it.

On supported platforms this is enabled by default.

Parameter Description
shouldUse Turns the default unhandled exception filter on or off.
