Inheritance graph for efd::SimpleGraph

Description

The SimpleGraph class also represents a collection of RDF triples.

It is not optimized in its internal storage mechanisms, and is only suitable for small graphs, such as those used for per-asset-file metadata. The main difference in the interface arises from the fact that SimpleGraphs can include any mix of un-related triples, whereas the Graph class only stores sets of triples that share the same predicate value.
