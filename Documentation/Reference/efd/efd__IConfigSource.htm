Inheritance graph for efd::IConfigSource

Description

Interface for configuration sources used in the Configuration Manager (IConfigManager).

Public Types

efd::IConfigSource::tSourceType

Definition of the default source types.

Addition sources types may be to the "Configuration Manager" (IConfigManager) but need not be added to this enumeration. It is merely a convenience.

Name Description
kSourceUnknown Unknown source.
kSourceCommandLine Command-line source.
kSourceIniFile INI file source.
kSourceEnvVar Environment variable source.
kSourceTypeMax Assign new source types any greater number.

efd::IConfigSource::tSourcePriorities

Definition of the default source priorities.

Higher numbers equal higher priority source that will overwrite lower priority source information.

Name Description
kSourcePriorityLow Low priority source.
kSourcePriorityNormal Normal priority source is used for the default ini source file.
kSourcePriorityHigh High priority source.
kSourcePriorityCmdLine Command-line source priority is the highest.

Public Methods

virtual int GetPriority() const = 0

Returns the priority for this configuration source.

A higher number is a higher priority and overwrites lower values.

Parameter Description
Return Value Priority value for this source.

virtual unsigned int GetSourceType() const = 0

Retrieves the source type of the configuration source as defined in tSourceType.

Other source types may be added and they should be assigned numbers starting with tSourceType::kSourceTypeMax.

Parameter Description
Return Value Source type.

virtual const efd::utf8string& GetSourceName() const = 0

Returns the string name assigned to this configuration source.

In general, it will contain the file name or address of the data used by this configuration source.

Parameter Description
Return Value Name of the source.
