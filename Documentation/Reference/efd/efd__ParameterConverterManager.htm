template <typename FROM , typename TO > static void MakeDefaultConverter()

Generate a converter to allow translation of one type to another type.

The types used must support the efd::ConvertingAssignment templated method. To use your own custom type just ensure the efd::ConvertingAssignment specialization is defined prior to calling this method.

Generate bi-directional converts for any two types that support the efd::ConvertingAssignment templated method.
