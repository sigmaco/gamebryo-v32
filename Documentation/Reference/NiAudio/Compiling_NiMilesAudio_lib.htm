Compiling NiMilesAudio.lib

The NiMilesAudio source project is included in the Gamebryo UtilityLibs project workspace files. However, it is not compiled as part of the standard build configurations because the Miles Sound System is not shipped with a Gamebryo distribution. Gamebryo licensees wishing to compile NiMilesAudio.lib must do so either by using the Miles-Bink build configurations or by choosing the NiMilesAudio project target in the CoreLibs project workspace.

Gamebryo project files use environment variables to determine the install location of Miles:

  • MILES_PATH should point to Miles for Win32 install location
  • MILES_PATH_XENON should point to the Miles for Xbox 360 install location
  • MILES_PATH_PS3 should point to the Miles for PS3 install location

Because Developer Studio reads all environment variables at startup, be sure you restart Developer Studio after creating the environment variable.
