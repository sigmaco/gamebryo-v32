NiAudioListener

NiAudioListener is used to represent the sound-receiving object of the application. Like NiAudioSystem, there is a single instance of the NiAudioListener object in an audio enabled Gamebryo application. This instance is created automatically at application startup by the audio system and is accessed with methods available through the audio system.

However, unlike NiAudioSystem, the listener is derived from NiAVObject and is typically attached to a Gamebryo scene graph to define its location. Similar to the Gamebryo camera class, NiCamera, the NiAudioListener has a coordinate frame that defines its position and orientation in 3D space. This design enables an application to model the orientation of a character's head or the position of whatever sound receiver the application needs to use.

It is suggested that NiAudioListener::Update() be used to update the listener's position and orientation before any 3D sources begin playback.
