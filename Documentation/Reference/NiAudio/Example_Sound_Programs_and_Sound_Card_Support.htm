Example Sound Programs and Sound Card Support

MetalWars and several other GameDemos demonstrate how to use the NiMilesAudio library to add sound to an application. Because the sound-enabled configurations for these samples require the Miles Sound System (which is not shipped with Gamebryo), these sound-enabled configurations are not built by default by the Samples workspace.

Developers holding a Miles SDK license, who wish to recompile the sound-enabled configurations for these sample applications, must do so manually using the appropriate .dsp file.

Emergent Game Technologies does not explicitly test or certify NiMilesAudio with a matrix of sound cards or operating systems, as is done using the Win32 graphics renderers. Licensed developers of the Miles SDK should consult Miles sources for details on sound card and operating system support.
