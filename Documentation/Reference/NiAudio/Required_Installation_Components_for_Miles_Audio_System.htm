Required Installation Components for Miles Audio System

The Miles Sound System SDK is required in order to compile the Gamebryo NiMilesAudio library and run the NiMilesAudio example programs. The Miles Sound System SDK is available from RAD Game Tools (http://www.radgametools.com). Contact sales@radgametools.com for details on their licensing terms and evaluation SDKs.

Compiling and linking with the MSS requires the Miles include files along with the Miles library file (mss32.lib for Win32, mssxenon.lib for Xbox 360, mssps3.a for PS3). Additionally on Win32, the mss32.dll file must reside in a directory accessible to the executable file. Furthermore, it is necessary to have the contents of the Miles redistribution directory accessible to the application. This directory contains the necessary data files for the various providers along with various filter data files.

The MSS documentation that comes with Miles will be extremely useful when programming sound with NiMilesAudio. Information on all Miles functions as well as the 3D audio providers is contained in the MSS documentation. Additionally, information regarding using, debugging, and distributing the MSS can also be found in the MSS documentation.
