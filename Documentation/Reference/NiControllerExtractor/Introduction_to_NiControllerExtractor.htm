NiControllerExtractor Class Reference: Introduction

The NiControllerExtractor library implements the code that the Controller Extractor export tool plug-in uses. The Controller Extractor uses this code to create NiSequenceData objects from an animated scene graph. It also uses the code to remove sequence controllers from the scene graph. Removing sequence controllers facilitates saving extracted NIF and KF files.
