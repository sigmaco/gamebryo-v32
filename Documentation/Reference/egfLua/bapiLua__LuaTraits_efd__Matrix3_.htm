lua_State* L,
int pos,

Get an efd::Matrix3 from the Lua stack.

This is represented as either a Matrix3 object or a nested table with 3 rows of tables with 3 values in Lua.

This does not pop the stack.

Parameter Description
L Lua runtime state.
pos position on the stack.
o_result efd::Matrix3 at pos or the default Matrix3 if the position doesn't contain one.
Return Value true if the conversion was successful.
