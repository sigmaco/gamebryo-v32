Inheritance graph for efd::LogService

Description

Log Service class - responsible for handling log configuration calls / messages, tracking log destinations, and log files.

This is a light weight control for an instance of the ILogger interface that runs as a SystemService. This allows the LogService to register for Category / Messages that can change the behavior of the underlying logger.
