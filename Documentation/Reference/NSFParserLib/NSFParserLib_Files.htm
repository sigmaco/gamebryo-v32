NSFParserLib Files

Description

NSFParserLib.cpp and NSFParserLib.h provide the interface function for static lib based builds of the shader library. It defines a single function for running the parser, which the application must call during RunParserCallback.

Functions

unsigned int NSFParserLib_RunShaderParser(const char* pszDirectory, bool bRecurseSubFolders)

This function is provided for static lib based builds of applications that wish to utilize the NSFParserLib. The function simply runs the NSF parser on any NSF files found in the pszDirectory path, and creates NSB files from them. Those files can then be loaded by the NSBShaderLibrary system to create NiBinaryShader objects.
