Renderer Options from the INI File

All NiApplication renderer settings can be saved to an INI file. The INI file can be used to preset the renderer settings dialog with default values, or to avoid showin the dialog at all and simply use the values specified in the file without user intervention.

The INI file is in standard Windows INI file format. It is a text file,divided into sections (specified as "[section_name]"), with each section containing some number of key value pairs in the form of "key=value". All renderer settings are stored under the "[Renderer]" section. By default, the INI file will be stored in the same directory as the executable file and will have the name "AppSettings.ini".

Following is an example of a typical INI file with comments.

Example

; Name of section for renderer settings.
[Renderer]
 
; Screen resolution
ScreenWidth=1024
ScreenHeight=768
 
; Minimum selectable screen resolution
MinScreenWidth=640
MinScreenHeight=480
 
; Enum values for vertex processing. 1 = hardware, 2 = mixed, 3 = software
Vertexprocessing=1
 
; Enumeration values, converted to integers for DX9 render target, depth and frame buffer formats. To edit by hand, look at corresponding enumerations.
DX9RenderTargetMode=3
DX9DepthSurfaceMode=75
DX9FrameBufferMode=0
 
; Display index for D3D10
D3D10OutputIdx=0
 
; D3D10 multisample settings
D3D10MultisampleCount=1
D3D10MultisampleQuality=0
 
; Enumeration values, converted to integers for D3D10 render target and depth buffer formats. To edit by hand, look at corresponding enumerations.
D3D10DSFormat=45
D3D10RTFormat=28
 
; Display index for D3D11
D3D11OutputIdx=0
 
; D3D11 multisample settings
D3D11MultisampleCount=1
D3D11MultisampleQuality=0
 
; Enumeration values, converted to integers for D3D11 render target and depth buffer formats. To edit by hand, look at corresponding enumerations.
D3D11DSFormat=45
D3D11RTFormat=28
 
; D3D11 feature levels
D3D11FeatureLevel9_1=1
D3D11FeatureLevel9_2=1
D3D11FeatureLevel9_3=1
D3D11FeatureLevel10_0=1
D3D11FeatureLevel10_1=1
D3D11FeatureLevel11_0=1
 
; Enumeration values, converted to integers for desired renderer.
; Specify 2 for DX9 renderer, 3 for D3D10 renderer, or 6 for D3D11 renderer
RendererID=2 

; Specify 0 for windowed mode and 1 for fullscreen
Fullscreen=0
 
; Specify 1 to use NVPerfHUD if available, 0 otherwise
NVPerfHUD=0
 
; Set to 0 to use HAL device (if available), 1 for REF.
RefRast=0
 
; Set to 1 to create HAL device as pure.
PureDevice=0
 
; Set 1 to lock to VSync
VSync=0
 
; Multithreading enabled. Default is 1.
MultiThread=1
 
; Renderer dialog will be shown, if this key is set to 1.
RendererDialog=1
 
; Default value for "Save settings" checkbox.
SaveSettings=1
