NiCursor

: public NiRefObject

<NiCursor.h>

Description

The NiCursor class is used to display and move a cursor around the screen.

Constructors

The constructor for this class is protected. To create an NiCursor object, the following static NiCursor functions are used rather than an NiCursor constructor.

static NiCursor* Create(NiRenderer* pkRenderer, NiRect<int>& kRect, unsigned int uiHotSpotX, unsigned int uiHotSpotY, const char* pcImageFile, const NiRenderTargetGroup* pkTarget = NULL)

pkRenderer points to the renderer that will be used to render the cursor. kRect is an NiRect object that contains the bounds of the cursor, in screen space pixels. Normally this will be the size of the entire back buffer, but it can also be a subset of it, or it can even extend beyond. The hot spot of the cursor, which indicates where on the image file the "true" location of the cursor is found, is indicated by uiHotSpotX and uiHotSpotY. The source of the cursor image itself is the texture file indicated by pcImageFile. The NiRenderTargetGroup pkTarget is the buffer that the screen space pixels specified in kRect refer to, and it is the surface that the NiCursor will be displayed on. If pkTarget is NULL, then the default render target will be used.

static NiCursor* Create(NiRenderer* pkRenderer, NiRect<int>& kRect, unsigned int uiFlags, unsigned int uiHotSpotX, unsigned int uiHotSpotY, NiTexture* pkTexture, const NiRenderTargetGroup* pkTarget = NULL)

This version of the Create function takes the cursor image as a pre-loaded and pre-initialized NiTexture object via the pkTexture parameter. All other behavior of this function is identical to the first version.

Supported Member Functions

virtual int Show(bool bShow)

Indicates to the NiCursor object that the cursor should be displayed if bShow is true, or removed if bShow is false. The function returns an integer representing how many more times the function has been called with true than with false.

virtual bool SetPosition(float fTime, int iScreenSpaceX, int iScreenSpaceY)

Sets the position of the hot spot of the cursor to the coordinates indicated by iScreenSpaceX and iScreenSpaceY. The fTime parameter indicates the current time, and can be used by a derived class to perform some animations.

virtual bool Move(float fTime, int iDeltaX, int iDeltaY)

Moves the position of the hot spot of the cursor from its current location by the amounts indicated by iDeltaX and iDeltaY. The fTime parameter indicates the current time, and can be used by a derived class to perform some animations.

virtual void Draw()

This function draws the cursor image in the correct location to the backbuffer.

virtual bool SetImage(NiTexture* pkTexture)

Sets the image used by the cursor to a new image. If no references remain to the previous image, it will be deleted.

const int GetShow() const

Returns how many more times Show has been called with true than with false.

void GetHotSpot(unsigned int& uiX, unsigned int& uiY)

Returns the location of the hot spot in the cursor image in the uiX and uiY parameters

void GetScreenSpacePosition(int& iX, int& iY)

Returns the location of the cursor in screen space in the iX and iY parameters

void SetRect(NiRect<int>& kRect)

Sets the bounds of the cursor, in screen space pixels, beyond which the cursor will not move. Normally this will be the size of the entire back buffer, but it can also be a subset of it, or it can even extend beyond.

NiMeshScreenElements* GetScreenElements()

Returns a pointer to the NiMeshScreenElements that will be used to render the cursor.

const NiRenderTargetGroup* GetRenderTargetGroup() const

Returns the NiRenderTargetGroup that the NiCursor expects that it will be drawn to.

void SetRenderTargetGroup(const NiRenderTargetGroup* pkGroup)

Sets the NiRenderTargetGroup that the NiCursor expects that it will be drawn to.
