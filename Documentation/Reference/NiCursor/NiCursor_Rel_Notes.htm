Gamebryo 2.5.0

  • The NiCursor class had the previously unsupported flags enumeration and its related API removed. This was not previously supported in any version of Gamebryo and so this is not a change in functionality, just a change in API.

Gamebryo 2.3.0

  • NiSystemCursor::Create(..) now correctly sets the cursor type.
  • Samples/Data/SystemCursors.tga has been updated to improve cursor contrast on very dark or very light backgrounds.

Gamebryo 2.1.0

  • Added ZBuffer property so the cursor will always draw and added access function so the application can directly manipulate the screen elements.

Gamebryo 2.0.0

  • Added image file parameter to NiSystemCursor so you can specify the texture to be used.
