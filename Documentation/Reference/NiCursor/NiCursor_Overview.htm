NiCursor Overview

The NiCursor library provides the ability to display a cursor and move it around the screen. On Windows, it can be used to replace the cursor with an application-defined pointer icon. On a console, it can be used to create and move a cursor around the screen. The cursor image can be any size, although some platforms require that its dimensions be powers of two. The cursor image can be animated.

The cursor image is displayed as an NiMeshScreenElements object, rendered internally by the NiCursor class. Because it is a rendered object, the NiCursor will only update as fast as the application's frame rate. This can be slower than the default system cursor will update under Win32.

The following code demonstrates how an NiCursor object can be created and used. In this sample, m_spRenderer points to the application's instance of NiRenderer. The cursor image is located in the "../../Data/Cursor.tga" file. The "hot spot" of the cursor, which is the point that represents the actual pixel that the cursor is indicating, is located at 8 pixels from the left and 10 pixels from the top of the image.

NiRect<int> kRect;
kRect.m_left = 0;
kRect.m_top = 0;
kRect.m_right = m_spRenderer->GetBackbufferWidth();
kRect.m_bottom = m_spRenderer->GetBackbufferHeight();
 
NiCursorPtr spCursor = NiCursor::Create(m_spRenderer,
    kRect, NiCursor::IMMEDIATE, 8, 10, "../../Data/Cursor.tga");
 
spCursor->SetPosition(0.0f, 320, 240);
spCursor->Show(true);

The hot spot of the cursor is placed at pixel (320, 240) in screen space, and the cursor is marked to display. Now, the application can move the cursor at any time in the application by calling Move as follows:

spCursor->Move(fTime, iDeltaX, iDeltaY);

When the application has finished rendering its scene, it will display the cursor by calling Draw. Note that the input parameter to Draw is currently unused.

spCursor->Draw(0);

When used in conjunction with NiInput under Win32, NiCursor can easily be used to replace the system cursor. Be sure to set the mode on the mouse to Exclusive mode so that the system cursor doesn't display.
