NiCursorRenderClick

: public NiRenderClick

<NiCursorRenderClick.h>

Description

This class is a render click that holds and renders a list of NiCursor objects.

Overridden Virtual Functions

virtual unsigned int GetNumObjectsDrawn() const

virtual float GetCullTime() const

virtual float GetRenderTime() const

Overridden from NiRenderClick. These functions return the rendering statistics from the most recent frame. These values are stored in member variables in this class and are computed each frame during the PerformRendering function.

virtual void PerformRendering(unsigned int uiFrameID)

Overridden from NiRenderClick. This function calls SetScreenSpaceCameraData on the renderer and iterates through the list of NiCursor objects, calling Draw on each.
