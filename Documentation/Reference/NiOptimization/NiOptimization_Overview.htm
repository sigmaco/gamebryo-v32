NiOptimization Overview

The NiOptimization library is used to create optimized content for use with Gamebryo. Optimization functions are typically executed with the SceneGraphOptimization tool plug-in, which is run by default upon export from 3ds max or Maya. All the optimization functions in NiOptimization can also be called from custom applications, although their performance is not intended to be suitable for execution in real-time applications. In other words, the functions in NiOptimization are not intended to be called from within a game application.

Please refer to the NiOptimization class reference for more information about each optimization function and what it does.
