NiCodeBlock

: public NiRefObject

<NiCodeBlock.h>

Description

This class represents a fragment of GPU shader code that will be used by the NiFragmentMaterial to construct a shader program. An NiCodeBlock object is held onto by an NiMaterialFragmentNode object in the shade tree.

Constructors

NiCodeBlock()

Constructs a new NiCodeBlock instance.

Supported Member Functions

const NiFixedString& GetLanguage() const

void SetLanguage(const NiFixedString& kStr)

Get and set a string that represents the programming language that the NiCodeBlock represents. Valid values include "HLSL" and "Cg".

const NiFixedString& GetPlatform() const

void SetPlatform(const NiFixedString& kStr)

Get and set a string that represents the platform that the NiCodeBlock can work on. Valid strings include "DX9", "Xenon," and "PS3".

const NiFixedString& GetTarget() const

void SetTarget(const NiFixedString& kStr)

Get and set a string that represents the shader target that the NiCodeBlock can be compiled against. Valid strings include "vs_2_0", "ps_2_a", and other platform-specific strings that denote shader versions. Note that it is expected that an NiCodeBlock can be compiled against any shader target equal to or later than the one indicated by this string.

const NiString& GetText() const

void SetText(const NiString& kCode)

Get and set the text of the GPU shader code itself.
