NiVSMShadowTechnique

: public NiShadowTechnique

<NiVSMShadowTechnique.h>

Description

This class is a specialize shadow technique that inherits from the NiShadowTechnique class and uses a variance shadow mapping solution. This class makes use of the NiVSMBlurMaterial to perform as post-process Gaussian blur on shadow maps.

Supported Virtual Member Functions

virtual void AppendPostProcessRenderClicks(NiShadowMap* pkShadowMap, NiShadowClickGenerator* pkClickGenerator)

This function adds two render clicks to the set of render clicks that need to be performed to render the shadow maps for the scene. One render click performs a horizontal Gaussian blur while the other performs and vertical Gaussian blur.

virtual void PrepareShadowMap(NiShadowMap* pkShadowMap)

This method sets the filter mode for the provided shadow map to FILTER_BILERP.

Supported Member Functions

void SetBlurKernelSize(unsigned int uiKernelSize)

unsigned int GetBlurKernelSize()

These functions specify the blur kernel size that will be used by NiVSMBlurMaterial when performing the post process Gaussian blur. This kernel size must be an even number from 2 to 16.
