NiDefaultShadowVisitor

: public NiShadowVisitor

<NiDefaultShadowVisitor.h>

Description

The default shadow visitor used by the shadowing system. The NiDefaultShadowVisitor overrides ShouldProcessNodeChildren and returns false for any node that inherits from NiSwitchNode. This prevents NiSwitchNode based objects, such as NiLODNode, from having all of their children used as shadow casters and receivers when only one of the children needs to be used.

Supported Static Member Functions

static NiShadowVisitor* CreateDefaultShadowVisitor()

Default callback for specifying the NiShadowVisitor to use. This default callback creates the NiDefaultShadowVisitor. This callback is registered with the shadowing system at static data manager initialization time in NiMainSDM::Init.

Supported Member Functions

virtual bool ShouldProcessNodeChildren(NiNode* pkNode)

Returns false for any node that inherits from NiSwitchNode. This prevents NiSwitchNode based objects, such as NiLODNode, from having all of their children used as shadow casters and receivers when only one of the children needs to be used.
