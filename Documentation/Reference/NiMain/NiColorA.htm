NiColorA

: public NiMemObject

<NiColor.h>

Description

NiColorA represents floating-point RGBA colors. Normally, colors are assumed to be in the range [0.0,1.0] for each component, but this assumption is not required inherently. The data members r, g, b, and a are all declared public for application convenience (allows structure-like access).

Constructors

NiColorA(float fR = 0.0f, float fG = 0.0f, float fB = 0.0f, float fA = 0.0f)

This constructor is the default and member-wise constructor. Given no arguments, it constructs translucent black (the zero color).

Supported Member Functions

void Clamp()

Ensure RGBA components are within the range [0.0,1.0] by clamping any components above 1.0 to 1.0. Note that all the components are assumed to be ≥ 0.0, so no such check is performed.

void Scale()

Ensure RGBA components are within the range [0.0,1.0] by scaling down all components by the max component. This function sets the components in the correct range and preserves the hue of the final color. Note that all the components are assumed to be ≥ 0.0, so no such check is performed.

void GetAs(NiRGBA& kRGBA)

This function converts this object from a floating point RGBA components to a unsigned integer color value. This function does not clamp the values. If they are outside of the range of 0.0 - 1.0 problems will exist. It is a good idea to clamp the value before converting to an NiRGBA object.
