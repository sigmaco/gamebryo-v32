NiFrustum

: public NiMemObject

<NiFrustum.h>

Description

This class defines the view frustum of an NiCamera object. See the camera documentation for details on the frustum model in Gamebryo. The data members m_fLeft, m_fRight, m_fTop, m_fBottom, m_fNear, m_fFar, and m_bOrtho are declared public for ease of access

Constructors

NiFrustum(bool bOrtho = false)

This constructor is the default and member-wise constructors. All members, other than m_bOrtho, are uninitialized upon creation.

NiFrustum(float fLeft, float fRight, float fTop, float fBottom, float fNear, float fFar, bool bOrtho = false)

This constructor allows the setting of each of the data members for the view frustum of a camera..
