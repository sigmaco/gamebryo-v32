NiRenderObjectMaterialOption

This class manages the strings that are used by Gamebryo to query meshes and mesh modifiers for material options.

Supported Static Functions

static const NiFixedString& TRANSFORM_SKINNED()

This option specifies to transform vertices with Gamebryo skinning.

static const NiFixedString& TRANSFORM_INSTANCED()

This option specifies to transform vertices with Gamebryo instancing.

static const NiFixedString& MORPHING()

This option specifies to transform vertices with Gamebryo morphing.

static const NiFixedString& GetString(Enumeration e)

Converts an NiRenderObjectMaterialOption::Enumeration to a predefined NiFixedString

static Enumeration GetEnumeration(const NiFixedString& kString)

Converts a predefined NiRenderObjectMaterialOption NiFixedString to an NiRenderObjectMaterialOption::Enumeration. Returns E__Invalid if kString is not a supported option.

CloningBehavior

This enumeration lists the predefined NiRenderObject material options.

Enumeration Meaning
E__Invalid Unrecognized material option.
E_TRANSFORM_SKINNED Transform vertices with Gamebryo skinning.
E_TRANSFORM_INSTANCED Transform vertices with Gamebryo instancing.
E_MORPHING Transform vertices with Gamebryo morphing.
