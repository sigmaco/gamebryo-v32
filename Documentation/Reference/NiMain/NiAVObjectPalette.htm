NiAVObjectPalette

: public NiObject

<NiAVObjectPalette.h>

Description

NiAVObjectPalette is a pure virtual interface for indexing NiAVObjects by name. It is used by NiControllerManager or your application to look up objects in the scene graph based on their name. The default implementation of an NiAVObjectPalette can be found in NiDefaultAVObjectPalette.

Supported Member Functions

virtual NiAVObject* GetAVObject(const char* pcName) = 0

Takes a string name and returns a pointer to an NiAVObject.

virtual void SetAVObject(const char* pcName, NiAVObject* pkObj) = 0

Stores an NiAVObject associated with the specified string name.
