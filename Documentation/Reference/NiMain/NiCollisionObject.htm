NiCollisionObject

: public NiObject

<NiCollisionObject.h>

Description

This class is an abstract base class that forms the basis for collision information and data that is associated with a particular object.

Constructors

Since this class has pure virtual functions, it cannot be instantiated. However, documentation for its constructor is given here to assist applications in creating their own subclasses of NiCollisionObject.

NiCollisionObject(NiAVObject* pkSceneObject)

Constructor that creates an NiCollisionObject with pkSceneObject as the target object.

Supported Member Functions

NiAVObject* GetSceneGraphObject() const

void SetSceneGraphObject(NiAVObject* pkSceneObject)

Retrieve and Set the NiAVObject that is associated with the collision information.

virtual void UpdateWorldData() = 0

A pure virtual function that is called whenever NiAVObject::UpdateWorldData is called and the NiAVObject has an associated NiCollisionObject.

virtual void RecreateWorldData() = 0

A pure virtual function that recreates the collision object from scratch.
