NiRect

: public NiMemObject

<NiRect.h>

Description

NiRect is a templated rectangle type whose specializations are used throughout the library. The member data is declared public, to allow it to be accessed using simple struct-like methods. Note that it is assumed that the parametric type T can support construction from a numeric type.

Constructors

NiRect(T left = T(0), T right = T(0), T top = T(0), T bottom = T(0))

This constructor is the default and member-wise constructor. The default constructor (no arguments) constructs and zeros the elements of the rectangle.

T GetWidth()

Returns the width of the rectangle.

T GetHeight()

Returns the height of the rectangle.
