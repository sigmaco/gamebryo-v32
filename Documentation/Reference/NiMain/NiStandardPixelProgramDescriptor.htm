NiStandardPixelProgramDescriptor

: public NiGPUProgramDescriptor

<NiStandardPixelProgramDescriptor.h>

Description

This class describes how a pixel program should be configured to match the settings on a given piece of geometry using the default pipeline.

Constructors

NiStandardPixelProgramDescriptor()

Standard constructor. Zeroes out the bitfield.

Supported Member Functions
