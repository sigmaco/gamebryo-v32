NiDepthStencilBuffer

: public Ni2DBuffer

<NiDepthStencilBuffer.h>

Description

This class represents an interface to a depth/stencil buffer on the graphics hardware. As with Ni2DBuffer, there is no capability of accessing the pixels contained within the buffer.

Constructors

No public constructors are available, as the creation functions must be able to fail in some cases. As a result, all creation is handled via the static Create functions below.

Static Member Functions

static NiDepthStencilBuffer* Create(unsigned int uiWidth, unsigned int uiHeight, NiRenderer* pkRenderer, const NiPixelFormat& kFormat, Ni2DBuffer::MultiSamplePreference eMSAAPref = Ni2DBuffer::MULTISAMPLE_NONE)

This create method attempts to create an NiDepthStencilBuffer with the preferred multisampling level matching the NiPixelFormat kFormat. If the renderer is unable to create a buffer matching the specifications, this method will return NULL. Note the preferred multisampling level must match that of the render target otherwise no multisampling will be used. Please refer to the release notes for each renderer that is being used to check multisample antialiasing support for rendered targets.

static NiDepthStencilBuffer* CreateCompatible(Ni2DBuffer* pkBuffer, NiRenderer* pkRenderer)

This method will attempt to create an NiDepthStencilBuffer with an NiPixelFormat that is compatible with the input buffer. If the renderer is unable to create a buffer of this type, this method will return NULL.
