NiTPointerList

: public NiTPointerListBase

<NiTPointerList.h>

Description

This class is a templated, doubly linked list intended to hold pointers, smart pointers, floats, ints, or, in general, any data with a size of 4 bytes. All list items are block allocated from a shared pool of memory. The interface is defined by the parent class, NiTPointerListBase. The templated type T has the following restrictions:

  • It must have a "0" element (i.e. T var; var = 0; is supported).
  • It must have a constructor, and that constructor must be able to create the "0" element (i.e. T var(0); must be a valid declaration).
  • It must have a valid copy constructor.
  • The destructor must be able to completely destroy the element without additional input.
  • The operator==, operator!=, and assignment operators must work correctly.

Constructors

NiTPointerList()

The default constructor creates an empty list.
