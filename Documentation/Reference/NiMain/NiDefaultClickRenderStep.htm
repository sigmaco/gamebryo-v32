NiDefaultClickRenderStep

: public NiClickRenderStep

<NiDefaultClickRenderStep.h>

Description

This class is an NiClickRenderStep that holds an internal array of render clicks that it returns for rendering. This class is the most common type of render step used.

Overridden Virtual Functions

virtual const NiTPointerList<NiRenderClickPtr>& GetRenderClicks()

Overridden from NiClickRenderStep. This function simply returns a reference to the stored list of render clicks.

Supported Member Functions

void AppendRenderClick(NiRenderClick* pkRenderClick)

void PrependRenderClick(NiRenderClick* pkRenderClick)

void RemoveRenderClick(NiRenderClick* pkRenderClick)

void RemoveAllRenderClicks()

NiTPointerList<NiRenderClickPtr>& GetRenderClickList()

const NiTPointerList<NiRenderClickPtr>& GetRenderClickList() const

NiRenderClick* GetRenderClickByName(const NiFixedString& kName) const

NiTListIterator GetRenderClickPosByName(const NiFixedString& kName) const

NiRenderClick* GetRenderClickAndPosByName(const NiFixedString& kName, NiTListIterator& kIter) const

These functions provide access to the list of render clicks. The GetRenderClickAndPosByName function returns the first render click in the list with the specified name, storing the list iterator to that render click in the supplied variable. GetRenderClickByName and GetRenderClickPosByName both call GetRenderClickAndPosByName under the hood and are provided for convenience purposes.
