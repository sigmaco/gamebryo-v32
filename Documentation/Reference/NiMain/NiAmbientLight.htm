NiAmbientLight

: public NiLight

<NiAmbientLight.h>

Description

NiAmbientLight is an object that represents a source of ambient light for a scene. This object is the simplest type of light object, and has no location, direction, or attenuation.

Constructors

NiAmbientLight()

Creates a NiAmbientLight object, which does nothing beyond creating an NiLight object.
