NiBackToFrontAccumulator

: public NiAccumulator

<NiBackToFrontAccumulator.h>

Description

NiBackToFrontAccumulator is an accumulator that is designed to simply render all registered objects in far-to-near order.

Constructors

NiBackToFrontAccumulator()

This default constructor creates an NiBackToFrontAccumulator.

Supported Member Functions

There are no supported member functions for this class.

Notes

Rendering Behavior

The NiBackToFrontAccumulator does not render any objects during the calls to RegisterObjectArray. All rendering is deferred until FinishAccumulating, because all registered objects must be available before any depth sorting can be completed.
