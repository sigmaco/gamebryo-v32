NiTexturePalette

: public NiRefObject

<NiTexturePalette.h>

Description

Abstract base class that presents the interfaces of a texture lookup table. The table maps filenames and paths to NiTexture objects via a set of virtual functions. The NiStream objects contain pointers to objects of this type to implement texture sharing.

Constructors

NiTexturePalette()

Default (and sole) constructor. Creates an empty texture palette

Supported Member Functions

virtual NiTexture* GetTexture(const char* pcPath, NiTexture* pkSelf) const = 0

Returns the texture (if any) in the palette that is associated with the given path.

virtual void SetTexture(const char* pcPath, NiTexture* pkTex) = 0

Sets the texture associated with the given path name. The particular subclass on NiTexturePalette may choose to ignore these calls—they are hints, not requirements.
