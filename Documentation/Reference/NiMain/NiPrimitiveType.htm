NiPrimitiveType

<NiPrimitiveType.h>

Description

NiPrimitiveType is a container class that holds the enumerations that describe various ways of interpreting the topology of vertex data. These enumerations are most commonly used by NiMesh objects to describe the structure of the triangle or line meshes they represent. Not all primitive types are directly supported on all platforms.

Enumerations

Enumeration Meaning
PRIMITIVE_TRIANGLES Individual triangles
PRIMITIVE_TRISTRIPS Triangle strip
PRIMITIVE_LINES Individual lines
PRIMITIVE_LINESTRIPS Line strip
PRIMITIVE_QUADS Individual quads
PRIMITIVE_POINTS Individual points
PRIMITIVE_MAX Invalid value used as a sentinel

Supported Static Functions

static const char* GetStringFromType(Type eType)

Returns a string that describes the NiPrimitiveType that is passed in as the eType parameter.
