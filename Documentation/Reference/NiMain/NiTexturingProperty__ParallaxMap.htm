NiTexturingProperty::ParallaxMap

: public NiTexturingProperty::Map

<NiTexturingProperty.h>

Description

NiTexturingProperty::ParallaxMap is a nested class of NiTexturingProperty and a subclass of the nested class NiTexturingProperty::Map. It adds a new data member that is specific to Gamebryo parallax maps an offset value that can be used by Gamebryo shaders to scale a parallax map.
