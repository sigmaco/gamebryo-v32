NiStringsExtraData

: public NiExtraData

<NiStringsExtraData.h>

Description

Extra data in the form of an array of arbitrary size of text strings that can be attached to aNiObjectNET.

Constructors

NiStringsExtraData(const unsigned int uiSize, const char** ppcValue)

Creates an extra data object that stores an array of arbitrary size of text strings.  A copy is made of the input array of text strings.

Supported Member Functions

void GetArray(unsigned int &uiSize, char** &ppcValue) const

void SetArray(const unsigned int uiSize, const char** ppcValue)

These functions retrieve and set the extra data as an array of arbitrary size of text strings.  For the latter, a copy is made of the input array of text strings, and any existing data in the object is deleted.

char* GetValue(const unsigned int uiIndex)

bool SetValue(const unsigned int uiIndex, char* pcValue)

These functions retrieve and set an individual entry in the array of text string extra data.

void AddValue(const char* pcValue)

Add an additional string to the array. The size is increased by one.
