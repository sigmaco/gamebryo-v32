NiShaderLibraryDesc

: public NiShaderRequirementDesc

<NiShaderLibraryDesc.h>

Description

NiShaderLibraryDesc provides a description for a shader library.

Constructors

NiShaderLibraryDesc()

The default constructor creates an uninitialized NiShaderLibraryDesc with no shader descriptions.

Supported Member Functions

const unsigned int GetShaderCount() const

Returns the number of shaders in the library.

const char* GetShaderName(unsigned int uiShader) const

Returns the name of the shader at index uiShader.

NiShaderDesc* GetShaderDesc(unsigned int uiShader) const

Retrieves the shader description at the given index, uiShader.

NiShaderDesc* GetShaderDesc(const char* pcShaderName) const

Retrieves the shader description for the given shader named pcShaderName. Returns NULL if the shader is not in the library.
