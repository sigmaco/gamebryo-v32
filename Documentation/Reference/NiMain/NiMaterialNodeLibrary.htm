: public NiRefObject

Description

This class contains a set of NiMaterialNodes.

Constructors

NiMaterialNodeLibrary()

The default constructor.

Supported Member Functions

unsigned short GetVersion()

Returns the version number for this library. This version number is used to help keep cached programs in sync between runs of the application.

unsigned int GetNodeCount()

Returns the number of material nodes contained within this library.

NiMaterialNode* GetAttachableNodeByName(const NiFixedString& kName, bool bClone = true)

Retrieves a material node by name. If specified, this node is cloned to create a unique instance.

NiMaterialNode* GetNode(unsigned int uiNode)

Retrieves a material node by index in the library.

void AddNode(NiMaterialNode* pkFrag)

Inserts a material node into the library.
