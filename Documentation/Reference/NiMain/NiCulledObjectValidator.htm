NiCulledObjectValidator

: public NiRenderClickValidator

<NiCulledObjectValidator.h>

Description

This class is a render click validator that uses the visibility of a particular NiAVObject scene graph to determine whether or not to execute its render click. It is provided with an NiAVObject, an NiCamera, and an NiCullingProcess, and it performs a culling pass to determine if any geometry objects are visible in that scene graph. The value to return when the scene graph is culled is configurable.

Constructors

NiCulledObjectValidator(NiAVObject* pkObject, NiCamera* pkCamera, NiCullingProcess* pkCullingProcess, bool bReturnOnCulled = false)

The sole constructor for this class takes a scene graph, a camera, and a culling process. It also takes a Boolean variable indicating the value that should be returned from the ValidateClick function in the case that the scene graph is entirely culled (no geometry objects are visible).

Overridden Virtual Functions

virtual bool ValidateClick(NiRenderClick* pkRenderClick, unsigned int uiFrameID)

Overridden from NiRenderClickValidator. This function uses the camera and culling process to perform a culling pass over the specified scene graph. If no geometry is returned from the culling pass, the function returns the value returned by GetReturnOnCulled. Otherwise, it returns the opposite of that value.
