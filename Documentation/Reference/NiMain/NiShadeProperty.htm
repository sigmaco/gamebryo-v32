NiShadeProperty

: public NiProperty

<NiShadeProperty.h>

Description

An NiShadeProperty allows applications to select Gouraud or flat shading.

Constructors

NiShadeProperty()

Default constructor. Constructs a property with smooth (Gouraud) shading enabled.

Static Member Functions

static int GetType()

static NiShadeProperty* GetDefault()

See NiProperty for details on these non-virtual functions.

Supported Member Functions

bool GetSmooth() const

void SetSmooth(bool bSmooth)

Retrieve and set the value of the smooth shading enable flag. If set to true, then smooth shading will be used when drawing polygons in the subtree to which the property is attached. If the flag is set to false, then flat shading is used.
