NiIntegersExtraData

: public NiExtraData

<NiIntegersExtraData.h>

Description

Extra data in the form of an array of arbitrary size of integer values that can be attached to a NiObjectNET.

Constructors

NiIntegersExtraData(const unsigned int uiSize, const int* piValue)

Creates an extra data object that stores an array of arbitrary size of integer values. A copy is made of the input array of integer values.

Supported Member Functions

void GetArray(unsigned int &uiSize, int* &piValue) const

void SetArray(const unsigned int uiSize, const int* piValue)

These functions retrieve and set the extra data as an array of arbitrary size of integer values. For the latter, a copy is made of the input array of integer values, and any existing data in the object is deleted.

int GetValue(const unsigned int uiIndex)

bool SetValue(const unsigned int uiIndex, int iValue)

These functions retrieve and set an individual entry in the array of integer extra data.
