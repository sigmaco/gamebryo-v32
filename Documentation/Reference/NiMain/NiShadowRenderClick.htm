NiShadowRenderClick

: public NiViewRenderClick

<NiShadowRenderClick.h>

Description

This class is a view render click that holds a reference to the NiShadowGenerator object that contains it, a dirty flag that defines if the click needs to be executed, and a force render flag that forces the click to always be executed. Shadow render click objects are used to render geometry to shadow maps for the shadowing system.

Constructors

NiShadowRenderClick()

The sole constructor for this class initializes the background color to NiColorA(1.0f, 1.0f, 1.0f, 1.0f), sets the use renderer background color to false, marks the click as dirty, and sets the stored NiShadowGenerator pointer to NULL.

Overridden Virtual Functions

virtual void PerformRendering(unsigned int uiFrameID)

This function call its parent class PerformRendering() function and sets the shadow render click's dirty flag to false.

Supported Member Functions

void SetGenerator(NiShadowGenerator* pkGenerator)

NiShadowGenerator* GetGenerator() const

These functions set and get the NiShadowGenerator used by the shadow render click. A shadow render click must have a valid NiShadowGenerator associated with it.

void SetForceRender(bool bForceRender)

bool GetForceRender() const

These functions set and get the force render state on the shadow render click. If this state is true, the NiShadowClickValidator will automatically allow the shadow render click to be executed.

void MarkAsDirty()

bool IsDirty() const

These functions set and query the dirty state of the shadow render click. A shadow render click will only be executed if its forced render state is true or if its forced render state dirty flag is true.

unsigned int GetLastRenderedFrameID()

This function returns the FrameID during which the shadow render click was last rendered.
