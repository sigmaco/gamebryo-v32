NiRenderClickValidator

: public NiRefObject

<NiRenderClickValidator.h>

Description

This class is an abstract base class for all render click validators. Each NiRenderClick holds a pointer to a render click validator to help determine whether or not that render click should render for any particular frame. The pure virtual ValidateClick function should be overridden by derived classes to return the appropriate value.

Supported Virtual Functions

virtual bool ValidateClick(NiRenderClick* pkRenderClick, unsigned int uiFrameID) = 0

This pure virtual function is called by NiRenderClick::ShouldPerformRendering and returns whether or not the specified render click should be rendered for the specified frame.
