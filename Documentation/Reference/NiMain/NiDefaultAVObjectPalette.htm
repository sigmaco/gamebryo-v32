NiDefaultAVObjectPalette

: public NiAVObjectPalette

<NiDefaultAVObjectPalette.h>

Description

NiDefaultAVObjectPalette is the default implementation of the NiAVObjectPalette interface. It contains an NiTStringPointerMap in which it stores NiAVObject pointers hashed on the name of the NiAVObject. It also contains a pointer to the root node of a scene graph. When GetAVObject is called with a particular name, it first looks up that name in the NiTStringPointerMap. If it doesn't find it there, it calls NiAVObject::GetObjectByName on the root of the stored scene graph to find the object. If it is found then, it stores that name/object pair in the hash table for fast lookup later. This class can be used to quickly lookup objects by name and provides a real benefit when the same object needs to be resolved multiple times.

Constructors

NiDefaultAVObjectPalette(NiAVObject* pkScene = NULL)

The constructor takes a pointer to the scene that should be used for looking up objects by name.

Supported Member Functions

virtual NiAVObject* GetAVObject(const char* pcName)

This function is overridden from the NiAVObjectPalette interface. When called, it first tries to look up the specified name in a hash table to return the proper NiAVObject pointer. If there is no entry in the hash table for the name, it calls NiAVObject::GetObjectByName on the stored scene graph root to attempt to resolve the pointer. If the object is still not found, the function returns NULL. If it is found, an entry for the name/object pair is added to the list, and the NiAVObject pointer is returned.

virtual void SetAVObject(const char* pcName, NiAVObject* pkObj)

This function is overridden from the NiAVObjectPalette interface. When called, it adds an entry to the hash table for the name/object pair, removing any previous entry that might have existed.

virtual void SetScene(NiAVObject* pkScene)

virtual NiAVObject* GetScene()

These functions get and set the stored scene graph pointer that is used for looking up NiAVObjects by name with the NiAVObject::GetObjectByName function.
