NiOutputStreamDescriptor::VertexFormatEntry

: public NiMemObject

<NiOutputStreamDescriptor.h>

Description

NiOutputStreamDescriptor::VertexFormatEntry objects describe a single entry in an NiOutputStreamDescriptor object. It is a public class that keeps track of a semantic name/index value, and a format/component count.
