NiRenderCallContext

<NiRenderCallContext.h>

Description

This class is a convenience class with public member variables that represents the collection of data needed to render a single render object. This class is used by the shader during a RenderMeshes call to minimize the number of parameters passed to sub routines.

Members

NiRenderObject* m_pkMesh;

Object currently being rendered. Assigned at beginning of render call.

const NiBound* m_pkWorldBound;

World bound of object.

const NiDynamicEffectState* m_pkEffects;

Effects of object.

const NiPropertyState* m_pkState;

Property state of object.

const NiTransform* m_pkWorld;

World transform of object.

const NiVertexDeclarationCache* m_pkMeshMaterialBinding;

Mesh material binding of object.

unsigned int m_uiPass;

Current pass, initialized to zero.

unsigned int m_uiSubmesh;

Current submesh, initialized to zero.

unsigned int m_uiActivePhases;

A bitmask of currently active phases. For a list of phase enumerations, see NiRenderer::RenderingPhase

Constructor

NiRenderCallContext()

The default constructor, clears all member values.
