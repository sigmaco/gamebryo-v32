NiStandardVertexProgramDescriptor

: public NiGPUProgramDescriptor

<NiStandardVertexProgramDescriptor.h>

Description

This class describes how a vertex program should be configured to match the settings on a given piece of geometry using the default pipeline.

Constructors

NiStandardVertexProgramDescriptor ()

Standard constructor. Zeroes out the bitfield.

Supported Member Functions

void SetOutputUVUsage(unsigned int uiWhichOutputUV, unsigned int uiInputUV, NiStandardMaterial::TexGenOutput eOutputUV)

Sets how a given input UV coordinate is transformed to produce a given output UV coordinate.

void GetOutputUVUsage(unsigned int uiWhichOutputUV, unsigned int& uiInputUV, NiStandardMaterial::TexGenOutput& eOutputUV)

Gets how a given input UV coordinate is transformed to produce a given output UV coordinate.

Notes

Bitfield

In order to compress the storage of the data for describing the vertex program, the data has been stored into a 128-bit bitfield. The values are:
