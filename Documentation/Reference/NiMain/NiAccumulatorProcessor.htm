NiAccumulatorProcessor

: public NiRenderListProcessor

<NiAccumulatorProcessor.h>

Description

This class is a render list processor that wraps an NiAccumulator. It uses that accumulator to process the list of geometry objects during rendering. The processor will optionally call FinishAccumulating on the accumulator once all geometry objects have been processed.

Constructors

NiAccumulatorProcessor(NiAccumulator* pkAccumulator = NULL, bool bFinishAccumulating = true)

The sole constructor for this class takes a pointer to the NiAccumulator to use during rendering. bFinishAccumulating indicates whether or not the processor should call FinishAccumulating on the accumulator at the end of the PreRenderProcessList function. if the accumulator pointer is null, this processor will simply pass all input geometry directly to the output array for later rendering.

Overridden Virtual Functions

virtual void PreRenderProcessList(const NiVisibleArray* pkInput, NiVisibleArray& kOutput, void* pvExtraData)

Overridden from NiRenderListProcessor. This function calls StartAccumulating on the stored NiAccumulator, if necessary, and then registers the pkInput array with that accumulator and optionally calls FinishAccumulating based on the value returned by the GetFinishAccumulating function. At the end of this function, all geometry objects in pkInput will have been rendered (if GetFinishAccumulating returns true). If there is no stored accumulator, all geometry objects in pkInput are added to kOutput, and the function returns without rendering anything. This function calls the GetCameraData function on the renderer to populate a temporary camera object that is used by the accumulator.

Supported Member Functions

These functions set and get the NiAccumulator object to use during rendering.

void SetFinishAccumulating(bool bFinishAccumulating)

These functions set and get whether or not FinishAccumulating should be called on the stored NiAccumulator object at the end of the PreRenderProcessList function.
