NiRenderTargetGroup::CubeMapSetupHelper

: public NiMemObject

<NiRenderTargetGroup.h>

Description

This is a helper class that is used to construct a series of NiRenderTargetGroup objects to represent the faces of an NiRenderedCubeMap. The reason this class should be used rather than creating the NiRenderTargetGroup objects is that using this class allows the engine to make renderer-specific optimizations in the resource allocations.

Constructors

CubeMapSetupHelper(NiRenderedCubeMap* pkCubeMap)

The constructor takes a pointer to the NiRenderedCubeMap that will be used to create the series of NiRenderTargetGroup objects.

Public Member Variables

NiRenderTargetGroupPtr m_aspRenderTargetGroups[NiRenderedCubeMap::FACE_NUM]

When the constructor returns from creating an instance of this class, this array will be filled with the six NiRenderTargetGroup objects that can be used to render to each face of the cube map.
