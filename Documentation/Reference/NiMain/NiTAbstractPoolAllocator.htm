NiTAbstractPoolAllocator

<NiTAbstractPoolAllocator.h>

Description

This class is a templated class used by NiTList and NiTMap for block allocation support.

Macros

Required Macros

NiAllocatorDeclareStatics(T, BLOCKSIZE)

This macro declares the static member variables defined by the NiTAbstractPoolAllocator class. This macro must be specified once for each instantiated NiTAbstractPoolAllocator type T. If customers instantiate NiTAbstractPoolAllocator types beyond what Gamebryo already instantiates, this macro must be used. For example, a customer who defines a container class of NiTObjectList<MyClass> would need to declare:

NiAllocatorDeclareStatics(MyClass, 256);

Where 256 is the pool block size.

NiAllocatorShutdown(T)

For each type that is used with NiTAbstractPoolAllocator, NiAllocatorShutdown must be called or the application will leak that pool's memory at application end. In general, this is only called for size_t automatically.

Optional Macros

NiAllocatorSetBlockSize(T, BLOCKSIZE)

This macro may be called to adjust the pool block size for type T at runtime.

NiAllocatorEnsureFreeMem(T)

This macro may be called to prime the allocator pool for type T at runtime.
