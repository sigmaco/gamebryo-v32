NiDitherProperty

: public NiProperty

Description

NiDitherProperty allows the application to turn the dithering of interpolated (or "iterated") triangle colors and fog values on and off. Dithering can be used to make the noticeable "banding" often apparent in 16-bit images effectively disappear.

Constructors

NiDitherProperty()

This default constructor creates a property with the dithering enable flag set to false (off).

Supported Member Functions

bool GetDithering() const

void SetDithering(bool bDither)

Retrieve and set the value of the dithering enable flag. If set to true, then the renderer is recommended to use dithering when drawing polygons in the subtree to which the property is attached. If the flag is set to false, then the renderer is recommended to avoid using dithering.

Notes

Hardware/Software Performance and Dithering

Current 3D hardware tends to have only minor losses in performance when dithering is used; however, a significant fraction of current 3D hardware does not support dithering. Software rasterizers tend to see significant losses in performance when dithering is used.
