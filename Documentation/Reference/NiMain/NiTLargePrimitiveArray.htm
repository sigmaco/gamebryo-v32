NiTLargePrimitiveArray

: public NiTLargeArray

<NiTLargeArray.h>

Description

This class is an extension of NiTLargeArray that is used to allocate intrinsic types like char, float, int, and C-style pointers (char*, NiObject*, and so on). Unlike NiTPrimitiveArray, this class can hold more than 65535 objects.
