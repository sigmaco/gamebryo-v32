PhysX Prop Service (PropService) Quick Reference

For more information about this service, see the ecrPhysX::PropService class reference documentation.

Library/Namespace

ecrPhysX

Display Name

PropService

Default Priority

1740

Priority for PhysX services varies depending on whether PhysX is being run synchronously or asynchronously; see PhysX system service setup for sample orderings. Although the Prop Service priority is not expected to vary, the function SetAsynchronousSimulation() must be passed a value consistent with the priorities of other PhysX services.

Notes

  • Receives SceneGraphAdded and SceneGraphRemoved messages from SceneGraphService; on SceneGraphAdded will process the NIF to find PhysXProp objects and construct corresponding NiPhysXProp objects.
  • SceneService handles the pure PhysX features of the PhysXScene model; PropService handles the additional features of the model used to construct an NiPhysXScene object.
