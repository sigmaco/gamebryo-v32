ecr::SceneGraphService::HandleSceneGraphFunctor

<SceneGraphService.h>

Inheritance graph for ecr::SceneGraphService::HandleSceneGraphFunctor

Description

A functor class for iterating over all scene graphs with handles.

Public Methods

virtual efd::Bool operator()(
const SceneGraphHandle handle,
const efd::vector<NiObjectPtr>& objects) = 0

Derived classes must implement this operator.

During scene graph iteration, the operator is called once for every handle that has a non-empty scene graph.

The operator can prevent further iteration by setting the return value.

Parameter Description
handle The handle for the scene graph objects.
objects The scene graph objects for the entity.
Return Value True if iteration should stop, or false if it should continue.
