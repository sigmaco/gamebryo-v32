Description

The SceneGraphAddedMessage message is sent by the SceneGraphService service when a new scene graph has been loaded and is available for use.

Other services should subscribe to this message if they need to take action when a new scene graph is available. For example, the Light Service looks for SceneGraphAddedMessage messages to add lighting to the newly added scene graph.

All functionality is inherited from the base class except the class ID accessor.
