Inheritance graph for ecr::RenderSurface

Description

This class wraps the concept of a render window or render surface.

This is useful for things like tools that may want to render to several different windows or controls. Most client side game related applications would only have one of these that represents the primary surface.

Public Methods

virtual ~RenderSurface()

Destructor.

The back buffer is not destroyed if this render surface refers to the primary back buffer. The renderer itself is responsible for this behavior.

NiCamera* GetCamera()

Gets the default camera used for rendering to this render surface.

On construction of a new RenderSurface object, the default camera is a newly allocated camera with all default parameters (see the NiCamera class documentation).

Parameter Description
Return Value The default camera.

void SetCamera(NiCamera* pCamera)

Set the default camera to be used for rendering to this render surface.

Parameter Description
pCamera The new camera to use when rendering to this render surface.

Gets the window handle for this render surface.

Note, on some platforms this handle has no meaning.

Parameter Description

Frame Rendering Interface

NiViewRenderClick* GetSceneRenderClick() const

Get a pointer to the render click used to draw the main scene.

void SetMainRenderClick(NiViewRenderClick* pRenderClick)

Set the primary 'scene' render click.

The existing click is replaced in the render step object's list of clicks, if present, and most of its properties are copied to the new click. Otherwise, the new render click is appended to the end of the render step obejct's list of clicks.

NiRenderTargetGroup* GetRenderTargetGroup()

Gets the render target group with which the render surface is bound to.

All rendering is done to this render target group.

This render target group is the primary render target group in the render frame. There may be more render target groups in a frame depending on the render steps that have been added. However, this render target group always refers to the back-buffer of the render surface.

Parameter Description
Return Value The render target group for the render surface.

NiRenderFrame* GetRenderFrame()

Gets the render frame for this render surface.

The render frame is the primary component of the frame rendering system and controls the rendering of a frame.

NiRenderStep* GetRenderStep()

Get the default NiRenderStep object for this surface.

By default, a click render step is created and registered with the render frame on this surface. The render target group for the default render step is attached to this surface's back buffer.

Parameter Description
Return Value The default render step for this render surface.
