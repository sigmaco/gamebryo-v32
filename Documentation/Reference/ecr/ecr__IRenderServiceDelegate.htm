ecr::IRenderServiceDelegate

<IRenderServiceDelegate.h>

Inheritance graph for ecr::IRenderServiceDelegate

Description

The IRenderServiceDelegate interface allows for a class to implement callback logic for common operations within the RenderService, such as RenderSurface creation and destruction.
