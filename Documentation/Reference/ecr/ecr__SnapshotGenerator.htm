Inheritance graph for ecr::SnapshotGenerator

Description

The SnapshotGenerator is used to produce live images from the game for display in a connected, Toolbench-hosted Simulation Debugger.

These images can be generated from arbitrary world locations to give a better idea of the state of the game beyond screen space.
