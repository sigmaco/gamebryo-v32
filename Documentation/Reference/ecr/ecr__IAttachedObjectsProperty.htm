Inheritance graph for ecr::IAttachedObjectsProperty

Description

The IAttachedObjectsProperty class provides the data structures used by built-in models that include an associative array of AttachedNifAsset data.

The current implementation is a simple data container, with convenience functions for models that use it. The class must be derived from to access the data - see the class header file for details.
