NSBPackingDef

: public NiMemObject

<NSBPackingDef.h>

Description

NSBPackingDef provides a definition of how to pack the geometry into vertex buffers in the Gamebryo engine.

Constructors

NSBPackingDef()

The constructor for NSBPackingDef creates an 'empty' packing definition instance.

Supported Member Functions

const char* GetName()

Retrieves the name of the packing def.

void SetName(const char* pcName)

Sets the name of the packing def to the given pcName.

bool GetFixedFunction()

Returns true if the packing definition will be used in the fixed-function pipeline as well, false otherwise. If the flag is true, the packing definition should adhere to the proper register usage for each vertex elements. Please refer to the proper system documentation for the proper values.

void SetFixedFunction(bool bFixedFunction)

Sets the fixed-function flag to the given bFixedFunction value.

bool AddPackingEntry(unsigned int uiStream, unsigned int uiRegister, unsigned int uiInput, NSBPackingDefEnum eType, NiShaderDeclaration::ShaderParameterTesselator eTesselator, NiShaderDeclaration::ShaderParameterUsage eUsage, unsigned int uiUsageIndex)

Adds a packing definition entry with the given parameters. Please refer to the NiShaderDeclaration for more information on the parameters.

static const char* GetTypeName(NSBPackingDefEnum eType)

Returns a printable string for the given packing definition type. Used strictly for debugging purposes.

static const char* GetParameterName(NiShaderDeclaration::ShaderParameter eParam)

Retrieves a printable string for the given parameter type. Used strictly for debugging purposes.

Notes

PackingDefType Enumeration

The PackingDefType enumeration provides ALL available types, regardless of platform. Several types are only supported on specific platforms; refer to those platforms' documentation for more details.
