Inheritance graph for etk::RCSStatus

Description

This class is used to return the revision control system status of a file.

The caller passes this object to the RCSProvider::Status() method to determine the current RCS state of a file.

Public Types

etk::RCSStatus::RCSAction

The current RCS state of a file.

Name Description
RCS_ACTION_UNKNOWN Indicates that file is not currently under revision control.
RCS_ACTION_NORMAL File is under revision control.
RCS_ACTION_ADD File is scheduled to be added to revision control.
RCS_ACTION_DELETE File has been marked for deletion.
RCS_ACTION_EDIT File has been locally modified.
RCS_ACTION_BRANCH The file has been branched from another file under revision control.
RCS_ACTION_INTEGRATE The file is schedule to be integrated into an existing file in revision control.
