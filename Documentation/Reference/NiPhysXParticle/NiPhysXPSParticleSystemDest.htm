Description

NiPhysXPSParticleSystemDest objects are responsible for interfacing between the PhysX scene and the Gamebryo particle system.

As a destination object, its update function is called upon NiPhysXScene::UpdateDestinations (assuming is it active).

An object of this type is created and stored by NiPhysXPSParticleSystem and NiPhysXPSMeshParticleSystem objects when they are created or their physics scene is changed. Applications never should need to create one of these. Documentation is provided for cases where the class must be extended.
