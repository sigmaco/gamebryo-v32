Description

NiPhysXPSParticleSystemProp objects hold information about a PhysX particle system prop.

Each prop stores an array of physics-based particle systems that are part of the prop, and implement callback functions to attach the systems to a scene, or remove them from a scene.

Particle systems are added to the prop via the function NiPhysXPSParticleSystem::AttachToProp, and functionality is provided here only to access the particle systems.
