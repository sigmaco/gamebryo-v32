Inheritance graph for NiPhysXPSSimulator

Description

NiPhysXPSSimulator objects add additional sync points to a NiPSSimulator mesh modifier.

These objects are created as part of physics-based particle or fluid system construction, and should never be explicitly created by an application.

Public Methods

NiPhysXPSSimulator()

Default constructor.

virtual ~NiPhysXPSSimulator()

Virtual destructor.

virtual bool Attach(NiMesh* pkMesh)

Attach function calls attach on the NiSPSimulator object and adds a completion sync point for PHYSICS_SIMULATE.

virtual bool Detach(NiMesh* pkMesh)

Detach is passed through to the NiSPSimulator object, and then the PHYSICS_SIMULATE complete point is removed.
