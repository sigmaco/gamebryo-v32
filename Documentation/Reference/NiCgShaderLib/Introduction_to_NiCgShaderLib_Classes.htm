NiCgShaderLib Class Reference: Introduction

The NiCgShaderLib is largely an internal-use only library. Most of its classes and functions are not documented. The purpose of the library is to allow Cg shader programs to be compiled and loaded in the same manner that shader programs that are written in HLSL or assembly are compiled.

For an overview of the interfaces that applications need to use, see NiCgShaderLib Files.

For information on how applications can use the library, see NiCgShaderLib.
