NiCgShaderLib Files

Description

NiCgShaderLib.cpp and NiCgShaderLib.h provide the interface functions for applications that will use the NiCgShaderLib. This library defines a single function for registering objects that will load Cg files.

Supported Library Functions

void NiCgShaderLib_RegisterShaderCreator();

This function registers the specialized NiD3DShaderProgramCreator objects that will load text and binary Cg shader programs. It associates the ".cg" file extension with text Cg shader programs. It also associates the ".cgo" file extension with binary Cg shader programs. This function must be called prior to loading any shaders that use Cg shader programs.
