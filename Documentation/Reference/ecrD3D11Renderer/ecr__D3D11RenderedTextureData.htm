Description

ecr::D3D11RenderedTextureData manages the D3D11 resource and resource view representing a rendered texture.

Textures of all dimensions, including cube maps, can be represented with this class. Textures that are rendered to using multiple resource views can be managed by separate NiRenderedTexture objects that refer to the same D3D11 resource but have different resource views.
