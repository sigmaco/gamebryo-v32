Description

D3D11 implementation of NiShaderLibrary.

The class introduces a directory member, ms_pDirectory, which is intended to provide derived libraries with a common place to look for shaders. The platforms the library is valid on are also identified in the static member ms_platform.

The description of the library is held in the m_spShaderLibraryDesc member, which is an instance of an NiShaderLibraryDesc class.
