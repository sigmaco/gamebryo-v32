Introduction to the D3D11Renderer

The Gamebryo D3D11-based Direct3D renderer is designed to give developers access to the latest features available on D3D11-capable systems. NiApplication links in the ecrD3D11Renderer by default.

The ecrD3D11Renderer requires the same version of the DX SDK to be installed on development machines that was used originally to build the library. It also requires the D3D11 runtime corresponding to this version of the DX SDK to be installed on end user machines. Refer to the documentation on DirectX Support for more details.

The ecrD3D11Renderer, like the D3D11 libraries themselves, is only supported under Windows Vista or later. Refer to the documentation on Windows Support for more details.
