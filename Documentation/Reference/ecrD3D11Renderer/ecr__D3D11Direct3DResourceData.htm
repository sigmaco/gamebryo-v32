Inheritance graph for ecr::D3D11Direct3DResourceData

Description

D3D11TextureData-derived class that manages a D3D11 resource provided by the application.

This allows applications to create D3D11 resources in ways unexpected by Gamebryo but still allow Gamebryo to interact with them as it does with other resources.
