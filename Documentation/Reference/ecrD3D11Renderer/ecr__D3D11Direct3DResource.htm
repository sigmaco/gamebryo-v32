Inheritance graph for ecr::D3D11Direct3DResource

Description

NiTexture-derived class that can be created by the renderer to wrap a D3D11 resource provided by the application.

This allows applications to create D3D11 resources in ways unexpected by Gamebryo but still allow Gamebryo to interact with them as it does with other resources.

Public Methods

virtual efd::UInt32 GetWidth() const

Returns the width of the resource if it is a texture, or the number of elements for a buffer.

virtual efd::UInt32 GetHeight() const

Returns the height of the resource if it is a texture with 2 or more dimensions, or 0 otherwise.

virtual efd::UInt32 GetDepth() const

Returns the depth of the resource if it is a texture with 3 dimensions, or 0 otherwise.

virtual efd::UInt32 GetArrayCount() const

Returns the number of elements in the resource if it is a texture array, or 1 otherwise.
