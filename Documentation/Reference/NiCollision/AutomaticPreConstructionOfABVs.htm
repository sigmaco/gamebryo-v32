Automatic Pre-Construction of ABVs

While it is possible to create ABVs at run time with application code, this approach is inefficient and will lengthen the time for the application to start, since the operation requires processing the scene graph, calculating the ABVs, removing proxy geometry, etc. By using the automatic ABV construction facilities provided with the Gamebryo exporter plug-ins, ABVs can be created and streamed to NIF. Streaming the ABVs is faster than processing the scene graph and calculating the ABVs at run-time.
