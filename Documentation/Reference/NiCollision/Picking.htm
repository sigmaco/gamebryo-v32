Picking

Picking is the detection of an intersection of a ray with an object in the scene. In Gamebryo, picking works with the default bounds (bounding volumes) of the scene graph. It can also be setup to work with ABVs.

Picking is a largely independent subsystem of the Gamebryo collision detection system. However, it uses many of the same concepts and low-level mathematical operations. See NiPick for more detailed information on picking.

Access Flags and Triangle Collision

Meshes used with picking must have a specific set of access flags and data streams. See the Collision System Usage page for details.
