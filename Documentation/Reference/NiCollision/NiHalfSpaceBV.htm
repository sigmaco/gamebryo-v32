NiHalfSpaceBV

: public NiBoundingVolume

<NiHalfSpaceBV.h>

Description:

The NiHalfSpaceBV ABV is useful for handling a (nearly) planar object without having to process the individual triangles that make up that object.  For example, the four walls, floor, and ceiling of a room can be added as six separate planar objects, each consisting of a pair of triangles, to a NiCollisionGroup.  An alternative would be to provide NiNode parents for each of the six planar objects and attach NiHalfSpaceBV ABVs to them.  Intersection tests for bounding volumes and planes are generally less expensive than tests for bounding volumes and triangles.

A half-space has a plane boundary.  The normal vector of the plane points to the half space.  If the plane is:

Dot(N, X) = c,

then a point P is inside the NiHalfSpaceBV whenever:

Dot(N, P) >= c.
