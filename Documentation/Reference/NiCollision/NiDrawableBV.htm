NiDrawableBV

: public NiMemObject

<NiDrawableBV.h>

Description:

NiDrawableBV is a static class that is intended for debugging use. Its members may be used to generate geometric representations of several of the ABV shapes. This capability is useful for applications to enable visually debugging of collision detection.

Static Member Functions:

static NiNode* CreateWireframeBV(const NiBoundingVolume* pBound, const NiColor& color, const float fMult, NiAVObject* pkObject = 0)

This function may be used to create an NiNode whose geometry leaf nodes represent the specified bounding volume. The NiNode will have appropriate properties attached to render it in wireframe in the specified color. To aid in visualizing the wireframe, an optional multiplier may be set for the bound so that it can be distinguished from the underlying geometry, in case they are of the same exact shape. The curved surfaces already have a slight multiplier applied to try to address this issue by offsetting the wireframe line segments so they will be visible.

Notes:

NiDrawableBV does not support Unions. CreateWireframeBV will return a NULL pointer in these cases.
