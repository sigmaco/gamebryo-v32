Triangles

The Gamebryo collision detection system is capable of performing triangle level collision detection against other collision types. It should be noted, however, that raw triangle collision detection can be the slowest method of collision detection. In nearly all cases, Oriented Bounding Boxes, are favored over strict triangle collision because that approach can converge much faster on the solution.

However, it may be prudent to question whether triangle collisions are really needed, given the performance hit they represent. Alternatives such as 2D collision or using ABVs will typically be far more optimal solutions.

Access Flags and Triangle Collision

Triangle meshes used in collision must have a specific set of access flags and data streams. See the Collision System Usage page for details.
