Limit Depth Traversal for Approximately Flat Objects

Maximum depth of traversal indicates the number of levels in an OBB tree that are to be traversed by the collision detection system. This parameter is specified when adding an object to a NiCollisionGroup.

The number of levels to be traversed is specified by setting the parameter to a positive number. The default value of -1 instructs the collision detection system to traverse the OBB tree for an object all the way to the leaf nodes.

For triangle meshes that are approximately flat (for example, sections of canyon walls or ground), a collision between an object and an OBB near the top of the OBB tree is typically accurate enough for most applications.
