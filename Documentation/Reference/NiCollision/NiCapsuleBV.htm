NiCapsuleBV

: public NiBoundingVolume

<NiCapsuleBV.h>

Description:

A capsule is defined as all points equidistant from a line segment. The NiCapsuleBV can be useful for providing a tight fitting bound around body parts of characters, around an entire character, or around any object for which one dimension exceeds the other two. The NiCapsuleBV is specified by a center, a major axis, an extent along that axis, and a radius.

Constructors:

NiCapsuleBV(float fRadius = 1.0, float fExtent = 1.0, NiPoint3 kCenter = NiPoint3::ZERO, NiPoint3 kAxis = NiPoint3::UNIT_X)

This constructor builds the NiCapsuleBV with a default origin at [0, 0, 0], a major axis aligned with the x-axis with an extent of 1.0 unit, and a radius of 1.0.

NiCapsuleBV(const NiCapsuleBV& kABV)

This copy constructor duplicates all the capsule parameters of the specified NiCapsuleBV.

Supported Member Functions:

void SetCenter(const NiPoint3& kCenter)

const NiPoint3& GetCenter() const

While the capsule will likely be fully defined at creation and never changed, the various GetValue and SetValue functions allow the individual parameters to be modified, causing the internally-stored capsule that is used for collision testing to be recomputed. In this case, these functions access the center of the capsule (its location relative to its parent).

void SetAxis(const NiPoint3& kAxis)

const NiPoint3& GetAxis() const

These functions access the major axis unit-length vector of the capsule.

void SetExtent(const float fExtent)

float GetExtent() const

These functions access the half-length extent of the major axis.

void SetRadius(float fRadius)

float GetRadius() const

These functions access the radius of the capsule.
