: public NiMemObject

<NiIntersector.h>

Description

The class NiIntersector is an abstract base class that encapsulates a numerical root finding technique, using backward differences.

NiIntersector and its derived classes are Gamebryo internal-only classes used for collision detection, and should not be used directly in an application. NiIntersector has no supported constructors or member functions, other than for internal use only.

Each derived class represents a pair of object types. Such derived classes implement a pseudo-distance function for the two object types and a function to compute a contact point and a contact normal at the first time of contact.

It is worth noting that squared distance calculations are used to avoid the square root calculation (for performance), and relative measurements are used so that this calculation does not vary with the scale of the objects.
