Inheritance graph for egmPhysXTerrain::PhysXTerrainService

Description

The PhysXTerrain service extends the Gamebryo Terrain service to support PhysX interactions with the terrain.

It adds the terrain custom data policy and loads physx data when capable

Public Methods

PhysXTerrainService()

Constructor.

Initializes members, but otherwise does nothing.

virtual efd::list<NxActor*> GetPhysXTerrainActors(egf::EntityID entityID)

This helper function returns a list of all actors created for the terrain associted with the given entity ID.

The list will be empty if there are no terrains for this entity ID or the terrain is not a PhysXTerrain.

Parameter Description
entityID The entity ID associated with the terrain we want the actors for
Return Value The list of NxActor created for the associated terrain
