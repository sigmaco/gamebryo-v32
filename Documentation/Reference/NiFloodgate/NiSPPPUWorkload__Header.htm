NiSPPPUWorkload::Header

<NiSPPPUWorkload.h>

Description

The header for a workload data structure. PS3 only.

Public Attributes

void* m_pvBaseAddress

Base address of the data stream.

NiUInt32 m_uiDataOffset

Offset in bytes from the base address to the section of the data the active kernel is working with.

NiUInt32 m_uiPadding

Padding in bytes to keep the data stream properly aligned in memory.

NiUInt32 m_uiBlockCount

The number of blocks the active kernel is operating on.

NiUInt32 m_uiStride

The size in bytes of each block.
