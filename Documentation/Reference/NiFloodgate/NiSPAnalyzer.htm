Description

This class handles the analysis phase of Floodgate workflow submission.

Analysis is the process of determining an execution order of tasks within a workflow such that the dependencies between the inputs and outputs of the tasks are satisfied. Applications do not need to call this function - analysis is automatically managed by the stream processor.
