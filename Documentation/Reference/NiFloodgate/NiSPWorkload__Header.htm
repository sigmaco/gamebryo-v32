NiSPWorkload::Header

<NiSPWorkload.h>

Description

A header to manage workload meta information within the NiSPWorkload.

Public Attributes

uint32_t m_uiBaseAddress

This class is a platform specific implementation. The documentation given here is PS3 specific. For platforms other than PS3, this documentation has incorrect types. Please refer to the platform specific header file for complete documentation. Base address of the data stream (main memory effective address).

uint32_t m_uiDataOffset

Offset in bytes from the base address to the section of the data the active kernel is working with.

uint16_t m_uiPadding

Padding in bytes to keep the data properly aligned in memory.

uint16_t m_uiBlockCount

The number of blocks the active kernel is operating on.

uint16_t m_uiStride

The size in bytes of each block.

uint16_t m_uiReserved

Unused bits to keep the data structure 128 bit aligned.
