NiSPThreadPool

: public efd::MemObject

Inheritance graph for NiSPThreadPool

Description

Maintains a pool of efd::LoopedThread objects used for executing tasks, each on a separate thread.

Public Types

typedef void(* AssignWorkerThreadAffinity)(NiUInt32 uiWorkerThreadID, NiUInt32 uiMaxWorkerThreads, NiProcessorAffinity &kProcessorAffinity)

Call-back function signature for assigning the processor affinity for a worker thread.

Parameter Description
uiWorkerThreadID Input. The ID of the worker thread to assign a processor affinity too. This value will range between 0 and uiMaxWorkerThreads - 1.
uiMaxWorkerThreads Input. The maximum number of worker threads that will be available.
kProcessorAffinity Input/Output. The NiProcessorAffinity that will be configured by this function.
