NiD3D10EffectShaderLib Class Reference: Introduction

The NiD3D10EffectShaderLib is largely an internal-use only library. Most of its classes and functions are not documented. The purpose of the library is to load FX files and create NiShader-derived objects so that they can be used to render Gamebryo objects.

For information on using FX shaders with Gamebryo in general, see here.

For an overview of the interfaces that applications need to use, see NiD3D10EffectShaderLib Files.
