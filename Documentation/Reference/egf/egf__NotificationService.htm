egf::NotificationService

: public efd::ISystemService

<NotificationService.h>

Inheritance graph for egf::NotificationService

Description

The notification service is responsible for monitoring changes to entities and notifying other entities and services about those changes.

It can send notifications by message, or entity behavior invocation.

Entity Change Handlers

void HandleEntityDiscoverMessage(
const egf::EntityChangeMessage* pMessage,
efd::Category targetChannel)

This message handler is called when a new entity is discovered.

It then checks for any registered discovery notifications that match and schedules the notification to be sent on the next tick.

Parameter Description
pMessage The message containing the discovery details.
targetChannel The channel the discovery message arrived on.
