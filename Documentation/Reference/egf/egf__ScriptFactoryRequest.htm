Description

The ScriptFactoryRequest is used to request an asynchronous script file load.

These requests are passed to the AssetFactoryManager for processing. The resulting world information is returned to the caller in a ScriptFactoryResponse object.

The requester only needs to provide the urn to the script file. The AssetFactoryManager is responsible for providing the fully qualified path to the asset.
