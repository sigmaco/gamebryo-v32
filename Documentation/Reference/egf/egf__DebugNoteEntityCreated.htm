Description

Notification message that alerts the Toolbench Sim Debugger that an entity has been created.

toolID The 128-bit tool-time ID, if the entity was generated from a data file.
modelName The name of the entity model used to create the new entity.
position The initial position of the new entity.
facing The initial facing angle of the new entity.
