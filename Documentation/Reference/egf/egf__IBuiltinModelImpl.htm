Description

A helper class to implement all of the property access methods needed in order to implement IBuiltinModel.

By deriving from this class you can then use the various EE_BUILTINMODELPROPERTY_* macros to easily bind properties to either raw variables or accessor methods.

See egf::IProperty and egf::IBuiltinModel for descriptions of the methods implemented by this class.
