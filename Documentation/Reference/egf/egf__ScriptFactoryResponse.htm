Description

Response generated by the ScriptFactory to reply to a ScriptFactoryRequest.

The ScriptFactoryResponse contains the original request URN and responseCategory. It includes a result indicating the status of the load. If the load was successful, the ScriptFactoryResponse message contains the size and contents of the script file, along with all of the other script files required by the script.

Construct a new ScriptFactoryResponse.

Each response contains the request URN and response category and a result. Successful loads will have at least one ScriptContent for the request script, and zero or more additional ScriptContent for dependent scripts.

Parameter Description
urn original request URN.
scriptName Name of the script, not including extension.
responseCategory original request response category.
result result of the load.
assetPath Fully qualified path to the asset, if available.
isReload hint to the factory indicating this is a reload.
scriptList Contents of script files, listed in breadth first order. The last element in this list is the requested script. Elements [0 to n-1] are dependent scripts.
