egf::IPropertyCallbackInvoker

<IPropertyCallbackInvoker.h>

Inheritance graph for egf::IPropertyCallbackInvoker

Description

The IPropertyCallbackInvoker class provides common functionality for managing property callbacks in built-in models.

It has no public methods. Classes that need to send property update callbacks should derived from this class.

The protected member function InvokeCallbacks is used to send the callbacks.
