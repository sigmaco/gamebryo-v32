IsStatic Boolean Bakeable False The IsStatic property is used to indicate that an entity will not be updated (moved, animated, etc) after it has been created. Only entities with this property true will be baked. Only entities with this property false will be updated by the Scene Graph Service on every frame. It is a major performance advantage to mark all static objects as such. Note that Rapid Iteration updates are still applied to static objects.
