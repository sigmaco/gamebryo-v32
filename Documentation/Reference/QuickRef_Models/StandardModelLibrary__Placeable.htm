Placeable Model

Found in project: StandardModelLibrary

Traits: Abstract, Active, BuiltinModel, ReplicaBuiltinModel, ToolBuiltinModel

Mixed Into: Renderable, PhysXProp, PhysXShape

Description

Provides position, rotation and scale information for placing objects in a 3D space.

Note: When mixing this into a model in another entity modeling project, it is advisable to not mix in directly. Instead, create a new model, like "MyCustomPlaceable", and mix-into that model. Then mix "MyCustomPlaceable" into all relevant models in that project. This is useful when refactoring large numbers of placeables at once.
