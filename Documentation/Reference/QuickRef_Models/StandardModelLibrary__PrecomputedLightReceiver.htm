PrecomputedLightReceiver Model

Found in project: StandardModelLibrary

Traits: Active

Mixes In: PrecomputedLighting

Mixed Into: PCLActor, PCLMesh

Description

Indicates that an entity is a receiver of precomputed lighting.

Properties

Name Type Source Model Default Value Traits Description
UseLightProbes Boolean PrecomputedLightReceiver False Indicates that this entity will use light probes rather than object-specific baked lighting for precomputed illumination. This will allow its lighting to change as it moves throughout the scene.
UseForPrecomputedLighting Boolean PrecomputedLighting True Indicates that this entity will be used in the generation of a precomputed lighting solution.
