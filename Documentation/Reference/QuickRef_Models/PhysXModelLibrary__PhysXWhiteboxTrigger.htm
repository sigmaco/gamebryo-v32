PhysXWhiteboxTrigger Model

Found in project: PhysXModelLibrary

Traits: Abstract, Active

Mixes In: PhysXTrigger, PhysXBox, Whitebox

Description

A PhysXBox that gets the dimensions from the Whitebox mixin so that it can quickly be created in World Builder and implement an OnPhysXTrigger behavior to react to collisions.
