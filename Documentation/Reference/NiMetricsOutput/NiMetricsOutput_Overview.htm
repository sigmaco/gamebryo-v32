NiMetricsOutput Overview

The NiMetricsOutput library is a collection of default output modules for the NiMetricsLayer class. To use any of these output modules, first include the relevant output module header file(s) and, if this is a DLL build configuration, define NIMETRICSOUTPUT_IMPORT. Within the application, instantiate the output module and add it to NiMetricsLayer using the NiMetricsLayer::AddModule function. Once this is done, the NiMetricsLayer will pass all instrumentation of the codebase or the application to all of the output modules that have been added.
