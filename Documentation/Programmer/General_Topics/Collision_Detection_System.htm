Collision Detection System

Gamebryo includes a robust and flexible collision detection system that can make use of bounding spheres, alternate bounding volumes, oriented bounding boxes, and triangle-triangle intersection testing.  Implementation for the Gamebryo-supplied collision detection system is documented in the ne;">NiCollision Engine Library Details and Reference manual.

The main scene graph system also includes the flexibility and interfaces for developers to link in a third party or internally developed collision system.
