Specular Lighting

To use specular highlighting, the following must be done:

  • Ensure that regular diffuse lighting is on (the NiVertexColorProperty lighting setting is LIGHTING_E_A_D).
  • Attach an NiSpecularProperty property to the subtree to be lit with specular lighting and turn it on.
  • Set the material specular color to a nonzero value on the object to be rendered as shiny.
  • Set the specular color of at least one light to be nonzero.
