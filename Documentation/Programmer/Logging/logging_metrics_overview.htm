Metrics Logging Overview

Metrics logging is a specialized application of logging. In addition to serving as a useful debugging tool (and a "poor man's profiler"), it demonstrates how to extend Gamebryo LightSpeed's logging system. When activated, logging calls placed strategically throughout the code generate structured log messages in order to measure various data points, which we call metrics probes.

Metrics logging must be enabled at build time. Once metrics logging is active, you can log an individual probe by calling the macros defined in efd/Metrics.h.
