Terrain with the Game Framework

The most common and recommended way of adding terrain support to your application is by registering the egmTerrain::TerrainService service with the main service manager of the application. The following code may be used to acheive this in a generic Game Framework application.

// The TerrainService handles Gamebryo terrain
egmTerrain::TerrainServicePtr spTerrainService = EE_NEW egmTerrain::TerrainService();
m_spServiceManager->RegisterSystemService(egmTerrain::TerrainService::CLASS_ID, pTerrainService);

Once added, the terrain service will manage the loading and unloading of all terrain related data whenever an entity containing the Terrain model is added/removed from the scene.

Please consult the emgTerrain Library documentation for more information.
