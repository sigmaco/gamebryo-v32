Removing behaviors

To remove a behavior, it must be removed from the Toolbench project. Open the project and model that contains the behavior in Toolbench, select the behavior in the behaviors tool window and press the "remove" button. Save the changes in Toolbench. The behavior is no longer available in the game and can no longer be called through behavior methods.

Now that the behavior is no longer available, find all the code and scripts that invoke it and remove them. The behavior's implementation, the script or C++ function, does not have to be removed from the code, but it is a good idea. Removing the function will prevent a script writer from assuming that it is a valid behavior that can be used.
