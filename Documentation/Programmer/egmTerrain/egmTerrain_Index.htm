  • EnvironmentService
    The egmTerrain::EnvironmentService interfaces the Game Framework to the Core Runtime Environment System. This implementation also includes:
    • support for a time of day file format to save and load any animated keyframes
    • time of day entity functor that provides basic functionality towards animating any entity property
    • time of day messages to ensure the environment system can provide users with rapid iteration functionality.
