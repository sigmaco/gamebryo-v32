Using the NiSceneGraphUpdateService Library in an Viewer

The NiSceneGraphUpdateService library can be used in a cross-platform viewer (or game) to display a scene graph that is being edited in a Windows-based tool. Once the NiViewerSceneGraphUpdate singleton in the viewer has successfully connected to an editor, the viewer has access to the replicated scene graph.

Optionally, the editor may set a camera that can be used (or ignored by) the viewer. The root node (and all the nodes under it) are automatically kept in sync with editor. Refer to the RemoteViewer class in the %EMERGENT_PATH%\Source\Tools\RemoteViewer project for an example of a viewer that simply renders the entire scene represented by the root node and uses the camera controlled by the editor.
