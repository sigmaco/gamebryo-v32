Shadow Click Validator

All NiShadowRenderClick objects produced by the shadowing system are assigned an NiShadowClickValidator object. The shadow click validator ensures that there is at least one visible shadow receiver for the NiShadowGenerator object referenced by a the shadow render click. If there are no visible shadow receivers, there is no need to render the shadow map.

When a shadow render click executes, its shadow click validator will be used to determine if the click actually needs to render. The validator first checks the forced render flag on the shadow render click. If this flag is true, the validator immediately returns true. If the flag is false, the validator iterates through the list of shadow receivers for the shadow generator associated with the shadow render click being executed. Each shadow receiver is culled against the active camera. If none of the shadow receivers are visible, the validator will return false indicating that the executed shadow render click does not need to be rendered.
