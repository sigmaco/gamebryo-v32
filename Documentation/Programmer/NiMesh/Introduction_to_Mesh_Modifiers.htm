Introduction to Mesh Modifiers

A mesh modifier is an object that dynamically modifies an NiMesh object at run-time. Gamebryo uses pre-defined mesh modifiers to implement skinning, morphing, particle systems and instance culling. Applications can define new mesh modifier classes to implement dynamic mesh functionality. For example, the Mesh Creation sample distributed with Gamebryo uses a mesh modifier to generate the water wave effect.

Mesh modifiers are designed to work with the Floodgate library, enabling easy-to-use parallel processing on mesh data.

This section of the programming guide includes the following topics:

  • Mesh Modifier Execution Model discusses the invocation and execution of mesh modifiers, including the model for controlling parallel computation.
  • Associating Mesh Modifiers with a Mesh provides details on the methods for attaching a modifier to a mesh.
  • Mesh Modifier Interactions with Tools describes the methods provided for checking, within content tools, whether the modifier is appropriate for a certain mesh.
  • Creating a Custom Mesh Modifier gives an overview of the functionality that must be provided to implement a new mesh modifier, and provides a set of best practices for developers.
  • Standard Gamebryo Mesh Modifier Requirements summarized to aid in the development of Mesh Profiles.
