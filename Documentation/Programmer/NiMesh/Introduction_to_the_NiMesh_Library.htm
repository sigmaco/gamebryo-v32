Introduction to the NiMesh Library

The Gamebryo NiMesh library contains a cross-platform set of classes for interacting with geometry.

  • The NiMesh Programmer's Guide topics cover the architecture of the system in detail, as well as the integration into the renderer. It is suggested that the topics be read in the following order:
    • NiMesh Primer
    • Creating an NiMesh Object
    • Working With NiMesh Data
    • Modifiers
    • Instancing
  • The NiMesh class reference documentation describes each of the classes in the library in detail.
