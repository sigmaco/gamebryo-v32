NiGPUProgram

This class is the base implementation for shader programs such as pixel or vertex shaders in the Gamebryo shader system. The class contains the name of the shader, the full name of the shader program (typically the file name), and members for tracking the code of the program and setting the constants that the program will use.

For a complete description of the functions and members contained in this class, please refer to the NiGPUProgram class reference.
