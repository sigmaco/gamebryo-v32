Description Classes

Several classes are provided for the tool chain usage of the shader system. These classes include the following:

  • NiShaderDescBase—the base class for all descriptions. Contains a name and description string.
  • NiShaderRequirementDesc—derived from NiShaderDescBase, this class extends the information included to cover the hardware requirements of the shader/library. This information includes pixel and vertex shader versions, platforms supported, bones per sub-mesh required (for skinning), the BinormalTangent method to use for setting up the geometry, and a user-defined version value.
  • NiShaderAttributeDesc—derived from NiShaderDescBase, this class provides a description of required attributes of the shader. This information includes an initial value for the attribute, as well as optional range values.
  • NiShaderDesc—derived from NiShaderRequirementDesc, this class describes a specific shader. This information includes the number of implementations the shader supplies and descriptions of each. A list of attribute descriptions (described above) is also included in this class.
  • NiShaderLibraryDesc—derived from NiShaderRequirementDesc, this class describes a shader library. This information includes the number of shaders the shader library contains, and a description for each one.
