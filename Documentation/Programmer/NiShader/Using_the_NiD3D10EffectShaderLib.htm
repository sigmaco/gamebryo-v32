Using the NiD3D10EffectShaderLib and ecrD3D11EffectShaderLib

The NiD3D10EffectShaderLib performs the same tasks for D3D10-oriented FX shader files as the NiD3DXEffectShaderLib does for D3D9 FX files. The ecrD3D11EffectShaderLib does the same for D3D11 FX shader files.

The D3D10 and D3D11 libraries can have difficulty loading FX files written for D3D9, and vice versa. As a result, Gamebryo implements a distinction between such files though file extensions. By default, Gamebryo assumes that files with the extension ".FX" (text) or ".FXO" (compiled) have been authored for D3D9; files with the extension ".FX10" (text) or ".FXO10" (compiled) have been authored for D3D10; and files with the extension ".FX11" (text) or ".FXO11" (compiled) have been authored for D3D11.
