Compiling the NiD3DXEffectShaderLib

The NiD3DXEffectShaderLib library makes direct use of DirectX9 interfaces in order to load the D3D Effect files. For this reason, the library requires that the same version of the DirectX9.0 SDK that was used to compile the rest of Gamebryo be installed in order to compile the library. Different DX9 SDKs may compile, but were not tested with the latest release of Gamebryo and may exhibit problems. It also requires that the DX9.0c runtimes be installed in order to load and utilize the files. This does not, however, imply that all tools that make use of the NiD3DXEffectShaderLib library must be recompiled using DX9.

For further information on versioning, refer to the DirectX Support or Build Prerequisites for Gamebryo LightSpeed for Win32 sections.
