Introduction

This document describes how the renderer interacts with shader instances in the Gamebryo engine.

Each subclass of NiShader instance via its parent interface. On D3D-based renderer platforms, this pointer can actually be treated as a pointer to an NiD3DShaderInterface, which is the level at which the renderer interacts with the shader system.

All rendering at this time gets funneled into derived renderer versions of Do_RenderMesh. We will use the DX9 renderer implementation of that method for this examination of the renderer interaction with shaders.
