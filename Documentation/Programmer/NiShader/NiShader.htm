NiShader

The base class for a Gamebryo shader is the NiShader class, contained in the NiMain library. This class is essentially a platform-independent interface class that provides a common interface for NiShader-NiRenderer interaction. An example of this interaction can be seen in the renderer interaction discussion.

Because NiShader is intended to be extended through derivation, the class makes use of Gamebryo's RTTI functionality to allow for usage of the NiIsKindOf family of functions.

The renderer-specific classes derived from NiShader will have their own mechanisms for handling concepts such as render passes and groups of render states to apply. All of these mechanisms will be enabled by the renderer using the common interfaces, though. There are, however, platform-independent concepts such as the NiGPUProgram and NiShaderConstantMap that provide similar functionality to all renderers.

For a complete description of the functions and members contained in this class, please refer to the NiShader class reference.
