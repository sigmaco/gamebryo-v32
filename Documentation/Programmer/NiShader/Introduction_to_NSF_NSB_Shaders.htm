Introduction to NSF/NSB Shaders

The NSF/NSB shader format was created in order to provide a data-driven method for generating and using Gamebryo's shader system. The system allows shaders to be written in text files known as NSF files, which are parsed into binary files known as NSB files. The NSB files can then be used to generate shaders that are applied to objects in a scene.

Shaders written in the NSF format can reference external files to act as vertex or pixel shader programs. These external files can be code written in shader assembly language, or a higher-level shader language like Microsoft's HLSL. They can even be compiled shader object code.
