NSFParserLib

The NSFParserLib is a library that will search a directory for valid NSF files, and generate an NSB file for every valid shader found in an NSF file that it finds. An application should refer to this library as "NSFParserLib" when calling NiShaderFactory::LoadAndRunParserLibrary.

The library consists primarily of a single interface function that will search for all NSF files in a given directory (optionally recursing any sub-folders found), and parses them into NSBShader objects, which are then saved back out to disk in NSB format.
