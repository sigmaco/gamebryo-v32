Frame Rendering System Components

The Gamebryo frame rendering system moves the complexity of setting up rendering effects to application initialization time rather than requiring custom code in the render loop. To achieve this, the system is broken up into several components, each of which represent a logical piece of the puzzle. See the Anatomy of a Frame section for a general overview of how the frame rendering system components interact. All classes associated with the frame rendering system are located in the NiMain library.
