Using PhysX built-ins in script

The PhysX system services expose a number of built-ins for scripted control of physics.

The egfPhysX libraries allow your script to turn simulation on and off for particular scenes, to check the current PhysX time, and to control the interactions of collision groups.

The ecrPhysX libraries allow your script to apply forces to entities, to vary the scale at which PhysX simulates them, and to reset their state to one of a number of "snapshots" stored in a NiPhysXProp.
