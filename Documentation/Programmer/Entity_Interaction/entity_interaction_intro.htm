Entity Interaction: Introduction

There are two main ways of interacting with Entities:

  • Behavior Interaction: An Entity or system service can interact with an Entity by invoking a Behavior which could be implemented in either C++ or script.
  • Direct State Change: An Entity or system service can interact with an Entity by getting and setting properties directly on that entity.
