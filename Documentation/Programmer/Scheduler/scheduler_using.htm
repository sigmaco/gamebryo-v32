Using the Scheduler

If you plan to use entity behaviors in your application, then you will need to create an instance of the Scheduler to handle the run-time queueing and invocation of the behaviors. You will also need to register any scripting run-time engines that your game will require. This is done by adding code similar to the following to your application Main.

    SchedulerPtr spScheduler = EE_NEW Scheduler();
    m_spServiceManager->RegisterSystemService(
        Scheduler::CLASS_ID,
        spScheduler);

    // Since we want to be able to run Lua behaviors we must add the Lua scripting runtime to the
    // scheduler.
    spScheduler->RegisterScriptingRuntime("Lua", BehaviorType_Lua, EE_NEW SchedulerLua());
