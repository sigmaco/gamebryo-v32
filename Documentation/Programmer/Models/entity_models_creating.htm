Creating models

  • There are step-by-step instructions for creating Models using Toolbench in the Hello World Tutorial. It also shows some Model hierarchy in a practical application.
  • Here are the basic steps for creating a Model in Toolbench.
    • Start the Toolbench application.
    • Choose File > New. Then select Game Solution. Follow the instructions on the dialogs to create your empty project.
    • Double-click on the *.emtproj in the Game Solution to load it.
    • Right-click on the *.emtproj in the Game Solution, choose Add > New Model.
    • The middle, canvas panel should now have a box labeled "Model". Fill in details in the right-hand panel, the Entity Model Editor, to customize this model.
    • Note: There are file layout planning steps that should occur before creating models. See Asset Runtime Service for more detail.
  • Models are mixed together by dragging from the Model Palette to the area above or below the diagram of the current Model.
