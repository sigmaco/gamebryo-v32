Replication Requirements

The table below lists the libraries, system services, and header files that are required to work with the replication service:

Library System Services Header Files Description
eon
  • Replication Service
  • eon/ReplicationService.h
 Implementation of the replication system service, needed to create and register the service.
eonLua, eonPython [optional]
  • eon/bapiOnline.h
 Lua and Python bindings for replication-related built-in APIs. Bound into the bapiOnline module for use from scripts.
