Foundation Programming: Introduction

Foundation provides a base layer of software on which the rest of Gamebryo LightSpeed is built. It provides a generic set of functionality that is useful for a wide variety of software applications. Key functionality includes: logging, messaging, memory management, and asset management. It also provides the System Service framework. This framework implements the main loop of the application and provides processing time to all of the other services that are defined by the application. Messaging is implemented using a System Service and uses a publish-subscribe model.
