Networking Requirements

The following table lists the libraries, system services, and header files that are required to work with remote messages in a C++ module:

Library System Services Header Files Description
efd
  • Message Service
  • MessageService.h
  • Provides access to the core message service functionality. Includes headers for IMessage, MessageHandler and associated macros.
efdNetwork
  • Net Service
  • Channel Manager Service
  • NetService.h
  • ChannelManagerService.h
  • Provides access to the remote message service functionality.
  • The Channel Manager Service is required only on one of the processes forming the online gaming environment.
  • The headers files are needed when instantiating these services. They are not needed for using remote messages.

The following table lists the libraries, system services, and header files that are required to work directly with the network system in a C++ module when you need to access low level information such as connection status:

Library System Services Header Files Description
efdNetwork
  • Net Service
  • NetService.h
  • Provides access to the networking functionality.
