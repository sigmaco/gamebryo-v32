egmTerrain

Description

The egmTerrain lua library provides the core functions for interacting with terrain from behavior script code. These built-in API methods interact with the egmTerrain::TerrainService system service. Below are the supported functions.

Library Functions

boolean, number, table FindTerrainIntersection(
table point)

Picks against the terrain in the -Z direction from the given point. Returns true if the terrain is intersected.

ParameterDescription
pointA Point3 used as the starting point for the query.
Return ValueDescription
boolean True if the terrain was intersected, false otherwise.
numberIf the terrain is intersected, this number contains the height or Z value of the intersection.
tableIf the terrain is intersected, a Point3 containing the surface normal at the intersection point is returned.
