SearchOptionsFileFilter (Extension-Point)

Emergent.Toolbench.Search.UI.SearchOptions.FileFilter

This extension point is used to specify sets of file extensions to be listed in the Toolbench Find in Files dialog. 

<extension point="Emergent.Toolbench.Search.UI.SearchOptions.FileFilter">
  <file-filter persist-key="egt.codeeditor.lua"
               filter="*.lua"
               priority="50">
  </file-filter>
</extension>

file-filter

Attribute Type Description
persist-key String Name used to reference the file filter when user state is stored.
filter String Semicolon-delimited list of file extensions to search.
priority Integer Determines the display order of the file filters, lowest first.
