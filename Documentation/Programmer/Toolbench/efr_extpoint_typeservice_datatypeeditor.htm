DataTypeEditor extension-point

The data type editor extension point allows you to specify an editor which can be used to edit specific data types.

<extension point="Emergent.Toolbench.TypeService.DataTypeEditor">
  <editor id="Emergent.Toolbench.TypeService.Editors.BooleanEditor"
          group="TypeService"
          class="Emergent.Toolbench.TypeService.Editors.BooleanEditor">
  </editor>
</extension>

editor

Identifier Type Description
id String The unique identifier of the editor.
group String A group name that the editor belongs to.
class String The full name of the class that gets instantiated to represent the editor.
