ContentCategory (Extension-Point)

Emergent.Toolbench.ContentService.ContentCategory

Content categories are a mechanism for organizing content. This extension point facilitates creating new categories in which content can be added. Content can then be queried via category using the content service (Emergent.Toolbench.ContentService.IContentService).

<extension point="Emergent.Toolbench.ContentService.ContentCategory">
  <category name="General"/>
</extension>

category

Attribute Type Description
name String The name of the category that content types should be listed under when presented to the user.
