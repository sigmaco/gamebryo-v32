Extension

To implement existing functionality in your plug-in you will need to add extensions to your plug-in manifest. The extension element allows plug-ins with extension-points to scan your plug-in and understand what features of their plug-in you have implemented.

<extension point="Emergent.Toolbench.Workbench.ActionSet">
    ...
</extension>

extension

Attribute Type Description
point String The unique identifier of the extension point this extension implements.
