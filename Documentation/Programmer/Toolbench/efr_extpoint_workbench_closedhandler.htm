ClosedHandler (Extension-Point)

Emergent.Toolbench.Workbench.ClosedHandler

The closed handler extension point allows you to hook via an extension the closed event generated by Workbench. This event is generated immediately after all of the closing handlers have been called, assuming none of them have cancelled the closing operation. If you need to shutdown any UI related things, you should do them here.

Note: Your plug-in must have been loaded for your closed handler to be called.

<extension point="Emergent.Toolbench.Workbench.ClosedHandler">
  <handler class="Emergent.Toolbench.UI.OutputView.ExitHandlerAction"/>
</extension>

handler

Attribute Type Description
class String The full name of the class to use when calling the closed handler. The class must implement the Emergent.Toolbench.UI.Workbench.IWorkbenchClosedHandler interface.
