DynamicContextMenuProvider (Extension-Point)

Emergent.Toolbench.Workbench.DynamicContextMenuProvider

Toolbench provides an extension-driven dynamic context menu service. This can be used to add new commands to an existing context menu without directly modifying the plug-in that first introduced the menu. The DynamicContextMenuProvider extension point exposes this functionality.

<extension point="Emergent.Toolbench.Workbench.DynamicContextMenuProvider">
  <provider name="ModelingProjectRevisionContextMenuProvider"
              type="Emergent.Toolbench.GameSolution.GameSolutionItem"
              key="Emergent.Toolbench.EntityModeling.EntityModelingProject"
              class="Emergent.Toolbench.GameSolution.UI.RevisionContextMenuProvider" />
</extension>

provider

Attribute Type Description
name String The name of the menu provider.
type String The type of item for which the provider will supply a menu.
key String The key of items for which the provider will supply a menu. This is a logical key, often referencing a class, that can be used to further distinguish which types are applicable for this provider.
class String The full name of the class implementing IDynamicContextMenuProvider that provides the context menu.
priority Integer The priority of the menu provider. This affects how the generated menu item group will sort compared to the other groups of dynamic context menu items. Lower priority places items closer to the top of the menu.
