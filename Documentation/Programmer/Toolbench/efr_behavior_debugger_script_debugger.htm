Toolbench Script Debugger

The Toolbench portion of behavior debugging is implemented in three plugins.

  • Emergent.Toolbench.ScriptDebugger—manages the tool-side state of the debugger.
  • Emergent.Toolbench.ScriptDebuggerMessaging—handles communication with the connected game and its behavior debugging module.
  • Emergent.Toolbench.ScriptDebugger.UI—presents the graphical script debugging interface to users.
