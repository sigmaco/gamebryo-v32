ContentWizard (Extension-Point)

Emergent.Toolbench.ContentService.ContentWizard

The ContentWizard extension point provides a mechanism for creating helper wizards which will be shown when creating content of certain types. As an example, the Game Solution Document requires the user to choose multiple options when it is created; these options are presented via a wizard which is specified using this extension.

<extension point="Emergent.Toolbench.ContentService.ContentWizard">
  <wizard content-id="Emergent.Toolbench.GameSolution.GameSolutionDocument"/
             class="Emergent.Toolbench.GameSolution.UI.GameSolutionWizard>
</extension>

wizard

Identifier Type Description
content-id String The content-id is the full name of the content type which uses this wizard for creation.
class String The class is the full class name of the content wizard class. The wizard class must implement the Emergent.Toolbench.ContentService.IContentWizard interface.
