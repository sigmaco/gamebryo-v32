BrushTypeUI (Extension-Point)

Emergent.Toolbench.WorldBuilder.Terrain.UI.BrushTypeUI

The Terrain BrushTypeUI extension point is used to define a custom user interface to configure a particular brush type. The user interface is displayed in the Terrain Editor panel whenever the user selects to use that particular brush type. The class defining how the brush type is applied to terrain data is defined in another extension point 'BrushType'. Both extension points must be used together.

The class indicated by the class attribute of the extension point must refer to a class implementing either the UserControl class. 

<!-- New brush type UI. -->
<extension point="Emergent.Toolbench.WorldBuilder.Terrain.UI.BrushTypeUI">
    <brushtypeui class="Emergent.Toolbench.WorldBuilder.Terrain.UI.BrushTypes.NewBrushTypeControl"
            brush-type="NewBrushTypeName"/>
</extension>

BrushTypeUI

Attribute Type Description
class String The class name of the UI object to be created to represent the given brushtype.
brush-type String The name of the BrushType registered in the BrushType extension point that this UI is to be associated with
