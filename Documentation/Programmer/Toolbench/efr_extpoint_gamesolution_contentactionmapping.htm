ContentActionMapping (Extension-Point)

Emergent.Toolbench.GameSolution.ContentActionMapping

The ContentActionMapping allows content types to be mapped to actions. This way the solution can display options specific to a piece of content in something like a context menu or some other UI.

<extension point="Emergent.Toolbench.GameSolution.ContentActionMapping">
  <content-type> 
    <action id="Emergent.Toolbench.GameSolution.UI.DefaultActivateItemAction"/>
    ...
</extension>

content-type

Identifier Type Description
id String [optional] The id is the full name of the content type to which this action mapping applies. If no id is given, the mapping is considered default.

action

Identifier Type Description
id String The id is the full name of the action to apply.
command enum [optional] If this is "activate" the content is set as the active content. If the command is "remove" then the content is removed. If nothing is set, the item is created but the content is closed after creation.
