View (Extension-Point)

Emergent.Toolbench.Workbench.View

This extension point is used for adding new views to Toolbench. A view is a windows form that you want to be created and instantiated in a data driven fashion.

<extension point="Emergent.Toolbench.Workbench.View">
  <view id="Emergent.Toolbench.UI.OutputView"
        name="Output View"
        class="Emergent.Toolbench.UI.OutputView.OutputView"
        default-dock="DockBottom"
        icon="Icons\icon.png"/>
</extension>

view

Attribute Type Description
id String The id is a unique identifier used to refer to this view.
name String The name of the view is the display text used in the title bar of the view and other areas where a name is needed to identify a view.
class String The full name of the class to use when instantiating the view. The class must inherit from the Emergent.Toolbench.UI.Workbench.AbstractView class.
default-dock enum The default dock of the view is the location at which the view should appear by default.
  • Float
  • DockTopAutoHide
  • DockLeftAutoHide
  • DockBottomAutoHide
  • DockRightAutoHide
  • Document
  • DockTop
  • DockLeft
  • DockBottom
  • DockRight
content-bound Boolean [optional] [default: false] Content bound views will be instantiated on a per-content basis. So if a view were content bound to the image content type and we opened 3 images, 3 separate instances of that view would be created. It also means that if those views are closed, their content is closed as well.
default-visible Boolean [optional] [default: true] The default visibility of the view.
allow-float Boolean [optional] [default: true] Determines if the view should be allowed to be torn out and float above the other windows outside the primary Toolbench window.
allow-docking Boolean [optional] [default: true] Determines if the view should be allowed to be redocked by the user in a different location than the default.
icon String [optional] The path to the icon associated with this view. The path should be relative to the directory of your plug-in.
hide-on-close Boolean [optional] [default: false] Determines if the view should be hidden instead of closed when the view is closed.
default-width int [optional] [default: 300] Determines the view's default width. This option will only be used by views that have their default dock set to float.
default-height int [optional] [default: 300] Determines the view's default height. This option will only be used by views that have their default dock set to float.
default-location-x int [optional] [default: 30] Determines the view's default location along the x axis. This option will only be used by views that have their default dock set to float.
default-location-y int [optional] [default: 30] Determines the view's default location along the y axis. This option will only be used by views that have their default dock set to float.
