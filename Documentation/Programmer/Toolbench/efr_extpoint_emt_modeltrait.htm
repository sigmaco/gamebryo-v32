ModelTrait (Extension-Point)

Emergent.Toolbench.EntityModeling.ModelTrait

The model trait extension point allows you to create traits which can be specified on a per model basis that your game will interpret.

<extension point="Emergent.Toolbench.EntityModeling.ModelTrait">
  <model-trait name="TraitName"
               tool-time="false"
               default-on="false">
    <description>Trait Description.</description>
  </model-trait>
</extension>

model-trait

Attribute Type Description
name String The unique name of the trait.
tool-time Boolean [optional] [default: false] The tool-time property controls whether the trait is available only for tools. Tool time traits will be stripped out in the export process and not be written to the flat model files.
default-on Boolean [optional] [default: false] The default-on property controls whether the trait is on by default.
