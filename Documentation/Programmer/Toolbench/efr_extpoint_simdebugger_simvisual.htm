SimVisual (Extension-Point)

Emergent.Toolbench.SimDebugger.UI.SimEvent

The SimVisual extension point defines new "layers" for the Simulation Debugger World View visualization. SimVisuals generally render one type of game-related information, such as entity positions. In addition to rendering, SimVisuals can implement picking against the items they have drawn. SimVisuals can be toggled on and off via toolbar buttons, which serve as filters for the World View display. 

<extension point="Emergent.Toolbench.SimDebugger.UI.SimVisual">
  <sim-visual class="Emergent.Toolbench.SimDebugger.UI.GridVisual"
              priority="10"
              setting="Emergent.Toolbench.SimDebugger.UI.ShowGridVisual">
  </sim-visual>
</extension>

sim-visual

Attribute Type Description
class String The class factoried to render elements to the World View.
override String [optional] A SimVisual class to be completely replaced by this extension.
priority Integer [optional] [default: 0] Determines the draw order of SimVisuals, lowest first.
setting String [optional] A boolean PerUserPerContent setting that toggles display of the SimVisual.
