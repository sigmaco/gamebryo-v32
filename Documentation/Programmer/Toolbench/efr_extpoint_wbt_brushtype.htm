BrushType (Extension-Point)

Emergent.Toolbench.WorldBuilder.Terrain.UI.BrushType

The Terrain BrushType extension point is used to define a custom BrushType for manipulating terrain data through the painting interfaces. A brush type defines a kernel region over which the brush will take effect when painting. The BrushType will be displayed at the top of the Terrain Editor panel for the user to select. If the BrushType being added requires a user interface to customize various settings, then the BrushTypeUI extension point should also be used.

The class indicated by the class attribute of the extension point must refer to a class implementing the AbstractBrushElement class. 

<!-- New Brush Type Logic Object. -->
<extension point="Emergent.Toolbench.WorldBuilder.TerrainCore.BrushType">
    <brushtype class="Emergent.Toolbench.WorldBuilder.Terrain.UI.BrushTypes.NewBrushTypeClass"/>
</extension>

brushtype

Attribute Type Description
class String The class name of the object to be created. This object handles connecting and disconnecting the brushtype when the user changes the brushtype selection.
