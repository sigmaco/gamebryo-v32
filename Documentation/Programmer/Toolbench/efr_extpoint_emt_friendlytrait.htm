FriendlyTrait (Extension-Point)

Emergent.Toolbench.EntityModeling.FriendlyTrait

The friendly trait extension point allows you to customize text that is shown for the Property, Model, and Behavior traits in Entity Modeling Tool. A more user-friendly name and description can be provided via this extension point. Additionally, a studio can customize whether or not a trait is defaulted to on.

<extension point="Emergent.Toolbench.EntityModeling.FriendlyTrait">
        <friendly-trait trait-name="ReplicationGroup00" trait-group="PropertyTrait" 
                        friendly-name="FastMoving" default-on="false" visible="true" priority="50">
            <friendly-description>For fast moving entities</friendly-description>
        </friendly-trait>
</extension>

friendly-trait

Attribute Type Description
trait-name String The unique name of the trait to override.
trait-group String The type of Trait that is being overridden. Possible values: PropertyTrait, ModelTrait, BehaviorTrait.
friendly-name String The user-friendly name to use instead of the trait's default name.
friendly-description String The user-friendly text to describe this trait to a user.
default-on Boolean [optional] [default: false] The default-on property controls whether the trait is on by default.
visible Boolean [optional] [default: true] The visible property allows customers to hide or show specific traits to end users.
priority Unsigned integer [optional] [default: 100] The priority property helps determine which override to use in the event of conflicts. The lowest priority declared always wins.
