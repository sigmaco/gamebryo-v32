BrushElement (Extension-Point)

Emergent.Toolbench.WorldBuilder.Terrain.UI.BrushElement

The Terrain BrushElement extension point is used to define custom paint filters used to filter terrain data when painting on the terrain. Multiple brush elements may be active on the paint brush at once, and are available for selection from the Terrain Editor panel menus. Each BrushElement defines a specific operation to perform on the terrain data. Existing BrushElements include elevation, noise, smoothing, and painting elements.

New brush elements need to implement the IBrushElement and IPaintDataFilter interfaces. The abstract base class new brush elements should derive from is AbstractBrushElement.cs. Refer to the documentation for these classes for greater detail. 

<!-- New brush element logic object. -->
<extension point="Emergent.Toolbench.WorldBuilder.TerrainCore.BrushElement">
    <brushelement name="New IBrushElementName"
                  class="Emergent.Toolbench.WorldBuilder.Terrain.UI.BrushElements.NewBrushElementClass"
                   icon="Icons/NewBrushElementIconFile.png" 
               category="IBrushElementCategory">
        String containing New IBrushElement Description. 
    </brushelement>
</extension>

BrushElement

Attribute Type Description
name String The unique identifier for this brushelement mode. This name will be displayed to the user for selecting the brushelement.
class String The class name of the object to be created. This object handles connecting and disconnecting the brushelement when the user changes the brushelement selection.
icon String The path to the icon resource file that display the icon for this brushelement.
category String The catetory of interface to implement for this brushelement.
