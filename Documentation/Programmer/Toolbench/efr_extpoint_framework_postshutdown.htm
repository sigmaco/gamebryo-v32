PostShutdown (Extension-Point)

Emergent.Toolbench.Framework.PostShutdown

The PostShutdown extension point provides a mechanism for executing any shutdown code after the framework (including the service manager and all the services) has been shutdown.

<extension point="Emergent.Toolbench.Framework.PostShutdown">
  <handler class="Emergent.Toolbench.Framework.PostShutownClass"/>
</extension>

handler

Identifier Type Description
class String The class is the full class name of the post shutdown class. The post shutdown class must implement the Emergent.Toolbench.Framework.IFrameworkShutdownHandler interface.
