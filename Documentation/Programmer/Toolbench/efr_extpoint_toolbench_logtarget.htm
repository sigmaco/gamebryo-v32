LogTarget (Extension-Point)

Emergent.Toolbench.LogTarget

When a plug-in logs something using the standard Toolbench logger, log targets intercept the log action and do something useful with them. By default Toolbench doesn't actually log anything when log is called. It is up to the log tagets to store the information somewhere the user can locate it. However, there are several standard log targets included in other plug-ins with Toolbench.

<extension point="Emergent.Toolbench.LogTarget">
  <log-target class="Emergent.Toolbench.ConsoleLogger.ConsoleLogger" />
</extension>

log-target

Attribute Type Description
class String The class is the full class name of the log target. The log target class must implement the Emergent.Toolbench.ILogTarget interface.
