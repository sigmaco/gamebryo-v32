Localization

The localization element is used to tell Toolbench the fallback culture to use for your plug-in when resources are requested from your plug-in from the Platform.Resources property.

<localization fallback-culture="en-US"
              resource-base="Emergent.I18nSample.Localization.Resources">
</localization>

localization

Identifier Type Description
fallback-culture String The fallback-culture corresponds to the culture of the satellite assembly to load. Resources in the fallback culture will be used if no satellite assembly exists for the current culture. However, it will also be used to access individual elements from the resources if those resources can not be found in the active cultures satellite assembly, or any of that cultures parent satellite assemblies.
resource-base String The resource base corresponds to the resource manifest name of the resources in your satellite assembly.
