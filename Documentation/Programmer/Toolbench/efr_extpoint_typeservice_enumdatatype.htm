EnumDataType (Extension-Point)

Emergent.Toolbench.TypeService.EnumDataType

The EnumDataType extension point allows plug-ins to create custom enumerations which can then be used as data types for properties. These enumerations consist of a collection of strings each with an ordinal number associated with them. The extension point allows the enumeration's name to be different than the name used when the enumeration is exported.

<extension point="Emergent.Toolbench.TypeService.EnumDataType">
  <enum name="Enum">
    <literal name="One" ordinal="1"/>
    ...
</extension>

enum

Identifier Type Description
name String The user-friendly name of the enumeration.
export-name String [optional] The name used for the enumeration when properties using this enumeration as a data type are exported.

literal

Identifier Type Description
name String The user-friendly name of the literal in the enumeration.
ordinal Integer The ordinal number assigned to the literal in the enumeration.
description String [optional] Describes the literal in the enumeration.
