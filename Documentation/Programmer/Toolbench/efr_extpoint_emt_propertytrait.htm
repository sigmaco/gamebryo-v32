PropertyTrait (Extension-Point)

Emergent.Toolbench.EntityModeling.PropertyTrait

The property trait extension point allows you to create traits which can be specified on a per property basis that your game will interpret.

<extension point="Emergent.Toolbench.EntityModeling.PropertyTrait">
  <property-trait name="TraitName"
                  tool-time="false"
                  default-on="false">
    <description>Trait Description.</description>
  </property-trait>
</extension>

property-trait

Attribute Type Description
name String The unique name of the trait.
tool-time Boolean [optional] [default: false] The tool-time property controls whether the trait is available only for tools.
default-on Boolean [optional] [default: false] The default-on property controls whether the trait is on by default.
