BehaviorTrait (Extension-Point)

Emergent.Toolbench.EntityModeling.BehaviorTrait

The behavior trait extension point allows you to create traits which can be specified on a per behavior basis that your game will interpret.

<extension point="Emergent.Toolbench.EntityModeling.BehaviorTrait">
  <behavior-trait name="Trait Name"
                  default-on="false">
    <description>Trait Description.</description>
  </behavior-trait>
</extension>

behavior-trait

Attribute Type Description
name String The unique name of the trait.
default-on Boolean [optional] [default: false] The default-on property controls whether the trait is on by default.
