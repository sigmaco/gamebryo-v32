PhysX Debug Rendering

PhysX can produce renderable geometry for the purposes of debugging various scene parameters, such as collision shapes and orientation axes (see the PhysX documentation for a full description). Gamebryo-PhysX provides support for rendering this geometry inside Gamebryo.

To enable this feature, call NiPhysXScene::SetDebugRender on the scene that you wish to visualize. The geometry produced by PhysX is then automatically drawn into the Gamebryo scene. Note that you must enable debug visualization in the PhysX SDK separately.

Debug geometry has a significant negative performance impact, particularly on hardware scenes or when large numbers of actors are involved.
