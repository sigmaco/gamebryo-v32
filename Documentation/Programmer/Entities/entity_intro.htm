Entities: Introduction

The game logic is decomposed into Entities, also known as game objects. The designer decides which aspects of the world will be represented, and how to represent the behavior of the objects.

Note

The Game Framework entity system is a replacement for the earlier NiEntity library, adding new features and integration with the Entity Modeling Tool and World Builder.
