Modifying custom message types

You can modify an existing custom message type if the use of the custom message needs to change or expand. A common example of the use of a message changing would be a message going from being just a notification to needing to have a payload, or the payload of a message changing.

To modify a custom message type

A custom application specific message requires only that you modify the ToStream and FromStream methods and then set any additional variables in the message before sending and get any additional variables in the message handler (see Defining application-specific messages).

Changing the contents of a StreamMessage requires you to find all the places it is streamed into and out of.
