Proxy Service Overview

Overview

The proxy service (egmVisualizers::ProxyService) is a service used by tools to manage graphic representations of entities which would not be represented graphically in a running game (for instance, a light or camera). This service acts on entities which have 3DProxy or 2DProxy as a mix-in and adds the proxy asset to the scene graph for visualization. This visual representation can be selected and manipulated by the user (translated or rotated) much like a normal entity.

Entity Handling

The proxy service handles entity discovery/removed messages, adding/removing the appropriate proxy asset to the scene graph respectively. The service also handles entity update messages, updating the position, rotation, scale, and visibility of the proxy in response to entity changes.
