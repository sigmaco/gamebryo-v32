The egmWhitebox Library

The egmWhitebox::WhiteboxService class is the runtime component of the whitebox tools that exist in World Builder. When an entity is created that mixes in the Whitebox model, the service creates a scenegraph containing mesh geometry and associates that scenegraph with the entity via the SceneGraphService. The service also listens for property changes for those entities and modifies their scene graphs to reflect the new property values.

The primary purpose of the Whitebox service is to enable designers to quickly create shapes that represent an environment and interact with that environment in game. It is expected that the Whitebox service is removed prior to shipping a game because all whitebox entities have been replaced with actual art assets.

The WhiteboxService doesn't have any relevant public interfaces or built-in functions. The only method of interaction with Whiteboxes is through the entity properties. For more information on authoring whiteboxes, see Working with Whiteboxes.
