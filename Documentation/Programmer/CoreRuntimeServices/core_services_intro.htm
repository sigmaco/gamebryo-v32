Core Runtime Services: Introduction

Core Runtime Services wrap Core Runtime functionality, providing Game Framework objects such as services and scripts with access to Core Runtime objects and data. For example, the Animation Service in the egmAnimation library provides a service that manages Core Runtime animation assets and objects. It also provides handling of Entities with animation related properties, and script access to C++ functionality. Additional services are provided for rendering, scene graph management, input, terrain, and many other modules of the Core Runtime.

This section of the documentation is a guide to the services, including a high level description of their functionality and techniques for programming with them. Summary information and links to the API reference can be found in the Quick Reference To Services section.
