Core Runtime Functionality Provided by Services

The following table summarizes the relationship between various Core Runtime libraries or systems and the Core Runtime Services. It is constructed from the perspective of a user familiar with the Core Runtime.

Core Runtime System Core Runtime Library (primary) Core Runtime Service Functionality Provided
Scene Graphs NiMain ecr::SceneGraphService
  • Asset management of NIF files and the scene graphs within them
  • Updating of Scene Graphs
  • Modifications to scene graphs, such as attaching one to another, or adjusting properties based on entity properties
  • Access to scene graphs for other services and the application
  • Script API for scene graphs (see bapiCoreRuntime)
Rendering NiMain and platform-specific renderer libraries ecr::RenderService
  • Configuration, creation and management of the NiRenderer-derived object
  • Rendering the scene on each tick
  • Management of render targets
  • Access to rendering objects for other services
Cameras NiMain ecr::CameraService
  • Creation and updating of NiCamera objects
  • Access to cameras for other services
  • Static functions previously found in the NiViewMath library.
Picking NiCollision ecr::PickService
  • Built-ins for picking from script and other services.
Lighting and Shadows NiMain ecr::LightService
  • Creation and updating of NiLight-derived objects
  • Management of area of effect and other lighting controls in response to changes to entities
  • Management of the shadowing system and the properties on entities that control shadows
Shader System NiMain and various platform-specific shader libraries ecr::ShaderService
  • Loading and management of shader assets
Input System NiInput ecrInput::InputService
  • Handling of user input from keyboard and mouse on PC, and console controllers
Animation System NiAnimation egmAnimation::AnimationService
  • Loading and management of actor managers bound to entities.
PhysX NiPhysX NiPhysX ecrPhysX::PropService and related services
  • Loading and simulation of PhysX assets streamed in NIF files.
Terrain NiTerrain egmTerrain::TerrainService
  • Loading and management of terrain objects
PhysX Terrain NiTerrainPhysX egmPhysXTerrain::PhysXTerrainService
  • Loading and simulation of PhysX data associated with terrain objects
Visual Tracker NiVisualTracker egmVisualTracker::VisualTrackerService
  • On-screen visual trackers for various metrics such as service tick durations or the frame rate
Geometry NiMain egmWhitebox::WhiteboxService
  • Creation of box meshes for whitebox entities.
Geometry NiMain NiPhysX egmPhysXWhitebox::PhysXWhiteboxService
  • Creation of box meshes for Whitebox entities and box rigid bodies for PhysXWhitebox entities.
