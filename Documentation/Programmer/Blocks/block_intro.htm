Block Files: Introduction

Block files are used to effeciently create sets of entities using a data driven method. Block files are created in World Builder inside of Toolbench and are exported as .xblock files for use by the run-time. This documentation section discusses the run-time usage of block files, if you are interested in the tools side of block creation see World Builder: Introduction.
