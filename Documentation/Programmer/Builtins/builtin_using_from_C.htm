Using Built-in API Functions from C++

Built-in API functions are primarily intended for use from scripted behaviors, but it is valid to call these functions from C++ behaviors as well. It is not, however, valid to call built-in API functions from outside of the context of a running behavior as many built-in API functions use the current behavior context to figure out details like which entity should be used. Most built-in API functions are simply script-friendly wrappers around existing system service APIs which you would normally just call directly when using them from C++ code. In fact built-in API functions are often used specifically to limit what functionality from a system service is exposed to script to a safer subset of the total functionality.
