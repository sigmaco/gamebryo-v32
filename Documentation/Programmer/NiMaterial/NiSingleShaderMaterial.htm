NiSingleShaderMaterial

This material class is designed to easy wrap the application of an NiShader to a piece of geometry. The primary mechanism for creation of NiSingleShaderMaterials is the NiSingleShaderMaterialLibrary, which searches through all NiShaderLibrary objects in the NiShaderFactory.

The NiSingleShaderMaterial's GetCurrentShader method retrieves the NiShader matching the implementation defined in the NiMaterialInstance's extra data field from the NiShaderFactory. The shader is initialized if needed.
