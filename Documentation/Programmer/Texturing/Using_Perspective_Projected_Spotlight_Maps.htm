Using Perspective Projected Spotlight Maps

Effect

A perspective spotlight map creates an effect of point light source shining through a hole, scrim, photographic slide, window, etc. The light rays all meet at the center of the effect—the projected effect changes size with distance from the source.

Filter

Any filter mode may be used.

Clamping

Slide projector effects require CLAMP_S_CLAMP_T mode, for the same reasons as it was used in the parallel case.

Effect mode

PROJECTED_LIGHT

Texture coordinate mode

WORLD_PERSPECTIVE

Image

The image used may be any image that the application wishes to project into the scene, with one important restriction. The edge pixels of the image (left, right, top, bottom) must all be black. If the image itself is stretched to the edge of the texture, then clamped texture mode will cause the pixels at the edges to be "smeared" out past the boundaries of the desired projection frustum. The black edges ensure that any objects outside of the projection frustum of the texture that receive the projected texture will not be illuminated by the effect.

Transforms

Simply use the matrix described in the above section "Perspective Matrices".
