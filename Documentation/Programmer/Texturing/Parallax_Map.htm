Parallax Map

The ParallaxMap is an RGB map that defines the perceived heightfield applied to the surface of a geometric object. This heightfield can be controlled by the NiTexturingProperty::ParallaxMap bias. Rather than creating micro-triangles, as displacement mapping does, parallax mapping will perturb the UV coordinates of all textures sharing the same UV set as the parallax map. This perturbation gives the impression that the object has height from the texture, but in fact is an optical illusion.
