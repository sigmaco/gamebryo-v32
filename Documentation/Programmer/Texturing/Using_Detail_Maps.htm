Using Detail Maps

Image: An image representing the fine-scale surface texture of the object. For example, in the case of a brick wall, the base texture would represent the brick & mortar colors (for example, red and beige), while the Detail Map would represent the fine-scale bumps and cracks in the mortar and bricks.

Filter: A mipmapped mode is required, as the effect should blur into a solid color, from a distance. If a non-mipmapped mode is used, the effect will "shimmer" badly in the distance as the camera moves.

Clamping: Detail maps use WRAP_S_WRAP_T, as they are almost always mapped many times across a surface.

Texture Coordinates: Generally some large multiple of the base texture coordinates, often 32-64 times the base texture coordinates.
