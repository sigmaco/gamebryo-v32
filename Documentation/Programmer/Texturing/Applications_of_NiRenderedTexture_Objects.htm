Applications of NiRenderedTexture Objects

Applications use NiRenderedTextures to represent image data that is generated by rendering a scene. For example, an application might use an NiRenderedTexture to represent the shadow cast by a character. In some cases, they might represent the image coming from a closed-circuit television camera (which is then textured into an object in the main scene). Or, an application might render its main scene to a large texture, and then render that texture to the screen, applied to a polygon mesh. The resulting mesh can be deformed, allowing the application to warp the scene on the fly for "heat rippling", "drunken view", or other effects/tricks.

NiRenderedTexture effects are generally more complex than basic texturing effects, as they require (in most cases) two rendering operations per frame; one to update the texture by rendering to it, and then a second rendering to render the main frame that includes the texture.
