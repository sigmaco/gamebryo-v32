Multitexture Hardware 3D Support

Applications wishing to use multitexture should be prepared for a wide range of single-pass multitexture support levels ranging from 2 up to 16 texture multitexturing. The following sections on multitexturing serve as an introduction to the use of multitexturing in Gamebryo, as well as a set of guidelines. Applications should follow these to ensure consistent application behavior, appearance, and performance on the widest range of display hardware.
