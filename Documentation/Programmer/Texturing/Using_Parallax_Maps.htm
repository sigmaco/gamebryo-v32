Using Parallax Maps

  • Image: The image used is a grayscale heightfield wherein black is coincident with the triangle surface and white is a scaled height above the surface. This map has a similar visual effect to a displacement map, but it accomplishes the height changes by perturbing UV coordinate look-ups as if they were on a more tessellated surface. This effect can break down when the eye is almost tangent to the surface of the triangle. An NBT frame is required in the geometry for parallax maps to be functional, as it is necessary to map from triangle space to UV space.
  • Filter: Any mode is reasonable, dependent upon the desired effect.
  • Clamping: Completely dependent upon the effect.
  • Texture Coordinates: Completely dependent upon the desired effect. However, another texture must share the same UV set as the parallax map in order to be influenced by it. Texture transforms can cause the UV coordinates to no longer be shared between maps.
