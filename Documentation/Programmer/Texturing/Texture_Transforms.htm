Texture Transforms

Both WORLD_PARALLEL and WORLD_PERSPECTIVE mappings use a user-supplied texture coordinate transform. Although this may seem daunting at first, the following sections describe how applications can use this powerful feature to map textures into scene in a wide variety of ways. In general, the default texture coordinate transforms will almost certainly not produce the desired effect for any form of WORLD_PARALLEL or WORLD_PERSPECTIVE texture effect. Note that projected effects generated in the Gamebryo 3ds max Plug-in or the Gamebryo Maya Plug-in set their own matrices, so the application need not worry about these.

The following sections detail the way the texture transforms are used, give examples of transforms for numerous common effects, and explain how these effects can be extended.
