Preparing to Ship Your PC Title

Redistributable requirements:

  • DirectX Redistributables. The title installer must package the corresponding redistributable for the DirectX SDK version that was used to compile Gamebryo and the title. More recent or older versions may not be include the appropriate DLLs (specifically for the release versioned D3DX libraries). For more information on the DirectX SDK used to compile this version of Gamebryo, see the DirectX Support page. For more information on Microsoft's redistributable policy for DirectX, please see DirectX Installation for Game Developers (msdn.com).
