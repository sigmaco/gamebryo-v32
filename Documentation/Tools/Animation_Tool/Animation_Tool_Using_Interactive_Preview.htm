Using the Interactive Preview in the Animation Tool

You can use the Animation Tool to preview transitions between animations in a free-form environment.

To use the interactive preview

  1. Launch the Animation Tool and load a .KFM file that you want to work with. For example:

    %EMERGENT_PATH%\Samples\Models\AnimationToolExamples\Tutorial\Tutorial2.kfm

  2. Select Interactive Preview in the Project View. (You can also select this from the main toolbar.)

    The application enters Interactive Preview mode. You can see this by examining the Edit View.

    Note that nothing is animating. This is because the bind_pose sequence is the current Idle Sequence, and it has no animations. The Idle Sequence is the default sequence that the Interactive Preview tries to play when no other sequence is active. Select a different sequence. For example, if you are working with the Tutorial2.kfm sample, select idle01_front from the Idle Sequence drop-down list. The idle01_front sequence starts animating.

  3. There are two ways to preview sequences and their transitions in the Interactive Preview. The first way is to click the sequence that you want in the Current Sequence list. For example, if you are working with the Tutorial2.kfm sample, click the walk sequence. Notice that the character transitions from theidle01_front sequence to the walk sequence. The engine will continue to play that sequence until you select a different item from the list. If no transition is specified from the currently animating sequence to another sequence in the list, that sequence will be grayed out in the list and you will not be able to select it.

    Change the Current Sequence back to its previous state. If you are working with the Tutorial2.kfm sample, select idle01_front for now.

  4. The second method of activating sequences in Interactive Preview is through keyboard shortcuts. Open the Keystroke Settings rollout. This rollout contains all of the currently enabled keyboard shortcuts for the Interactive Preview. If you are working with the Tutorial2.kfm sample, this rollout is currently empty.
  5. If necessary, drag the size bar at bottom of the dialog to fit all the entries on the screen without needing a scroll bar. The size bar is the grey rectangle at the bottom of the grid. (Many controls in the Animation Tool contain size bars like this that let you resize their contents to suit your preferences.
  6. Double-click the sequence entry to which you want to assign a keystroke. Then, press the keyboard key that you want to assign to the sequence. For example, if you are working with the Tutorial2.kfm sample, press the Up arrow key on the keyboard. This changes the keystroke assigned to the walk sequence in the chart to Up.

    Assign other keystrokes as needed. If you are working with the Tutorial2.kfm sample, double-click the run sequence, hold down the Shift key, and press the Up arrow key. This changes the keystroke assigned to run in the chart to Shift and Up simultaneously. The Animation Tool detects Shift, Ctrl, and Alt as modifier keys when you hold them down while specifying a keystroke.

  7. Click either the Render View or the Edit View. Keyboard shortcuts only work in these two views.
  8. Test the keystrokes that you assigned. For example, if you are working with the Tutorial2.kfm sample, press the Up arrow key. The Interactive Preview transitions from idle01_front to walk. When you release the Up arrow key, the Interactive Preview returns to the idle01_front sequence. Hold down the Up arrow key again. This activates the walk sequence. Now hold down the Shift key without releasing the Up arrow key. This triggers the transition from walk to run.
  9. Save the current project.
