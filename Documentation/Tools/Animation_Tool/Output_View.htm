Output View

You can use the Output View in the Animation Tool to review useful messages about what the application is currently doing, warnings that the application has encountered, and other context sensitive feedback.

To display this view, select View > Output View or click the Output View button on the main toolbar.

Output View in the Animation Tool

Output is provided in several channels, such as General, Warning, or Events. Output entries are color-coded per-channel. You can use the drop-down selector at the top of the view to filter which output channel entries are visible.
