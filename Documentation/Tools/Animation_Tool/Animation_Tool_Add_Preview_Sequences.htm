Adding and previewing sequences in the Animation Tool

You can use the Animation Tool to add and preview sequences.

To add and preview sequences

  1. Launch the Gamebryo Animation Tool.
  2. Load a character model by selecting Character Model in the Project View. There are two components to the character model: the NIF file associated with the character; and the node in the scene graph that is the root of the animation hierarchy. Because neither of these have been set up yet, both fields state that the values are <UNDEFINED>. The Edit View should now look like the example below.
  3. Click the Select File button in the Edit View. In the Open Character NIF dialog box, navigate to and double-click the NIF file associated with the character. For example:

    %EMERGENT_PATH%\Samples\Models\AnimationToolExamples\Killbot\Killbot Default_Pipeline\Killbot.nif

  4. Once you have selected the file that is associated with the character, you need to select the root of the animation hierarchy. If you have set up your animations in a content creation package like 3ds max or Maya, this node will be the same node that was specified as the actor root in those tools. In most cases, this will be the parent node to the node with "NonAccum" in the name. In the Select Character Root dialog box, expand the scene graph and select the node named Sequence node and then click OK.
  5. The Edit View should now look like the image below. You can hover your mouse over the killbot.nif filename to view the full path to the file within a ToolTip.
  6. In the Project View, click Sequences in the tree. The Edit View should now match the image below. Click the Add Sequences from KF File(s) button in the Edit View. (For convenience, this can also be done from the main toolbar or by right-clicking Sequences in the Project View.)
  7. In the Open KF File(s) dialog box, choose a .KF file that contains the sequences that you want to work with. For example:

    %EMERGENT_PATH%\Samples\Models\AnimationToolExamples\Killbot\Killbot Default_Pipeline\Killbot.kf

    The sequences will now be visible in the Project View, and, if you selected the Killbot.kf example, you will also see default transitions created between the sequences.

  8. Select a sequence from the Project View. The Render View updates to reflect your sequence selection. For example, if you are viewing the samples that we've previously mentioned, you can select walk from the Project View.

    Notice that a time slider has appeared at the bottom of the Render View. The time slider has several usages in the Animation Tool. It displays the current position in the sequence as the diamond shaped bar. The downward facing arrows indicate the location of the text keys in the sequence. Moving the mouse over a text key marker will display the text of the key as well as the exact time at which it occurs in a ToolTip. The animation playback controls are located to the right of the time slider.

  9. Click the Pause button. You can drag the current position marker on the time slider. The playback will change to match the time slider. The buttons on either side of the time slider allow you to step time one frame.
  10. Click the Play button. If you want to see animation play back faster or slower, drag the slider at the bottom right corner of the Render View.
  11. Review sequence properties, evaluators, and text keys in the Edit View. For example, in the image below, the Bone Pelvis evaluator is expanded to show information about the keys used by that evaluator. The Sequence Text Keys rollout provides information about the various text keys that occur in the sequence.
  12. Select File > Save As to save your project.

    KFM files use relative paths to reference the NIF and KF files that you have selected, so you can save the KFM file anywhere you want and it will be able to load the NIF and KF files when you open the file again. However, you cannot move the KFM file to a different location and expect it to still be able to find the NIF and KF files. In this case, the Animation Tool will prompt you to find these files when it loads the KFM file.

You have now added and previewed a sequence and you have saved your project so you can return to it later.
