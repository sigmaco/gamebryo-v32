Working with physical forces in the Animation Tool

Within the Animation Tool's Physics View, you can specify forces that you want to exert on objects while they are simulating. The forces are saved and re-applied every time the simulation restarts. This enables you to apply a force and then change the view to see from various angles how the object behaves under the force.

After you click the Apply Forces button in the Physics View, you can use your mouse to apply forces:

  • Left Button applies forces parallel to the screen.
  • Middle Button applies forces into and out of the screen.

In the Render View, click an object and drag to apply the force. The force is applied at the point under the mouse when the button is first depressed. If there is no dynamic or cloth object under the mouse, no force is applied.

The Animation Tool estimates how much force to exert based primarily on the mass of the object to which force is applied. If the force is too strong or too weak, use the Strength combo box to modify the magnitude. Each unit increment doubles or halves the effective force.

Remember, forces are saved and re-applied each time the animation resets. This is so that you can see the result of forces from a different view which may not be perpendicular or parallel to the original force direction. Forces might not give exactly the same result each time the simulation is reset because PhysX is sensitive to things like the frame rate. Click Clear button to remove any saved forces.
