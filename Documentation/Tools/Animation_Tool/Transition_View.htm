Transitions View

The Transitions View in the Animation Tool consists of a matrix of all the possible transition combinations that can exist in the system. Further, it displays these transitions in such a way that the user has a quick overview of all the various types of transitions that currently exist or have not yet been defined. The Transitions View operates as a sort of extension to the Project View in that it allows the user to select transitions that will populate the Edit View and the Render View. The grid is structured such that the first column (or row headers) lists all the source sequences while the first row (or column headers) list the destination sequences.

To display this view, select View > Transitions View or click the Transitions View button on the main toolbar.

Transitions View in the Animation Tool

Each cell in the Transitions View displays text indicating the type of transition (if it has one). Each cell is color-coded based on the transition type. This color-coding enables you to quickly interpret the matrix without having to read the text of individual cells. The color that is used for each transition type is user-customizable. To change the color associations, simply right-click anywhere on the matrix and choose Edit Table Settings. To change the transition type of a single cell, double-click the cell and choose the desired transition type from the dialog that appears. Alternately, you can change the transition type of a single or multiple cells by right-clicking and choosing one of the available Convert To options or by pressing the appropriate hot-key (listed next to each item on the context menu). You can delete single or multiple transitions by right-clicking and choosing Delete or by pressing the Delete key.
