Edit View: Sequence Groups

The actions that you can perform in the Edit View change depending on the type of object that is active in the Animation Tool. When you select a sequence group, you can use the Edit View to edit and test the parameter settings for the selected sequence group.

In the sequence group context, the Edit View contains two roll-outs.

  • Sequence Group General—contains general information about the sequence group, including its ID and name.
  • Sequence Group Properties
    • Active Status of Sequences—In this section, you can review a list of the sequences contained in the sequence group, and you can review or modify sequences' weights.
    • Properties—In this section, you can review and modify sequence group properties, including the following:
      • Synchronize To—Select a sequence to synchronize to; or, if no synchronization is necessary, you can leave the field blank.
      • Priority
      • Weight
      • Ease-In and Ease-Out
      • Additive—If checked, this sequence will be blended additively.
      • Key Assignment—This is the key that can be used to activate and deactivate the sequence within the Sequence Group. This value can be changed in the Keystroke Settings.

You can also assign keystrokes in this editing context.
