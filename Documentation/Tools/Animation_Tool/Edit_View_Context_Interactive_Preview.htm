Edit View: Interactive Preview

The actions that you can perform in the Edit View change depending on the type of object that is active in the Animation Tool. If you select the Interactive Preview mode, you can use the Edit View to activate sequences via keyboard or mouse input. The appropriate transitions between each sequence will occur automatically.

In the Interactive Preview context, the Edit View contains two roll-outs.

  • Interactive Preview—In this roll-out, you can select the idle sequence and you can activate available sequences. Sequences that cannot be transitioned to from the current sequence are grayed out in the Current Sequences list. The Idle Sequence is the sequence that is activated when no keyboard keys are pressed.
  • Keystroke Settings—In this roll-out you can review, edit, and assign keystrokes to selected sequences. When you enter a keystroke, the input focus moves to the next keystroke field in the list. This enables you to rapidly modify all settings. At any time, you can press the ESC or ENTER key or click elsewhere with the mouse to remove focus from the field. Note that for number pad keys, NUMLOCK affects which key is generated by the Windows OS. With NUMLOCK on, SHIFT + NUMPAD8 will generate the UP key. If NUMLOCK is off, SHIFT + NUMPAD8 will generate the SHIFT + UP keys.
