Saving custom workspaces in the Animation Tool

You can customize the layout of the Animation Tool by resizing and repositioning various components of its interface. You can save these custom settings to use whenever you run the Animation Tool.

To save your custom workspace settings

  1. Launch the Animation Tool.

    By default, the Animation Tool's various views display in the locations shown in the example above.

  2. Customize the screen layout to match your preferences. You can do the following:
    • Show or hide any of the views via the View menu or main toolbar.
    • Resize the dimensions of the views by dragging their borders.
    • Dock or un-dock (float) views within the parent application window by double-clicking the view title bars.
    • Dock multiple views in tabbed views.

    In the following example, the Output View is now visible and all of the views have been grouped and are available as tabs.

  3. When you've arranged the interface components in a way that suits your needs, save the workspace by selecting Edit > Docking Management > Save Dock Configuration.

You have now saved the settings in the Animation Tool workspace. When you next launch the application, all of the screen elements will be sized and located in the way that you prefer.

You can reset the application to its defaults at any time by selecting Edit > Docking Management > Reset Dock Configuration. You will need to close and re-open the application to return the application to its defaults.
