ImageGenerator Command-line Usage

At the command line, run ImageGenerator.exe with appropriate command-line options for your scene. For example:

ImageGenerator.exe -nif "C:\My Assets\Scene.nif" -size 128

The following table outlines the command-line parameters for this application:

Command-line Parameter Description
nif The filename of the NIF file to load. This file should be generic or PC-compatible or the rendering will be incorrect.
size The size of the image, which is assumed to be square.
d3d10 [optional] If running on a computer capable of using D3D10, this flag determines whether or not to render using the NiD3D10Renderer.
d3d11 [optional] If running on a computer capable of using D3D11, this flag determines whether or not to render using the ecr::D3D11Renderer.
CoordType [optional] One of three coordinate systems used to auto-generate the default perspective. Possible values: "MAX", "MAYA", "XSI". If not specified, MAX is chosen by default.
shaderfiles [optional] Path to the shaders needed to render this NIF file.
dir [optional] Used to create images for all NIFs in a given directory and its subdirectories. Can be used instead of -nif.
camera [optional] The name of the camera to use when rendering. If not specified, a default camera will be constructed.
NodeNames [optional] When using -nif, this allows for multiple images to be generated (one per node).

Note

This tool is meant to be called from external applications and therefore does not generate a console window.
