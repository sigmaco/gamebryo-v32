NifViewer Command-line Usage

Typically, NifViewer is launched on the console by right clicking a .NIF file and selecting the target console from the Windows context menu. If you prefer to launch the NifViewer manually, you can do so. You can specify arguments at the command-line, or you can specify them in a file called cmdline.txt. A version of this file is located in the following directory:

%EMERGENT_PATH%\Source\Tools\NifViewer\Data

Arguments that you specify directly on the command-line will override commands in the cmdline.txt file.

Command Line Arguments

Acceptable command line arguments are as follows:

-bg fRed,fGreen,fBlue

Specifies the background color. Valid values for each color component are [0.0..1.0]. All three colors must be specified if this flag is used. The default background color is (0.5,0.5,0.5).

-near fNear

-far fFar

These specify settings for the near and far clipping planes. One or both may be specified.

file.nif

Specifies a NIF file to open. This must always be specified and should be the last argument.
