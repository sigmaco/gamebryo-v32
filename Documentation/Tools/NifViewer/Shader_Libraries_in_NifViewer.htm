Shader Libraries in NifViewer

The NifViewer supports different types of custom shaders depending on the platform. You can also modify the NifViewer to add support for your own custom shader libraries. The custom shaders supported by default are listed for each platform below in the order in which they are initialized.

Xbox 360 Shader Libraries

  1. NSFParserLib
  2. NSBShaderLib
  3. NiD3DXEffectShaderLib
  4. NiFXLShaderLib

PS3 Shader Libraries

  1. NSFParserLib
  2. NSBShaderLib
