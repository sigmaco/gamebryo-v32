Viewing information about local variables in the Script Debugger

When you are at a break in script execution, you can use the Locals View to view information about all local variables that are accessible at the current script context. For each local variable, the information includes the variable's:

  • name
  • type
  • value

For dynamically typed languages like Lua, the type of a variable may change over the course of its lifetime. Although complex data types like Lua tables are only summarized in the Locals view, you can use the Watches View to access them.
