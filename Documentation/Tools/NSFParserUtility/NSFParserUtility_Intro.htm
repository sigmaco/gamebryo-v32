NSFParserUtility: Introduction

The NSFParserUtility (NSFParserUtility.exe) is an application that batch converts NSF files to binary NSB files. The application operates on a directory level with optional recursion into subdirectories.

You can find this application in the following location:

%EMERGENT_PATH%\Source\Tools\DeveloperTools\NSFParserUtility\Win32\VC80
