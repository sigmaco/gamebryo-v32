NSFParserUtility Interface Options

You can use the NSFParser Utility to batch convert NSF files to binary NSB files.

NSFParser Utility

Use this field... To...
Folder Name Specify the folder that contains the NSF files that you want to convert. You can type the full path, or you can browse to the target folder.
Recurse Subfolders Have the application convert any NSF files within subfolders inside the target folder.
Run Start the conversion process.
Status Review the status of the conversion process.
Clear Status Clear the messages within the Status window. This command can be especially useful when you run the parser on several folders in a single run.
Exit Exit the application. (You can also exit by pressing the ESC key.)
