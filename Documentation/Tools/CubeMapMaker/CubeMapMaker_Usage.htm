CubeMapMaker Command-line Usage

At the command line, run CubeMapMaker.exe with appropriate command-line options for your scene. For example:

CubeMapMaker.exe -nif "C:\My Assets\Scene.nif" -node MyNode01 -size 128

If you invoke the CubeMapMaker from the command-line with no arguments it will display the following help screen:

While the tool is running, a CubeMapMaker item will appear on the Windows task bar. Depending on the scene's complexity, it may take the tool a few seconds to create the cube map and save the TGA image files. When the tool is finished, it will either display a message indicating the location of the cube map images or an error message indicating why a cube map could not be generated.

Note

The GamebryoMaterial node in XSI requires DDS files to be used for its Environment Map texture input. Additional tools must be used to convert 6 TGA image files into a single DDS cubemap for use with the GamebryoMaterial node in XSI, such as the NVIDIA Photoshop plug-in.
