CubeMapMaker: Introduction

You can use the CubeMapMaker command-line tool (CubeMapMaker.exe) to generate 6 Targa (TGA) format cube maps rendered by Gamebryo. You need to export the scene that a cube map will be generated for to a NIF file first. The tool loads the NIF file and generates a cube map that is centered on a named node in the scene that you specify.

You can find this application in the following location:

%EMERGENT_PATH%\Source\Tools\DeveloperTools\CubeMapMaker\Win32\<VC80 or VC90>
