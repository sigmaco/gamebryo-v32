DDE Header Generator: Introduction

You can use the DDE Header Generator (DDEHeaderGenRelease.exe) to generate data driven enumeration files, which provide a way to generate the many different constants that are used as identifiers in the system. By using this tool, you can ensure that there will never be an id conflict if the *.enum files are setup properly.

The *.enum files are an XML-based format. During the build process, the executable runs, parses the *.enum files, and creates proper header files for the rest of the project to include.

You can find this application in the following location:

%EMERGENT_PATH%\Source\Tools\DDEHeaderGen\bin

You can specify the value range for the IDs that the enum file will generate.
