Tools

In addition to its plug-ins for major digital content creation (DCC) applications, Gamebryo LightSpeed also includes a number of tools that you can use to fine-tune and streamline the creation of game assets. Many of these tools are command-line utilities that help you debug and optimize.

Tool details are outlined in the following sections:

  • Toolbench
    • Entity Modeling Tool
    • World Builder
    • Script Debugger
    • Simulation Debugger
  • Asset Viewer
  • Animation Tool
  • COLLADA to NSF Converter
  • CubeMapMaker
  • DDE Header Generator
  • DynamicCgCompiler
  • EEDeploy
  • ImageGenerator
  • MeshProfileCompiler
  • NiThumbnailExtractor
  • Scene Graph Printer
  • PhysX Scene Graph Printer
  • Tool Plug-in Testbed
  • Tool Plug-in Batch Tool
  • NifViewer
  • NiFontCreator
  • NiMaterialNodeXMLLibraryParser
  • NSFParser Utility
  • PhysX NIF Viewer
