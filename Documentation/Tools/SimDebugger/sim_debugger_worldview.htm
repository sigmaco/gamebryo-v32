Simulation Debugger World View

The World View is the primary view of the .simdebug session document. It is a top-down 2D visualization of the game world based on the state of the entity system.

This view is useful for getting a high level picture of game state and can show entity interactions in areas that may not normally be visible through an in-game camera.

Actor-based entities are displayed as circles with an approximate facing indicator. Mesh-based entities are displayed as squares. Other entities, including lights and triggers, are displayed as diamonds.

The view can be panned by holding and dragging the middle mouse button. Zoom is controlled by the mouse wheel or customizable keyboard shortcuts Ctrl+PageUp and Ctrl+PageDown.

During live debugging, the current game time is displayed in the upper left corner. The world coordinates under the mouse pointer are displayed in the lower left corner, along with an indication of the current grid scale.

Entities can be selected in the World View by clicking their icons. The selected entity is highlighted with a yellow square and determines the contents of the Entity View and Events View.

Shortly after connecting to a running game, a backdrop image will be generated for display in the World View. This image includes all static entities in the game at the time it is generated and provides additional context for the tool UI. The backdrop image is saved alongside the session document and will be reloaded when a session is reviewed offline.

When connected to a running game, double-clicking in the World View will display a rendered image of the game state at that location. The image is inset in the upper-left corner and can be dismissed by clicking anywhere in the World View.

The toolbars provide several buttons for filtering what information is displayed in the World View. There is also a button for reseting the view.

Icon Name Details
Toggle Backdrop Toggles display of the world backdrop image.
Toggle Heat Map Toggles display of the actor position heat map.
Toggle Grid Toggles display of the grid.
Toggle Entities Toggles display of entity icons.
Toggle Entity Summaries Toggles display of entity summary text.
Refresh Backdrop Requests an updated backdrop image from the connected game.
Reset View Centers the view at the origin with default zoom.

The Simulation Debugger accumulates position records for actor-based entities. This data can be visualized as a heat map by toggling on the appropriate filter in the toolbar. The collected data is continually re-normalized as game time progresses.

Warm colors indicate areas of greater activity and cool colors indicate less active regions. The granularity of the displayed data changes with World View zoom level. The size of the database backing the visualization can be tuned via user settings.

The World View is also useful for playback of Simulation Debugger sessions.
