Saving and Loading the Terrain

To save changes made to the terrain, click the 'Save' or 'Save All' icon in the toolbar, or choose those same options from the 'File' menu. Additionally, certain actions may cause terrain changes to be saved to disk, such as unloading a sector. In this case, a message window will provide a warning that saving is about to occur with the option to cancel.(See Unloading a sector.)

A prerequisite to loading a terrain is having a game solution created and opened. Open the game solution view from the menu option 'View'->'Game Solution'. Then, double-click on the .block file to open the block file and thus load the terrain. In order to edit a loaded terrain be sure to select the 'Terrain Edit gizmo' from the toolbar options.
