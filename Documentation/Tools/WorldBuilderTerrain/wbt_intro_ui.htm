Terrain User Interface

Terrain Menu

The 'Terrain Menu' options allow a user to create a new terrain, view terrain properties, or snap selected entities to the terrain.

Figure 1: Terrain menu

Terrain Material Editor

The Terrain Material Editor lists all the materials available to use with the Terrain brush. Here the user can manage the materials.

Figure 2: Terrain Material Editor

Terrain Editor

The Terrain Editor contains brush types and brush elements used for manipulating the terrain.

Figure 3: Terrain Editor

Terrain Edit Gizmo

To enable the ability to edit the terrain, the 'Terrain Edit Gizmo' must be selected from the toolbar options. 

Figure 4: Terrain Edit Gizmo
