Terrain Properties

Right-clicking on the terrain will open a menu with the option 'Terrain properties...'. Selecting this option will open the 'Terrain Properties' window where users can view and modify various properties of the terrain object.

Format

The 'Format' tab in the Terrain Properties window displays a collection of geometry settings, texture settings, and an overall summary of the current terrain object's format parameters. From this tab, users may change the low detail texture size by selecting a new one from the dropdown menu options provided.

Storage

The 'Storage' tab in the Terrain Properties window displays asset binding information, sector initialization settings, and and overall summary of storage sizes for sectors and the terrain archive. From this tab, users may change which sectors are initially loaded for a terrain. For further instructions, please read the section on Configuring the initially loaded sectors.

Low Detail

The 'Low Detail' tab in the Terrain Properties window displays Low Detail Specularity options. The default settings for both specular power and specular intensity are 1.00, but can be edited by typing new values into the textboxes provided.

Materials

The 'Materials' tab in the Terrain Properties window displays information only about materials currently on the terrain.

  • Layer: Displays the layer information which indicates the order the materials were applied to the terrain.
  • Name: Displays material names with the formatting of PackageName.MaterialName.
  • Status: Indicating if a material is loaded or if an error occurred in loading the material.
