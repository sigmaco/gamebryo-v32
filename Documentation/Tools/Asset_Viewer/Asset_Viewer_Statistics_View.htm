Asset Viewer Statistics View

You can use the Asset Viewer's Statistics View to view timing and polygon information about the current scene. The information in this view is especially useful when you need to analyze the performance of a scene.

Use this field... To...
FPS Review the number of frames that are being rendered per second.
Total Objects Review the total number of objects contained in all loaded NIF files.
Total Triangles Review the total number of triangles contained in all loaded NIF files.
Total Vertices Review the total number of vertices contained in all loaded NIF files.
Objects Drawn Review the total number of NiMesh objects that are rendered. This includes the screen element that displays the camera name, which will be listed even though no NIF files have been loaded.
Update Time Review the time it took for all scene graphs to be updated on the most recent frame.
Cull Time Review the time it took for a visible set of objects to be culled from all scene graphs on the most recent frame.
Render Time Review the time it took for all visible objects to be rendered on the most recent frame.
Swap Time Review the time it took for the renderer buffers to be swapped on the most recent frame.
