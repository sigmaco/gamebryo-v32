Asset Viewer Standard Toolbar

You can use the Standard Toolbar to perform common file operations and to view Asset Viewer version information.

Use this field... To...
New file Create a new scene graph, removing any NIF or KFM files that are currently loaded in the Asset Viewer.
Open file Open a NIF or KFM file, replacing any NIF or KFM files that are currently loaded in the Asset Viewer.
Add file Add a NIF or KFM file to the current scene, retaining any NIF or KFM files that are currently loaded in the Asset Viewer.
Remove file Remove a NIF or KFM file from the current scene.
About Display the Asset Viewer About Box.
