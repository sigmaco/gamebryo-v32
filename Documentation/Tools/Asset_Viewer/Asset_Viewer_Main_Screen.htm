Asset Viewer User Interface Overview

The Asset Viewer user interface is divided into eleven principal parts.

  1. Asset Viewer Menus
  2. Asset Viewer Standard Toolbar
  3. Asset Viewer Camera Navigation Toolbar
  4. Asset Viewer Mini-Animation Toolbar
  5. Asset Viewer Docking Views Toolbar
  6. Asset Viewer Collision Lab
  7. Asset Viewer Render View
  8. Asset Viewer Scene Graph Tree View
  9. Asset Viewer Scene Graph List View
  10. Asset Viewer Statistics View
  11. Asset Viewer Animation View
