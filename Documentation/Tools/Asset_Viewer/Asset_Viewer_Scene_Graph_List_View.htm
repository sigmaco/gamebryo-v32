Asset Viewer Scene Graph List View

The Scene Graph List View provides a searchable list of all the objects in the scene, arranged alphabetically by name.

Asset Viewer: Scene Graph List View

Typing a name in the edit box at the top will automatically scroll the list to the object with the closest matching name in the scene. Double-clicking on a list item or pressing "Enter" when a list item is highlighted will bring up the property sheet for that object. See Asset Viewer Property Sheets for more information about object properties. When an object is highlighted in the Scene Graph List View, it is also highlighted in the Scene Graph Tree View.

Clicking the "Options" button or pressing "Ctrl+F6" will bring up the Scene Graph List Options dialog box.

Asset Viewer: Scene Graph List Options

The Scene Graph List Options dialog box allows you to specify standard types of objects to display in the list or specify that only objects of a specific type or that have time controllers, properties, or dynamics effects of that type should be displayed. The dialog box also provides some other options to further narrow down the objects that are displayed in the list.
