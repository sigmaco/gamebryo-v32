Asset Viewer Mini-Animation Toolbar

The Mini-Animation Toolbar provides a simplified version of the animation controls contained in the Animation View.

Use this field... To...
First Frame Set the time slider to the animation's first frame.
Previous Frame Set the time slider to the animation's previous frame.
Play Toggle the animation ON/OFF.
Stop Stop animation playback and reset the time slider to the animation's first frame.
Next Frame Set the time slider to the animation's next frame.
Last Frame Set the time slider to the animation's last frame.
Playback Mode Set the animation playback mode. If you choose Loop, the animation will restart playback when the animation reaches the last frame. If you choose Continuous, the animation will never reset the time to the first frame and will keep playing until you stop the animation.
