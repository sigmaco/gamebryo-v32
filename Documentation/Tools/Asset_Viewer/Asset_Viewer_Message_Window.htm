Asset Viewer Message Window

The message window displays errors, warnings, and information output by Gamebryo during the process of loading and displaying your assets. For example, you can see error messages generated by shader parsers, and can see information about why an error shader is being displayed. Even if you don't have the window enabled, it will appear automatically whenever an error occurs in order to alert you that something went wrong.

Asset Viewer: Message Window

Right-clicking on a message will bring up a popup menu which allows you to copy its text to the clipboard. You can also right-click anywhere in the window to bring-up a pop-up menu that allows you to remove all the messages currently being displayed.
