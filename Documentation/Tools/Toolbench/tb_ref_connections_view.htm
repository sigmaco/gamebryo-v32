Toolbench - Connections View

The Connections view allows you to add, remove, and toggle the state of Asset Controller connections.

To see the Connections view, open the View menu and select Connections.

The top connection is active; the bottom connection is disconnected.

To add a new connection, press the Add Connection button (leftmost on the toolbar), and a dialog box will appear. Complete the dialog box by entering the address and port of the Asset Controller, and a name for the new connection.

To remove an existing connection, select it and then press the Remove Connection button (rightmost on the toolbar).

To connect to an existing Asset Controller connection, right-click it to open its context menu and select Connect.

To disconnect from all Asset Controllers, right-click in the list view and select Global Disconnect.
