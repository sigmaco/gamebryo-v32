Toolbench - Output View

Warnings, non-critical errors, and other messages generated by Toolbench appear in the Output View.

The Output View is shown by default, but if hidden can be restored by opening the View menu and selecting Output View.

The Output View consists of a toolbar and a scrolling list of messages.

Changing Which Message Types Are Shown

Messages are categorized by "channel." Channels are roughly equivalent to severity levels (for instance, there are Debug, Info, Warning, and Error channels), but some features may provide their own logging channels.

Display of the three most common channels, Info, Warnings, and Errors, can be toggled by pressing their respective buttons on the Output View toolbar. When the button appears pressed, its associated channel is shown. The number of messages logged on each channel is displayed with the channel buttons.

Note

Some plug-ins may add additional channels to the toolbar. These buttons function equivalently to the Info, Warnings, and Errors buttons.

Toggles for other channels can be found using the droplist at the end of the row of primary channels (small button between Errors and Filter). Select a channel from the droplist to toggle its status.

Searching, Sorting, and Clearing the View

To search the message list for a string of text, type into the "Filter" textbox. Only those messages containing the text you enter will be shown. To clear the filter, press the X button that appears next to the "Filter" textbox.

To sort messages by type, time, text, or details, click on the appropriate column header. "Type" is the empty button to the far left.

To clear the message log, including messages from channels not currently shown, press the Clear All button on the toolbar.

Use the Auto Scroll state button to automatically keep new messages in view as they arrive.

Interacting with Messages

Some messages perform actions when double-clicked. For example, double-clicking a Lua script compilation error shows you which line caused the error.

To copy a message to the Windows clipboard, right-click it and select Copy.

Extremely long messages may be shortened to improve visibility. To see the entire piece of text, click the "More..." link in the message.
