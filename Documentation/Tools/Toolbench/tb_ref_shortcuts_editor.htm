Toolbench - Shortcuts Editor

The Shortcuts Editor is used to add and change keyboard shortcuts for common actions in Toolbench and its plug-ins.

To access the editor, go to the Utilities menu and select Shortcuts.

Editable actions appear in the central list box with their current shortcuts, grouped by functionality. Most plug-ins expose one or more action sets. Toolbench will resolve collisions between shortcut keys by considering the current context, such the current type of open content.

To change the keyboard shortcut associated with an action or add a new one:

  1. Select the action in the list view. The current shortcut, if any, will appear in the textbox marked "Press keys."
  2. Click the textbox marked "Press keys."
  3. Press the new key combination you wish to assign to the action.
  4. Click the Assign button.

To remove a keyboard shortcut from an action:

  1. Select the action in the list view.
  2. Click the Remove button.

To reset all keyboard shortcuts to their defaults, press the Reset button.

When you are finished making changes, press the OK button. To abandon your changes without saving them, press the Cancel button.

Note

Keyboard shortcuts must include at least Ctrl, Alt, or a function key.
