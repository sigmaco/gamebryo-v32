Working with Toolbench Windows

Toolbench is a made up of a series of windows based on the active document. These windows are tightly integrated to improve workflow and increase overall productivity.

All windows in Toolbench can either be floating or docked into the parent application window. Floating windows "float" above the parent application window and can be moved around and resized. Docked windows can be resized as well by dragging the border of a window. Since the windows are docked, when you enlarge one window, the one beside it will automatically become smaller.

Resizing Toolbench windows

To dock/float a window

  • Double-click the title bar of a window.

The following occurs:

  • If the window was floating, it will now be docked into the parent application window.
  • If the window was docked, it will now be floating.

Note

If a docked window contains tabs, you can double-click one of the tabs to turn it into a floating window.
