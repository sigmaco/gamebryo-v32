Toolbench - Settings Editor

The Settings Editor is used to change various options provided by Toolbench and its plug-ins.

To access the editor, go to the Utilities menu and select Settings.

The editor has a tab for user settings and a tab for each open document that supports settings.

The tree view on the left lists the categories available. Selecting a category displays its settings in the right side of the dialog. All settings fall into one of three types:

  • PerUser settings are stored with your user profile. Changes to these settings affect all documents you edit, but do not carry over to other people.
  • PerContent settings are stored with the document being edited. Changes to these settings do not affect the rest of your Toolbench experience, but other users will see these changes when editing the document.
  • PerUserPerContent settings are stored in separate files paired with the documents they influence. These settings are helpful for restoring user preferences specific to particular content. Many of these settings are managed silently and do not require a document save to persist.

Select a setting from the list in the right side of the dialog to display a description of that setting. Click on the setting's current value to begin editing it.

Some settings may provide other means of editing than simple text. Color settings, for instance, provide a color picker. Such settings will show a button () when you begin editing them. Press the button to select a value using the picker.

To confirm your changes and close the Settings Editor, press the OK button. Press the Cancel button to close the Settings Editor without making any changes.

If you wish to save your changes and see their effects without closing the Settings Editor, press the Apply button.

Note

Changes to some settings may not take effect until Toolbench is restarted. The descriptions for these settings will mention this.

The Restore Default button undoes all changes to the current setting, returning it to the value it had when Toolbench was first installed.

Right-click on an entry in the tree view to bring up a context menu:

  • Reset Value resets the currently selected setting to the value it had when Toolbench was installed.
  • Reset Group undoes all changes to the open page of settings, restoring installed settings. It is the equivalent of the Restore Defaults button. Subpages will not be affected. The changes will not take effect until you press the OK or Apply buttons.
