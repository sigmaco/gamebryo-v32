Undoing and Redoing Actions in Toolbench

Toolbench allows users to inspect and operate on the undo/redo action stack via a few different UI elements. Each of these UI elements presents the action history of the active content, which generally means the document being edited in the center pane of Toolbench. When a document is dirty due to unsaved changes, an asterisk will appear next to the file name in the document tab. As undo and redo operations are performed, the document dirty status may toggle on and off. The action system will manage this status automatically based on the state of the action history.

Simple undo and redo commands are available from the Toolbench Edit menu. They are assigned the traditional shortcut keys by default.

To undo an action, click Edit > Undo or press Ctrl+Z.

To redo an action, click Edit > Redo or press Ctrl+Y.

There are also toolbar buttons that will drop down the action history for batch undo and redo operations. These buttons allow you to select several recent actions to undo or redo in one operation.

Note

You can view a history of actions available for undo/redo by clicking the expand icon beside either of the Undo/Redo toolbar icons.
