Using Lists

Many fields in Toolbench are actually drop-down lists. Lists are different than open entry fields in that you must choose from a series of pre-defined options. To display the list of options, click the arrow to the right of the field.

Example single option drop-down list

Two different kinds of lists exist in Toolbench: single option and multi-option lists. In single-option lists, only one option can be selected at a time. In multi-option lists, check boxes are displayed beside each option allowing you to select one or several options. When more than one option is selected, all option names are displayed in the field.
