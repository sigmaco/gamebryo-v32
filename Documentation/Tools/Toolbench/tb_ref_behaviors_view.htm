Toolbench - Behaviors View

The Behaviors view can be toggled via the Toolbench View menu. It displays a list of all behavior scripts in the open asset web.

This view can be a convenient way to access scripts and manage their revision control status.

The toolstrip at the top of the Behaviors view has controls for organizing and filtering the list of scripts. Scripts can be grouped by folder or in a flat list. A substring filter can also be used to limit the list of scripts.

Double-clicking a script will open it in the internal Toolbench editor and/or an external editor. This preference can be specified via Toolbench user settings under Code Editor | External Code Editor.

The context menu on script items gives explicit access to opening the desired editors. Revision control commands are also present here.

The Behaviors view is intended as a handy supplement to behavior management features inside the Entity Modeling Tool.
