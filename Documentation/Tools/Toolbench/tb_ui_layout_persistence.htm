Toolbench Layout Persistence

Toolbench can automatically remember the size and docking arrangements of views that you use. There are a few settings that control how Toolbench remembers the layout of views. These per-user settings can be found in the User > Toolbench > Docking settings category.

Layout Settings

Toolbench docking and layout settings

The Save Context Sensitive Dock Portions setting controls whether Toolbench saves view layouts on a per-content-type basis. If you would like Toolbench to have a different GUI layout for World Builder .block files than for EMT .emtproj files, make this setting True.

The Save Dock Layout setting controls whether Toolbench saves size and docking information or not. If you do not want Toolbench to remember the size and layout of various views from session to session then set this setting to False.

Restoring the Default Layout

If you want to revert Toolbench to its default layout settings for all tools, use the Restore Default Layout command from the Windows menu.
