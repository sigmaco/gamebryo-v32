Importing Content Libraries Into Your Game Solution

After using the Game Solution Wizard to import common libraries like the Standard Model Library, you may later want to re-import common libraries or import additional libraries.

To import/re-import content libraries to your game solution

  1. From the Toolbench menu bar, click Utilities > Integration Wizard.
  2. Select any content libraries that you want to import.
    • If the library selected depends on other content libraries, those dependent libraries will also be selected.
    • If the library already exists in your asset web, selecting it will allow it to be re-imported.
  3. Click Finish.
  4. If any of the libraries already exist in your asset web, you will be prompted to re-import and potentially backup your existing copies.

The selected content libraries are now added to your game solution. To use them in existing content, add them as references.

Note

The Integration Wizard assumes that all common libraries will be stored at the root of the asset web, which is the recommended location. If you want to use another location for such libraries, you will need to import the files by hand. This may also require fix-ups to asset references in the libraries. Depending on asset policies, asset references can break when content is moved to a new location on disk.
