Adding Content to Revision Control

As you create new content in Toolbench, it is automatically queued to be inserted into your revision control system. You can also explicitly add content to revision control at any time.

To add content to revision control

  1. In Toolbench, right-click the content that you want to add to revision control.
  2. From the context menu, select Revision Control > Add/Edit.
  3. When you are ready to do so, check in your changes. See Checking in Content to Revision Control.

Adding new content in Toolbench may affect related files in revision control. In this case, any related files will be checked-out automatically. For example, if you add a new entity model, the entity model project that contains the new entity model will be checked-out from revision control.
