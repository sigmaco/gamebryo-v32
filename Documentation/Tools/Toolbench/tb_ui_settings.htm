Toolbench Settings

Toolbench has many settings that are used to customize the behavior of Toolbench and the tools that run inside of Toolbench.

To access the Settings dialog, click Utilities > Settings...

The Settings dialog has a tab for user settings and a tab for each open document that supports settings.

In each tab, the settings that you can change are displayed in a tree on the left side of the Settings dialog. You expand each section of the tree to show settings in that category. There are two basic types of settings that you can change:

  • User Settings—These settings specify your preferences for working with Toolbench
    • These settings are stored under C:\Documents and Settings\<current user>\Local Settings\Application Data\Emergent Game Technologies\Toolbench\1.0 (varies by OS)
  • Document Settings—These settings apply to open documents
    • Document settings are embedded in the document file format or stored in a side-by-side file.

User Settings

User settings can be different on a per-user basis. They reflect settings that are personal preferences, such as the font to use in the Code Editor or the size of the toolbar buttons.

Document Settings

Document settings are specific to an open document. Changing a Document setting can modify the current document and requires you to save the document in order to save the settings changes. The settings you can change in this category will differ depending on the type of documents that are currently open.

Modifying Settings

After selecting the category of settings to change in the tree on the left side of the Settings dialog, a property grid is displayed on the right side of the dialog showing you all of the settings in that category. A description of each setting is displayed at the bottom of the Settings dialog.

Note

If you want to see the impact of your settings changes on Toolbench or the current document without exiting the Settings dialog, click the Apply button.
