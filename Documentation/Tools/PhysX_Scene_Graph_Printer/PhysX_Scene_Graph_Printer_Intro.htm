PhysX Scene Graph Printer: Introduction

The PhysX Scene Graph Printer (PhysXSceneGraphPrinter.exe) is a command-line application that prints a text representation of the PhysX-related content (all NiPhysX, NiPhysXParticle and NiPhysXFluid objects) in a NIF file, along with the scene graph.

You can find this application in the following location:

%EMERGENT_PATH%\Source\Tools\DeveloperTools\SceneGraphPrinter\Win32\VC80
