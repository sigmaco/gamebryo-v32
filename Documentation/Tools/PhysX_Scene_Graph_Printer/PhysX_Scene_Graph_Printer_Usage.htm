PhysX Scene Graph Printer Command-line Usage

PhysXSceneGraphPrinter uses the same method as SceneGraphPrinter to dump the scene graph portion of the file. All of the command line options for SceneGraphPrinter can be used with PhysXSceneGraphPrinter to control the output.

PhysXSceneGraphPrinter -in input_filename [-out output_filename] [-details] [-ts integer_tabstop_width] [-mesh] [-meshverbose] [-modifiers] [-ac] [-bs] [-mem] [-prop] [-trans] [-extra] [-notimecontrollers] [-skipnodes]

Required Arguments

-in input_filename

Specifies an input NIF file.

Optional arguments

-details

Show details on PhysX objects in the scene. This option causes all of the parameters of every PhysX object to be dumped. The only information not included is the vertex and mesh information of convex or mesh collision shapes.
