Tool Plug-in Batch Tool: Introduction

The Gamebryo Tool Plug-in Batch Tool (ToolPluginBatch.exe) is a command-line application that is designed to allow Gamebryo Tool Plug-in scripts (and thus their Tool Plug-in components) to be executed from a command-line and applied to data files. You can use the tool to create batch file-processing systems that are capable of a wide range of actions, including:

  • Optimization of existing NIF files (for example, post-export stripification or image format conversion for multiple target platforms)
  • Finalizing NIF files that have not previously had this done
  • Verification/profiling of existing NIF files (for example, printing statistics pulled from a NIF file)
  • Conversion of existing, legacy NIF files to the current NIF version
  • Conversion of non-Gamebryo files to NIF

The tool can itself be invoked from a batch file or other system script (such as Perl), allowing for fast and easy conversion or processing of an entire set of source files.

You can find this application in the following location:

%EMERGENT_PATH%\Source\Tools\DeveloperTools\ToolPluginBatch\Win32\<VC80 or VC90>
