Tool Plug-in Testbed: Introduction

The Gamebryo Tool Plug-in Testbed (ToolPluginTestbed.exe) is a small MFC application that is designed to expedite the Gamebryo tool plug-in development process. You can use the application to create, edit, and run tool plug-in scripts. Because the application is much smaller than 3ds max or Maya, it is easier to use when you are debugging custom tool plug-ins.

You can find this application in the following location:

%EMERGENT_PATH%\Source\Tools\DeveloperTools\ToolPluginTestbed\Win32\<VC80 or VC90>

Note

The Tool Plug-in Testbed always sets the target platform type to NiSystemDesc::RENDERER_GENERIC during the export process.
