Tool Plug-in Testbed Interface Options

You can use the Tool Plug-in Testbed application to create, edit, and run tool plug-in scripts.

Tool Plug-in Testbed Interface

Use this field... To...
Script Directory Specify the directory that contains your tool plug-in scripts.
  • If you have the Gamebryo 3ds Max plug-in installed, you can use its script directory. For example: <Max_install_directory>\plugins\MaxImmerse\Scripts
  • The scripts in the Gamebryo Maya plug-in scripts directory are not compatible with the Tool Plug-in Testbed. Setting the scripts directory to the Maya plug-in scripts directory will result in a process script error.
Note: All of the other fields in the interface are unavailable (grayed) until you specify the script directory.
Step 1: Import the Scene
Select the file to load Browse for the file that you want to import. In the Import File dialog box that displays, the type of files that are available correspond to the import plug-ins that were loaded by the system. When you click the Run button, the appropriate import plug-in for the file's extension will run to load the scene.
Step 2: Process the Scene
Selected Process Script Select the process script that you want to use. This combo box is populated with all the process scripts that were found in the designated scripts directory. You can click the Edit button to open the Script Editor dialog box. You can use this dialog box to edit the script or create new scripts of the selected type.
Step 3: View or Export the Scene
View the Scene Select the script that you want to use to view the scene. You can click the Edit button to open the Script Editor dialog box. You can use this dialog box to edit the script or create new scripts of the selected type.
Export the Scene Browse to the target directory and specify an exported file name. Select the script that you want to use to export the scene. You can click the Edit button to open the Script Editor dialog box. You can use this dialog box to edit the script or create new scripts of the selected type.
Run Run the application.
Exit Exit the application.
