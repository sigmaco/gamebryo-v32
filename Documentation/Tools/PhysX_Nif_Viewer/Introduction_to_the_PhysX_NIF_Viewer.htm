PhysX NIF Viewer: Introduction

The PhysX NIF Viewer is a cross-platform tool that you can use to preview Gamebryo PhysX assets. It enables you to load a scene, examine it, play the physics and animation, and view debug geometry. You can launch the tool from 3ds Max or Maya as a Gamebryo viewer, or you can launch it alone.

A basic force application interface is also provided to test the response of objects to external forces.

Note

At install, the Gamebryo installer creates an association between .nif files and the Asset Viewer application. NIF files that contain PhysX content also use the .nif extension. To open a .nif file with PhysX content, you should right-click the file and choose Open with to select the PhysX Nif Viewer executable.

You can find this application in the following locations:

  • Win32—%EMERGENT_PATH%\Source\Tools\DeveloperTools\PhysXNifViewer\Win32\VC80\PhysXNifViewer.exe
  • Xbox—%EMERGENT_PATH%\SDK\Win32\DLL\ToolPlugins\ConsoleViewers\Xbox360\PhysXNifViewer.xex
  • PS3—%EMERGENT_PATH%\SDK\Win32\DLL\ToolPlugins\ConsoleViewers\PS3\PhysXNifViewer.self

Note that when the application starts, motion is frozen to prevent things immediately dropping out of view. To un-freeze the motion, press the space bar, RRIGHT button on a Gamepad, or select Animation > Freeze.

The setting for gravity in the PhysX simulation depends on the command-line flags and the cameras:

  • The command line value for gravity is always used if provided.
  • Gravity is taken from the first non-default camera if gravity is not specified on the command-line and there is a camera in the loaded NIF file. Gravity is set to the down direction of the camera with a magnitude of 9.81. For best results, use a camera that is looking horizontally in the scene.
  • Gravity is set to (0, 0, -9.81) when no other option is applicable.

PhysX Nif Viewer creates all of the PhysX content in the primary simulation, and does not use any non-default compartments.
