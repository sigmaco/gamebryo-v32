PhysX Nif Viewer Controls

The keyboard, mouse and Gamepad controls on the PhysX Nif Viewer manipulate the view, apply forces, and access the menu system. The menus contain most of the configurable settings in the viewer. For information on how Gamebryo names for Gamepad buttons map to a specific platform's Gamepad, refer to the appropriate platform specific documentation:

  • NiInput on Win32
  • NiInput on PS3
  • NiInput on Xenon

When the application starts, motion is frozen to prevent things immediately dropping out of view. To un-freeze the motion, press the space bar, RRIGHT button on a Gamepad, or select Animation > Freeze.

Basic Controls

Keyboard Gamepad Button Action
space RRIGHT Toggle the frozen state of the animation. Equivalent to the Animation->Freeze menu item.
Tab Select or Start Enter or exit the menu system.
Left / Right Arrow LLEFT / LRIGHT Move between menus in the menu system or into and out of a sub-menu.
Up / Down Arrow LUP / LDOWN Move up and down within a menu in the menuing system.
Enter RDOWN Select a menu entry.
Esc None Exit the application. With a gamepad, use the menu File->Exit entry.

View Controls

Note that the view may be locked to keep gravity downward. This can be changed via the Controls->Gravity Down menu item.

Keyboard Gamepad Button Action
'Z' and 'X' Left V-Axis Translate the viewpoint in the X-axis direction.
'A' and 'D' L1 and R1 Translate the viewpoint in the Y-axis direction.
'W' and 'S' Left H-Axis Translate the viewpoint in the Z-axis direction.
'U' and 'I' RLEFT + Left H-Axis Rotate the view (pan) about the X-axis.
'J' and 'L' Right H-axis Rotate the view (pan) about the Y-axis.
'K' and 'M' Right V-axis Rotate the view (pan) about the Z-axis.

Force Controls

The force interface allows you to select an object and apply a force to it at the point of the selection. The interface for selecting differs for mouse or Gamepad control:

  • Mouse Picking—The mouse is used for picking if there is no Gamepad attached or if the -keyboard command line argument was used. To pick with a mouse, left click on the desired object. Force is applied while you continue to hold down the left mouse button, and stops when you release the button. The cursor shows a hand shape when you are applying a force.
  • Gamepad Picking— To enter picking mode on the Gamepad, push and release the RIGHT UP button. The cursor will appear as a target shape. While the cursor is a target the left stick is used to move it around the screen. When the cursor is over the point you wish to select, push and hold the RIGHT UP button to apply the force. The cursor will change to a hand to indicate that a force is being applied. Force application stops when you release the RIGHT UP button, or if there was no dynamic object under the cursor.

Forces are applied as if a spring was attached to the object where you first clicked on it, the attachment point, with the other end of the spring at the current cursor location. So moving the cursor further from the attachment point on the object applies larger forces. The force is always parallel to the screen in world space.

The magnitude of the force is 0.1 x Mass x Distance x 2exp where exp is thevalue shown in the "Force Exp" menu item under the SDK menu. Mass is the mass of the actor you selected, and Distance is the pixel distance between the cursor and the attachment point. If cloth is selected, the cloth density is used in place of mass.

You can only select and apply forces to dynamic actors or cloth.
