PhysX NIF Viewer Command-line Usage

Usage

PhysXNifViewer [-gravity x,y,z] [-ground] [-height float_ground_height] [-bg R,G,B] [-near float_near_clip] [-far float_far_clip] [-keyboard] [-debugText] [-debugGeom] [-full] [-resolution widthxheight] [-pausefordebug] [-shaderpath pathname] [-vrdHost hostname] [-vrdPort port] filename

Required Arguments

filename

Specifies a Nif file to load.

Optional arguments

-gravity x,y,z

Set the gravity vector. Note the commas between each coordinate, with no spaces.

-ground

Enable a ground plane for objects to fall onto. The normal of the plane is always pointed in the opposite direction to gravity. The ground plane is off by default.

-height float_ground_height

Set the height of the ground plane above the origin of the scene (as defined in the Nif file that is being viewed). Negative heights move the plane in the gravity direction, while positive heights move it against gravity. The height is in PhysX units, so if the loaded NiPhysXScene has a scale factor associated with it, the world height of the ground plane will be scaled by that amount.

-bg R,G,B

Specify the background color. Values R,G,B are floats in the range 0-1.

-near float_near_clip

Specify the near clip plane distance. Default is the near clip plane of any camera in the scene, or 5% of the radius of the scene's bound if no camera is present.

-far float_far_clip

Specify the far clip plane distance. Default is the far clip plane of any camera in the scene, or 40 times the radius of the scene's bound if no camera is present.

-keyboard

Specify a preference for keyboard over gamepad control. The keyboard is always used if no gamepad is present. When a gamepad is present it is used by default. This switch over-rides the default behavior.

-debugText

Produce debug output. The output is produced using NiOutputDebugString.

-debugGeom

Enable the rendering of PhysX debug geometry.

-full

Run in full-screen mode.

-resolution widthxheight

Set the resolution for either windowed (window size) or full screen modes. The size should be given as, say, 640x480 with a lower-case "x".

-pausefordebug

If present, this flag causes the program to pause immediately upon startup, giving a developer time to attach a debugger to the running process. It is intended to support debugging on consoles.

-shaderpath pathname

(PS3-only optional argument) If present, Specifies the directory from which to load custom NSF and NSB shaders.

-vrdHost hostname

If present, PhysX Nif Viewer will attempt to connect to a PhysX Visual Remote Debugger on the given host. This enables real-time visualization and recording of PhysX simulation data.

-vrdPort port

The port on the vrdHost to be used for connection. This argument is only used if the vrdHost argument is given. This argument may be omitted and the default port will be used.
