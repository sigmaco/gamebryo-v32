PhysX NIF Viewer Menus

You can access the PhysX Nif Viewer menu system by pressing the TAB key or Select or Start on a Gamepad. Arrow keys or the left directional pad move among the menus. Enter or the RDOWN button select entries.

Entries controlling numeric values, such as the ground height, have sub-entries that increase or decrease the value by a constant ratio (for example 1.2 or 1.0/1.2) and another sub-entry that resets to the default. If the numeric value is given as an integer, then each positive increment represents a further doubling of the value, and each negative increment is a halving. So a value of 2 for the force exponent results in four times the default force strength.

File Menu Description
Exit Exit the application.
View Menu Description
Camera Choose between the available cameras in the scene, and the default PhysXNifViewer camera.
Wireframe Toggle wireframe viewing mode.
Reset Reset the view to the active camera.
FPS Toggle the display of the frame-rate counter.
Sorter Select a depth sorter.
Controls Menu Description
Invert Toggle inversion of the gamepad camera controls.
Rotate Modify the camera rotation speed.
Translate Modify the camera translation speed.
Gravity Down Lock camera rotation so that gravity is always in the downward direction.
Animation Menu Description
Freeze Toggle animation activity.
Step Take a single time step.
Step Size Adjust the duration of the single step. The step is broken into sub-steps internally to ensure that the physics simulation is stable.
Reset Reset the animation and physics to any one of the initial states available in the NIF file.
Disable Keys Disable the key-framed animation on any object driven by PhysX. This is checked by default so that objects driven by both key-frames and PhysX are shown controlled by the PhysX. If this is turned OFF, the key-frames will drive the objects while the key-frame animation is active.
Speed Set the speed multiplier on the motion. Going faster than 100% is not advised because it may cause the physics system to take large time-steps.
SDK Settings Menu Description
Reset Defaults Reset the SDK to the default settings. The defaults are those found in the input NIF file, or the application defaults if the NIF file does not contain a NiPhysXSDKDesc object.
Force Exp. Adjust the strength of the applied force. Each increment represents a doubling in strength, each decrement a halving. For example, 0 is the default, -1 is half strength, -2 is quarter strength, 1 is double strength, 2 is quadruple strength, and so on.
Debug Viz On Toggle the rendering of PhysX debug geometry.
Debug Scale Adjust the scale for PhysX debug geometry (the NX_VISUALIZATION_SCALE parameter).
Ground Menu Description
Ground Plane Toggle the presence of a ground plane in the scene.
Ground Height Adjust the ground plane height. It is a known issue that if the height is negative, it will stay negative, and if positive it will stay positive.
