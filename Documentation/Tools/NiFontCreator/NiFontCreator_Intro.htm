NiFontCreator: Introduction

Gamebryo's NiFont library provides the ability to display 2-byte Unicode (UTC-2) and ASCII characters. However, other strings in Gamebryo, such as file names, file paths, object names, and so on remain in ASCII.

The NiFontCreator application (NiFontCreator.exe) runs on Windows. You can use it to create NFF files. These files contain font information for the NiFont library.

You can find this application in the following location:

%EMERGENT_PATH%\Source\Tools\DeveloperTools\NiFontCreator\Win32\<VC80 or VC90>
