Renaming Entity Model Projects

When you first create an entity model project, you must name it. You can, however, rename a project at anytime for any reason.

To rename a project

  1. In the Game Solution view, do one of the following:
    • Select the project node and press F2.
    • Right-click the project node and select Rename from the context menu.
  2. Type a new name and press Enter.

    The project is now renamed.
