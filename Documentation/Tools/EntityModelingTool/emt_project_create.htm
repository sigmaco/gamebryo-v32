Creating Entity Model Projects

When using EMT to manage the content in your game, it is a good idea to split up your game into more manageable projects. These projects can represent a variety of different parts of your game, including levels, zones, or missions. You can create as many entity model projects as you want. You can also add existing entity model projects to a game solution.

To create a new entity model project within a game solution

  1. Open a game solution.
  2. Right-click the game solution node in the Game Solution Explorer.
  3. From the menu, click Add > New Content.

    The New Content dialog box opens.

  4. In the Content Categories group box, select Game Content.
  5. In the Content Types group box, select Entity Model Project.
  6. In the Name text box, type a name that best represents the area of the game upon which you are working.
  7. In the Folder text box, type in the full path where you want the project file to be saved. (You can click the Browse button to browse through the folders on your computer or network to select a file.)
  8. Click OK

The new entity model project is created within the game solution.
