Renaming Entity Model Diagrams

When you first create a diagram, it is given a default name, such as Diagram1. You can, however, rename a diagram at anytime to give it a more appropriate name.

To rename a diagram

  1. In the EMT Project Explorer, do one of the following:
    • Select the diagram item and press F2.
    • Right-click the diagram item and select Rename from the context menu.
  2. Type a new name and press Enter.

    The diagram is now renamed.
