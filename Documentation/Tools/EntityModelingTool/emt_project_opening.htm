Opening Entity Model Projects for Editing

When you first open a game solution, none of the projects are opened for editing. Before you can add new content or edit the existing items within a project, you need to load the project first. After a project is loaded, you will have access to all of the project's content.

To open an entity model project for editing

  • In the Game Solution view, double-click the project you want to open for editing.

A tab with the project's name appears in the Toolbench document area signifying that the project has been opened for editing. You will now be able to access all items within the project from the EMT Project Explorer view.
