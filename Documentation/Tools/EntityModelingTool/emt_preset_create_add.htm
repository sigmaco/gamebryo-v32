Creating Presets

After you have decomposed your game into entities and corresponding models, and have defined all the properties and behaviors for these models, you can begin to create presets, which are basically derivatives of these models intended to be instanced in your game world. You can create new presets or re-use existing ones. The Preset Factory can also be used to rapidly stamp out many new presets based on different assets.

To create a new preset within a project

  1. Open a game solution.
  2. In the Game Solution view, double-click a project node to open it for editing.
  3. In the EMT Project Explorer, select Add > New Preset from the menu.

    The new preset is created within the open project.

  4. Specify a name for the preset.

Note

It is a good idea to add detailed descriptions to each preset that you create. This will help you locate and identify presets easily as the number grows within your project. This information is added in the Entity Model Editor.

To rapidly generate a set of presets with the Preset Factory

  1. Open a game solution.
  2. In the Game Solution view, double-click a project node to open it for editing.
  3. Click the Preset Factory toolbar item.

    The Preset Factory dialog box opens.

  4. Specify a parent entity model to be used as the basis for the new presets.
  5. Choose an asset property on the selected entity model from which to create preset variants.
  6. Pick the assets that should be assigned to the new presets.
  7. Click OK.

    The new presets are added to the open project.
