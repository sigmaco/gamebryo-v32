Moving Entity Models and Presets within the Diagram Editor

When experimenting with different mix-in possibilities in the Diagram Editor, it is important that you be able to move the different boxes around. This is why the Diagram Editor was made so flexible, so that you could easily re-arrange the different boxes in your diagram any way that you want.

The Mix-in Editor, on the other hand, displays an auto-generated layout of models. Models cannot be repositioned in this view.

To move models and presets within the Diagram Editor

  1. In the EMT Project Explorer, double-click a diagram to load it into the Diagram Editor.
  2. Mouse over the box that you want to move.

    The mouse pointer changes to the move (4-arrow) icon.

  3. Drag the box to a new location in the Diagram Editor.

    If the model or preset is already linked to another, it will remain linked.
