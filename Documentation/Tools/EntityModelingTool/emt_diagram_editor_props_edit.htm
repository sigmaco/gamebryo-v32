Editing Properties and Behaviors Directly in the Mix-in/Diagram Editor

Even though you can easily edit a model's properties and behaviors in the Entity Model/Preset Editor, there may be circumstances where it is easier for you to edit them directly in the Mix-in/Diagram Editor itself. In both editors, models and presets are represented by a box. This box contains the name of the model or preset along with a list of specific properties and behaviors, including the properties and behaviors defined for the current model/preset as well as any mix-in properties and behaviors that have been overridden.

To edit properties and behaviors directly in the Mix-in/Diagram Editor

  1. In the EMT Project Explorer, do one of the following:
    • Double-click a model or preset to load it into the Mix-in Editor.
    • Double-click a diagram to load it into the Diagram Editor.
  2. Zoom in to the model or preset whose properties and/or behaviors you want to edit.
  3. To display the properties and behaviors for a particular model or preset, click the Expand icon.

    The properties and behaviors defined for the current model/preset are displayed as well as any mix-in properties and behaviors that have been overridden.

  4. Double-click the property, behavior, or any value to enable it for editing.
  5. Enter in a new name or value and press Enter.

Note

If the model or preset has mix-in properties only the values of these properties can be edited. Also, mix-in behaviors cannot be edited here.
