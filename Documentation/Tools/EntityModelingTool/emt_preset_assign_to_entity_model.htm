Assigning Presets to an Entity Model

Since presets don't have their own properties and behaviors, you have to specify a parent model for them. When you setup this relationship, the preset automatically inherits the parent model's properties and behaviors. You can then customize each preset by overriding the property values.

To assign a preset to an entity model

  1. Load a preset into the Mix-in Editor. (You can also select a preset in a diagram to load it into the Entity Model Editor.)

    If the preset has already been assigned to an entity model, a list of the inherited properties will be displayed in the Entity Model Editor.

  2. From the Model list, select the entity model upon which you want to base your preset.

    The parent model's properties are displayed in the Property list of the Entity Model Editor.
