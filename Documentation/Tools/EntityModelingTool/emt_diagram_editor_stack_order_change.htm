Changing the Stacking Order of Models and Presets

When you have many different models and presets within the Mix-in/Diagram Editor, some models may begin to overlap others. If the boxes in the editor overlap, you can order the boxes by moving a model to the front or back of the stack or to any place within the stack.

To order the models and presets within the Mix-in/Diagram Editor

  1. In the EMT Project Explorer, do one of the following:
    • Double-click a model or preset to load it into the Mix-in Editor.
    • Double-click a diagram to load it into the Diagram Editor.
  2. Right-click the box whose order you want to change, select Order and then one of the following:
    • Bring Forward to move the current model up one place in the stack.
    • Bring To Front to move the current model to the top of the stack.
    • Send Backward to move the current model down one place in the stack.
    • Send To Back to move the current model to the bottom of the stack.
