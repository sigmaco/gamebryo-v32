Excluding Entity Model Projects from a Game Solution

If you no longer need a project in your game solution, but may need it at a later date, you can exclude it. When you exclude a project from a game solution, all entity models, presets, and diagrams are removed from the game solution as well. When you exclude a project from a game solution, it does not get deleted.

To exclude a project file from your game solution

  1. In the Game Solution view, right-click the project that you want to remove.
  2. From the context menu, click Exclude.

    Toolbench prompts you to confirm the removal.

  3. Click Yes.

The file is removed from your game solution.
