Entity Modeling Tool: Introduction

Welcome to the Entity Modeling Tool, a set of plug-ins for Toolbench. The EMT plug-ins combine to create a powerful tool that allows you to design and refine entity models, the building blocks of gameplay. Many different properties and behaviors can be applied to these entity models, which serve as blueprints for the actual Game Framework entities that populate the game world. This system is a replacement and expansion of the entity concepts originally introduced in the Scene Designer tool.

At the core of EMT is the Mix-in Editor, a graphical representation of the relationships between the entity models in your game. These relationships result in properties and behaviors being shared and customized from one entity model to another.

With its tight integration with World Builder and intuitive interface, EMT allows you to quickly iterate on entity models, both tuning existing content and introducing entirely new gameplay elements.

Note

Entity Modeling Tool is not a standalone tool! EMT functionality is accessed by using Toolbench to create or open an .emtproj file. For example, you may open an existing game solution (.gsl), then double-click an .emtproj file in the Solution View to launch the Entity Modeling Tool interface.
