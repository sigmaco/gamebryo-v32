Adding Content to your Project

After creating your entity model projects, you need to start populating them with the entity models, presets, and diagrams.

To add new content to your EMT project

  1. Right-click in the Contents sub-pane of EMT Project Explorer.
  2. From the context menu, click Add and then one of the following options:
    • New Model
    • New Preset
    • New Diagram
    • New Folder

    The new file is added to the project and will appear in the current context of the EMT Project Explorer.
