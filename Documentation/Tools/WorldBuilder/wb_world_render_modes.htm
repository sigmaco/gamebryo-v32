Rendering the World

In world builder, rendering the world is controlled using render modes which specify how the scene is rendered within a specific viewport. Several standard render modes are provided and among them are: "Default", "Flat Wireframe", "No Shadows", "No Textures", 'Per Vertex Lighting", and "Wireframe". Each viewport is allowed to have its own render mode setting which can be selected from a drop down combo box a the top of the viewport.

To specify a rendering mode for a given viewport

  1. First decide on the viewport whose render mode will changed. Locate the render mode selection combo box a the top of the viewport.
  2. Then simply select one of the available render modes.
