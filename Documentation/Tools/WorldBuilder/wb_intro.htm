World Builder: Introduction

Once you have created your game's art assets in the DCC tool of your choice and defined a suitable model hierarchy with the Entity Modeling Tool, it is time to assemble these components into scenes. World Builder comprises a series of Toolbench plug-ins that enable you to populate your game world with entity model instances.

In addition to defining the initial game simulation state, World Builder can connect to a running game. The two will synchronize, allowing you to place new entities or modify existing ones in World Builder and see the effects in your game.

To get started with World Builder, add a block file to your Game Solution in Toolbench and open it. New Game Solutions also add a block file by default.

Note

World Builder is not a standalone tool! World Builder functionality is accessed by using Toolbench to create or open a .block file. For example, you may open an existing game solution (.gsl), then double-click a .block file in the Solution View to launch the World Builder interface.
