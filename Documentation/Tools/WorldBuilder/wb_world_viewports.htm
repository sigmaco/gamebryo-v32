Working with viewports

Viewports in World Builder present the graphical representions of the world under construction. Viewports are available when "block" content has been opened under the "Graphical World Builder" perspective. World Builder allows viewing from the traditional perspective camera as well as the standard 6 ortho cameras (Front, Back, Left, Right, Top, Bottom). Additionally, World Builder provides 4 different viewport layouts where by each viewport can be specified to any of the standard cameras as desired.

To specify the viewport layout

  1. To change the viewport layout, first open the viewport layout menu.
  2. Each option represents 1 of the 4 possible viewport layouts, offering 1, 2, 3, and 4 viewport variants respectively. Simply select an option to have World Builder use that layout.

To maximize the active viewport

  1. To maximize the active viewport, first open the viewport layout menu.
  2. Select the Maximize Active Viewport option.
  3. To restore the original layout simply select this option again.

To specify the viewport camera type

  1. Position the mouse in the viewport whose camera type is to be changed.
  2. Right click to bring up the viewport context menu.
  3. Choose 1 of the 7 standard cameras under the "Select Camera" menu option. You can also choose 'Use Selection', which will use the currently selected camera in the scene, or you can choose a scene camera from the list of scene cameras. Hitting Esc will exit the scene camera mode and commit any changes to the camera entity.


    Note that camera positions and orientations for the standard cameras will reset to a default upon selection.
