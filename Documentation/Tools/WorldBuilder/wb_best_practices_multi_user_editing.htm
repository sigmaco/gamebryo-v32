Working with multiple users

In many studios, developers typically want to work on the same geographic area of the game at the same time. For this reason, World Builder supports the concepts of layer files. A World Builder block can be broken up into any arbitrary number of layers. The structure of layer files is entirely up to the development team. Here are some ideas for breaking down a block into layers.

  • A layer for lighting
  • A layer for static props
  • A layer for gameplay items (triggers, spawn points, etc)
  • A layer for terrain and vegetation

It is suggested that only one user check out and edit a layer at a time. This helps to avoid the need to merge layers. However, if a merge needs to happen, the XML has been structured to make this a fairly straightforward operation.
