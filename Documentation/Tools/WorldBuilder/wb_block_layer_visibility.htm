Changing Layer Visibility in World Builder

If you are working with large numbers of entities and layers in World Builder, you can limit the rendering view to a subset of the entire entity set. You can limit your view based on entities, but it is easier to do this by showing/hiding entire layers. Additionally, when you show/hide a layer, you can choose to hide its children as well.

World Builder: Changing layer visibility

Note

Showing/hiding is disabled on unloaded layers. Refer to Loading and Unloading Layers for more information.

To change a layer's visibility

  1. Run Toolbench (Start > Emergent > Gamebryo LightSpeed > Tools > Toolbench).
  2. Open your game solution (see Creating/Opening Game Solutions).
  3. In the Game Solution pane, expand the tree to view your game solution (.gsl) contents.
  4. If the Layer Editor is not already open, open it (View > Layer Editor).
  5. Change the layer's visibility by clicking its Show/Hide icon () or by right-clicking and selecting Hide Layer or Unhide Layer from the context menu.

To change a layer's visibility and the visibility of all child layers

Follow the steps above, but double-click on the Show/Hide icon or right-click and select Hide Layer and Children or Unhide Layer and Children from the context menu.
