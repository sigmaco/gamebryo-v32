World Builder - Palette View

The Palette View allows access to all non-abstract entity models and presets defined in any referenced EMT project. By default, it is visible in a left-docked pane when a block file is opened into World Builder.

Entity models and presets can be dragged from this view and dropped into the World Builder viewport to create new entities.

A thumbnail image and description for the selected content is displayed at the bottom of the view.

The view has two modes for finding palette content. They operate similarly to how Windows Explorer works with the file system.

Browse Mode

The Browse tab is used for navigating a virtual folder hierarchy of entity models and presets. This may be done exclusively in the Contents sub-pane to conserve screen space. Alternatively, the Folders sub-pane can be added on the left for greater flexibility. There is a draggable splitter between the two sub-panes to specify how the space should be shared. The content hierarchy viewed here is created and edited in the EMT Project Explorer within the Entity Modeling Tool.

Palette View - Browse mode

The root of the hierarchy contains read-only links to all EMT projects referenced by the open block file. Within these projects are entity models, presets, and virtual folders. Virtual folders can in turn contain additional EMT content, including virtual subfolders. Project references are managed in the Game Solution view.

There are several controls for navigating the content hierarchy in Browse Mode.

  • Use the Location breadcrumb control at the top of the pane to quickly move between folders and referenced EMT projects.
  • Use the Filter edit box to specify a substring that limits the display of items at the current level in the hierarchy.
  • Use the Folders sub-pane to expand, collapse, and view different folders and projects.
  • Double-click folders in the Contents sub-pane to navigate into them.
  • Use the backspace key to navigate up one level in the virtual hierarchy.

Search Mode

The Search tab is used to quickly locate palette content that might be scattered throughout multiple referenced EMT projects. The query-building controls provide several different criteria for searches. Matching entity models and presets are returned in the Results list.

Palette View - Search mode

A checkbox to the left of each option determines which search criteria are active. The Advanced rollout can be expanded for access to additional search options. The Execute Query button performs the search. If the Palette View has focus, simply hitting the enter key will also perform the search. The following search parameters can be specified.

Search Parameter Details
Search Under Specify an EMT project to search, or uncheck to search all referenced projects.
Name Includes Specify a substring to find in model and preset names.
Has Mixin Specify a model name substring that must be mixed in.
Has Property Specify a property name substring that must be present.
Has Behavior Specify a behavior name substring that must be present.

Results are listed with Name and Virtual Path. The path reflects the EMT project and virtual folders that contain an item.

Commands

Palette items listed in both Browse and Search tabs provide context menus with a variety of commands. The available commands depend on the type and context of content selected.

Command Details
Edit Opens the selected entity model or preset in the Entity Modeling Tool, where it can be edited.
Transmogrify Converts all selected entities to use the selected entity model or preset from the Palette View.
View Asset Opens an external tool for viewing the NIF or KFM asset associated with the selected entity model or preset. This feature is configured in user settings.
Browse to Location When used in Search Mode, this command will transition the view to Browse mode, arriving at the virtual path of the selected item.

Thumbnails View

Each mode of Palette View can be configured to use either of two visualizations. These visualizations are toggled via the Views dropdown. The default visualization is Details.

Thumbnails view can be useful for quickly locating entity models and presets that have associated visual assets.

Palette View - Thumbnails View

The Zoom slider controls the size of the thumbnail images, trading image detail for viewable image count. Hovering over the label for a thumbnail will display a tooltip with the full name and virtual path of the item.

The thumbnails are provided via the NiThumbnailExtractor COM DLL.
