World Builder Render View

The render view provides users with a WYSIWYG representation of their block file rendered with Gamebryo.  Each viewport supports being rendered from a separate camera with a separate render mode in each view.

Selection Modes

The selection modes in World Builder provide the user with different mechanisms for picking the entities they are interested in editing. All selection modes obey any locked or hidden effect currently being applied to the entities in the scene. Any entity with either of these effects will not be selected.

Selection Modes Details
Single Single selection is performed by left clicking on an entity in the viewport.
Multiple Multiple selection is performed by left clicking with the ctrl key held down on an the entities in the viewport.
Marquee Marquee selection is performed by holding down the alt key and clicking on a rendered view with the left mouse button and dragging the box over the entities to select. Marquee selection has the following modifier keys:
  • Shift - If held, the selection becomes additive; the selection will merge with any other selected entities that were previously selected instead of clearing the selection.
  • Ctrl - The selection toggles; if an entity within the box is not selected, it will be selected. Otherwise it will be deselected.
  • Ctrl + Shift - The selection becomes subtractive; if an entity within the box is already selected, it will be deselected.

Render Modes

The different render modes in World Builder provide the users with different ways to view the scene.

Render Mode Details
Default The default rendering mode is the standard Gamebryo render of the scene.
Flat Wireframe Renders the scene in wireframe where the wireframe representation is flat (all white without lighting or coloring).
No Shadows The no shadow rendering mode will not render any shadow generators.
No Textures The no texture rendering mode will not display any textures.
Per-Vertex Lighting Renders the scene with per vertex lighting.
Wireframe Renders the scene in wireframe.
