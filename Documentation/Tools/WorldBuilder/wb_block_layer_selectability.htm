Changing Layer Selectability in World Builder

If you are working with large numbers of entities and layers in World Builder, it can be useful to prevent visual selection of some subset(s) of the entire entity set. For example, if you are placing props on an outdoor environment, you might want to prevent selection of the terrain itself. You can prevent inadvertent selections by locking entity subsets. You can limit selectability based on entities, but it is easier to do this by locking/unlocking entire layers. Additionally, when you lock/unlock a layer, you can choose to lock/unlock its children as well.

World Builder: Changing layer selectability

Note

Locking/unlocking is disabled on unloaded layers. Refer to Loading and Unloading Layers for more information.

To lock/unlock a layer

  1. Run Toolbench (Start > Emergent > Gamebryo LightSpeed > Tools > Toolbench).
  2. Open your game solution (see Creating/Opening Game Solutions).
  3. In the Game Solution pane, expand the tree to view your game solution (.gsl) contents.
  4. If the Layer Editor is not already open, open it (View > Layer Editor).
  5. Change the layer's selectability by clicking its Lock/Unlock icon () or by right-clicking and selecting Lock Layer or Unlock Layer from the context menu.

To lock/unlock a layer and all child layers

Follow the steps above, but double-click on the Lock/Unlock icon or right-click and select Lock Layer and Children or Unlock Layer and Children from the context menu.
