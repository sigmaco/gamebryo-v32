World Builder Key Terms

Before continuing with World Builder it is recommended that you first read over the introduction documentation for the Entity Modeling Tool.

Entity Instance - "Entity"

Entities are instances of the entity models embodied in a simulator. The Game Framework documentation includes a thorough discussion of Working with entities. As the game simulation evolves, the entities will respond to events within the simulation.

Block and Layer Files

Block and layer files are tools-only file formats for defining initial conditions for a game simulation. These files are simple text XML files for ease of merging and authoring. Entity instances defined within these files only contain property values that are overridden from their parent entity models. See the Working with Blocks and Layers Introduction for more information.

XBlock Files

XBlock files are a runtime-only files for defining initial conditions for a game simulation. XBlock files are generated by collapsing block and layer files into a single file format in the export process. Xblock files are simple text XML files.

Palette

The World Builder palette consist of all instantiable (non-abstract) models and presets that are defined in any referenced EMT projects. The palette allows you to instantiate entities into the world. For more information, see the documentation on World Builder Palette View.

Viewports

Viewports in World Builder present the graphical representations of the world under construction.

Gizmos

Gizmos provide a means of interacting with viewports to accomplish various task. There are several gizmos, but the most common are the ones that allow a user to select, move, rotate, and scale entities. 

Camera Tools and Modes

Camera Tools and Modes provide a means of altering the views that viewports provide. In other words, they provide basic navigation capabilities such a being able to fly around, orbit entities, face toward selections, pan the camera, and what not. See the documentation Navigating Around the World for more information.

Selection Sets

Selection Sets are a named group of assets that are bundled together for ease of selection. See the Selection Set Editor reference documentation for more information.

Synchronizing Content

Synchronizing content is the process by which the World Builder's world is brought into sync with the models defined by EMT and the world used by the game. Synchronizing to the game amounts to exporting out the current state of the world while synchronizing from EMT amounts to updating World Builder's in-memory representations of models and presets. See Synchronizing Content with the Game and Synchronizing Content with Entity Modeling Tool for more information.
