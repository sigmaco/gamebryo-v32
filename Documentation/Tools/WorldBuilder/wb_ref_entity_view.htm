World Builder Entity View

The entity view is the primary mechanism users will use to navigate the entities in their block. Because users will be interested in different aspects of the data depending on what they are doing, there are three separate layouts in the entity view; folder, alphabetical and model layout.

Folder Layout

The folder view displays the folder information contained in the block files. The list of folders is displayed in the top pane with the list of entities and folders in that folder contained in the bottom pane. Users are free to drag the entities and folders around and drop them into a different folder of their choosing.

The numbers beside each folder represent the following (in order):

  • The number of entities in the folder with the filter applied.
  • The number of entities contained in the folder.
  • The number of entities contained in the folder and all of its descendant folders.

Right-clicking a folder in the Folder Layout presents a context menu with the following options:

  • Add Folder - Create a new folder.
  • Remove Folder - Deletes the selected folder.
  • Rename Folder - Allows the user to rename the selected folder.
  • Add Selection - Adds all selected entities in the scene into the selected folder.
  • Select All - Selects all entities in the selected folder and all subfolders.
  • Select All in Folder - Selects all entities in the selected folder only.

Alphabetical Layout

The alphabetical layout displays all the entities sorted alphabetically by their name.

Model Layout

The model layout shows all the entities in a block file broken up into groups based on the entity model that the entity was instantiated from.

Filtering

Filtering is performed by typing an entity name to search for in the search input field at the top. The displayed list will be trimmed in real-time as you type. To remove the filter just erase the text or hit the clear button. All searching is performed using a substring match on the entity name.

Picking

When using the EntityRef property editor, it is valid to pick an entity in the scene as well as from any entity contained in any of the entity layouts in these views.
