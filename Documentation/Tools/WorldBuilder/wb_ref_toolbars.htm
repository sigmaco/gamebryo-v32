World Builder - Toolbars

The World Builder toolbars provide access to most of the application's commands.

Note

By default, all of the toolbars use large icons. To change the default icon size, you can modify your Toolbench settings (Utilities > Settings > Toolbench > Toolbar > Use Large Buttons)

Standard Toolbar

Icon Name Details
Export Block Exports the current block to an xblock file.
Sync World Synchronizes World Builder to any newly saved versions of the EMT projects that World Builder references. This will update the world state to match the modifications to models that have been made. Performing this operation will clear the undo stack.
Transmogrify The transmogrifier is used to convert entities from one model to another. Select entities in the world, then select a destination type from the Entity Palette View. Clicking this button will then convert all the selected entities to the model selected in the palette.

Settings Toolbar

Icon Name Details
Toggle Default Lights Turns default lighting on and off. This illuminates the scene in the absence of lighting entities. The button icon changes to reflect the state of this setting.
Toggle Play Animations Turns scene animations on and off.
Toggle Grid Turns display of the grid on and off. This does not affect the snap settings.
Toggle Property Visualizers Turns display of property visualizers on and off.

Snap Toolbar

Icon Name Details
Free Placement Entities can be placed and moved freely.
Snap to Surface Entities will be translated toward the nearest surface but not rotated.
Align to Surface Entities will be translated toward and rotated with the nearest surface.
Toggle Translation Snap The toggle translation snap button toggles whether or not transformations that affect translation should snap to discrete units. The units can be specified to be any number in the settings, however there is also a quick access drop down next to the button for changing it to several useful defaults.
Toggle Rotation Snap The toggle rotation snap button toggles whether or not transforms that affect rotation should snap to discrete units. The units are in degrees can be specified in the settings, however there is also a quick access drop down next to the button for changing it to several useful defaults.
Toggle Scale Snap The toggle scale snap button toggles whether or not transforms that affect scale should snap to discrete units. The units are in percentages and can be specified in the settings, however there is also a quick access drop down next to the button for changing it to several useful defaults.
Toggle Snap Points When this button is pressed, models with snap points defined will automatically snap together during transform operations.

Transform Toolbar

Icon Name Details
Terrain Edit Gizmo This gizmo enables terrain editing features.
Selection Gizmo This default gizmo allows selection of entities.
Translation Gizmo This gizmo allows the user to manipulate the selected objects along the X, Y or Z axis or along any combination of the planes.

[World] - Transforms entities in world space.
[Local] - Transforms a selected entity along any local space rotations that have been applied to the entity. If multiple entities are selected, the local space flag is ignored and the entities are moved in world space.
Rotation Gizmo This gizmo rotates the selected objects.

[World] - Rotates the selected objects about the center of the selection or the averaged local origin of all selected objects.
[Local] - Rotates the selected objects about their local origin.
Uniform Scale Gizmo This gizmo allows the object to be scaled uniformly.

[World] - Scales the selected objects in relation to each other, so that the distance separating objects scales as well.
[Local] - Scales the objects in their local space and does not translate the objects.
Creation Gizmo The creation gizmo can be used to stamp a model into the scene several times. As a shortcut to the creation gizmo button, users can hold down Spacebar to toggle creation mode at any time.
Transform Space This box allows the user to change the transformation space that the gizmos will operate in, if applicable.

Camera Toolbar

Icon Name Details
Fly Level When the "Toggle Fly Level" icon is pressed, camera movements using the WASD keys will not affect the camera's height.
Standard Camera When in standard mode, clicking the left mouse button and dragging will rotate the camera. This effectively gives the ability to look around the world.
Orbit Camera When in orbit mode, if an entity is not selected, the orbit point will be determined by casting a ray from the center of the viewport into the scene. Whatever entity is hit first becomes the orbit point. If an entity is already selected or becomes selected, then that entity is used as the orbit point. The actual orbiting action is performed by clicking the left mouse button and dragging in the viewport.
Pan Camera When in Pan mode, navigation occurs solely along the viewport plane. Click the left mouse button and dragging in the viewport will move the camera along one or both of the two axes depending on the direction of the drag.
Move to Selection Moves the camera close to the selected entity while keeping the center of the entity within the center of the viewport. The camera's orientation is preserved.
Look at Selection Rotates the camera toward the selected entity. The camera's translation is preserved.

Whitebox Toolbar

Icon Name Details
Create Whitebox Activates the Create Whitebox gizmo which is used in place of any active transformation gizmos. Clicking and dragging in the viewport will begin creation of a new whitebox.
Edit Whitebox Activate the Edit Whitebox gizmo which is used to push or pull a single face of an existing whitebox.
Default Whitebox Model When the Create Whitebox gizmo is active, it will create entities that are of the model selected in this dropdown. It is automatically populated with every non-abstract model that mixes in Whitebox referenced by the block's EMT Project dependencies.
