Renaming a Layer in World Builder

You can rename a layer to better describe its contents and/or function. When you do this a new .layer file is created and the old .layer remains on disk.

World Builder: Renaming a layer

Note

An unloaded layer cannot be renamed. Refer to Loading and Unloading Layers for more information.

To rename a layer

  1. Run Toolbench (Start > Emergent > Gamebryo LightSpeed > Tools > Toolbench).
  2. Open your game solution (see Creating/Opening Game Solutions).
  3. In the Game Solution pane, expand the tree to view your game solution (.gsl) contents.
  4. If the Layer Editor is not already open, open it (View > Layer Editor).
  5. In the Layer Editor, locate the parent node of your layer. This could be the master block file or any other existing layer file.
  6. Right-click the parent node and select Revision Control > Add / Edit from the context menu.
  7. Right-click the layer and select Rename from the context menu.
  8. In the Set Layer Name dialog that displays, type the name of the layer. Only valid filename characters are allowed; 512 character maximum. If the layer name matches one that already exists in the block or on disk, you will be prompted to choose a different name. When you are done, click OK. This will branch the file if you have properly set up revision control.
