Synchronizing Content with Entity Modeling Tool

World Builder provides the ability to easily synchronize with content modified by the Entity Modeling Tool (models and presets). This is particularly useful during early development when model and preset development is taking place simultaneously with world construction. There are two primary methods for synchronizing the World Builder with the most current models and presets; one passive, one explicit.

To passively synchronize content with the Entity Modeling Tool

  1. Make changes to models and/or presets within the Entity Modeling Tool.
  2. Save the content changes that have been made.
  3. Select one of the "block" documents in the document pane to switch back to the World Builder context. When this is done, the following window will appear:
  4. Select "Yes" to synchronize World Builder with the Entity Modeling Tool's content changes.

To explicitly synchronize content with the Entity Modeling Tool

  1. Choose "Save All" from within World Builder
  2. Next, simply press the World Builder synchronize icon:



    Or alternatively, the synchronize command can be chosen from the World Builder menu:


  3. The World Builder will immediately synchronize itself with the most recent model and preset changes.
