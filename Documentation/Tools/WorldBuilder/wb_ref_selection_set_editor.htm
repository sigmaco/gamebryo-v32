World Builder - Selection Set Editor

The selection set editor allows the user to select a series of entities in the world and then save that selection for further use later under a specified name. The sets are stored inside the *.block file so that selection sets may be shared between all the users.

Button Name Details
New New Selection Set Adds the currently selected entities to a new selection set.
Delete Delete Selection Set Deletes the active selection set.
Add Selection Adds the currently selected entities to the active selection set.
Remove Selection Removes the currently selected entities from the active selection set.

Selecting

To select the entities in a selection set just double click the selection set in the list. If you hold down the ctrl key when double clicking you can merge the selection set with any currently selected entities in the world.
