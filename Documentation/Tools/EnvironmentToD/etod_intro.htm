Environment and Time Of Day Introduction

The Environment and Time Of Day features are as follows:

  • A flexible interface allowing the edition of animation keyframes through graphical display.
    • The interface can be used with other models.
  • A model has been added to facilitate the addition of an environment to a scene.
  • A model has been added to handle time of day animation.
  • An abstract model has been added to ease the creation of custom models that should be affected by the time of day.
  • Properties can hold a TimeOfDayAnimatable trait to allow for their animation.
