The TimeOfDay Model

A user can control several aspects of the time of day model which will allow for customization of the animation. Those can be set using the model's properties:

Attribute Type Description
TimeOfDayTimeScaleMultiplier Float This defines the speed at which the time of day animation should go. This will be the multiplier used in game. If this value is set to 1, then time passes at the same speed as real time. A value of 200 will speed up the animation to 200 times faster than real time.
TimeOfDayAnimate Bool Controls whether the time of day animation should be used or not.
TimeOfDayInitialTime Float This is the percentage of the time of day duration at which the time of day should be loaded.

Although future release might contain a TimeOfDayDuration property, it is not currently available. This would control the length of a single day. The default value currently in use is 24 hours.
