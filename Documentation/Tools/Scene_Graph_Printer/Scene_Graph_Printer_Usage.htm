Scene Graph Printer Command-line Usage

SceneGraphPrinter -in input_filename [-out output_filename] [-ts integer_tabstop_width] [-mesh] [-meshverbose] [-modifiers] [-ac] [-bs] [-mem] [-prop] [-trans] [-extra] [-notimecontrollers] [-skipnodes]

Required Arguments

-in input_filename

Specifies an input NIF file.

Optional arguments

-out output_filename

Provide a file name to which output will be dumped. For example:

SceneGraphPrinter -in model.nif -out model.txt

-ts integer_tabstop_width

Specifies a tab-stop width (default is 2).

-mesh

Show book-keeping information about the data streams attached to meshes.

-meshverbose

Show the data contained within data streams attached to meshes. This option implies -mesh.

-modifiers

Show information about the mesh modifiers attached to meshes.

-ac

Show each object's application-culled status.

-bs

Show object bounding sphere data.

-mem

Show object memory addresses.

-prop

List properties attached to objects.

-trans

Show object transforms.

-extra

List extra data attached to objects.

-notimecontrollers

Do not display time controller information.

-skipnodes

Filter out internal nodes in the scene graph, showing only objects with geometry, cameras, etc.
