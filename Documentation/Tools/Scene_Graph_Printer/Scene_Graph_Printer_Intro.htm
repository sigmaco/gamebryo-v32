Scene Graph Printer: Introduction

The Scene Graph Printer (SceneGraphPrinter.exe) is a command-line application that prints a text representation of a Gamebryo scene graph (contained in a NIF file) to the command line. The text version of the scene graph is formatted to represent the hierarchy of the scene graph's tree.

You can find this application in the following location:

%EMERGENT_PATH%\Source\Tools\DeveloperTools\SceneGraphPrinter\Win32\VC80
