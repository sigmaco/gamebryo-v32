Gamebryo Build Configurations

Gamebryo for PC ships with the following build configurations for Visual Studio 2005 (VC80) and Visual Studio 2008 (VC90). The "sdk Folder" column refers to the folder name within the sdk directory that contains the built libraries for that configuration. For VC 2005, that folder is %EMERGENT_PATH%\sdk\Win32\Lib\VC80" and for Visual Studio 2008, that folder is "%EMERGENT_PATH%\sdk\Win32\Lib\VC90".
