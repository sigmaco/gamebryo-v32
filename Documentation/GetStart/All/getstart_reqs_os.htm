Gamebryo LightSpeed Operating System Requirements

Gamebryo LightSpeed supports the following versions of Microsoft Windows as development and runtime environments:

  • Windows XP SP3
  • Windows Vista SP1

Notes

  • LightSpeed has been built and tested fully on Windows XP (using the DX9 renderer) and Windows Vista (using both the DX9 and D3D10 renderers). Any difficulties on these platforms are reported in the release notes.
  • On Windows Vista, LightSpeed should not be installed into the Program Files directory. User access controls prohibit executables that are built with Visual Studio 2008 from writing files into that directory.
  • Always run Setup.exe in order to install Gamebryo LightSpeed. Although an MSI is also included, running the MSI directly will fail under many versions of Windows, and doing so will also mean that the required pre-requisites will not be checked (nor installed) on all versions of Windows.
