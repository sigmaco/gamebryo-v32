Installing Gamebryo LightSpeed

Gamebryo LightSpeed is installed through a series of installers. The main installer (setup.exe), must be run first. This installer provides all of the documentation, tools, samples, and SDK that allow for LightSpeed development on the Windows platform.

Additional installers are available for the console platforms. These installers overlay files on top of an existing Windows install. When you run the installer for the console platforms, you must install their files in the same location as the original Windows LightSpeed installation.

Notes

You should always install Gamebryo LightSpeed via Setup.exe. An MSI file is also included, but running it directly may fail entirely on some versions of Windows. Or, even if the MSI runs, it may not include checks (and associated installs) for the required prerequisites.

The main installer will attempt to install prerequisites as described below and set environment variables as described here. In most cases, the environment will be set up correctly. However, it is recommend that the user reboot the system prior to attempting to rebuild any portion of Gamebryo.

Prerequisites

The LightSpeed installer will check for the following prerequisites before it runs. If these prerequisites are not found, the LightSpeed installer will install them.

  • DirectX Redistributable (February 2010) - Note that installing the redistributable will allow applications to run. In order to build applications, users should install the appropriate version of the DX SDK as noted here.
  • PhysX System Software
  • PhysX SDK Core
  • Visual C++ 2005 SP1 Redistributable
  • Visual C++ 2008 SP1 Redistributable
  • Windows Installer 3.1
  • .NET Framework 3.5 SP1

Note that Gamebryo LightSpeed is not designed to have multiple versions functioning at one time. It is recommended that the previous version be uninstalled before installing any new version. If multiple installations are desired during an upgrade, users should read the notes on working with multiple versions.

Features

If you choose to perform a custom installation of LightSpeed, you can select the following features:

Feature Contents
Documentation Product documentation.
Source Source code for LightSpeed libraries and applications. Only available for Source licensees.
SDK Headers and libraries. This is a required component, and will be included in all installations.
Samples Demos, tutorials, models, and sample game applications that demonstrate Gamebryo usage.
    Tutorials Set of tutorials to introduce new users to LightSpeed.
    TechDemos Technical demos that illustrate specific LightSpeed features.
    GameDemos More substantial Game Demos that show multiple LightSpeed features working together in a game-like environment.
    Artist Models Sample art assets that demonstrate how to implement LightSpeed features in the art packages.
Art Tools LightSpeed exporters for supported art packages.
    Autodesk 3ds Max LightSpeed exporters for 3ds Max.
    Autodesk Maya LightSpeed exporters for Maya.
    Softimage XSI LightSpeed exporters for XSI.
Tools LightSpeed Tools for developer and artists.
    Asset Viewer Tool for viewing NIF files.
    Animation Tool Tool for viewing and editing animation sequences.
    Toolbench Extendable tool framework for modeling entities and behaviors.
    Developer Tools Set of command-line tools and utilities that provide a variety of services.

Installation Performance

Installation and uninstall will run faster if you disable System Restore points before beginning installation.
