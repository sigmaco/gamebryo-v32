Gamebryo Resources on the Web

For registered evaluators and customers, the following web-based resources are available:

Note

If you are not already registered, it is very easy. Just start with the registration form.
  • Pulse - The Emergent Community—Emergent's user-to-user forum, where you can go to get the latest news about product updates and events. This web-based community also provides an events calendar, blogs, and downloads, including a number of video demos.
  • Support Portal and Knowledge Base—Emergent's support portal on the web. View the knowledge base and request support.
