Third Party Code

Gamebryo ships with some code that was not written by Emergent Game Technologies, including a number of open source libraries that were developed entirely by third parties. In all cases where Gamebryo code is either entirely or partially based on the work of others, full credit has been given and permission has been obtained where the code was not already put into the public domain by its authors.

The third party libraries that ship with Gamebryo are included in the Gamebryo distributions as a convenience and should not be considered an integral part of the engine or tools. The libraries are distributed as they were downloaded, except for perhaps some minor modifications required to get them to compile on various platforms, from the following sources:

  • Bison
  • DockPanel Suite

  • enet

  • ObjectDumper
  • Odyssey BreadcrumbBar
  • OpenEXR
  • Perforce

  • Puzzle.SyntaxBox
  • Python
  • Quaternion Interpolation Code
  • Rollup Control
  • SourceGrid .NET Control
  • STLPort
  • Subversion
  • SWIG
  • TinyXML
  • View Bar
  • Windows Presentation Foundation
  • WPF Model-View-ViewModel Toolkit
  • WPF TreeView Selection

Note that for OpenEXR, LibPNG, ZLib, and TinyXml, the distributions included with Gamebryo include project files that allow the libraries to be compiled using the supported compilers for the various platforms supported by Gamebryo. These project files are included in separate NiProject subdirectories.
