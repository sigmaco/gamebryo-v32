Core Runtime Overview

Gamebryo LightSpeed's Core Runtime subsystem provides geometry and scene graph management, animation, rendering, and other high performance runtime systems.

LightSpeed Core Runtime Subsystem

Key Core Runtime features include the following:

  • Hierarchical scene graph representation
  • Support for advanced 3D hardware acceleration on all platforms
  • High-end texturing and rendering effects
  • Geometry representation supporting instancing, per-frame modifiers, diverse data formats and hardware skinning.
  • Flexible rendering, sorting, and culling methods
  • Floodgate stream processing engine for effectively running multithreaded code on diverse hardware.
  • Controller-based animation system for m for motion, material parameters, and other data.
  • Robust particle systems for special effects.
  • A post-based Terrain System that supports LOD and material layers.
  • A Light Service that supports dynamic lighting and shadows

In addition, for an application that requires features that are not directly supported by LightSpeed, all objects are extensible through C++ class inheritance.
