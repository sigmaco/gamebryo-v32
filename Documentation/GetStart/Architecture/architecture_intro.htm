Gamebryo LightSpeed Architecture: Introduction

Gamebryo LightSpeed is divided into a number of major subsystems: Foundation, Core Runtime, Game Framework, and Online. Beyond these main subsystems are several auxiliary subsystems. LightSpeed also ships with powerful tools for artists and designers.
